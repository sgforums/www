    This is not going to be easy but from your account, it seems like the data from the jpeg files have already been corrupted before they were placed in the dvd. jpeg files consist of ascii codes written within them, so if even one character within that coding is changed, all hell breaks lose.

    It is difficult or near impossible to fix those files if they were corrupted from the source themselves usually resulting from a bad read and write of the data at the start.
