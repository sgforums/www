    well.. the PLAN was to start training 4months b4 the actual day..

    at that time i laid of running for some time already... so i started of with 3km.. three times a week.. and increase 1km per week...

    but then due to work, and other things cropping up.. i only ran on and off.. i only managed to run 3times a week on 2 occasions.. the other times, i didn't run or just ran once a week... but i managed to average 5km per week..

    the week before i ran part of east coast, for 10km...

    so i had poor training

    if u plan to do it well... here's something..

    start 16weeks before the competition.. try to run at least twice a week... each time run 30-40 mins.. go for running time instead of speed and distance... plan a distance run on weekends... start from 5km and increase every week by 1km...

    before the actual thing... try to have trial a 21km route

    oh another impt thing is to practise your form.. body upright, chest up, glide instead of bouncing on your run..
