    Features
    The Prolink Hurricane 9000 is an Ethernet ADSL Bridge/Router which fully complies to ANSI T1.413 Issue 2, ITU-T G.992.1 (G.dmt) and G.992.2 (G.lite) ADSL standards. It supports Full-Rate ADSL transmission at up to 8Mbps downstream and up to 1Mbps upstream. It can be connected to a small, home-office LAN or a single Ethernet-equipped PC to provide broadband services to a single/multi-user environment with support both Static and Dynamic IP Routing, Network Address Translation and Dynamic Host Configuration Protocol Server.
