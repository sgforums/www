    Here are the steps to posting up an image:

    1) Right click on the image and select " Copy Image Location ".

    2) Press the tree-like icon all the way to the left and on the same row as the 'Bold' ( B icon ), 'Italic' ( I Icon ) and 'Underline' ( U Icon ), which is just above the space you are posting in.

    3) Paste the link which you have copied into Image Url, and finally press "Insert".

    On a side note, you can use the rest of the functions to change the Image's size and positioning.
