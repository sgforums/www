    Before I touch on modification, let me make it clear that any form of modification will require the system to be opened, a chip to be installed and soldered.
    Please keep in mind that these modifications are done locally, most of the time by the shops themselves. So if you really intend to get a modified one, do get from a shop you trust.

    That being said, let me introduce 3 main types of modification:

    1) NeoKey / External Mod / USB Mod
    2) Orriga Chipset
    3) Messiah chipset

    The first form of modification is the cheapest of all. Known as a Neo Key or External Mod or USB Mod, it cost only about $60. However, it is also the worst mod chip of all.
    This mod chip requires an adaptor to be plugged into the front of the PS2 via the USB ports and the installation of a single jumper wire inside the system.
    Please take note that this mod chip only allows the system to playback copy CD-ROM games. Copy DVD games cannot be played at all. If we were to do a simple check, you will realize that most PS2 games are DVD games. So this makes the Neo Key a lousy mod.

    The Orriga chipset is an improved version of mod compared to the Neo Key. This requires no external adaptor but needs numerous wires to be soldered onto the motherboard. This modchip allows the system to playback CD-ROM and DVD games. BUT the booting of the system to play copy games is very tedious. To play CD-ROM games, we need to boot the system with either a Gameshark 2 or Action Replay 2 disc, after which we will need to swap the bootdisc with the CD game we want to play. If we want to play copy DVD games, we will also need an original DVD game with a larger capacity of the copy DVD game to boot up the system.
    All being said, the Orriga chipset is not very stable. Feedback from friends is that it will cause systems to hang quite often. Costs is around $200.

    The final modchip (messiah) is the one everyone is supposedly waiting for. It still needs major soldering of wires but the good thing is that after the modification, the system is supposed to behave like any original system. All games: originals, imports, copy CD-DVD ROMs can all be played without needing to do any disc swapping. Cost is around $200. However, please take note that this chip is designed specifically for the PAL system and thus may not work on JAP or US models.

    With so much said, some of you must think that I have tried all the different types of mods. No. I am a true gamer and I only play originals. On many occasions, the idea of modifying my system did cross my mind. But I never did want anyone to open my system and start putting extra wires especially since I paid so much money for it.

    So the question now is if you intend to buy an original system, which should you go for?

    First of all, when we purchase a console, itÂ’s intention is to play games. So the first objective you should keep in mind is that the system should have the games you want.
    Most games do appear in all systems, itÂ’s just a matter of when. The release dates of JAP and US games is usually not that far apart. The one major consideration is that JAP games are mostly in Japanese. And unless you can understand itÂ’s language, it might not be worthwhile to pay so much and in the end not knowing exactly what the game is all about.
