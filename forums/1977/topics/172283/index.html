    NTU Surf N Sweat 2006

    Date: 5 Feb 2006
    Venue: Sunset Bay, Sentosa
    Events: Surf n Run & Surf n Skate

    Surf n Run is a biathlon-like event that replaces the conventional swimming and running with participants paddling out to sea and back on a surf board followed by a run along the beach.

    Surf n Blade comprises of a paddling phase similar in distance covered by the main event, Surf n Run, followed by a blading phase of about 3Km.

    For this year, a NEW category, Lover's Challenge, has been added to increase the fun. Click here to check out the other categories out too.

    Event website: http://www.ntusportsclub.com/surfnsweat2006/Home.html
