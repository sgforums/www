    i'm using GMS standard key settings if notice it.

    ctrl -> basic attack
    shift -> main attack
    alt -> jump
    space -> hp skills
    ~, 1,2,3,4 -> supportive skills
    del -> main mp potion
    end -> main hp potion (backup potion for sie, since he doesn't need hp pots)
    pgdown -> backup pots
    z -> pickup
    x -> frequently used skills (like tele), or basic hp pots

    other less used pills and skills are placed at places which are easy to remember
