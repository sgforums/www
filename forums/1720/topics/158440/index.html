    Dear Maplers,

    There will be a complementary patch for MapleSEA Game Service on 20th October 2005, from 1400HRS (+8 GMT) to 1800HRS (+8 GMT). The Game Service will be down during this period. Please logout before this time to avoid loss of items.

    The upcoming update includes:

    Version .16

    1. Fixes for v.15 bugs
    a. Opening of the new Free Market (note: All entrance leads to the same Free Market)
    b. Opening of Utah's Pig Farm
    c. Huckle's Quest in Orbis Tower is back!
    d. Hinomaru Fan is back!
    e. Jay's quest disabled
    f. Rowen Milk quest fixed
    g. NPCs speaking ????? are fixed
    2. Launch of Orbis Regular Hair Coupon
    3. New Horoscope weapon for Crossbow available!
    4. End of Meso Magnet Promotion Period
    5. Removal of Lie Detector
    6. Removal of Tubes and Surf Board drops
    7. Re-opening of Florina Beach

    *NOTE: Alcaster Quest is made unavailable temporarily. Patch files are less than 2.5 MB, full client file is around 300MB.

    If you have problem running the auto-patch when starting the game through the desktop shortcut, restart your PC and follow the steps described in this link here: http://www.maplesea.com/support/faq.aspx#09 .

    If this fails, please download the manual patch. If manual patch does not work, please re-download the new game client. Check our FAQ section for more infomation.

    - MapleSEA Network Administrator
