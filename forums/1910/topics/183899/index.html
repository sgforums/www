Opening Ports For D-Link Routers

    How to open ports?

    1. Open your web browser and enter the IP address of your router
    (192.168.0.1).

    2. Click on Advanced and then click Virtual Server.

    3. - Create a new entry:
    - Click Enabled.
    - Enter a name (warcraft3).
    - Private IP - Enter the IP address of the computer you want to host the
    game.
    - Select Both for Protocol Type
    - Enter 6112 for both Private Port and Public Port
    - Click Always or set a schedule.

    4. Click Apply and then Continue.

    Commonly Used Ports

    - World Of Warcraft - Port 3724 , 6112 (Blizzard Downloader) ,
    6881-6999 (Blizzard Downloader)

    - Warcraft 3 - Port 6112 (When hosting game )

    - Diablo 2 - Port 4000, 6112-6119

    - Steam - 1200 (UDP) , 27000-27015 (UDP) , 27030-27039 (TCP)

    - Counterstrike - Port 27005 and 27015 (When hosting game )

    - Bittorent - Ports 6881-6999 (TCP)

    - Emule - 4662 (TCP) , 4672 (UDP) , 4711 (TCP)

    - Yahoo Messenger - Port 5000 (Chat), 5055 (UDP - Phone),
    5100 (TCP - Webcam)
