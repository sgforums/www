    How to open ports?

    1. Type http://192.168.1.1 into the "Address Bar" and click GO.

    2. Skip user name and type admin (admin is the default password) as the
    password, and click OK.

    3. Click on the Advanced tab to the very right. Once the "Advanced" tab
    comes up please go to the Forwarding tab.

    4. Type the ports you want to open.

    Commonly Used Ports

    - World Of Warcraft - Port 3724 , 6112 (Blizzard Downloader) ,
    6881-6999 (Blizzard Downloader)

    - Warcraft 3 - Port 6112 (When hosting game )

    - Diablo 2 - Port 4000, 6112-6119

    - Steam - 1200 (UDP) , 27000-27015 (UDP) , 27030-27039 (TCP)

    - Counterstrike - Port 27005 and 27015 (When hosting game )

    - Bittorent - Ports 6881-6999 (TCP)

    - Emule - 4662 (TCP) , 4672 (UDP) , 4711 (TCP)

    - Yahoo Messenger - Port 5000 (Chat), 5055 (UDP - Phone),
    5100 (TCP - Webcam)
