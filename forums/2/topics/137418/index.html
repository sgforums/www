    I think for new cars, you will still need to run in...

    This is because the engine block in a new car is still not "seasoned" and furthermore, the walls in the cylinders are not lubricated enough...so if you just don't care about it and rev your new engine, it will not be good for the engine... friction etc...

    It might not be obvious initially, but I've read before that in the long run, the engine will deteriorate faster.. so if you are kind to your engine in its early (infant) stage, it'll last longer...

    If I'm not wrong, running in for different cars varies...some cars require up to 3000km before you can rev your engine above 3000rpm... you have to check with the mechanics when you go servicing...
