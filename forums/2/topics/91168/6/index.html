    Original rims is 155/70R13

    rim diameter = 13 inch
    side wall height = (155/25.4) X 70% = 4.27 inch
    total wheel diameter = 13 + 4.27 + 4.27 = 21.54 inch = 547mm(1 inch = 25.4mm)

    The below are sizes available from yokohama website.
    http://www.yokohamatire.jp/yrc/asia/English/top.jsp;jsessionid=aaaHO25cedm5ESDYbAeh.?URL=tire_catalogue%2Fmain.jsp
    All the below is very thin and require rim protector against scratches.

    17 inch suitable size is 205/40R17
    If use the diameter is 595.8mm (+8.9%) Very big and Power

    15 inch suitable size is 195/50R15, 185/55R15
    If use 195/45R15 is 556.5mm (+1.7%) Very Good
    If use 195/50R15 is 576mm (+5.3%) too big
    If use 185/55R15 is 584mm (+6.8%) too big

    14 inch suitable size is 185/50R14, 185/55R14, 175/60R14,
    if use 185/50R14 is 540.6mm (-1.2%) Very good
    if use 185/55R14 is 559.1mm (+2.2%) good
    if use 175/60R14 is 565.6mm (+3.4%) a bit big

    If want to use 14 and 15 inch, must use very thin tyres or else, the overall wheel diameter will be big. If use bigger width of tyres like 195 or 185, must use aspect ratio of 50 %or 45% becos 60% is too big. But Singapore think the thinnest I saw b4 is 50% one on 195. Never see 45% b4.
