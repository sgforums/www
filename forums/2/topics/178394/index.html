    haha. to add in the recipe a bit....

    SBx = QBx = FBx = GBx i.e.
    SBA = QBA = FBA = GBA .....
    SBB = QBB = FBB = GBB .....

    i did not add in 'A' prefix for motorcycles though...

    the suffix is derived like this:
    1. 26 alphabets, remove 7 confusing letters i.e. F, I, N, O, Q, V, W.
    2. reverse order i.e. ZYXUTSRPMLKJHGEDCBA
    3. take alternate i.e. ZXTRMKHECAYUSPLJGDB

    there u have the 19-letter key to the suffix cycle.

    SBA1T
    SBA2R
    SBA3M
    SBA4K
    SBA5H
    SBA6E
    SBA7C
    SBA8A
    SBA9Y
    SBA10S
    SBA11P
    SBA12L
    SBA13J
    SBA14G
    SBA15D
    SBA16B
    SBA17Z
    SBA18X
    SBA19T
    SBA20M
    .......

    to count from 1 to 9999 is too long winded... there is a short mathematical method.... forming the 19-cycle with the first alphabet being the first suffix i.e. for SBA, it's "TRMKHECAYUSPLJGDBZX"

    nth letter in a/m cycle = (12 x thousands) + (11 x hundreds) + (2 x tens) + (1 x ones)

    to cut a long story short lah....
