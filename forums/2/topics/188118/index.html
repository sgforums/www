    Car = Class 3
    14footer lorry - 24footer up till 10tons = Class 4
    Cement mixer or lorry exceeding 10tons = Class 5
    Bus = Class 4A or otherwise know as PZ licence
    Taxi = Vocational Taxi licence

    SBS allow those who are qualified class 3 drivers to take a course before letting them drive.

    You must have class 3 with 12 points in your DIPS in order to enrol for class 4 and 5 at a TP counter beside Jurong swimming pool.

    Container trailer without the rear part = class 4
    *With container behind = A course in PSA (towing)
