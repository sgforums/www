    Just remember as a general rule, the more u depress the accelerator translate to the auto gearbox that you need more power; what happens when you need more power is that the gearbox will hold or downshift if necessary. So if u want to shift up, you should indicate to the gearbox that you dun need the power by slightly easing off the accelerator, provided there is already sufficient power for the next higher gear.

    You can actually leave the gearbox to D and control the gears using the accelerator, be it upshift or downshift

    Note too, that there are more responsive as well as less responsive gearbox, ie not all gearbox are the same. You might experience a slightly longer time for the upshft / downshift in a different cars.
