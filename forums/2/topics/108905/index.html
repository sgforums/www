    Originally posted by cornyfish2000:
    I think much more common are WRXs that stick on the STI badge. In fact i seldom see real STIs around, the ones with a single big tailpipe..

    btw, any1 knows how to identify a spec c?
    Spec C comes only in 2 colours, white, WR blue and RED.
    Spec C has a rood vent.
    Spec C's have DCCD.
    Spec Cs do not have rear wipers unlike the JDMs (light weight).
    Spec C's have no side skirts or even a hint of bodykit.
    Spec C's either run low gulwing spoilers or Original GT wing.
    Spec C's have the quick steering sticker on rear windscreen.
    Spec C's have thinner glass all round.
    Spec C's do not have the auto boot "open" lever
    Spec C's come with Optitron dash instrumentation meters
    Spec C's come with 12 litre intercooler tank in rear boot
    Spec C's has more postitive castor with longer wheelbase (visible to the naked eye when you see the front wheels more forward as compared with Motor Image Euro STis)
    Spec C's come with FORGED BBS Rims or the standard STi rims
    Spec C's has no climate control aircon, just the normal one

    With the above,
    Spec C limited comes with black seats with bold red stitched STI across
    Spec C limited comes with matt black dash panels
    Spec C limited comes with power windows and factory aircon which adds additional 20kg as compared to a original Type RA spec C
