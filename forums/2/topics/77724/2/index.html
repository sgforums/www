    You wanted some information on past Civic model codes and engines. The below list might help you a bit

    1992-1995 'Sports Civic'

    Export version / Singapore
    4 door

    LSi - 1.5 SOHC-D15A-105bhp - JHMEG4
    ESi - 1.6 SOHC VTEC - D16A-130bhp -JHMEH9
    VTi - 1.6 DOHC VTEC -B16A2160(1992)-170bhp(1994-1995) -JHMEG9

    3 door

    LSi - 1.5 SOHC - D15A105bhp -JHMEG4
    ESi - 1.6 SOHC VTEC -D16A130bhp -JHMEG5
    VTi - 1.6 DOHC VTEC-B16A2 - 160(1992)-170bhp (1994-1995) -JHMEG6

    JDM (Japan Domestic Model) Ferio = 4door

    VTi - 1.5 SOHC VTEC -D15A- 130bhp - E-EG5
    SiR / SiR II, Ferio SiR- 1.6 DOHC VTEC -B16A- E-EG6 / 9

    1996-2000 'Miricle Civic'

    Export and JDM (facelift 1999)

    EXi (no 3dr version export) - 1.5 SOHC -D15A2- 105bhp - JHMEK3, E-EK3 (GF-EK3)
    VTi, Vi RS - 1.5 SOHC 3 stage VTEC -D15A3-130bhp - JHMEK3, E-EK3 (GF-EK3)
    SiR, Ferio Si - 1.6 DOHC VTEC -B16A4- 170bhp - JHMEK4, E-EK4 (GF-EK4) , 3dr and 4dr have same code

    JDM only (3dr)
    Type R - 1.6 DOHC VTEC-B16B -185bhp - E-EK9 (GF-EK9)

    2001 - current

    Export & JDM

    4 Door (Ferio)
    EXi - 1.5 SOHC -D15A- 105bhp - CBA-ES1
    VTi - 1.5 SOHC VTEC-D15A- 115bhp - CBA-ES1
    VTi-S - 1.7 SOHC VTEC- D17A -130bhp - CBA-ES3

    2004 VTi - 1.6 SOHC VTEC- D16A- CBA-ES2

    JDM Civic Hybrid - 1.5 SOHC IMA with motor - 86bhp - CAA-ES9 (also used on Honda DIRECC cars)
    Malaysia & Thailand

    2.0 iS - 2.0 DOHC i-VTEC - K20A - 160bhp

    JDM 3dr

    Type R - 2.0 DOHC i-VTEC - K20A - 215bhp - LA-EP3

    In Japan there are 5door versions coded name EU and in the U.S. there are Civic Coupe, but we shall not go into that
