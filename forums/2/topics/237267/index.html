    Yes, the GST will surely motivate some people to book but the smart ones will know that the difference of 2% in GST only costs a few hundred dollars more to the distributor, as GST is levied only on the OMV and custom duty (which is 20% of OMV) i.e. 1.2 x OMV. So if OMV is $15k, the extra GST of 2% = 1.2 x $15k x 2% = $360 more, which can easily be covered if COE goes lower.
