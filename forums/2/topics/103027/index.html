  • superflydmw
    18 Nov 04, 22:38
    I applied for SSDC. This is my realistic assessment.

    1. Book and pass Basic is very fast, depends on availability of test date, which can span from days to 2 weeks. You just need to take the theory assessment class, which is almost everyday.

    2. Book and pass Advanced has the same factors, when I booked few days ago, the earliest date they give was 29 Nov. If you really KS, you might wanna book 2 days close to each other, because nowadays the test result is same day (instead of 2 weeks last time). I'm not sure if there's consequence by booking more than 1 theory test slot.

    3. 33 practical lessons is not a problem if you have time. I completed all within 12 sessions, now waiting for advanced theory test to pass. 12 sessions can be morning + afternoon, if you are crazy enough it can be done in 6 days.

    4. Once you pass advanced theory, you need to see capacity of test date. As of now, SSDC offer 24 Jan 05, but by the time I pass, date might shift, then it's another date (probably 6 Feb?).

    So... now is mid-Nov. Get license in end Jan. So roughly 2 months or slightly more is possible. Nonetheless, practice all the test routes, and book more practical trial tests, ask for fixed instructor if you find a very good guy during your 33 lessons.
