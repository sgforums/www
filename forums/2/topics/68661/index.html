    I think there is some discrepency here.
    Normally the PARF = 80% of OMV (for cars reg before May 02)

    So maybe that car has Parf = 15.9K that means the
    OMV = (100/80)*15.9K = $19.875K

    Assuming that OMV wrong but PARF correct
    Scrap value = (130/80)*15.9 + 0.5 * 30 = $40.8375K (looks too good to be true)

    BUT if you assume the OMV (27K) is correct and PARF is wrong:
    Scrap value = 1.3 * 27 + 0.5 * 30 = $50.1K
    which is higher than your buying price. So this is definitely wrong.

    So my conclusion is that the OMV = $15.9K

    As for the LTA WEB site (actually one.motoring)
    http://www.onemotoring.com.sg/vehDeReg/dereg_parf.asp
    There is an on-line check for PARF/COE rebates but only if you know the owner's IC.
