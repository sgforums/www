    Class : A
    Type : superminis
    Classification : the Baby-Benz is a supermini car

    Class : C
    Type : compact
    Classification : the C Class was sold as a compact car

    Class : E, CLS
    Type : mid size
    Classification : The E stood for Einspritzung, which is German for fuel injection, CLS for 4-door coupe

    Class : S
    Type : full size
    Classification : Full size, luxury car, in 1994 all Mercedes model names were rationalized, with the SE/SEL/SEC cars becoming the S-Class

    Class : SLK, SL
    Type : roadsters
    Classifications : the name was chosen to reflect the company's intentions for the car - that it be Sportlich (sporty), Leicht (light), and Kompakt (compact)

    Class : CLK, CL

    Class : SLR
    Type : supercar
    Classifications : SLR is a sports car, most people presume "SLR" to stand for "Sportlich, Leicht, Rennsport" - German for "Sport; Light; Racing"

    Class : G
    Type : SUV
    Classifications : G class referred to as the G-Wagen, short for GelÃ¤ndewagen (or Cross-Country Vehicle)

    Type : crossover SUV
