    today I went for a 2nd test drive - and thats when you can pick up things you missed out the first time.
    Realised there is a "gasping" sound each time you stab at the accelerator- so I ask the SE if this can be fixed or is it some air-leak in the air-intake or if this a characteristic that cant be fixed.

    Also find too much "hard-cheap looking" plastics used on the the lower dashboard section. And the locally fitted leather squeaky at the rear seats (like rubbing against each other when theres passenger in the rear.
    But at this price, I guess one has to be more tolerant for a 2.4L car with so many standard features. Maybe wait for Generation II even if it may cost slightly more.
