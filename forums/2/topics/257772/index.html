    A lot of things can help you save fuel..observing carefully the traffic flow can help..because when you observe traffic flow, then you know when to start moving off and when to brake..

    1) When moving off from a stop, move off gradually and slowly..can help to save fuel..

    2) When cruising on expressway and the road is clear, use cruise control as much as possible if your car comes with cruise control..can help save a lot of fuel too..

    3) When braking, try not to brake suddenly and very hard..try doing it gradually and letting it "crawl" to a stop..again, this can help save fuel..

    4) When you are waiting for someone, switch off engine, don't let it idle..

    5) If you are using RON 95/98 fuel, try pumping Shell's V-Power or Caltex's Techron Platinum occasionally every month once or so to help clean the engine to make it run smoother and save more fuel..
