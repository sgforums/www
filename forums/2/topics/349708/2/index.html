    Lol yup, not all repossessed cars are bad. In fact, I think there are a lot of good deals for higher end/luxury cars. Because reposessed cars have starting prices rather near their paper value, so the savings can be huge - possible even for 5 digit savings for the more high end continentals. However, if you factor in the fact that everyone else is looking for the same savings, in the end most auctions end up only marginally cheaper, if not equally expensive on the 2nd hand market (paper + market value), where market value is the premium one would pay above paper value for the car. 

    Thing about working with used car dealers, is that there is always a certain element of risk. I think it depends on the salesperson as well, as I have heard very good and yet very bad comments about the same used car dealer.
