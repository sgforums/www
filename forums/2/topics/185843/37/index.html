    Double layer SP + - is abt 15kg per door. Add on 60kg for 4 doors on your standard stock march curb weight of 975kg

    1. 975kg / 98 horse power = 9.9 kg per horse pull ( < 10kg per horse has good pick up )

    2. 975kg +60kg(SP) / 98 = 10.7KG ( per horse pull )

    ( Single driver )
    3. Actual weight : +-(FC) 40kg + 975 kg + 65kg (yrself) + (4rims) 36kg + ICE( 25kg ) + (SP) 60kg = 1201/98 = 12.3 kg per horse pull !
    That is abt average 2.4 kg onto 1 horse pull added when driving ?

    which is easier to pull with less effort, 12.4kg or 9.9 kg per horse ?
    For a small car it does make a little difference in terms of performance.
    Plus point: lesser wind noise, car more stable and more comfort if you SP !

    FC will increase, power to weight ratio will be effected ?
