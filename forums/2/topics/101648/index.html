    Originally posted by K_ge_zi_wang:
    whats the difference between a 2003 kelisa and a 2004 kelisa?
    The difference between them are,

    1) Front bumper
    2) Front grill
    3) Rear bumper
    4) Rim size (Older model is 13" and newer model is 14" )
    5) Meter (Older model using black colour and newer model using white as the background of the meter)
    6) Antenna (Front RHS on the older model and Front Center on the newer model)
