    Originally posted by y720:
    as far as i understand, that is not correct. this problem will only happen if you change the tyre rim size without changing the tyre profile (thickness). width will not affect the meters.
    The stock tyre of ALTIS is:

    185/70 R14, circumference is 1931mm.

    Assuming we have it changed to:

    195/70/R14, the cirucmference is now 1975mm giving an error of 2%. Not to also mention that such a size would be rare, maybe not manufactured too.

    Another scenario is we use a wider profile and reduce the aspect ratio (this scenario gives the smallest error):

    195/65/R14, the circumference is now 1914mm giving an error of -1%. Once again such a tyre size maybe rare thus cost more.

    Thus, the best way is to upsize the Rim size to keep the error to a minimum.
