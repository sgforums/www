    Last week, Transit Passengers's Skytrain has been replaced by the People's Mover System (PMS).

    Normal passengers's will be replaced next Friday (ST gave reports being next yr)

    The PMS, that will link T1, T2 and T3 together, will have 3 branch (For normal passengers) and 5 branch (Transit passengers)

    *The branch seen at T2 (Nearer to BT) is actually for Transit passengers

    Trains is made by Mitsubishi, model the same as Sengkang/ Punggol LRT only that it only has 8 seats!

    Operation Hrs: 0600 - 0130

    It will run 24 hrs once the other 6 branches linking T2 to T3 and T3 internal shuttle starts operating
