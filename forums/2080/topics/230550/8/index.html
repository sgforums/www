    021/022 has the ability to half throttle at last phase of acceleration. with this it could accelerate all the way to the next station before braking, though it usually just takes a longer time to accelerate slowly to the maximum allowable speed of the sector.

    this new ATO mode is compatible with current signalling systems and will not exceed current speed limits. it fact it takes a longer time to travel between stations and results in reducing staton dwelling time.

    it is not known if other refurbished trains feature this ATO mode. also, it went from the new ATO mode for SB trip to normal-fast ATO mode for NB trip.
