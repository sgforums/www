    Originally posted by SmRtFrEaks:

    guys, anybody could tell me how to read the train TRN's number..what does the number represents?? if there any meaning to it???

    TRN is Train Run Number, you can read it at the top right hand cornor of the head/tail of the train. TRN is like a number given to trains from the depot, to indicate which depot they're from, and also acts something like those kind of queue numbers.

    Note: Different trains can have different numbers, like Day 1, set 335/336 can have TRN of 053, Day 2, that same train, can have a TRN of 201 (For Lunch Time Xpress).
