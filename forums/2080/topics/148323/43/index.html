    Originally posted by Bus and Train LoVuR:

    So how can the parameter be reset? Using keys?

    Anyway , what's parameter?

    The parameter here refers to the train's running curve, which in turn control the train rate of acceleration, rate of deceleration and running speed. All of these information is stored in the onboard computer on the train itself, which in turns execute the whatever command and parameter given to operate the train automatically. If one were to reset this, alot of work needed to be done to re-program the whole system, which need quite some time. So instead of resetting, have a person who created the system or is good in the system to alter the parameter.
