    Hi ppl.. I would like to enquire about the fare payment that I need to make should I decided to use a Train Concession Pass (TCP) to travel on BOTH  the MRT and bus.

    I understand that distance based fare works in such a way that one is charged based on the distance travelled rather than the number of trips transferred.  

    1)Let's say I travel from Punggol MRT station all the way to Pioneer MRT station (by using a train concession pass, meaning that no fares would be deducted as long as I don't exceed the number of "free" rides that I am entitled to in a day) and then take a short bus trip ( a few bus stops), how much would I be charged?

    Distance From Punggol to Pioneer : 35.2KM

    Distance of bus trip (Bus stop at Pioneer station) : 4.6KM

    If i am not using TCP, I would be paying ($2.13 (Train)+$0.05 (Bus)=$2.18). Calculations taken from http://www.publictransport.sg/publish/ptp/en/distancefare.html

    If i am using TCP,does it mean that I would only be paying $0.05 for the bus trip??

    2)Let's say I travel the same route back.

    Distance of bus trip (Alighting at Pioneer MRT): 3.2KM

    Distance From Pioneer to Punggol :35.2 KM

    If i am not using TCP, I would be paying ($0.71 (Bus)+ $1.45 Train). Calculations taken from http://www.publictransport.sg/publish/ptp/en/distancefare.html

    If I am using TCP,does it mean that I need to pay $0.71 this time??

    This is really confusing. Lol. Hope some of you guys can enlighten me on this... Tks!

    The TCP is only valid on the train system, and any transfer rebate between train and bus is not applicable.

    The first journey consists of a train ride and a 4.6km bus journey.
    In this case, you will pay SGD 0.91 for the 4.6km bus ride.  

    The second journey consists of a train ride and a 3.2km bus journey.
    In this case, you will pay SGD 0.71 for the 3.2km bus ride.
