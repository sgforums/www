  • jcqh
    8 Sep 09, 14:24

    New releasal! You may replace your previous version with this. This train is compatible with both the CGL and the NSL route.

    The C151 train V0.6 (Set 131/132)

    New features include:

    • Improved and redesigned cab (Courtesy of Bus and Train LoVuR's image)
    • Full exterior body

    Improvements in later versions (Currently in progress):

    • Photorealistic interior and exterior
    • construction of underside of the train body (e.g. wheels)

    Note: This train does not have the same name as the one defaulted in the CGL/NSL routes, so you will have to either:

    • rename the train file to "c151"
    • rename the defaulted train in the route file to the current
    • manually select the train at your main menu

    Lastly, have fun and enjoy!!! Pls do leave comments too, and the BVE development team will try to fix the issues in the next version.

    http://www.4shared.com/account/file/131049619/7a4a419/131-132.html

    Notes to developers:

    • C151's wheels' tip is set as 0.15m, so the part where the tracks touches the wheels must be at 0.15m.
    • For every train exterior you make, make it 0.15m above as well.
    • When making routes, please note that the train run sound index is 1 for underground, 2 for above ground. If you don't specify any Train.Run commands, the old sound will be used.
