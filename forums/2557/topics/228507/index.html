    ok ok.... as the subject shows, this topic is about set pieces!

    Set pieces = free kicks, goal kicks, throw ins, corners and even the kick off!

    No doubt those who have been in this team as long as i have will probably have heard this millions of times.... but surprising, despite talking about it for the past few years, we still dun seem to be doing what we keep talking about!

    We are talking about the very basics of how the game should be played!
    How to defend set pieces.
    How to attack set pieces.

    ***I would like everyone to be alert and focus everytime the game is stopped, be it for a throw in, free kick etc. Dun walk away after we have conceded a set piece, instead immediately look for open opponents to mark.

    Similarly, when we win a free set piece , immediately free yourself for the quick free kick/throw in etc. Not only does this allow us to conter attack when the opponents are still reorganising, it also helps your players who are taking the set pieces. E.g, thrower can make a quick throw to the open teammate instead of waiting for all his players to be marked.*****

    So anyway, here it goes.
    ====================================

    - GOAL KICKS (Attacking)
    Left/Right fullbacks to spread out to give keeper option of short pass.

    Centre backs to drop deep to edge of penalty box to pick the ball directly from short goal kick or from full backs.

    Midfielders & attackers to either win the long goal kick in middle of the field or to pick ball from defenders.

    - GOAL KICKS (Defending)
    Attackers to pressure opponents defenders to deter any short goal kicks from the opponent's keeper.

    Midfielders to win the long goal kick in the midfield.

    ====================================

    I'll continue with the rest of the set pieces in the next few days.
    I'm sure i seem to have missed something out, so please feel free to add in the missing points!

    Thanks....
