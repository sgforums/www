    Heavy-weight training:
    I give example about bench-press, if maximum weight I can lift up is 100kg.
    Then measure 100kg*70% = 70kg
    Then measure 100kh*90% = 90kg
    Your range is between 70kg-90kg
    Your step range is (90-70)/5 = 4kg

    70kg = initial weight
    90kg = last weight
    4 kg = added every round

    so:

    Do about 6 sets of bench-press :
    1'st set = 70kg 7 times only
    2'nd set = 74kg 6 times only
    3'rd set = 78kg 5 times only
    4'th set = 82kg 4 times only
    5'th set = 86kg 3 times only
    6'th set = 90kg 2 times only

    if you cannot lift up 7 times at 70kg, it means your max weight is not 100kg, try reduce the amount ... but if you can lift up 70kg 10 times, then increase the max weight by 4kg.

    every set, let your body rest about 1 to 1.5 minutes. This kind of exercise can make your chest big and strong very fast, don't forget to eat a lot of protein source such as fish, and 2 eggs per day ...

    if your body is big enough, then you can start endurance exercise, it it to shape your body ...

    endurance training:
    I give example about bench-press, if maximum weight I can lift up is 100kg:
    Then measure 100kg*50% = 50kg
    Then measure 100kh*70% = 70kg
    Your range is between 50kg-70kg
    Your step range is (70-50)/5 = 4kg

    Do about 6 sets of bench-press :
    1'st set = 50kg 12-15 times only
    2'nd set = 54kg 11-14 times only
    3'rd set = 58kg 10-13 times only
    4'th set = 62kg 9-12 times only
    5'th set = 66kg 8-11 times only
    6'th set = 70kg 7-10 times only
