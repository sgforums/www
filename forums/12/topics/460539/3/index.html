    Hi Luleil,

    As you have mention, if you intend to study abroad, I will strongly recomend that you can contact The DesignSingapore Council when you have time so that you can plan ahead like financial aid, which art school around the world is best so that you can be well prepared, otherwise try to concentrate on your "O" level first before decided anything.

    After your "O" level ended, if you still want to improve your manga drawing, below are some of the resources you can explore :

    1) Manga Expert : http://www.mangaexpert.com

    2) Mad about Manga : http://madaboutmanga.com

    3) Cartoon Drawing Made Easy : http://www.cartoon-drawing-made-easy.com

    4) Manga Start : http://mangastart.com

    For more update for the trend of manga, you can visit mangamagazine.net : http://www.mangamagazine.net , The Daily Cartoonist : http://dailycartoonist.com or Otaku USA Magazine : http://otakuusamagazine.com

    If you want to read more about drawing manga :

    • http://www.hinklerbooks.com/OurProducts.aspx

    • http://www.kuretake.co.uk/products/new/ZIGCartoonist.aspx

    If you want to improve your drawing with better manga illustration tools : http://www.sakuraofamerica.com/manga-comic

    If you need apps to improve your drawing : http://artelplus.com  

    Just for your info, nowadays you can even use software to create manga instead of the conventional way of drawing : http://www.comipo.com/en

    Finally for your info, there is an anime event called Anime Festival Asia and they will had their roadshow at Singapore Expo Hall on 9-11 November 2012, you might like to visit those "anime superstars" for more idea : http://www.animefestival.asia
