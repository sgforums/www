    ok here's what's happening, grisoft has closed down 2 of 3 servers dedicated to the avg free version. This is to get people to buy the proffesional version instead. So well, too bad for you, you may have to just keep trying to update.

    anyway this may help :

    There are still actually 3 servers available. Here is a tweak that will enable you to use ALL of them...

    1) Navigate to your AVG install directory C:\ Program Files\Grisoft\AVG6 and rename your existing url.ini file to url.iniold

    2) Copy the following, then paste it into notepad, then save it in C:\ Program Files\Grisoft\AVG6 as url.ini

    [SERVER_NAME]
    1=free.grisoft.cz
    2=ftp.grisoft.com
    3=www.grisoft.com

    [SERVER_URL]
    1=http://free.grisoft.cz/softw/60/fe
    2=ftp.grisoft.com/pub/softw/60/fe/
    3=http://www.grisoft.com/softw/60/fe/
    Actual URL=3

    The above code will give you 3 optional update-servers to choose from, with server #3 {www.grisoft.com} being the default. Automatic updates should be less of a drag from then on.
