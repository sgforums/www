    see if there is metal tab label on the transformer.

    actually i cannot recall wat is label on the transformer... usually there is primary/secondary value of voltage, the secondary voltage should be the rated voltage if u consider it to be load side.

    as for power, usually it is given in apparent power wif rated kVA or VA.

    to find rated current:
    I = Power / Voltage (if it's single phase transformer)
    or I = Power / (1.732*voltage) (if it's 3-phase)
