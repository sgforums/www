    must be able to draw free-body diagrams well.

    know the different forces acting on an object, then relate all of them using an equation. after that, rearrange the equation to find the unknown.

    still got other tips but i shall elaborate about them later.

    btw circular motion only a few formulas to rmbr

    omega = 2 pi frequency = 2 pi / time perioud

    velocity = radius x omega

    acceleration = radius x (omega)^2 = (velocity)^2 / radius

    force = mass x acceleration
