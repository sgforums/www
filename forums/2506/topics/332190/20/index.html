    Q1. when your monitor goes blank, is the light on the monitor still on? what did you do to the monitor to get it back up and running?

    Q2. try this simple graphics card diagnostic. 

    1. start
    2. Run
    3. type in "dxdiag.exe" without the "  "  click enter.
    4. If a pop up asks you to use the web to search for drivers, just click yes (ignore if there isnt this.)
    5. An application named "DirectX Diagnostic Tool" will pop up.
    6. click the "display tab"
    7. look and see under the notes columm - it should say "no problems found"
    8. close it.

    Q3. How old is the monitor?

    Q4. Are there lines on the monitor? for example when you are surfing sgforums, do you see visible stripes like a greenline (other colours maybe) 

    Q5.

    any similar things like the above? & is there recently any message showing u of your graphics card driver stopping ect?

    Q6. Last but not least.. is it over heated? monitors that are failing or overheated will switch off. after awhile when it's cool, then it allows you to switch on. + does the screen flicker
