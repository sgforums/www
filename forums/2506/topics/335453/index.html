    Jagex will be releasing a new MMORPG game in March next year.

    It is speculated to be named - Mechscape. 

    Still browser based, using the same Java Platform. - but it would be totally different from Runescape. The game will be geared toward an older and more mature age group than Runescape is. More recently, Jagex hinted at the presence of space bases and creatures in Mechscape--and, presumably, space travel as well.
