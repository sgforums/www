    Well to tell the truth I'm really looking forward to the DVD release. The 3-DVD set will be released this November 2007, with the price of 8900 yen.

    Well if any of you have not heard of this before, basically Animelo Summer Live is a Live concert that features various artists that have sang any anime songs, regardless they're Seiyuu, Idol, or J-Pop artist. This concert started in 2005 and have been a yearly summer event ever since, because of its popularity. If I'm not wrong JAM Project is the one that came up with this project.
