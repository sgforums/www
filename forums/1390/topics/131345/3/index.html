    Life as a obese recruit.. (to those potential obese BMT enlistees)

    First 3 weeks:

    1. PTs
    2. Simple Drills
    3. Rifle Presentation

    Next 3 or 4 weeks:

    1. IMT
    2. First Aid Test
    3. Technical Handling of M16
    4. Live Ranges
    5. Short route marches (etc. 3km, 4km)
    6. PTs
    7. BCCT Lessons for 8 lessons thereof

    1. Field Crafts (man I hate camouflague lesson!)
    2. Field Camps (spilt into 2)
    3. PTs
    4. IPPT Evaluation

    1. Trench Digging
    2. SIT Test
    3. BIC
    4. BAC
    5. Route Marches
    6. Grenade Live Throw

    Last 3 weeks:

    1. Route Marches
    2. PTs
    3. Graduation March
    4. POP Parade Rehearsal
    5. Lots of Stand-by area
    6. Maybe at this time you have more admin time...
