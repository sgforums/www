    Look, imo, it is really quite irrelevent that you think you would enjoy army life. To a certain extent, i quite enjoyed army. That does not, however, make me think that i have not wasted 2.5 years for a country that really doesn't care.
    Remember, after 2.5 years, you will have almost gained no new transferable qualifications whatsoever.

    So unless you want to sign on as a regular, or is able to come up with some overwhelmingly good reason, i would advise against serving NS. If you think you would like to try a little soldiering, then just go try airsoft or go on some private courses run by ex-soldiers. If you would like to try something different, or seek a like changing experience, how about participating in some voluntary overseas expedition? That would be more worthwhile and meaningful. You would also be able to make more of a difference to the world.

    Hope you think you choices through carefully.
