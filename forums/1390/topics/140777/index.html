    my department also went through a LRI yesterday. but i was away for range so i wasn't around to see what are the things they will check up for.

    but basically since i've done sort of stores management as part of my vocation, i can get you some tips.

    make sure all the stores are there and accounted for, meaning they must be in working condition and u must know what they are used for and how to use them.

    make sure all parts are intact, they will probably like to ask things like how come this thing got no handle, no sling ah, wiring exposed ah, that 1 broken ah all that. so if it can be repaired, do so. if not hide it and take out/loan a new one.

    if u got a record book for the stores, loan book or such, make sure everything is filled in correctly, signed and endorsed.
