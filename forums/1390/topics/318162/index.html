    even if ur camp is on mob manning, so long as u r not in the nominal roll, u no need to report back... all mob manning personnel on standby r notified by sms a couple of weeks as reminders b4 the actual mob manning period...

    mob manning is for both reservists n nsfs ONLY WHEN their names r in the nominal roll namelist... so in short, if names r excluded, even really mobilised, u need not report back.

     

    so the rule is if ur names r not in the namelist nominal list, u need not worry...

    take this from a person who was once on standby recall for alert red and mob ops manning during nsf time, and now as a reservist...
