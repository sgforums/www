    31 Aug 08, 12:32

    How come your BMT PC is a CPT?

    Anyway, while the law says it is not illegal, it is SAF's practice not to allow non-weapon trained personnel to carry arms with live rounds.

    Basically, the order goes like this,

    1) practice and pass technical handling tests.

    2) passing technical handling test allows you to go for a live range.

    3) Firing a single live round in a live range after having passed the technical handling test constitute your ability to use that weapon.

    4) you are allowed to perform duty that involves possession of live rounds (eg Guard duty)

    However, carrying of rifles without having passed handling test is not an offence so long you are permitted to do so.

    In recent years, some Camps have prohibited guard duty personnel from doing sentry with a baton/wicker shield. This usually apply to more sensitive camps.
