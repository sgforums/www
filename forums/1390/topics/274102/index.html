  • venny
    15 Aug 07, 01:35
    so when u ord? anyway, u need to go for ur blood test first to book an ffi date. mus be at least 1 mth be4 u ord.
    once u get the clearance form, go see all the accessible ppl first, like ur rsm, co and oc to sign for u.
    then u need to get all ur stuff to return beforehand, eg: et blade n stick, groundsheet, rck, rifle sling, etc. u will need to look for ur cq or rq and finally ur base logistics manager or depot manager.
    if u have drawn equipment before like if u frm signals etc n got draw things like signal sets etc, u got to return to the base or clothing store and then get him to sign again.
    next is if u are frm a combat unit and hav arms, u need to clear ur weapons for sending back to the base armoury. this is only done 2 wks before ord. make sure u clean ur rifle....
    collect ur cos frm ur unit. normally my unit has a small ceremony where the co presents the cos to the men.
    finally, go to ns portal website and do the nsmen thingy. its mainly for u to read and a quiz to do. print out ur results. bring the printout on ur final day to ur unit s1 or base s1 to collect ur ic.
    exit and shout, ORD LOH!
