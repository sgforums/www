    Originally posted by Detached:

    Everytime you go overseas for more than 24 hours, overseas notification at ns.sg website.

    No mob manning until a code has been given to you.

    No SAF emart credits.

    No unit. But not clear of NS obligations before 40. You can be posted to a unit anytime before 40 for WOSEs / 50 for officers.

    No IPPT if PES C2 and below, if not IPPT and RT applies yearly.
