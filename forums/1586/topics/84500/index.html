    Sharing of printer
    ==============
    Qn: Local or network printer?
    For Local Printer :=
    Solution 1 : Attach ur printer to either ur PC or laptop and shared it out as a shared printer on the same network that the PC and laptop is on.
    Requirement : Need a network.
    Con : the host(PC/laptop) connected to the printer must be power up for the other one to use the printer.
    Solution 2 : Buy a printer switch box. This allows both PC/laptop to use the printer.
    Requirement : A printer switch box.
    Con : Only 1 can print to the printer at a time. have to toggle the switch for the other to use.

    For Network Printer :=
    Solution : Configure the IP of the printer, then just install the printer driver to both PC/laptop and add the printer to both the PC/laptop by specifying the IP of the printer.
    Requirement : -
    Con : -

    Sharing of boardband connection
    =======================
    Qn: Direct connection to modem or thru a router?
    For Direct Modem Connection := (PC is connected to the boardband modem.)
    Solution 1 : Set up Internet Sharing Connection on ur PC. Allows ur PC to share out its internet connection to other PC/laptops.
    Solution 2 : Set up software proxy on PC. Allows ur PC to share out its internet connection via a proxy.
    For both solutions :-
    Requirement 1 : 1x Cross ethernet cable, 1x lan card(for PC if dun have intergrated lan on motherboard), 1x PCMCIA lan card for laptop(if dun have internal ethernet port)
    Requirement 2 : 2x straight ethernet cable, 1x hub, 1x lan card(for PC if dun have intergrated lan on motherboard), 1x PCMCIA lan card for laptop(if dun have internal ethernet port)
    Con : internet host(PC) have to be on for laptop to access internet.

    For Router Connection :=
    Solution : Configure both PC and laptop's gateway to the router.
    Requirement for single ethernet port Router : 1x straight ethernet cable for PC, 1x lan card(for PC if dun have intergrated lan on motherboard), 1x PCMCIA wireless lan card or USB wireless lan adapter for laptop(if dun have wireless)
    Requirement for more than 1 ethernet port Router : 2x straight ethernet cable, 1x lan card(for PC if dun have intergrated lan on motherboard), 1x PCMCIA lan card for laptop(if dun have internal ethernet port)
    Requirement for laptop with built-in wireless connectivity : 1x straight ethernet cable, 1x lan card(for PC if dun have intergrated lan on motherboard)
    Con : -

    Equipment if wish to go wireless :
    Options for PC: USB wireless adapter, wireless PCI lan card
    Options for laptop: USB wireless adapter, PCMCIA wireless lan card, built-in wireless connectivity
