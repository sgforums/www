    Do you mean IFSHLP.SYS?

    Fatal Exception 06 in VXD VFAT(01)

    If the Ifshlp.sys file is missing from the Window's folder, extract a new copy of the file from your original Window's disks or CD-ROM to the Window's folder.

    When reinstalling Windows, Setup will delete the Ifshlp.sys statement in the Config.sys file and not replace it. Add the Ifshlp.sys statement back to the Config.sys file.

    DEVICE=C:\WINDOWS\IFSHLP.SYS

    If this line is already in the Config.sys file, then remove it and reboot. Config.sys file points to Previous Version of Ifshlp.sys file. (Windows automatically loads this driver when you start your computer.)

    There is another Fatal Exception with VFAT when you copy WIN95/98 from a FAT16 Partition, to a FAT32 partition. This is due to 16bit clusters verses 32bit clusters. If your WIN operating system was FAT16, and you got a new hard drive, it must also be formatted FAT16 for the BOOT partition. Use the FAT32 conversion program that comes with WIN98 to upgrade to a FAT32 operating system, do not partition copy a FAT16 partition to a FAT32 partition.
