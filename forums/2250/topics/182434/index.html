    Windows Live parental controls due this summer

    Microsoft has announced that it will release parental control software to monitor children's surfing habits. It lets people filter online content and get activity reports on Web sites that were visited by others.

    Not only so, an update will be out later this year, called "contact management", which will let parents approve contacts from Windows Live Mail and Windows Live Messenger. This feature also gives control over who can access their kids' blogs on Microsoft's Spaces service.
