    I would suggest you not to do anything with it unless it is causing you problems, such as having a hidden trojan embedded insided it. But, since you had deleted it. It's best to leave it alone for now.

    While deleting the program icon and its shortcut icons from folders and desktop may seem to "delete" the file, it does not delete the "processes" that came with the program. As such, your computer may be "unstable" whenever it tries to run these "processes." It's like the heart is beating, but there is no head and no legs. Best thing is to simply run the program through a reputable antivirus, antispam, antispyware program to ensure it is harmless. Then, leave it there lor.
