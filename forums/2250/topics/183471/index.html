    If you're using Win XP..

    Go to Control Panel > Regional and Language Options > Languages Tab

    Check the box for "Install Files for East Asian Languages" and Windows will do the job for you. Will require the Win XP CD though.

    Choosing the language to display >>

    Eg. Using Internet Explorer, just right click > Encoding, then choose the Language you want.
