    http://computer-services-birmingham.com/anti-virus-and-spyware/215-remove-resycled-boot-com.html

    resycled/boot.com may be a worm that spreads itself around local fixed HDD drives and any removable USB and drives and also any other media that is connected to your computer such as SD cards and memory sticks. Note that the spelling is wrong "resycled". DO NOT CONFUSE THIS WITH RECYCLED.

    resycled/boot.com will infect all drives via an autorun.inf file it creates that runs a command each time the drive is accessed. Malicious files will be copied to a drives attached on infected computer. resycled/boot.com also embeds itself in various places in the windows registry.
