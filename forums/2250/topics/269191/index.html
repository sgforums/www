    New or old Singnet users?

    New users - if using Singnet's routers/modems, enter details in router/modem. You will be automatically connected. Please wait for up to a week for service to be activated if you are a new user.

    Old users - those using WinPOET. Install the WinPOET client, run it and dial in.

    If by any chance you lost it... buy a new router/modem... enter the Singnet details like those for new users and should be able to connect.
