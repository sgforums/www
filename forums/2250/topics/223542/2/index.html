    Either you increase virtual RAM or you upgrade your RAM.

    Increase virtual RAM:

    Right click My Computer > Properties

    Click on the Advanced tab. Under Performance, click Settings.

    Another window will open. Under Virtual Memory, click Change.

    Another window will open again. Select the drive where Windows is installed (by default it's C drive)

    Under Paging file size for selected drive, select Custom size. Enter at least 1024 for Initial size. Maximum size set it to 2048MB.

    Click OK all the way to apply the settings.
    _________________________________

    Upgrade RAM:

    Warranty expires, go any tech shop to upgrade it. Spoil not my business.

    Warranty haven't expire, go IBM authorized service centres.
