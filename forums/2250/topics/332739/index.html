    Everyone who is facing this problem of being unable to read email on Hotmail this few days must be a Singnet user (for once, the blame is not on Microsoft). Singnet had silently made changes to the server side to have caused this issue.

    To resolve this problem, configure your browser to surf using the singnet proxy. For SNBB users on IE:

    Go to IE-->Tools-->Internet Options-->Connections tab--> LAN Settings--> tick the Use proxy server...--> address: proxy2.singnet.com.sg port:8080 (for all)--> press Ok and restart IE

    Similarly for Firefox, go Edit->Preferences->Advanced->Network->Connection setting->Manual proxy configuration:proxy2.singnet.com.sg port:8080 (check box for all)->Click Ok and restart

    Tested and proven to be working. I initially thought is a system problem and wasted me tons of precious time doing many uninstalls and reinstalls on my PC. No more reason for anyone to go through the same ordeal.

    Down with Singnet. :(
