    Despite all the hype, it only looks better. Frankly, I see that it's no better.

    Security wise, which is the main focus, managed to kill my interest for it.

    While Windows XP has a firewall that only blocks inbound attacks, Windows Vista has a firewall that blocks both. However, by default, this feature is NOT ON. So I see no point in getting an OS that has security features of its predecessors.

    Plus the fact that everything is still registry-based, it should be easy for viruses to turn off the damn firewall... just like in XP.

    The idea of running on a limited account by default is a good idea, but is implemented so badly that it's near useless. They should take a cold hard look at how their UNIX counterparts implement it.

    Antispyware... another resource hog. I don't see why it's compulsory. Although it could be turned off... but not completely at all.

    It's said that Vista could handle resources better... but I'm seeing the opposite. Perhaps technology has not mature yet... it needs some machine with pretty good specs to run smoothly.
