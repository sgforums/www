    To fix the error 8007000e MSN Messenger check that you have internet connection and you do not have the option of Internet Explorer "Work offline."

    To do this:

    1. Open an Internet Explorer window and click the "File".

    2. This opens a menu. At the bottom make sure the option "Work Offline" is turned off (without a tick on the left).

    3. You may need to restart your computer to make sure that everything returns to work properly.

    If the problem persists reinstall Internet Explorer and check if you have to install updates Visit Windows Update at http://windowsupdate.microsoft.com.
