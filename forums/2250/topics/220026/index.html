    Restart got a few reasons:

    1. tio virus, go scan and remove anything found. Remember to update before scanning.

    2. overheat
    - heat sink may be faulty, if so, replace heat sink
    - you overclock, either reset back to the original settings or add more internal fans.
    - too much dust, please clean up your PC.

    3. hard disk spoil

    4. motherboard spoil

    Number 3 and 4 are quite rare, unless your PC parts are old.

    Originally posted by phanta5y:
    in my case. my comp completely shuts down, i have recently changed my graphics card, so i know it can't be that problem, and i have problems using the tools posted above, also the event viewer doesnt shows any error, so can anyone help me on this?
    If you have problems with the tools, try the manual way.

    For RAM, take out one stick and put in another slot. Repeat for the other sticks if you have more than one. Same for the slots.

    Make sure your hard disk is a Seagate hard disk before using the tools.

    Go to the website provided and choose the first option, which is SeaTools Online.

    Under that column, you will see Select a language. Select a language which you are comfortable in.

    You will be brought to another page. Select Drive Self Test (DST) and File System Check. Do each of them one by one.

    Make sure JavaScript is enabled before you do the tests.

    Also, check the temperature of your CPU. It shouldn't be running higher than 70 degrees. Restart your computer. When you hear it beep once, press the Del key to enter BIOS. Navigate around and find the temperature. Monitor it for a while.

    Get Speed Fan to monitor your CPU temperature. Don't do anything to it, just monitor.

    If it gets quite high after a few hours of usage, it's time to check your heatsink and fan. It's also cleaning up time as dust doesn't help in airflow.

    If all results are good... it may be due to motherboard... or software corruption.
