    after getting your cross cable ethernet cable, go to control panel, network setup wizard, choose the options without internet connections, select 'enable file sharing'.

    remember to do this to both of the computers. Off your firewall, go to ''view network connections'' and ''view workgroup computers''.

    You will need to ''share the folder'' that you are transferring.
    To do this, right click on that folder, go to 'sharing' tab and enable file sharing
