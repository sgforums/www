    The 1394 net adapter is a firewire adapter with networking capability. The system control panel lists this under network adapters, even though the 1394 adapter probably isn't being used as a network adapter for the computer, unless you changed something to make it as the network adapter.

    When did this error start? Have you tried a system restore? A system restore may help to solve this problem.

    If system restore doesn't solve it, try uninstalling this device and reinstalling it again.
