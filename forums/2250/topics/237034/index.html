    I am assuming u are going an upgrade, if so, please do take note of ur current memory configuration.

    Meaning, are u going to keep ur existing memory? If so, what configuration are they in? Dual Channel? If yes, what brand is being used? Current type? You have to sort this out before your purchase or else adding a mismatch dual channel configuration, will either make ur computer cannot start or start to give you attitude problems (suddenly reboot).

    Or u add on a single RAM to a dual-channel configuration, then u immediately loses ur dual channel. Dual channel doesn't work with 3 RAM configuration.

    ROT is to add on RAM chips that is the similar capacity and type or totally replace the current RAM configuration.
