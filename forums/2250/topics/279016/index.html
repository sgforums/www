    No prob. Just treat me dinner if I ever see you on the streets.

    Install the second com with Alcohol 120%. Trial or full or Captain Hook doesn't matter for now.

    http://www.alcohol-soft.com/

    Insert the game disc into this com and open up the Image Making Wizard. Choose the appropriate datatype for the disc you want to be read, like for example, for Frozen Throne, you will need to use Securom 4 (it's some tech used to protect discs from piracy - didn't work that well anyway). Such info are usually found by Googling something like "alcohol [name of disc]". If you are still unsure, just use General (but this may result in a non-working disc image).

    http://dl.latecomerx.com/images/Screenshot_2007-10-20_11-26-52.jpeg

    After extracting the disc data and saving the disc image somewhere, copy it to a thumbdrive and plug that into your laptop; copy the disc image over. Install Daemon Tools, which can "transform" disc images into physical discs. After installation, probably requiring a reboot, run the program and left-click on its taskbar on the bottom-right (the one that looks like the logo for PAP) and choose a drive to mount the image on. They will then ask you to browse for a file - select the disc image which you've just transferred. And the cliche "Voila!" comes in, when My Computer shows you a game disc in the virtual CD drive.



    http://daemon-tools.cc/dtcc/download.php?mode=ViewCategory&catid=5

    Regarding the Daemon Tools, get the X86 if your Windows is 32-bits/normal, X64 for 64-bits (duh).
