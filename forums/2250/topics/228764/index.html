    Right track, but calculations wrong.

    Everything in computer starts from 0.

    Binary = Base 2 has only 0 and 1
    Octal = Base 8 has 0 to 7
    Decimal = Base 10 has 0 to 9
    Hexadecimal = Base 16 has 0 to 15

    So A = 10, B = 11 and so on and so forth. Counting 0 and 1 only has 2, that's why it's base 2 for binary. Counting 0 to 15 has altogether 16 numbers in between them, so it's in base 16.
