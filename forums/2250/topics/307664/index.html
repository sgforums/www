    All Windows installation CD comes with a partition manager that is just text-based. Installation process is the same as how you install Windows XP.

    1. Boot up to the CD

    2. Read the license agreement, press F8

    3. Choose the Install option

    4. Select the first partition, press D to delete it. Repeat the same for all partitions

    5. When done, you will be left with an unpartitioned hard disk. Press C to create your first one. If you need more, input the numbers carefully

    6. Select the partition you want to install on and press Enter to install. It will prompt you to format in either NTFS or FAT32 (if I remember correctly, not sure if FAT32 is still available in Windows 2K3.)

    After formatting, it will install automatically.
