    Well for those of you who download files from the website www.themexp.org

    If you dont know, the exe file that u downloaded not only contains the wallpapers or themes, but the installation package also install spyware and trojans at the same time.Here's a way to rip apart the installation package so you only get the wallapeprs/themes and leave the trojans and spyware:

    1)First download WinRar

    2)Download your file from themexp.org.you should get an exe file BUT LEAVE IT UNTOUCHED!

    3)Rename your downloaded file name to a.exe

    4)Put the file in your local disk drive main folder(Eg.C:\, D:\, etc etc)

    5)Now comes the most important part:

    Click 'Start' 'Run'

    Type '(Local disk drive letter, where you put your downloaded file):\a.exe /x (Local disk drive letter, where you put your downloaded file):\a' without the quotations

    For example, for my case, mine would be like: 'C:\a.exe /x c:\a' as the letter C is my local drive letter.

    6)Now, check your local drive again.There should be a new folder named 'a'

    7)Go to the new 'a' folder .You should see a file named '%EXENAME%' and all the other spyware and trojan installtion files.LEAVE THE SPYWARE AND TROJAN FILES UNTOUCHED.

    8 )Now, double-click on your '%EXENAME%' file.In the 'OPen With' Window, Select 'WinRAr ARchiver'

    9)There you go.Once you opened the file with WinRar, you can see your files that you want inside.
