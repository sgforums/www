    Bridging connections? You will have to unbridge...

    1. Disconnect from the Internet.
    2. Go to Start > Control Panel > Network Connections
    3. Right click on your connection and select Remove from bridge.
    4. Restart the PC.
    5. Go to Start > Run and type in cmd
    6. Type in ipconfig. If you see an IP address that starts 169.X.X.X, your router has a problem. Reset the router. If you see a default gateway such as 192.168.X.X, go to your web browser and see if you can start configuring it.
