    1. It could be the Recovery data is corrupted. To format, you will need an original XP disc. Call Gigabytes and ask them to provide you with the disc. Otherwise, visit one of the reputable shops in Sim Lim and buy a XP disc.

    As per usual, get into BIOS, set the CD/DVD drive as first boot device. Before saving the settings, put in the XP CD. Save the settings and exit BIOS.

    Windows will run from CD and you will be prompted to read a license agreement. Read through it, press F8 to agree. Delete all partitions, it will be merged automatically into one big partition. If you want more than one partition, press C and follow the instructions.

    Once done, select the first partition, followed by Format and Install Windows. If you need to share files, make sure you format it as NTFS, otherwise, FAT32 will suffice.

    Windows will now format and install Windows, which can take up to a few hours. In the meantime, watch TV or do something else.

    2. Since you said it's working now, I believe that you managed to do a successful reinstallation? Are you using an external sound card? Or is the sound integrated into the motherboard?

    Also, if the PC hasn't been shut down, shut it down. Wait for around 5 minutes and boot it up.

    Then see if the sounds work.
