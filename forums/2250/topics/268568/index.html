    0X0000000A error - does it have any file? It's usually a .sys file.

    Stop Windows from restarting after a BSOD:

    1. Right click My Computer, select Properties.
    2. Select the Advanced tab.
    3. Under Startup and Recovery, click on the Settings button.
    4. Under System Failure, untick this box: Automatic restart.
    5. Click OK to apply the settings.

    Whenever Windows crashes, note down the full 0X0000000A error. After noting down the error, restart the computer manually. Post back the error message.

    0x000008E error - either one of your RAM slots are faulty, or one of your RAM sticks are faulty, based on your description.

    Since you are getting the 0X0000008E error, it's clear that it's not your RAM that is facing problem, but the RAM slots. A new motherboard is required.

    Try booting up into Safe Mode and let it run. Run programs that are in Windows. Like Solitaire, Freecell, etc. If it doesn't give blue screen, go to either C:\i386 or C:\WINDOWS\ServicePackFiles\i386 folder and copy win32k.sys to C:\WINDOWS\system32

    Also, if your onboard graphics card is by Nvidia... then... erm... it's common. A lot of problems have been reported with Nvidia cards, especially win32k.sys giving BSOD...

    Usually, reinstalling Windows will help. Try copying the win32k.sys file over and see how first. If it doesn't solve the problem, try reformatting and reinstalling Windows. Back up your files before doing so. If it's not possible due to the crashing... split the backing up job separately.
