    Solitaire and all are present on Ubuntu. So yes.

    Dota, Warcraft and the likes, no. If you insist, get Wine.

    To dual boot, depends on what OS.

    To dual boot Windows and Linux

    1. Install Windows
    2. During the partitioning hard disk, create at least 3 partitions. Create more if you need them for backup purposes
    3. Install Windows on the first partition and leave the other 2 blank
    4. Insert the Linux CD, and follow installation guide
    5. Install Linux on the next blank partition
    6. 3rd partition is your swap. It's the Windows version of virtual memory
    7. Format these 2 partitions to install Linux

    To dual boot Windows and Mac

    1. Make sure Mac OS is installed first
    2. Get Boot Camp and follow the instructions to install Windows
