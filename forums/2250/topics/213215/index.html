    Try this.

    Download http://windowsxp.mvps.org/reg/sharedaccess.reg and save it to your Desktop.

    Double click sharedaccess.reg and answer Yes when asked to merge it into the registry.

    Reboot.

    Once back in Windows click Start > Run and type in cmd and hit Enter. Once the command windows opens type in this...

    NETSH FIREWALL RESET and hit Enter.

    Wait for the command to complete. Exit the command box once it has completed. You should be able to access the Windows Firewall now.
