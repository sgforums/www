    9 Jan 06, 15:27
    Originally posted by OldMan:
    i tot pc-3200 is still sdram?
    Both are still considered as sdRam.

    But the main difference is at the clocking rate...
    SDR - Single Data Rate, 1 Complete clock pulse give access to 1 address.
    DDR - Double Data Rate, 1 Complete clock pulse give access to 2 address.

    Pc-166 and below are all SDR clock rate.

    Apart from the apparent speed gain, the timing, voltage and physical dimension is quite different as well.

    Originally posted by OldMan:
    my motherboard is gigabyte ga-7vtx, do you think it can support pc-3200 ram?
    According to the MB specs:-
    Memory

    1. Supports up to 3GB DDR SDRAM
    2. 3 DIMMs of 184-pin DDR sockets
    3. Supports either PC1600/DDR200 or PC2100/DDR266 DDR SDRAM
    4. Supports 72bits ECC type DRAM integrity mode

    So safest bet is to search for PC1600 or the PC2100 RAM types.
