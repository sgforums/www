    Restore without the Recovery CD in Acer

      [*]Turn on the PC and access the BIOS (Press F2 when you see the screen prompting you to do so)
      [*]Check to see if Acer disc-to-disc Recovery function is enabled.
      [*]Make sure the Acer D2D Recovery setting in Main is set to Enabled.
      [*]Exit BIOS and save the changes. The system will reboot.
      [*]When you see the screen prompting you to press F2, press Alt + F10 to enter the recovery process.
      [*]The system will prompt you: "The system has password protection. Please enter a password. Enter 6 zeros and press Enter to continue.
      [*]You will see Acer Self-Configuration Preload. Use the arrow keys to select an operating system and press Enter to select it.
      [*]Follow the on-screen instructions.
