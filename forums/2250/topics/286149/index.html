    It's a virus.

    Plug in your thumbdrive and hold down the Shift key to prevent it from auto running.

    Open Command Prompt (Start > Run and type in cmd).

    In Command Prompt, type in the following line by line, pressing Enter after each line:

    X: (replace X with the correct drive)
    dir /s /a:h > %systemroot%\look.txt
    start notepad %systemroot%\look.txt

    Look at the notepad results

    You may find some unknown files there. temp1.exe, temp2.exe, autorun.inf. Delete all unknown files this way

    del /q autorun.inf (this one must go)
    del /q file (any unknown file found)

    After you've deleted the files, type in exit and press Enter to exit.

    You will need to restore the functionality of your hard disks that may be damaged by the viruses.

    http://www.techsupportforum.com/sectools/sUBs/Flash_Disinfector.exe

    Restart your computer after running the tool.
