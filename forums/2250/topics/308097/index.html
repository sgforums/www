    Hmm... I wonder if IE7 itself is corrupted.

    First, download IE7 and save it.

    Next, open Add/Remove Programs. Find Windows Internet Explorer 7 and uninstall it. If it's not shown there, try ticking the Show Updates box at the top right hand corner.

    After that, restart your computer. Double click on the IE7 installation file that you've downloaded earlier and install it.

    If all else fails, download Winsock XP Fix and save it. Double click to run it. Click on Fix. After that, restart your computer.
