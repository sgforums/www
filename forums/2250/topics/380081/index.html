    And for those who can afford nothing more than 99 Zimbabwean Dollars for a PDF editor, myself included, use these two tools instead:

    http://www.pdftoword.com/ (Online PDF to Word converter)
    http://www.primopdf.com/ (Desktop Anything to PDF converter)

    Both tools above are free and created by the same software company, Nitro. So basically your workflow for editing a PDF file would be:

    PDF to Word -> Edit in Word -> Word to PDF

    Kinda tedious, but if you rarely edit PDFs, this saves you $99 (or more).
