    Some suggestions ....

    Check out this HP webspage for your printer

    Installation issues for Software & Drivers
    HP Photosmart C7280 All-in-One

    http://h10025.www1.hp.com/ewfrf/wc/d...le=1891&lc=en&
    cc=us&dlc=en&os=2093&product=3204785&lang=en

    and in particular:

    HP All-in-One Products - An Error Message Appears on the Computer
    While Installing the Software in Windows

    http://h10025.www1.hp.com/ewfrf/wc/d...5478&lc=en&cc=
    us&dlc=en&product=3204785&os=2093&rule=1891&lang=en

    Here's an off the top of my head thought:

    You are installing software for an All in One printer onto a VISTA Home
    series system and the Home series VISTA does not include Fax & Scan so
    the installer is looking for something that is not there.

    Since I've seen references to a similar problem and Image Zone which is
    part of the HP package, try uninstalling everything HP you can see in
    Remove Programs and deleting any HP icon there may be in the Printers
    folder after unplugging the USB cable from the PC end.

    Then reboot, download the simplified package:

    HP Photosmart Basic Drivers
    HP Photosmart C7280 All-in-One

    http://h10025.www1.hp.com/ewfrf/wc/s...softwareitem=m
    p-52292-3&lc=en&cc=us&dlc=en&product=3204785&os=2093&lang=en

    Make sure you use Download Only and Save, not Run when the choices
    appear.

    Note where you saved the file to and install that -- do not plug the
    USB cable from the AIO to the PC until the install procedure tells you
    to or until you get to [Finish].

    Good luck.
