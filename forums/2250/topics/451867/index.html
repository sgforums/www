    Usually there are two main failure with regards to the fan motor.

    First one would be the starting coil failure, Second one would be the running coil failure.

    Symtomps of starting coil failure would be everytime you start the fan, you hear a loud humming sound, but the fan won't spin. Then you have to manually spin the fan to get it started, then the fan will spin continuously.

    Symtomps of running coil failure would be everytime you start the fan, it spins up, then slows down, then spins up again.

    Also if the ball bearing is stuck it would cause the fan not to spin as well.
