    P1: Sounds like a prank that runs a "shutdown.exe -l -f -t 00" on startup to me.

    Try booting into safe mode by spamming the Fate (F8) button when Windows is loading. Remove all startup items under the Start > Run > msconfig > Startup tab and try booting into Windows normally again. If it still doesn't work,

    1. Run an anti-virus scan on safe mode.
    2. Wait for anty's reply. She would probably have a better solution for you.
    3. Wipe out your disk and do a reformat, and reinstall Windows. (last resort)
