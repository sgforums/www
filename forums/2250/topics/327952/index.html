  • ndmmxiaomayi
    23 Aug 08, 13:08

    Also, disable the Show MSN Today or whatever that crap is.

    Tools > Options

    On the left, select Sign In.

    Untick Show Windows Live Today after I sign to Messenger box.

    Click OK to apply the settings.

    For now, also disable script debugging in IE.

    Tools > Internet Options

    Select the Advanced tab.

    Untick both of these boxes - Disable script debugging (Internet Explorer) and Disable script debugging (Others)

    Click OK to apply the settings.
