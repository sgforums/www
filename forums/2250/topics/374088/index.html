    Originally posted by melissatoh:

    starExcaliblur, Once i on my computer it's stuck at black screen shown above. I cant do anything else. & when i choose to start my windows normally or any other options given, it just hangs.

     


    You don't have to fully boot into windows to put the windows installation CD into your CDROM / DVDROM.

    When your computer is starting, press the Cdrom / Dvdrom eject button to push out the disc loading tray, then insert the windows installation CD into the disc loading tray and press the eject button to close the Cdrom / Dvdrom.

    After your windows installation CD is in the Cdrom / Dvdrom, just either press the reset button or power off and power on the computer. Your Cdrom / Dvdrom should boot up first, thus allowing you to run windows xp setup program from the CD

    PS: Remember to set the boot sequence to Cdrom / Dvdrom first, then harddisk in the bios.
