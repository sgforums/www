    Under Item, select Icon.

    Under Font, leave it as per usual. Next to it, there's a size field. Change that value to 8.

    Same thing, right click on desktop and select Properties.

    Select the Appearance tab. Click on the Advanced button.

    Under Item, select Active Title Bar. Next to it, there's a size field. Choose 25.

    Fourth item, Message Box. Font size is 8.

    Fifth item, Menu. Size is 19. Font size is 8.

    Sixth item, Scollbar. Size is 17.
