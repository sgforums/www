    Whatever your edition is, Windows File Protection should be working. It should automatically be replaced. Refresh the system32 folder every few minutes.

    If it's not there, find it in X:\Windows\system32\dllcache folder. Replace X with your Windows drive. By default, it's C drive.

    The Dllcache folder is hidden. You need to set Windows to show hidden files and folders to see it.
