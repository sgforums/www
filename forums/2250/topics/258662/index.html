    if im more wrong, u are saying that when u double click that icon, it will open up the folder

    open a pic that u like in ms paint

    u needa create a pic 32x32 pixel(click image, then attributes)

    then, #save# it as *youriconname*.ico to somewhere convinent

    bear in mind that ico means icon file

    then right click the folder u wanna change the pic, select properties

    then select advanced tab, at the bottom, u can change the pic to the pic that u just created and name *sth*.ico

    *youriconname* and *sth* means u can change anything, no ned put the astreick sign

    #save#- in paint, click save as, leave the "save as" selection as 24 bitmap
    then, name the file that u wanna save wid the extension as .ico
