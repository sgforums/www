    1st one I can help. Second one I don't know.

    Use If Then Else

    Assuming that the drop-down box name is ComboBox, has 2 items single and married:

    ComboBox.SelectedItem = "Single"
    Then MsgBox("Your salary has not been increased.")
    Else
    MsgBox("$500 has been added to your salary.")

    When they select either value, a popup will appear, telling them whether their salary has been changed. For singles, no change. For married couples, it will popup telling that it has been changed.

    You can edit it to add names.

    For example,

    Dim name As String
    name = textbox.Text
    ComboBox.SelectedItem = "Single"
    Then MsgBox("Dear " + textbox.Text + ", " + " Your salary has not been increased.")
    Else
    MsgBox("Dear " + textbox.Text + ", " + " $500 has been added to your salary.")

    This will produce a popup saying "Dear (name), your salary has not been increased or Dear (name), $500 has been added to your salary."
