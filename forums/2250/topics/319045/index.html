    No idea, could be infected. Some of these buggers prevent a format from happening.

    Try running this tool first - http://www.techsupportforum.com/sectools/sUBs/Flash_Disinfector.exe

    After that, show hidden files.

    • Open My Computer.
    • Go to Tools > Folder Options.
    • Select the View tab.
    • Scroll down to Hidden files and folders.
    • Select Show hidden files and folders.
    • Uncheck (untick) Hide extensions of known file types box.
    • Uncheck (untick) Hide protected operating system files (Recommended) box.
    • Click Yes when prompted.
    • Click OK.

    Open the thumbdrive. Delete all .COM files and any unknown .EXE file(s).

    If unsure, post back the file name.
