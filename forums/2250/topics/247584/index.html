    What LED lights are present on your router?

    For example:

    1. Power
    2. Sync
    3. Online
    4. WLAN

    Typically, the first 3 needs to be on if you are using wired connections. If you are using wireless, WLAN LED lights need to be on.

    Number 2 may or may not be present depending on routers...

    In short, all LED lights need to be on for your internet to work.

    If after checking that your cables are plugged in properly and all LED lights are on but you can't access the Net, try downloading this program to restore your connections:

    http://www.snapfiles.com/get/winsockxpfix.html

    Save it to somewhere convenient and run it. Follow the prompts.

    Once you are done, check your connections.

    If your connections are working properly, delete the tool away. Don't keep this around.

    Might be good to follow basic network tools like Ping
    go to start, run, type 'cmd", at dos prompt, type "ipconfig /all"
    take note of the IP address, subnet mask, gateway address, are they assigned correctly from the router itself else, try "ipconfig /release" and then "ipconfig /renew"
    try ping the gateway address as well (gateway = router in this case)
    if there's reply from gateway, probably is the router to modem connection.
    If that's the case, check the WAN IP of the router, might not be getting proper address from the modem.

    My 10cent worth of information.
