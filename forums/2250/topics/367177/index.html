    Pls refer below to clear the "Unable to Clean" error:

    1. Enter MAINTENANCE MODE: Press MENU, MONO START 4x on ARROW UP within 2-3 sec
    2. Using the ARROW UP botton, press botton and wait for the ff #:
    a. 8 then press SET
    b. 0 then press SET
    3. Using the MONO START, just press until the word "PURGE XXXXX" comes out
    4. Using the ARROW UP botton, press botton and wait for the ff #:
    a. 2 then press SET
    b. 7 then press SET
    c. 8 then press SET
    d. 3 then press SET
    e. Press Stop/Exit
    (This is to reset the purge counter)
    5. Using the ARROW UP botton, press botton and wait for the ff # to appear:
    a. 0 then press SET
    b. 1 then press SET (to reset unit)
    c. 9 then press SET
    d. 9 then press SET (to exit maintenance mode)

    If after this procedure you still encounter problem, see your service center
    Hope this will solve your problem "unable to clean".
