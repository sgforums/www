    Did you publish the movie?

    If you published the movie, what format is it in?

    If you have something like .avi or mpg, maybe your office computer doesnt have the required codec(something to decode the video file) to play those files, so you need to download them.

    K lite media codec is quite good, but make sure to check with your IT engineer to see if you can download and install the codecs.( i know some of them will say it is against the company's policy to install things)

    If all else fails, try to encode your video in .wmv(windows media video it im right) format, that format should be able to be played by most computers

    Or did you jsut copy out the .mswmm file? If you just copied the .mswmm file it is not going to work, as that file is just like an instruction to tell the computer where the original file is and what you have edit to them
