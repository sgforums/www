    Then looks OK.

    XP can try this registry fix:

    Open Notepad and copy and paste the bold parts into Notepad:

    Windows Registry Editor Version 5.00

    [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced\Folder\Hidden\SHOWALL]
    "CheckedValue    "=dword:00000001

    Click on File > Save As

    Save it fix.reg

    Select All Files from the drop down list.

    Click Save.

    Then double click to run it.

    Make sure your Notepad is maximized while copying and pasting it in.
