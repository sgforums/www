    The link to Microsoft's guide seems professional, but thats not the way I did mine. (guess thats why i am nv a pro at these kind of things)

    Here is how I did mine. I have tried these steps with Vista and Xp, OS don't make a difference here for me. I used a wireless router btw.

    Switch both machines on and connect to router.

    1) On my laptop (XP + wireless connection), Control Panel -> Network Place -> Set up a home or small office network

    2) Click through the options. You have the option to name your laptop here. Make sure you enable file and printer sharing.

    3) When done, just click finish

    4) On my desktop (Vista + wired to router), check under Shared Documents, you should be able to see the laptop you named earlier. Create a txt file, and if you see this file in the Shared Documents of your laptop, it means its pretty much done.

    5) Don't think which machine the printer is connected to matters. When printing on a non-printer machine, just make sure when printing, select the correct printer. It should be somethings like
    Printer on

    ok this is my version for win Xp home for the computer which connects to the printer.

    1. ON the main computer(the one with printer),go to Control Panel>Network Setup Wizard

    2. Create a new network and remember to allow printer sharing

    3. DO this for the second computer also. IN this case, my second computer is WinXp home also. Make sure the network name u name it is the same with the main computer one.

    4. on the second comeputer, press [Windows]+[R]

    5. input: cmd

    6.a command prompt will appear. input this into the command prompt: net view.

    7. you should be able to see your main computer name. eg: \\dad's computer

    8. if you can't see, try restarting both machine and repeat step 5 to 7.

    9. still on the second computer, go to Control Panel> Printers and Faxes

    10. right click on the background > add printer

    11. browse for your network printer. add it. and its done.
