    Hmm... looks weird, and it reminds me of some virus, which I can't recall at the moment.

    Try this:

    Go to Start > Run and type in C:\

    If you can open it without the installer thingy, then go in to change some settings.

    Click on Tools > Folder Options

    Select the View tab

    Scroll down to Hidden files and folders

    Select the Show hidden files and folders radio button

    Untick the Hide extensions for known file types and Hide protected operating system files (recommended) boxes. When it prompts you, just click Yes.

    Click OK and close.

    Look around your desktop and see if your folders have a .exe extensions.

    Folders don't require any files to open... it's the .exe that requires.

    Since there's no way you can check whether important files are deleted, might as well just scan for viruses. There's one virus which was notorious for doing that.

    Only do this in IE, as it uses ActiveX.

    Click here to do a scan with Kaspersky.

    Then click on Kaspersky Online Scanner button.

    Accept the License Agreement. It will start installing the scanner. When IE prompts you about some ActiveX component, allow it.

    Then it will start to download the definitions. After that, click Next.

    Now click on Scan Settings.

    In the scan settings make that the following are selected:

    Scan using the following Anti-Virus database:

      Extended (if available otherwise Standard)
    Scan Options:
      Scan Archives
      Scan Mail Bases


    Click OK

    Now under select a target to scan:

    Select My Computer

    The program will start scanning. It takes a few hours to complete.
