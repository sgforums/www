    On your the other computer, first follow the instructions manual to install the wireless drivers properly. Usually plug and play would work, but if it doesn't, there should be a CD that you can insert and install.

    After that, reboot.

    Next, on your computer, log in to the router. Look around the settings and find the wireless settings.

    Wireless network name is usually called SSID. If you find such a field, enter a wireless network name for your wireless network. Give it a unique name so it won't clash with your neighbours and cause you to lose connections, lose signal strength or even to the extent of not able to connect due to different security settings.

    After that, navigate around to find the wireless security settings. Some routers put them together with the wireless settings page, some put it in the advanced page.

    If both computers are using XP, I will recommend you to use WPA. When selecting WPA, please choose the Pre-shared Key (PSK) option. The other options will require a server.

    For the PSK field, treat it as a normal password field and enter a password. Write it down somewhere because you will need it.

    If both computers are using Vista, you can set it to use WPA2, which is more secure than WPA. Again, you should choose the PSK option.

    After that, save the settings and restart the router (if necessary).

    Once that is done, try connecting to your wireless network. Enter the PSK that you've written earlier and set it to remember so you don't have to always re-enter it.

    From your first post, I gather that you have at least two computers. Please correct me if I'm wrong.

    To log in to the router from your computer (not the other computer that you want to connect via wireless) :

    1. Open Command Prompt

    2. Type in ipconfig and press Enter.

    You should see something like:

    Default Gateway . . . . . . 192.168.X.X (X are numbers)

    Type in this address into your web browser. Log in to the router. Router username and password is in the manual if I remember correctly.

    That done, you should be able to navigate around the router pages and set the wireless settings.

    If you've forgotten your WEP/WPA key, I'm afraid you have to reset the router and re-do everything. Unless... your router offers a way to view the PSK in plain text.

    If you use Firefox, install this addon - https://addons.mozilla.org/en-US/firefox/addon/10174

    Restart Firefox. After that, click on the password field. It will show you the password in plain text. Do this for the PSK field...

    Copy down the PSK and pass it when prompted to enter this key.
