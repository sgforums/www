    Also, try other wireless hotspot if can (If not study tertiary or working a company that uses wireless broadband then try Wiress@SG). Rationale: If you can connect, then is not Windows 7 problem.

    Also, do remember to check in device manager see if the driver for the wiress is there or not. I got a friend who is "joker" that has his Wiress network adapter's driver missing and cannot go wireless internet at all (On the school). He is using Vista and all I asked him is to search for device manager for updates....... (And vola, problem fix after installed the driver automatically)
