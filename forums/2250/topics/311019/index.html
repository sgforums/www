  • ndmmxiaomayi

    For those who have automatic updates turned on and your PC didn't crash after the February automatic updates, you should be using Vista SP1 now.

    The reason is simple. Microsoft removed the patch for Vista SP1 after it crashed a lot of PCs. Those who are lucky and survived the patch will be fine and Vista SP1 should be downloaded and installed automatically for you.

    Those whose PCs that didn't survive the patch, then my condolences. If you reinstalled Vista, Microsoft have already removed the update, so you won't get the patch at all.

    The patch is only downloadable via MS website now, manually. It's not available via Windows Update anymore.

    Although Vista SP1 is already released to manufacturers, I haven't seen one installed with Vista SP1.

    Microsoft estimates that Vista SP1 will be released in mid-March. I believe that this deadline is still valid despite the bad crashes.

    Filesize is about several hundred megabytes...
