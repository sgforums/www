    Err... I don't know how to retrieve data from the physical RAM.

    That's because physical RAM is volatile, meaning that when the power is cut off, all data will be gone.

    It's always refreshing, in order to make sure that the data is the newest.

    As for virtual RAM, I only know there's a file called pagefile.sys in Windows XP.

    In Windows 9x and ME, it's called Win386.swp

    As for memory cards... just Google.
