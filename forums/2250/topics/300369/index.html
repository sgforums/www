    Got a few methods to backup DVDs.

    1. Burn an image (iso) of the DVD:

    To do this, make an image of the DVD using example nero burning rom http://www.free-codecs.com/download/Nero_Burning_ROM.htm

    Then use daemon tools to mount the DVD and play it like a DVD
    http://www.disc-tools.com/download/daemon+sptd+md5sum

    2. Reencode. This one is more tedious. You can either wait for other forumers to help you if you choose this choice or go forum.doom9.org to see what you need. Dont recommend you to try this unless you want to have a headache

    ===
    btw, dvd shrink only shrink them into 1 dvd size... not produce a copy of your dvd.
