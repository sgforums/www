    LOL... if you don't know, means it's most likely turned on. By default, it's turned on.

    For XP

    • Right click on My Computer and select Properties.
    • Select the System Restore tab.
    • If the Turn off System Restore on all drives box is not ticked, then System Restore is turned on.

    For Vista

    • Right click on My Computer and select Properties.
    • Click on the System Protection link on the left.
    • If UAC prompts, allow it.
    • Select System Restore tab.
    • If your C drive partition has a tick, means that System Restore is turned on.

    Since System Restore is turned on... you will find your hard disk space decrease. This is because System Restore will automatically create a new restore point when it detects changes made, like when you installed a new program or uninstalled some unwanted programs.

    To clear them is simple.

    For XP

    • Click on Start > All Programs > Accessories > System Tools > Disk Cleanup.
    • Select your C drive and click OK.
    • Let Windows finish calculating. When done, click on the More Options tab.
    • Under System Restore, click on the Clean Up button.
    • This will clear all the old restore points except the latest one and hopefully give you back some space.

    For Vista

    • Click on Start > All Programs > Accessories > System Tools. Right click on Disk Cleanup and select Run As Administrator.
    • UAC will prompt. Allow it.
    • Select your C drive and click OK.
    • Windows will start calculating. When done, click on the More Options tab.
    • Under System Restore and Shadow Copy, click on the Clean Up button.
