    Do you have a firewall? If yes, when you start the pinging, tell the firewall to allow it.

    Here's how it would look like if it's successful:



    You will see 4 replies from 216.109.112.135 (that is Yahoo! IP address).

    If you get a very complete different set, create the following file and run it.

    Open Notepad. Use notepad and no other editors or else it doesn't work. Copy and paste the code below:

    tracert yahoo.com > results.txt
    notepad results.txt
    Save this file.

    For the File name, type in: trace.bat
    For the Save as type: All Files

    Click Save. results.txt should appear once the tracing is done. If it asks you to create a file, click Yes. If the firewall prompts you, allow it.
