    Originally posted by chanff8:
    There is no way to do it using iTunes because of copyright protection. If you try to explore it using Windows Explorer you'll find the files hidden and the song file names will be replaced by numbers or letters in random order.

    If you are using Windows, use Yamipod. Shut off your iTunes without ejecting your iPod before launching this application. No installation required.

    If you are using Linux, there is Yamipod for Linux as well

    If you are using Macintosh, you can either download Yamipod, or use the more popular Senuti.
    If you reformatted your computer which is synced with your iPod, the alert about whether you want to pair with the computer should appear, when you plug the iPod into the newly formatted computer for the first time.

    To be on the safe side, plug your iPod into ANOTHER computer, select no when the alert to sync pops up, go to iPod options and change the setting to "Manually manage my songs and playlists"
