    1. Start > Control Panel > Regional and Language Options
    2. Select Regional Options tab. New window will open
    3. In the new window, select the Time tab
    4. In the Time Format box, select either h:mm:ss:tt or hh:mm:ss:tt
    5. Click OK twice to save the settings. If prompted to reboot, reboot.
