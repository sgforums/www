    Lotus Symphony needs Java to be installed... otherwise, each time you open it, it will throw out thousands of Java programming errors.

    It will still run though... (just create a new document) but with some errors.

    As it's written in Java... expect it to have some lags.

    On my P3 and 300++ MB of RAM, it takes about 3 minutes to start up properly. Slower than MS Office and Open Office.

    Looks wise, it's a cross between MS Office and Open Office.
