    For Windows, refer here

    For Task Manager, Ctrl + Shift + Esc.

    When typing in forums, if you want to go to the end of a sentence, just press the End button. Go to the first letter of the sentence, press Home button.

    Go right to the top of a paragraph, Ctrl + Home.

    Go right to the end of a paragraph, Ctrl + End.

    Select a whole word, Ctrl + Shift + left or right arrow key.

    Select letter by letter, Shift + left or right arrow key.

    Find something, Ctrl + F.
