    In my first reply to you - the link.

    For your HijackThis log, open up HijackThis.

    Put a tick in the box for these entries:

    F3 - REG:win.ini: load=C:\windows\system32\wincfgs.exe
    O2 - BHO: CVirtualDNSObj Object - {86C510E9-97EF-4749-914F-0280247BE3A6} - C:\WINDOWS\VirtualDNS.dll (file missing)
    O4 - HKLM\..\Run: [RavMont] C:\WINDOWS\system32\RavMon.exe

    Click Fix Checked.

    Go to Start > Control Panel > Add/Remove Programs

    If you find NewDotNet or New.Net or anything that sounds like it, uninstall it.

    Boot into Safe Mode and delete these files:

    C:\windows\system32\wincfgs.exe
    C:\WINDOWS\VirtualDNS.dll
    C:\WINDOWS\system32\RavMon.exe

    Also download Winsock XP Fix in case you lose your connections. Run this tool if you can't access the Internet after uninstalling New.Net.
