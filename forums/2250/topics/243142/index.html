    Singnet users use proxy. Singnet's proxy is proxy.singnet.com.sg. Port number 8080

    For setting proxy in Firefox, see the Firefox sticky thread.

    For IE, refer to xiaodingdung last post.

    For Avant browser, Tools > Proxy > Customize. In Address, type in proxy.singnet.com.sg

    For the Port field, type in 8080. Click Add. Then click on Tools > Proxy. Make sure proxy.singnet.com.sg:8080 is ticked

    For Maxthon, Tools > Manage Proxy List. Click on Add. For Address field, type in proxy.singnet.com.sg

    For Name, type in proxy.singnet.com.sg

    For Type, choose HTTP. Tick Use this proxy after adding it box.

    For Opera, Tools > Preferences. Click on Advanced tab. On the left hand side, select Network. Click on Proxy Servers. Tick the HTTP box. Type in proxy.singnet.com.sg for and 8080 for Port. Tick Enable HTTP 1.1 for proxy.

    Click OK.

    Other ISP proxy servers if you know, please share.
