    1st problem: If you're using Vista, plug in the HDD into the PC. Next, open up the start menu and right-click on Computer. Select Manage. A new Window should open up. Look for Disk Management on the Left. Click it. Look for your portable HDD (Should be labeled Disk 1 or Disk 2 or so on. See if it has any unallocated space. If so, right click on the 299GB partition and select Expand Volume. Follow the prompts and then you're done.

     

    2nd problem: If it's a portable hard drive, open it up and extract the HDD inside. The HDD should be fine, and only the USB interface of the enclosure should be damaged. The data can be recovered by plugging the bare drive into a computer.
