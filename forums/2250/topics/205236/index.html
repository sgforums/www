    1st problem: Make sure that the file is not not read-only. It must open normally.

    For example, you are opening homework.doc. The title should read homework.doc, not homework.doc (read-only). If it's opened as read-only, you can't copy and paste anything.

    2nd problem: This is pretty common with Windows Explorer. All applications are running normally, except that you don't see them in the system tray. If you need them, just restart your computer.
