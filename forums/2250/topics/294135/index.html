    The evil solution...

    Uninstall the USB controller drivers and SM Bus controller drivers.

    Restart computer. Windows should start installing some drivers. Then check Device Manager again to see if the USB and SMB bus controller drivers are working fine.

    If they are, plug in the removable drive.

    Doesn't work... find the motherboard drivers... download it.

    Uninstall the USB and SM Bus controller drivers, restart computer.

    Windows will install them again... cancel the installation or just let it move on. Install the motherboard drivers and restart computer.
