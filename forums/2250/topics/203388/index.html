    Try this:

    Go to Start > Printers and Faxes

    If you did not put this in your Start menu, go to Start > Control Panel > Printers and Faxes.

    Right click on your printer and select Printing preferences.

    Select the Layout tab. Under Orientation, select Portrait.

    Next, select the Paper Quality tab. Click on Advanced button.

    In Paper/Output, make sure Paper Size is A4.

    Try printing again.

    If it doesn't work, change the Orientation to Landscape. Try printing again.

    Fail again, copy everything to Microsoft Word and print. It should work.

    If it doesn't work, drag the ruler in Microsoft Word to align properly. Open up Print Preview by going to File > Print Preview. Zoom in to 100% so that it will show you how it would look like once it's printed. If some words are off, re-align them again till you get it right.
