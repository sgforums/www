    1st of all...how sure are U that the CPU set that came in is working in the first place?

    I wouldnt want to buy something that has a HDD from overseas.Even if not spoilt...the number of bad sectors will make ur heart ache.

    I think now the best option is to take out the individual parts out and see what works and what is not...

    1.PSU (Power Supply Unit)

    The US is using 120V...We are using 240V...if U neber set the correct fuse on ur PSU...then there will be a loud boom...at least a blown fuse and a spoilt PSU(unless U know how to replace).

    2.Motherboard.

    Simpler if ur mobo has integrated VGA output. Just put in ur CPU chip and RAM and power it on...if nothing comes out on ur monitor...then no need to put the rest in liao...

    if no integrated VGA output...then have to plug in graphics card and power on. if still no monitor output, then either ur graphics card is gone or something on ur mobo(mobo itself,RAM,CPU chip) is gone. The sickening part is finding out exactly which one.Having spare hardware to help U test is very helpful in this case.

     

    And if U buy a new mobo and AGP card...then wouldnt it be about the same as writing off this set and buying a new one?Start praying my friend...
