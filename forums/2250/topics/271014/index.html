    Originally posted by Darkdrium:
    Yah.. it only showed for a less than 1 sec.
    OK. Do this in order to note the error down.

    1. Right click My Computer and select Properties.
    2. Select the Advanced tab.
    3. Under Startup and Recovery, click the Settings button.
    4. Under System Failure, untick the Automatically Restart box.
    5. Click OK all the way to apply the settings.

    When you get the blue screen of death, note down the error.

    There are two parts which are important:

    0X8 alphanumeric
    File name if exist

    The 0X8 alphanumeric looks something like this: 0X0000008E
