    I believe Sony has their own wireless management software. Can you check if it's turned on.

    Also... if using XP, make sure that the Wireless service is turned on.

    1. Start > Run and type in services.msc, press Enter

    2. Scroll down to Wireless Zero Configuration

    3. See the Status column. It should be Started. If it's not, double click on it.

    4. Startup Type - Change it to Automatic

    5. Click on the Start button to start the service

    6. Click OK to apply the changes

    If Vista...

    1. Start > Run and type in services.msc, press Enter. Can't find your Run button, kindly look at the Vista sticky.

    2. Scroll down to WLAN AutoConfig

    3. Check the Status column. It should be started. Otherwise, start this service and change the Startup Type to Automatic.

    Neither works... just run all Sony programs. One of them should be managing it, play around with the settings.
