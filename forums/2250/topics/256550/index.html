    Download ISO version of Memtest and burn it to CD or Floppy version and save it to a floppy.

    Unzip the files before saving.

    Boot up your PC, get into BIOS. Usually you will need to press a button to get into BIOS, look at laptop screen once it boots up and press the corresponding key.

    Set the first boot device to either CD or floppy (depending on which version of Memtest you saved earlier). Put in your CD or floppy, save the settings and exit.

    This will check for memory errors.
