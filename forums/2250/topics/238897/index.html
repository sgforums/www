    Me. XP and Xubuntu.

    Anyway, it's quite easy... you will only have driver problems with Linux...

    To dual boot two Windows OS, you have to install the older one first.

    Let's say you want to dual boot Windows 98 and XP.

    First, overwrite everything (which is reformat)

    Create a few partition. Install Windows 98 in the first partition.

    Once done, put in XP CD. Follow the prompts. When you reach a stage about choosing how to install, don't select upgrade. Select New Installation. Choose the partition you want XP to be installed on and just follow instructions.
