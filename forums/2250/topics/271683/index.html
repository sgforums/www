  • ndmmxiaomayi
    1 Aug 07, 22:46
    1. Registry takes up space. Registry is more than the Registry Viewer, there is actually quite a few files which Registry Viewer grabs the files from.

    2. There's something called logs. Some programs don't remove logs on uninstallation.

    3. There are other hidden folders which these programs created, mainly for configuration. These folders are not gone on uninstallation as well, so that they could take the settings of your previous version and not let you re-configure everything again. For lovers of Firefox, this should be very familiar as your profile corrupts all too often.They are known as Application Data folder, and could be accessed by typing %appdata% into the Run dialog box.

    4. Lastly, there's the Event Viewer which keeps track of everything you do. Again, the Event Viewer grabs its logs from various files and in real-time as well.
