    That is very vague.

    Is any file listed when it blue screened?

    1. Click on Start > Control Panel.

    2. Double click on System.

    3. Click on Advanced System Settings on the left. If UAC prompts, allow it.

    4. Under Startup and Recovery, click on the Settings button.

    5. Untick Automatically Restart box and apply the settings.

    When it blue screens again, note down the file and error.

    Usually something like 0x000000a
