  • ndmmxiaomayi
    9 May 07, 21:40
    Ever tweaked your Firefox for one PC and want the same settings for another PC/user account?

    You can import all Firefox's tweaked settings into another PC by locating the prefs.js file and copying it into the Firefox's folder.

    This trick only works if you've tweaked, but not installed any add-ons. If you've installed any add-ons, then you might have some trouble with Firefox.

    The location of this file is at: C:\Documents and Settings\Your Username\Application Data\Mozilla\Firefox\Profiles\XXXXXXXX.default

    Copy the prefs.js file into your thumbdrive and paste it into the user Firefox's folder or the other PC Firefox's folder.

    Paste it here: C:\Documents and Settings\Your Username\Application Data\Mozilla\Firefox\Profiles\XXXXXXXX.default

    Replace C: drive with the drive where Firefox is installed.

  • ndmmxiaomayi
    15 May 07, 20:20
    Tools > Options

    Select the Security tab at the top.

    Under Passwords, click on the Exceptions... button

    You will see sites that you've chosen not to remember passwords for. Click once to select the site, then click on Remove Site button.

    Go to that website that you want passwords to be saved. Log in as per usual, Firefox will prompt you.

    This time, select Remember passwords.
