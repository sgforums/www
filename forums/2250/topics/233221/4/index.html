    Running 2 different versions of Firefox

    For the bored... really. Those who wants to use FF2 and wants FF3 for testing, you will definitely need more than one profile, otherwise, it will start crashing.

    1. Close all Firefox windows.

    2. Click on Start > Run. Copy and paste the following in: firefox.exe -ProfileManager

    3. A new window will open. Click on Create Profile. Follow through and you're done. Remember your profile name, because you need it.

    4. Start your Firefox this way:

    a. Click on Start > Run.

    b. Type in the following: path to your Firefox -P Profile_Name -no-remote and press Enter. You should be prompted to select a profile, select the one you want and start it.

    Path to your Firefox - By default, it's in C:\Program Files\Mozilla Firefox\firefox.exe

    Profile_Name - if you want to start the default, it will be named default. Otherwise, it's the one you created earlier.
