    I assume that you have this laptop setting: When you close the lid, it will be on standby mode.

    So, although you ask the laptop to shutdown, you close the lid first, causing some confusion for the laptop. It chooses a standby instead of a shutdown.

    I guess you leave it in a cool place, running. It can take really to cool down, so if it's an air-con room, it would greatly reduce the time. There is no way to turn the fan off, it would be disastrous.
