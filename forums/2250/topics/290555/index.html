    Not that more and more will be moving to Vista, but just curious what kind of softwares you used are compatible with it.

    Found one list here: http://www.computerforum.com/79375-vista-software-compatibility.html

    Other than MS products, I find that these programs worked flawlessly for me:

    1. Notepad++
    2. nPad2
    3. 8Start
    4. Firefox
    5. Winpatrol
    6. Spybot S&D
    7. Download Express
    8. 7-zip beta

    For Firefox, if you are a tweaker, especially if you mess with the configuration files, you need to run as Administrator, otherwise the settings won't be saved. It will also give out an error about some files not found.
