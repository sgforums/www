    [b]Can I put ddr2 ram at my empty slot 2?    [/b]
    Judging from what you said earlier, you are using DDR I RAM. So, no you cannot use DDR II RAM, since your motherboard do not support it and it will not fit into the slot.

    About which DDR I RAM to buy, just buy any DDR I RAM for laptop will do.

    Yes,I know that there are many types like for example DDR 133 and so on. Even if you buy a ram with a higher timing, it will still work, it just that the ram will not run in full capability (For example, DDR 266 RAM will not run in 266 MHz, instead it will run in 133 MHz, which your laptop can support).
