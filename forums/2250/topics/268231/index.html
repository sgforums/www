  • ndmmxiaomayi
    15 Jul 07, 20:09
    Start > Control Panel > Network Connections

    Double click on your wireless network

    Select the General tab

    Under This connection uses the following items: double click on Internet Protocol (TCP/IP)

    A new window will open

    Select the General tab

    Are these two options being selected?

    1. Obtain an IP address automatically
    2. Obtain DNS Server address automatically

    If yes, it's dynamic

  • ndmmxiaomayi
    16 Jul 07, 13:02
    Try this in Command Prompt:

    1. Start > Run and type in cmd
    2. Type in ipconfig /release, then press Enter
    3. Type in ipconfig /renew, then press Enter

    If no error at Step 3, type in exit and press Enter

    If got error, post back the error message.
