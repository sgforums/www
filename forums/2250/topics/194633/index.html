    After reinstalling, the bookmarks should still be there because Firefox creates registry entries. Try waiting for a while, the bookmarks may need some time to show.

    C:\Program Files

    xxx is the username.

    Find the Firefox folder and delete everything.

    After that, reinstall Firefox.
