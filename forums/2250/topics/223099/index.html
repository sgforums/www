    Try this to off Windows Firewall totally:

    Start > Run and type in services.msc

    Scroll down to Windows Firewall/Internet Connection Sharing (ICS)



    Double click it.

    Click on Stop.

    Once it had been stopped, changed the Startup type to Disabled. Click OK to apply the settings.
