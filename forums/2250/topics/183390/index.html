    For those who don't like IE title (xxx provided by Microsoft Internet Explorer), here's one way you can change it.

    Open up registry editor (Start, Run, type in regedit in the textbox).

    Go to this key:

    HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main

    Change (or create) the value of the string "Window Title" to whatever you want on the titlebar of Internet Explorer.

    To have no title except the title of the web pages you are browsing do not enter anything for a value.

    Note: Do not edit the registry if you don't know how.

    A better disk cleanup

    This will free up a lot more disk space for you.

    Assuming that Windows XP is installed and is in the default folder (C drive), this is how it works.

    1) Open up notepad and type this in:

    c:\windows\system32\cleanmgr.exe /dc /sageset: 1
    c:
    cd \
    cd c:\windows\prefetch
    del *.* /q

    2) Save the file, changing the extension from .txt to .bat
    For this tip's purpose, we will call it clean.bat

    3) Execute the file. This will run the Disk Cleanup program in a special mode which asks what items you will want cleaned when Disk Cleanup performs an unattended cleanup. Check the items you wish to have cleaned up, then click OK.

    4) Right-click on the clean.bat file and click edit. Change the first line to read:

    c:\windows\system32\cleanmgr.exe /dc /sagerun: 1

    5) Save the file. You can execute this file in place of running Disk Cleanup, or, to have this program run unattended, run the Add Scheduled Task Wizard in the Control Panel and create a scheduled task using the clean.bat file you just created.

    Changing folder icons

    Open up MSPaint (Start, Run, mspaint)

    Copy or design in MSPaint. After that, save it as x.ico (where x is your file name).

    Open up Windows Explorer. (Start, All Programs, Accessories, Windows Explorer).

    Right click on a folder that you want and select Properties. Go to the Customize tab. Under Folder Icons, click Change icons. Find that icon you just designed.

    Click OK and there you goes. A new folder icon.

    For those who need to type password but the Num Lock is always not on try this:

    1) Open up Registry Editor.

    2) Navigate to HKEY_USERS\.Default\Control Panel\Keyboard.

    3) Change the value for InitialKeyboardIndicators from 0 to 2.

    For those unfamiliar with registry, don't edit it. There's a script from Microsoft that will help you overcome it.

    http://support.microsoft.com/directory/article.asp?ID=KB;EN-US;Q262625

    Warning: No abusing of this knowledge.

    This will allow one to change password without knowing the previous password. This is good when you forget your admin password.

    1) Log on to the computer and run Command Prompt. (Start, All Programs, Accessories, Command Prompt)

    2) Type in net user xxx * (xxx is username, put in quotes if it contains spaces. e.g. net user administrator *, net user "Ang Hwee Ching" *)

    3) It will prompt you to take the new password. To see all users, just type in net users. To see the different user's password, type in net user xxx (xxx is the username)

    This is a good tweak for anybody working in an environment with MSN or any other kinds of IM service disabled. 2 disadvantages:

    1) If your boss shares the same domain as you, and you send this to everybody, good luck.

    2) If you want to send messages to everybody, but not your boss, you are going to type in the others' IP addresses a lot of times just to send one message.

    Here's the instructions:

    1) Open up Command Prompt.

    2) Type in net send * (your message) without the brackets.

    Wait for a while and it will be sent.
