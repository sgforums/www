    I suppose you'll need to install the necessary language support into Windows.

    Go to Start > Control Panel ( > Date, Time, Language and Regional Options ) > Regional and Language Options > Languages tab

    Under Supplemental Language Support section, tick both. They may need you to follow certain installation instructions which would require you to have a XP CD at hand.

    Within the same tab and above that section, there's a Details... button. Click that and there will be another window. Go to the Advanced tab and click on the checkbox within the Configuration Compatibility section.

    For anyone else who wish to type Chinese characters, you'll need to set the text input options accordingly under the Settings tab of the same "Text Services and Input Languages" window.
