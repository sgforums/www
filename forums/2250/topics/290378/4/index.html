    Originally posted by AndrewPKYap:
    another thing.... if you have auto update on... firefox will revert back to FF2.x ... and replace your FF3.0b2
    Install FF3 somewhere else to solve the problem. And .. use a separate profile for testing purposes.

    Start > Run

    "path to firefox" -P -no-remote

    Click on Create Profile. Make sure your new profile has no spaces otherwise you will meet with odd problems like mine.

    To run Firefox with that particular profile

    "path to firefox" -P profile name -no-remote
