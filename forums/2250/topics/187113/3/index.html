    Use more anti-spywares to scan.

    Adware.Navhelper

    1) Go to Control Panel, Add/Remove Programs. If NavHelper is present, uninstall it.

    2) Restart the computer.

    3) After the computer starts up, click on Start, Run. Type in regedit.

    4) Find these keys:

    HKEY_CURRENT_USERS\Software\NavExcel Ltd
    HKEY_LOCAL_MACHINE\SOFTWARE\NavExcel
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\NavHelper

    Delete the ones in red.

    Spyware.CometCursor

    The solution for this is too long. You read through Symantec's writeup, and ask me when you don't understand.

    http://www.symantec.com/avcenter/venc/data/spyware.cometcursor.html
