    How to setup a server with win 2000 server and set up all end-users pc as wireless?
    Should be the same as a wired network.
    The network connection is something like this: Server<--->Network Switch<--->Wireless Access Point<--->End-Users. The only special configuration required is at the end-user side since you need to install the wireless drivers + software.

    Note: Make sure you are using an access point and NOT a wireless router.

    I am not sure why a company will allow wireless devices to connect to a Windows Server but take note that current wireless technology are not very secure even if all transmissions are heavily encrypted. In addition, it is relatively easy for an attacker to disable your network just by blasting powerful jamming signals from a distance. Even if your network is down unexpectedly for just 5 minutes, it is still consider as a breach of security. If security is important to you, I suggest you use Ethernet.

    And if you are going to use SonicWALL Unified Threat Management Solution, it means security is important to you. You should be able to receive training to configure the device from the company who sells you the solution.

    In my opinion, you are going to have a weird network configuration since to me: on one side, you have your network expose to attack, and on the other side, you have a smart firewall that detects and drop any packets in the "wanted list".
