    Had a similar thing happen to me with my Compaq. Would ask the check disk for errors, and hang. Skip, and would hang 5 mins later. Turns out the drive failed.

     

    To see if it is software problem or HDD problem,

     

    On booting up the PC, as soon as the manufacturer logo appears, press Del, F10 or whatever other combo that might be used to enter BIOS

    Next, Scan through each tab until you see one that refers to the hard drive. Press enter if able. If it is about boot preferences, wrong one. If it says S.M.A.R.T support or something, you got it. Most, if not all, recent hard drives support SMART. Run the self-test. If it fails, your HDD spoilt.
