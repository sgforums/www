    Try this:

    Right click on the file or folder that you want to compress.

    Select Add to archive.

    Select the Advanced tab, and click on Set Password

    Enter the password you want, re-enter it.

    Click OK throughout to apply the settings.

    This is becasue winRAR will use the entered password as default after you typed a password. For example, you chose 'a' as password. After that all password protected rar file will use 'a' as password.

    Just set the password as empty again in winRAR press enter. and now go to pass-protected file again, it should be pass-protected.
