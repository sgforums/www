    This could be because your registry hive is deleted or corrupt.

    Try running the chkdsk command from Recovery Console and reboot the system.

    The Recovery Console can be accessed from:

    1) Your Computer

    Reboot your computer
    Before the logon screen, press F8

    2) Your Windows CD

    At the Welcome to Windows Screen, press F10 or "r"

    More on how to use the Recovery Console is found here.

    http://support.microsoft.com/default.aspx?scid=kb;EN-US;314058
  • Conner MacDermott
    8 Apr 07, 19:43
    Once you got into Recovery Console, select the operating system you are using. By default, there should only be one. Unless you have a multiboot system. Post back to tell me if this is the case.

    Now, you will be asked for an administator password. This is usually blank unless your itchy hands changed it. In that case, post back to tell me if that is the case.

    Once inside, type "chkdsk /p" and enter. Please type exactly as shown, excluding the parenthesis but including the space.

    Reboot your computer.

    If this does NOT solve your problem still.

    Type "CD C:\system~1\_resto~1"
    Type ">dir"

    You will see a list of Restore Points (rp) by now.

    Select the second highest Restore Point. (Eg: rp8

    Type "cd rp88" <- Change the number to the second highest one
    Type "cd snapshot"
    Type "copy _registry_machine_system c:\windows\system32\config\system" <- Don't copy """
    Type "y"
    Type "exit"
    Press any key to initiate reboot.
    Remove CD from Drive.

    If this also cannot, I have no other means liao.

    Cheers.
