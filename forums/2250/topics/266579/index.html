  • ndmmxiaomayi
    7 Jul 07, 23:40
    You have to log in to your router. Typically the IP address is 192.168.1.1, but may differ from products to products.

    To check, open Command Prompt (Start > Run and type in cmd)

    In Command Prompt, type in ipconfig /all

    You will see a Default Gateway. Type in that Default Gateway IP address to log in to your router.

    Check your manual for the username and password to log in.
