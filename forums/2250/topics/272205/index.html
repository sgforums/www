    Copied from http://www.computing.net/windowsxp/wwwboard/forum/72810.html

    some folks figure your hard drive is shot, but try some tweaks before you give up on your HD;

    1. change the boot sequence in the BIOS to: 1 - CD, 2 - floppy, 3- HD

    2. the disk cabling should be checked for problems.

    3. use your Antivirus Emergency Boot Disk, if you have one.

    4. boot to the Options menu (use F8 key repeatedly during boot) and use the arrow keys to select Last Known Good Configuration and press the Enter key.

    if that allows you to boot to XP, run Error Checking in My Computer, Properties, Tools.

    5. may have a bad RAM chip; try swapping them to see if one is faulty.

    Sidenote: Acer was never very reliable... My desktop is Acer, and so is my gf's. Both have problems with hardware occasionally...

    Let's try this.

    Go into BIOS again.

    Check the hard disk mode, whether it is set as Master, Cable Select or Slave.
