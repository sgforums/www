    Though it would be cheaper in cost to upgrade with your mobo, but i think if you can afford it, you should go for Sandy Bridge upgrade.

    If you want to upgrade only CPU, i say go for the 45nm since power consumption will be lower. Though Core 2 Quad is possible, but i would recommend core 2 duo for LGA 775.
    Core 2 Quad is non-native quad core, which means under home conditions, you wont see much difference in speed. For Core 2 duo,  E8600 is the highest you can go.

     

    http://windows.microsoft.com/en-US/windows7/products/compare

    eBay; you are buying from another end user, so the software would be second hand, though not really important, but i'll recommend you not buy a second hand one.
    64-bit is better.

     

    3. Yes,6GB RAM. For home use, actually 2-4GB ram is sufficient.
