    Hhahaa very nice joke,anyway i have solve the problem already,i got it from the web paste it here in case others may encounter this problem too

    Hive: HKEY_CURRENT_USER
    Key: Software\Microsoft\Windows\CurrentVersion\Policies\System
    Name: DisableTaskMgr
    Type: REG_DWORD
    Value: 1=Enablethis key, that is DISABLE TaskManager
    Value: 0=Disablethis key, that is Don't Disable, Enable TaskManager

    Just double click on the grey area.

    From Windows 2000 onwards, there's no such thing as Ctrl Alt Del to Task Manager. It will give you options to lock the PC, log off, shut down, open task manager, etc.

    To access Task Manager without pressing Ctrl Alt Del, just press Ctrl Shift and Esc simultaneously.
