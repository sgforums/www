  • ndmmxiaomayi
    11 May 07, 19:06
    With Firefox, see sticky.

    With IE:

    Tools > Internet Options



    Select Connections tab

    Under Local Area Network (LAN) Settings, click on LAN Settings... button



    Under Proxy Server, tick this box: Use a proxy server for your LAN

    In the Address field, copy in proxy.singnet.com.sg
    In the Port field, type in 8080
    Tick the Bypass proxy server for local addresses box

    Click on the Advanced... button



    In the HTTP field, type in proxy.singnet.com.sg for the Address field, 8080 for the Port field

    Same for Secure and FTP fields

    Under Exceptions, type in your router's address (default is 192.168.1.1)



    Click OK all the way to save the changes. Restart IE.
