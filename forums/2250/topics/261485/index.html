    http://kmeleon.sourceforge.net/

    Finally, a good alternative to Firefox. It's faster than Firefox even with all the plugins that are loaded.

    There's not much difference between Firefox and K-Meleon as it's based on the Gecko engine (the engine that powers Firefox).

    It's interface, however, looks like IE. I almost thought I'm using IE.

    Tweaking K-Meleon is the same as tweaking Firefox, not too bad. Except some advanced tweaking, which requires you to edit the commands yourself. Unless you know K-Meleon well, the advice is don't touch it.

    Support for extensions is lacking though, but I don't think this is an issue, yet.

    One new thing - tabs are called layers, otherwise, nearly everything is the same.

    No spellchecker, but doesn't matter. Who spells correctly any way?
