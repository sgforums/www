    there's no how much you must oc...up settings at regular intervals one at a time..run benchmark tests (which may last for hours or even days) and wait for error or hangings. if after a specified hours or days u desire to set the tests run stable you can up the settings again until you hit one that cuases the computer to hang or crashes or behave funny...then back down to the previous notch setting and run test again to double confirm. one this is set (e.g cpu speed u tried), proceeed to ram/ graphics card etc and repeat again. different types of benchmark tests are named and available for different parameter tweaking so you may have to get them all over the internet. common ones are usually free to download and use. once all the parameters are tweak and passess the tests (note usually starting with the core to ram to graphics), the earlier settings are left untouch while a new parameter setting is been tested. then you can start all over again with the initial parameter in the chain and see if you can squeeze more performance outta the system. if it hangs back up the setting to the last one and try again. switch to the nex parameter to see if you can up the system higheruntil the limit and perfect OC settings are reached.

    stock cooling solutions on some hardware and cards facilitates OC-ing to a small amounts...serious OC-kers get seaparate cooling hardware from OEMs.

    if u r truely in to oc-ing..that pc should be left alone to run the tests because serious oc-ing tests takes days and weeks. work on another computer for troubleshooting or when you need to gain access online for help when u encounter probs during the OC process.

    graphics engine clock can be pushed up to the specified levels or scale indicated with the OCing programme u use on the card.

    as for memory you can increase the voltage or buy modules that comes with specified timings to have more options to fine tune.

    as for the 0 degress celcius clocking on 4 Ghz mark its hardly possible with most boards sold. mayb only specific boards with specific cpu are able to do that. first clocing and timings are increase with the preset or allowable options available on the cpu, board, cards and rams. increasing the voltage means increasing the timing thus the term overclocking. with this definitely more heat will b produced thus advance and appropriate devices have to be used to take away the heat generated. even if you have a vanilla ice cooled device able to keep the system and core temp as low as 0 degrees centigrade, a board or core lacking enough voltage tweak to tune up the speed of the system means that it will only overclock up to x speed and runs stable with y duration of the test/s.

    all the above has nothing to do with overclocking. its just some advance settings available and coupled with the graphics card user menu. some oof the settings (e.g 'disablepower management'and 'disable screen saver') can be found in the OS's control panel.

    to overclock a graphics card is as simple understanding basic mathematics. what is most crucial in overclocking is the appropriate cooling device add-ons, a spare computer just for the overclocking so that you can still work on another machine, time, patience, cautions, the benchmarking software, the appltcation to be run with the test installed, latest graphic drivers updated, the appropriate OC programme/ software. read http://callforhelptv.com/callforhelp/guests/0120A.shtml if you are really trying to overclock your card because OC-ing a graphics card simply means increasing the core clock and memory clock. the better time of the process is spent in monitoring your test results and restarting everythin over agin when a test is completed or the system responded otherwise. make sure you back up your system and also the test results after every round/ session of the benchmarking. anyway simply overclocking the graphics card alone will not increase or make noticeable performance increases. you have to OC the whole system to see a difference. but before u go on make sure you have an appropriate cooling solution on your card first and foremost, not because it should be maintained at 0 degrees centigrade but because when your pc crashes/ hangs, the clock cycle still goes in loops and there is still heat produced. you would want to be able to force a shut down and OC/ use the card again with chips not BBQ-ed.
