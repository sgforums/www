    There could be 2 possible probs

    1) bios showing wrongly the processor speed

    2) system information/device manager showing the wrong processor speed.

    the 2nd problem is more often seen, so we will check for the 2nd problem first.

    ________________________________________________________________

    Try using CPU Z to check first.

    http://www.cpuid.com/cpuz.php

    if it's still Atom 270.

    Try this.

    Go to your device manager.

     Look under processors, check if it's Atom 270 or If it's 280, then update it. what you have to do is to uninstall the driver under the processor tab and reinstall it after a restart. It will normally show the updated processor.

    Normally this happens, when manufactuers install the image of a different system that has the same drivers for all of the rest except it has a different processor and it results in wrong information shown in system manager and device manager.

    I personally installed a wrong image before into my laptop, and it results in my processor being wrongly recognised in the device manager.

     

     update: If cpu z and updating the drivers still show it's Atom 280, it would be your bios showing the wrong information.
