    i've got a video file that i downloaded from a site and i wanted to upload it to photobucket. the file is 'loading-a-bike.mpa'...apparently mpa isn't a valid video extension.

    is there any way to give it a valid extension or am i just out of luck?

    this is what photobucket tells me:
    Upload Error
    The file "loading-a-bike.mpa" does not have a valid video extension.
    Valid video extensions are: 3g2, 3gp, 3gp2, 3gpp, 3p, asf, avi, divx, divx, dv, dvx, flv, gif, moov, mov, mp4, mpeg4, mpg4, mpe, mpeg, mpg, qt, swf, wmv, xvid.
  • ditzy
    16 Jan 07, 14:53
    Under the folder where the file it, click on "Tools"--->"Folder options"--->"View" tab, and uncheck "Hide file extensions for known file types", then "ok". Now you can see the extension of the video file you specified as blahblahblah.mpa. Rename it to blahblahblah.mpg. tada!
