    Install the Language Packs...

    XP, can't recall off hand, but I think is something like this:

    1. Start > Control Panel > Regional and Language Options
    2. Languages tab, tick Install East Asian Languages
    3. Insert CD, let Windows loads the files, restart computer
    4. Start > Control Panel > Regional and Language Options
    5. Languages tab, click on Details button
    6. New window open, click on Add
    7. Locate Chinese (PRC) (Note: Other Chinese languages won't work except Chinese Traditional)
    8. Click OK to apply the settings

    Vista Business. I'm using the Classic Start Menu, so you navigate around to find the settings.

    1. Start > Settings > Control Panel
    2. Double click on Regional and Language Options
    3. Select Keyboard and Languages tab
    4. Click on Change Keyboards
    5. Click on Add
    6. Scroll all the way down to Chinese (PRC)
    7. Click on + sign to expand Chinese (PRC). Tick these boxes:
    a. Chinese Simplified - Microsoft Pinyin IME
    b. Chinese Simplified QuanPin (Version 6.0)
    c. Chinese Simplified ShuangPin (Version 6.0)
    d. Chinese Simplified ZhengMa (Version 6.0)
    8. Click OK to apply the settings. Let Vista install the files, restart computer.
