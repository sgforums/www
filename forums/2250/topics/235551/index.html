    Originally posted by asdfzhao:
    how 2 check if i set it 2 prompt me?
    (sry 4 e late reply)

    n y does opra hang sometimes when i try 2 reply on sgforums?
    it usually hangs at night...
    Go to Tools > Preferences

    Select Advanced tab

    On the left hand side, click on Downloads

    Untick Hide file types opened with Opera

    Then under Mime type, select application/x-msdownload

    Click on the Edit button on the right.

    Under Action, the selected Option should be Show download dialog

    If it's not, then it's set to run with the browser... those settings can be changed one by one. Leave them as default is just nice.

    As for the hanging at night, perhaps more users are surfing the Net at night, reducing speeds and causing it to hang for a while.
