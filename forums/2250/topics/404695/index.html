    BIOS, which stands for Basic Input and Output System, is a firmware interface with information for system start-up.

    Its primary function is to load and start an operating system(OS) from a boot device such as a HDD.

    For school assignments, usually referring to Wikipedia alone is enough.

    For the first question, you missed out the primary functions of BIOS.

    BIOS

    • Identify and initialize hardware
    • Load OS from boot device
    • Pass control to OS

    Sources:
    http://en.wikipedia.org/wiki/BIOS

     

    As for the second one, system software not only includes OSes, but also device drivers and utility programs.

    System software

    • Operate computer hardware (eg. graphics card driver, sound card driver)
    • Provide and maintain platform for application software (eg. Java Runtime Environment, disk defragmenter)

    Sources:
    http://en.wikipedia.org/wiki/System_software
    http://en.wikipedia.org/wiki/Operating_system
