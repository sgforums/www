    It's a good price for such a laptop with these specs. However, it's not really complete. Is it duo core or single core? If it's single, probably a bit too slow in today's standard. If it's duo core, it's one of the best.

    What are your school work like? Does it require you to use resources intensive programs like Photoshop often? If yes, you will need one with more memory, 512MB will seriously kill it after a while. If it's some programming stuff, 512MB should be enough. As for gaming, since you are not too hardcore, 512MB is enough, the graphics card is a good one also. But it takes up too much resources, you might want to consider increasing it.

    Acer doesn't spoil easily if you take good care of it. Don't let it overheat, don't overcharge and it shouldn't be a problem. Shutting down properly will also make it last.
