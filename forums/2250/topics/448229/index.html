    If you mean you want to reinstall Windows, yes, that's doable. It's not impossible. New PCs and laptops these days comes with a recovery button. You can press that button and the reinstallation process will start. It will wipe everything and start all over.

    However, please backup your files before doing so.

    If you mean to clean up some of the unwanted stuffs on your PC, you can do so as well. I'm not sure which version of Windows you are running, so I'm going to assume that it's Windows 7. I'm also going to assume that your PC is not infected with any virus. Because doing any of these stuffs before cleaning up your PC can make things worse.

    If you want to be sure, we will go through the procedure of checking your PC first, followed by a performance tuneup.

    Step 1

    Click on Start > Control Panel > Programs and Features

    Remove any unwanted programs from there.

    Step 2

    Download CCleaner and remove temp files. Save this file to a thumbdrive and extract it.

    • Right click on CCleaner and select Run As Administrator
    • When it prompts to intelligently scan for cookies, select No
    • Select Options on the left
    • On the Cookies to Delete panel, look for any sites that you log in regularly (e.g. Gmail, Yahoo, Hotmail, forums). Select the site and click on -> button to keep the site
    • Click on Cleaner on the left
    • Under Internet Explorer, keep this box unchecked: Saved Passwords
    • Click on Applications tab
    • If you have Firefox and/or Chrome installed, keep these boxes unchecked: Saved Form Information, Saved Passwords and Compact Databases
    • Under Internet, select Sun Java checkbox
    • Uder Utilities, check Adobe Air
    • Click on Run Cleaner button at the bottom right hand corner

    Step 3

    Run Windows Disk Defragmenter (Start > All Programs > Accessories > System Tools > Disk Defragmenter)
