    Hi, chances are, you will not find any adware related to this problem. Windows XP has a bad habit of associating video and music files to another program and then forgets to release the association, thus causing you to be unable to move, delete or rename the file. It tells you that the file is currently in use despite all the programs being closed.

    I do not know how to release the association but instead, I use a program downloaded from the internet, GiPo@moveonboot to delete/move/rename the file at the next boot up.
