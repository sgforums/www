    TS-Since you have the money, why not go for Quad Core and X38 chipset motherboard which is DDR3 capable, more future proofing

    Q6600+ASUS P5K-E

    Hard Disk-ok

    RAM-2Gb ok

    Graphics card- Ok but get the Xpertvision one because it has a custom cooler which is better than the reference cooler

    Case-No experience with thermaltake casing, but to me it feels rather overpriced.Rather get a Lian-Li casing and then get a separate PSU

    PSU- Enhance 500W ENP-5150GH (PFC, 80Plus certified)

    Sound-No point getting a 5.1 setup if you are running onboard sound.Get a better soundcard before you get the speakers.The Auzentech Prelude is good for gaming and music if you got the $$$. If not just get a 2.1 setup, the logitech X240 is good enough

    Display-You are not doing justice to your graphics card if you are using 19 or 20" display, better get a 22" one.Trust me it's worth it

    Optical drive- Just get any SATA ones can already

    The above setup can run all the games now mostly with max settings.
