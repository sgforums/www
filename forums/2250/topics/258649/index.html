    Ports are basically communication channels. Think along the way of USB ports, PS/2 ports, parallel ports, serial ports, etc. All these ports that you see on a PC, are physical ports; they allow communication between the computer and the different devices.

    For the Internet, they are also used for communication. The numbers will determine what kind of program you are using. For example, port 23 for Telnet, port 25 for FTP, port 80 for browsers (IE, Firefox, etc).

    If you use a firewall, it will prompt you what kind of program is trying to access the Net, together with the port number.

    For example Sygate firewall. When you attempt to connect to www.google.com, it will prompt you that iexplore.exe (Internet Explorer) is trying to access the Net using port 80.

    what is worm
    A worm is a virus, but is spread via networks. Blaster is one famous one.

    I do download stuff from the net , how do i know what im downloading is actually virus or backdoor , trojan etc
    Download from trusted sources. Open-sourced programs such as those from Sourceforge, programs from Pricelessware have been tried and tested to be spyware free. If unsure, do a Google search for user reviews and read their reviews. If the review is good, most likely it can be trusted. If you find mixed reviews, just post in the forum. Somebody will evaluate the product.

    If it's a file, upload it to Virus Total or Jotti for a scan. If the results are clean, chances of it being a clean file is high. However, these results are no guarantee that the file is clean because malware mutates fast and antivirus companies jsut can't catch up to produce signatures to tell if a file is malicious.
