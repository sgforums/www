    1. Overheating

    2. Virus

    3. If you are on Windows, for some particular updates, it will automatically restart your computer if you don't cancel it

    4. Hardware or software problems, especially those that have blue screens, commonly known as BSOD

    5. A failed upgrade

    6. Not enough power supply. Usually, this will halt the starting up of a PC, but sometimes, it doesn't. If it only occurs while you run certain programs... you know what is using so much energy to trigger that.
