    Depending if your windows 8 tablet has an ARM processor (type found on phones) or a X86 Processor (think of it as a normal processor)

    You can't transfer per say, transfer Windows 7 from your laptop - if it's a x86 processor, you can install windows 7 from an image - dvd , then key in the CD key - but please note, if you got your laptop with windows 7 preinstalled , the cd key found on the label at the base, will not work. However if you bought the windows 7 installation disc and installed in your laptop, then yes you are able to then transfer it to your tablet.

    Take note though, Windows 7 is not optimised for touch - drivers and operating software wise.

    It's advisible not to do so.
