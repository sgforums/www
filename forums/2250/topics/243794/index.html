    Edit your system registry... if you dare

    START>RUN>
    Type in the box the word and click OK: Regedit

    When reg editor opens up, click Edit>Find

    Type in Find What and choose Match Whole String Only the words: RunOnce

    On the left hand side you will see the folder RunOnce, highlight and delete all the entries on the right hand side.

    Better if you make a system restore "restore point" before doing it.
