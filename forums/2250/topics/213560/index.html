    The technician is right. It has nothing to do with the RAM. If you wish to improve speeds, you can upgrade the RAM. HP has added loads of rubbish which will load during boot, that's why it's CPU intensive. Nothing else. If there's anything, it's HP.

    If your warranty has not expired, please do not upgrade your RAM elsewhere, other than at authorized HP dealers, resellers or repair shops. If you do so, warranty will be void. If your warranty has expired, go ahead and find someone you trust to upgrade the RAM for you. Check HP website for the correct RAM or the manual that comes with it.

    System Idle is always taking up the highest CPU time if you are not doing anything that is intensive enough. If you do, like using Photoshop, you will notice that the System Idle process CPU time drops, and your laptop's performance will drop too.
