    for all routers.. generall all have the same function... especially for home usage..

    1. find out the MAC address for each of your computer in your home.
    2. write it down on a piece of paper or something.
    3. type http://localhost in ur internet explorer or web browser..
    4. go to the wireless settings and disable ESSID broadcast
    5. go to the security settings and remove the WEP encryption
    6. go to the MAC address tab
    7. enter the mac addresses in step 2
    8. save the mac addresses
    9. reboot

    for step 3, please use IE because some other web browsers MIGHT not support the GUI of ur router. routers have been designed and tested using IE. some routers do not have http://localhost and u must check for the IP address in ur user manual

    step 4. prevents people from seeing ur network name. hackers or nothing-better-to-do people can still sniff ur network name.. but then again.. chance that u have a neighbour like me is low.

    step 5. because u are only allowing stations to connect using MAC address.. u do not need an encryption key for secuirty reasons.

    last but not least, they will say that ur network is un-secure and some errors.. but it is ok.. nobody else can have the same mac address as u and only ur computers will be able to connect..

    if in help, please post questions here and i might be able to help u.. i will try my best to answer ur question. the above guidlines are just a GENERAL help and each router differs.. so... ya.. please do refer to ur user manual before attempting the above steps.

    reading the manual gives u a better overview and understanding b4 u attempt ur router configurations.

    if ure unsure of any settings, please do not do, if u do that, u might even lock urself out of the network and may need to do a firmware restart..

  • zerocool85
    14 Mar 06, 16:17
    1. Click the Start menu button on the Windows taskbar.
    2. Click Run... on this menu.
    3. Type cmd in the text box that appears. A command prompt window launches on the desktop.
    4. In this command window, type ipconfig /all. Details are shown for each of the computer's network adapters. Computers installed with VPN software or emulation software will possess one or more virtual adapters.
    5. The IP Address field states the current IP address for that network adapter.
    6. The Physical Address field states the MAC address for that adapter.
