    Hi, TS i'm using the same router as u also. and i did transferring files in home my network too. this is how i do it... BTW, i'm using XP Home, so if u using other OS, u might not get into the same steps.

    and also, i assume all computers/printer connects directly to the router.

    Do this to every computer as follow below

    1. go to "My Network Places"
    2. on the left panel, click on "setup a small home or small office network"
    3. click "next" (This is what u see at the title bar: "Beore u contimue...."
    4. click "next" again (This is what u see at the title bar: select a connection method")
    5. click "next" again (This is what u see at the title bar: "Give this computer a description and name")
    6. click "next" again(This is what u see at the title bar: "Name your Network")
    7. click "next" again(This is what u see at the title bar: "file printer and sharing")
    8. click "next" again(This is what u see at the title bar: ready to apply network settings...")
    9. (This is what u see at the title bar: "Please wait...")
    10. (check the last option: "Just finish the wizard...")(This is what u see at the title bar: you're almost done)
    11. Click Next (This is what u see at the title bar: Completing the network setup wizard)
    12. click "Finish"

    after doing for all computers, you will need to check whether all computers are under a network.

    this can be done by..

    1. Start>run>input: cmd
    2. a command prompt will be there. input: net view
    3. u should be able to see all the listing computers including yours inside. if its not there, this computer or the other computer might not be able to send files to each other.

    once checking is done, now, proceed to sharing..

    1. right click a folder which u want to share
    2. click on "properties"
    3. click on "sharing" tab
    4. under "network security and sharing", click on "if u understand the security risk ....."
    5. check "just enable sharing", and click "ok"
    6. Then still under under "network security and sharing", click "share this folder on the network"
    7. then click "ok" and u are done.. all computers under the same network in your router is able to see this folder in their My Network Places

    Then for example u want to take retrieve a file from computer A, in computer b, go to :My Network Places. u will see a folder something like "zzzzzzzz on Computer A". double click it and drag into where u like in Computer B.
