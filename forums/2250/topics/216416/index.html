    CCleaner

    If you do not want the Yahoo! toolbar, untick the checkbox for Yahoo! toolbar during installation.

    CleanUP!

    ATF Cleaner

    Instructions for ATF Cleaner

    Instructions for using CleanUp!

    Go to Start > All Programs > CCleaner > CCleaner.

    You should see this window. Click on Analyze. If you see some files that you don't want CCleaner to clean, untick one of the boxes. If you want a pretty clean system, but don't mind it being slow, you can tick the Old Prefetch data box as well.

    After analyzing and you are sure that you want CCleaner to clean those files, clean on Run CCleaner.



    In this Application tab, it list some of the applications that are installed and files that can be safely cleared. Do not tick the Saved Form Information box if you don't want to retype your password the next time you log in.



    Click on the Options button at the extreme left hand side. You should see Settings, Cookies, Custom, Advanced, About and Version History.

    Select Cookies. You will see the cookies from the different websites. If you trust a website, you can don't remove it. If you don't trust it, keep it in the Cookies to Delete column.

    If you want to select more than one item, hold down the Ctrl key and click. After that click on the -> button to put them to the Cookies to Keep column.

    To know more about cookies, read this article.
