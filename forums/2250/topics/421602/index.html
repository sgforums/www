    What operating system (Xp/vista/ Windows 7) are you using? 

    Can you access safe mode?

    Start, once the Dell screen boots up, immediately repeatedly press F8. 

    If you get to a screen titled "Advanced boot options" 

     

    1. Try entering into safe mode with Command prompt
    2. When you successfully logged in start, find your command prompt, then type   Chkdsk /R (enter) and Chkdsk /F (enter)
    3. or if there is a repair option, go to it. Then access the command prompt
    If there's a repair option under the Advanced boot options, click on that and go to command prompt.
