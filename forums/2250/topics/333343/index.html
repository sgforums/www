    Flashy folder? Any screenshot to show? Try accessing the folder by entering the path in the Address Bar (eg. "F:\DCIM"). If it does not exist, the folder is probably deleted already. In this case, you may want to try PhotoRec to recover the deleted files.

    http://www.cgsecurity.org/wiki/PhotoRec

    Regarding the "500 MB used" info, it could be redundant files created by the virus stored in the XD card which are hidden away from view. The easiest way to remove such crap is to copy whatever files you need out and perform a reformat on the card.
