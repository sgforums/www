  • PedoBear

    the speedtouch modem is running a customised firmware to allow the miobox network access without interent service

    ie - miotv works even without the internet connection as long as your adsl line/miotv service works.

    so the answer is no, only Singtel's speedtouch modem can run miotv

    even if you are to buy the exact speedtouch modem from thomson (distri), miotv will still not work as it does not have the customised firmware

    hope this helps


    addition
    in case there is any confusion about Singtel's mio service modem, there a 2 brands
    1. small basic set from speedtouch without wireless AP
    2. bigger advance set from 2wire with wireless AP
