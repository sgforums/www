    ... showndown.exe is a program which can shutdown your computer with selected preference. windows shutdown is a simpler type(start>shtdown), but then they didn't show extra preference

    the usage for it is (u have to type it in cmd or in run)

    shutdown [function]

    the "function" is the type of action u command shutdown.exe to do

    function action
    -------------------------------------------------------------------------------------
    -i Display GUI interface, must be the first option
    -l Log off (cannot be used with -m option)
    -s Shutdown the computer
    -r Shutdown and restart the computer
    -a Abort a system shutdown
    -m \\computername Remote computer to shutdown/restart/abort
    -t xx Set timeout for shutdown to xx seconds
    -c "comment" Shutdown comment (maximum of 127 characters)
    -f Forces running applications to close without warning
    -d [u][p]x:yy The reason code for the shutdown
    u is the user code
    p is a planned shutdown code
    xx is the major reason code (positive integer less than 256)
    yy is the minor reason code (positive integer less than 65536)

    forexample, i wish to shutdown my dad's computer using my computer, and tell him to have a good rest.. here it goes...

    shutdown -s -m \\daddycomputer -c "daddy, have a good rest!=)"
