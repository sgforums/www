    80048820 is usually something about your system clock.

    Check that your system clock is correct. Change it to the correct timezone also. Singapore's time is GMT + 8.

    If this doesn't work, it's just MS. Some problems with their DLLs.

    Do each of the following line by line. When you receive a prompt about it being successful, then move on to the next line. If it isn't successful, don't move on. Post back.

    Go to Start > Run and type in:

    REGSVR32 softpub.dll

    REGSVR32 wintrust.dll

    REGSVR32 initpki.dll

    REGSVR32 Rsaenh.dll

    REGSVR32 Mssip32.dll

    REGSVR32 Cryptdlg.dll

    REGSVR32 Dssenh.dll

    REGSVR32 Gpkcsp.dll

    REGSVR32 Slbcsp.dll

    REGSVR32 Sccbase.dll

    For Netgear router users, try this:

    1. Enter IP address 192.168.1.1 on the address bar.
    2. Enter your user name and password that your ISP has provided.
    3. Enter the administrator username and password.
    4. GOTO advance menu (left hand side of screen)
    5. GOTO WAN set-up
    6. GOTO MTU size, and change to 1400
    7. Apply, OK.
