    Let's try changing your proxy settings first. Restart your computer after that to see if SP has locked down certain changes to Internet Explorer.

    1. Open Internet Explorer. Click on Tools > Internet Options.

    2. Select the Connections tab.

    3. Click on LAN settings button.

    4. Untick the Use a proxy server for your LAN (These settings will not apply to dial-up or VPN connections) box.

    5. Click OK to apply the settings.

    6. Restart the computer. See if you can access the Internet now.

     

    If it doesn't work, try this:

    1. Open Control Panel, double click on Network Connections.

    2. Right click on your connection and select Properties.

    3. Double click on Internet Protocol (TCP/IP)

    4. Select Use the following DNS server addresses radio button.

    Add in these 2 IP addresses:

    208.67.222.222
    208.67.220.220

    Click OK to apply the settings.
