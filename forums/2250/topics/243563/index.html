    You said about turning off the router before Windows fully shut down, and seems like this is causing the error.

    Did you install any router software? If yes, try removing it in Safe Mode.

    Also, do you have your network card installation CD with you?

    If Windows give an error that you can't uninstall, and you can't boot into Normal Mode, do this:

    Go to Start > Run and type in msconfig

    Select the General tab. Under Startup Selection, select Diagnostic Startup.

    Now select the Services tab. Tick this box: Windows Installer. Click OK to apply the settings. Windows will prompt you to restart. Select Yes to reboot.

    Then uninstall the router's software after Windows restart.

    Do not proceed with this step if you don't have your network card installation CD. Only proceed on if you have the CD

    Right click My Computer and select Properties. Select the Hardware tab and click on the Device Manager button.

    Device Manager will open. Scroll down to Network adapter. Click on the + sign to expand. Right click on the adapter shown and select Uninstall.

    Reboot your PC back to Normal Mode. See what happens.
