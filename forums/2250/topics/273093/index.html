    Now explanation

    1. Disc problem - have you tried with other discs? Does it produce the same error? Also try lowering the burn speed. If it's set at 8x, try to set it lower, say at 4x or 2x.

    2. Not enough power - rare problem, but some do face this problem. If you are unsure, please post your full computer specs here to determine if it is the cause.

    3. Firmware outdated - common problem. All softwares, drivers and firmwares are not without problems. Go to the manufacturer's website and check if there are any software, drivers and/or firmware upgrades. If there are, download them and install and try again.

    4. Too many burning softwares installed - due to programming. Windows has a set of codes to "tell" the program how to burn a disc. If you have too many burning programs, and they are running (check your startup programs and running processes in Task Manager), they may interfere and produce an error.

    Uninstalling the extra programs will help. Or if they are starting up when Windows starts, try disabling them, restart the computer and try burning your DVD Data disc again.
