    Download Link

    About the same MSN Messenger... except that quite a few things change for the installation.

    1. License agreement no longer presented directly to you, you must now click on the links provided.

    2. No more choosing of installation path.

    3. Takes a tad too long to install. 10 minutes was how long it took, even without installing all the rubbish it provided.

    4. No longer at Start > All Programs > Windows Live Messenger. It's now in Start > All Programs > Windows Live > Windows Live Messenger.

    Interface wise, very Vista like. Surprisingly it has no lags or crash. However it's quite ugly.





    Signing in wise, very fast. It took me 3 seconds to sign in. And no lags when it starts. A first for any Microsoft programs that I tested.

    Windows now add a function to search, just click on the magnifying icon.

    As for chatting, all the display info (which is whoever is writing a message, last chat at time) has moved from the bottom of the chat window directly into the main chat window.



    The notification messages, however, looked very nice. It gives a metallic feel, like Mac OS (I think...). Neat, clean and simple.
