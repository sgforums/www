    That's because Windows Automatic Update is enabled.

    Just disable it, don't run this script. This script will damage quite a lot of things.

    Go to Start > Run and type in services.msc

    Locate Automatic Updates. Double click it.

    Change to the Startup Type to Disabled from the drop-down list.

    Then click on Stop to stop this service.

    WinDizUpdate is for those using a fake version of Windows. If yours is legal, ignore it. Otherwise, Microsoft will detect your legal copy as illegal and you are going to have headaches.
