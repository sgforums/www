    i'm using maxtor onetouch III..great use if u are looking to use it as a back-up harddisk

    pros,
    - love the one-touch feature, juz pre-define the directories u want to save and press the button and the software u will automatically copy the directory over
    - if u edit a certain file and save it on ur notebook, the next time u back-up ur system, the software will copy over the newer file w/o deleting the older one (based on the number of revision u want, the software will keep a certain back-dated versions of the file)
    - no software installation needed for these back-up functions, everything's on the external harddrive, great if u work on multiple computers (e.g. work & home)
    - you can set password in order to access the external drive, great if u got files which u dun want ur collegues or love ones to access

    cons,
    - expensive
    - no leather case for protection
    - a bit bulky compared to other usb harddisk
