    If it doesn't work, try this instead:

    1. Do a scan with Kaspersky first.
    2. Locate the virus file.
    3. Delete it.

    Before this, show hidden files.

      [*]Open My Computer.
      [*]Go to Tools > Folder Options.
      [*]Select the View tab.
      [*]Scroll down to Hidden files and folders.
      [*]Select Show hidden files and folders.
      [*]Uncheck (untick) Hide extensions of known file types.
      [*]Uncheck (untick) Hide protected operating system files (Recommended).
      [*]Click Yes when prompted.
      [*]Click OK.
      [*]Close My Computer.


    Then re-scan with Kaspersky.

    If it detects some virus in C:\System_Volume_Information, that's your System Restore point that is infected.

    Turn off System Restore, restart PC and turn on System Restore will reset the System Restore and create a new clean System Restore point.
