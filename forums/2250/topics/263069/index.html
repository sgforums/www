    1. Need to install drivers. Check the modem brand and model and go to the manufacturer's site to download the drivers if you have lost the CD. After installing, restart the PC.

    2. The limitation of wireless is speed. For wired connections, say the maximum speed transferring by cable is 100Mbps, you will get that. For wireless, no. At any one time, the stated bandwidth is always chopped into half. Say the maximum is 54Mbps, the best you get is 27Mbps. Coupled with reality, it can drop to a few megabits per second.

    3. That's the Internet Connection Gateway. If you are using a modem, this will show up. Usually when this shows up, you might have problems with connections. Disabling it and enabling it will sometimes solve the problem, but most of the time this doesn't happen. This has to disappear before you can use the Net properly.
