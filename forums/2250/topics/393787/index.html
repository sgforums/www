    Option 1: Use Dreamweaver or other HTML editors to work with web pages. There's an option in Blogspot where you can copy out the template code. Paste it into a text document and open it up in a HTML editor should get you started. For music players on web pages, usually they come from a music sharing site which would provide detailed instructions on placing their flash music player on your web page.

    Option 2: Pick up basic knowledge on HTML and CSS. You may want to begin with tutorials like these:

    http://htmldog.com/guides/htmlbeginner/

    http://htmldog.com/guides/cssbeginner/

    Option 3: Post your blog URL here and let the forum community help you.
