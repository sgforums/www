    Task Manager Problem

    Check your registry to make sure that nothing is changed.

    Go to Start, Run and type in regedit.

    Make sure the values in your registry tally with those in blue:

    [HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\System]
    "DisableTaskMgr"=dword:00000000

    [HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Group Policy Objects\LocalUser\Software\Microsoft\Windows\CurrentVersion\Policies\System]
    "DisableTaskMgr"=dword:00000000

    [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\system\]
    "DisableTaskMgr"=dword:00000000

    [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon]
    "DisableCAD"=dword:00000000

    If they are not, right click, and select Modify. Type the correct values accordingly.

    Can't find much info about drsmartload758a.exe, could be a malicious program. Update your antivirus and anti-spywares and do a scan for your computer.

    Originally posted by mhcampboy:
    how do a system get corrupted?? by deleting files izzit?
    A few ways.

    One way would be to delete important system files, especially the files located in C:\Windows. It's best that one not touch the files there, unless you know what you are doing.

    The second way would be through viruses or worms. Some viruses or worms can modify system files, causing them to be corrupted.

    The third way is defragmenting the PC with programs running. During defragmenting, a lot of files would be moved. By not closing programs, some programs might be corrupted in the process.

    The last way, and one of the most common way, is updating of drivers and system. Some patches create more problems rather than solving problems, so it would be wise to wait a while to see if others have problems with the new patches before proceeding to install the new patches.
