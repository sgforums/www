    Hello.

    Go into the Tools Menu in Internet Explorer (I assume that's the browser you're using) and then Internet Options. On the General tab you will find a button labelled Delete Files. Pressing that button will empty the cache and restore IE to normal operation allowing you to save images properly.

    If that doesn't solve the problem then go to the Advanced tab and check that "Do not save encrypted pages to disk" is not selected. Next press F5 to reload the page and try again.
