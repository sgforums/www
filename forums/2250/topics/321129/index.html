    I don't know whether it works for you, but you can try this out.

    Before you try anything please close all visible programs.

     

    -1a. For Vista : Click "Start" icon then type "regedit" in the box then press enter.

    -1b. For XP : Click "Run" then type "regedit" in the box then press ok.

    A window called "Registry Editor" will open.

    -2. Go down the folders:

    HKEY_CURRENT_USER->software->microsoft->office->12.0->word

    -3. Remove the folder called "Data".

    -4. Close "Registry Editor".

    -5. Open up MS Word and try typing things into it.

     

    It's no garantee so try this at your own risk. Be careful not to remove the wrong folder.

    If all else fails, uninstall ms office, restart then reinstall it again.
