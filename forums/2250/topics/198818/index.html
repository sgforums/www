    Ok, so your mysql server is actually configured since you are able to view it access..

    Try to add one more line in your httpd.conf:
    code:
    AddType application/x-httpd-php-source .phps


    And this line in your php.ini:
    extension_path = %apache directory%/php/extensions/
    extension=php_gd2.dll
    extension=php_msql.dll
    extension=php_mysql.dll

    I think maybe your version of MySQLi needs mysql too..
    Then now add the MySQL as a service and restart the server...

    In ms-dos:
    >cd %mysql_directory%
    %mysql_directory%>mysqladmin -u root shutdown
    %mysql_directory%>mysqld-nt --install
    %mysql_directory%>net start mysql

    Then should be okay le ba... Haiz.. really hope this helps....

    Does Apache support any sort of database integration?

    No. Apache is a Web (HTTP) server, not an application server. The base package does not include any such functionality. See the PHP project and the mod_perl project for examples of modules that allow you to work with databases from within the Apache environment.
