  • ndmmxiaomayi
    21 Feb 08, 17:32

    Overheat either hangs a machine, shuts it down or restart. It depends on the hardware maker.

    I doubt that's the cause. I think it may be due to corruption, because you shut it down forcefully when Firefox crashed the machine. As the hard disk is still writing data and power is suddenly lost, it can cause corruption.

    The corruption can be fixed by booting up to Last Known Good Configuration. Keep on pressing F8 when booting up, before the Windows logo shows. Then select Last Known Good Configuration and press Enter. It takes a while for Windows to repair itself if you are using NTFS. If using FAT32, might be gone case liao.
