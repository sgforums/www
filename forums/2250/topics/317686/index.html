    Well, I use Runtime software.

    http://www.runtime.org/

    It's a free recovery tool... in a sense. If you don't pay, don't expect auto recovery to work. I had to manually save 1000++ files when I reformatted a wrong PC. If you want it free... I guess that's the price of it. If you don't mind paying for it, then it's a great tool.

    Another one is AccessData.

    http://www.accessdata.com/

    This is used by computer forensics scientists. It's a great tool as well, but if you don't pay, it limits itself to scanning the first 5000 files. No idea how they scan, but I don't remember having seen any of my deleted files within the 5000 files scanned, so I guess it's not of much use. You could try, no idea what you could yield from it.

    There's no license, you pay for a USB dongle (much like a thumb drive), which costs a few thousands bucks to unlock all its features.
