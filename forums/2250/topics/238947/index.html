    I only know of one update that will screw it up - Daylight Savings Time update.

    Because of changes in US laws. If your OS is screwed up in any way, look for an update called 928388 or 931836 under Updates for Windows in Add/Remove Programs

    http://support.microsoft.com/kb/928388

    http://support.microsoft.com/kb/931836

    If you have Microsoft Outlook as well, look for 931667 under Office Updates.

    http://support.microsoft.com/kb/931667/

    Because Chinese have no need for DST if I'm not wrong, so this update sort of screw it up as it changes some things in registry for the different time zone.
