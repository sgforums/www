    What's the difference between Parallel ATA 133 and Serial ATA 133 harddisk?
  • ndmmxiaomayi
    23 Apr 07, 22:58
    PATA is IDE, SATA is SATA.

    The difference between them is minor or major, depending on how you see it.

    For PATA hard disks, you will need to set the jumpers accordingly if you are using more than 2. For SATA, that isn't an issue.

    Also, there's a speed difference. SATA is faster than PATA...

    PATA disks also have more pins, more specifically, 40 pins, in 2 rows, 20 pins each.
