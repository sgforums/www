    Originally posted by limpshaggy:
    i found the copy dvd option in nero, what is the meaning of source and destination drive? for source, the only option is drive e which is the dvd drive. For destination, it is either my drive e again or image recorder. Which is the correct one to choose?
    source -> the drive with the original dvd which u wan to copy. say.. drive e.
    destination -> the drive which u want to copy into, with the blank dvd. u can put it as drive e too.

    so u should choose drive e for both.

    after it read the original dvd, it will make a copy in your harddrive automatically with the correct file structure, everything.

    den it will eject the drive and ask u to put in a blank dvd-r.

    u put it in, den it will write the dvd data into this drive.

    wala. done!
