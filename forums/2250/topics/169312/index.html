    If it appears again, try either one of the following methods:

    1. Disconnect the LAN cable (If your using broadband). If this can't work, refer to 2.
    2. Go to Start --> All Programs --> Accessories --> Command Prompt and type "shutdown -a" without the quotes and press enter. The shutdown process should be cancelled.

    Now to check whether you have Blaster, which is rather unlikely, see if the process msblast.exe is in your Task Manager (Ctrl+Alt+Del).

    If so, report back here...

    Oh, and are you using Internet Explorer? You said it appears when you log on to the net. Can state internet connection?

    Watch out also for these processes:
    rpc.exe
    rpctest.exe
    dcomx.exe
    lolx.exe
    worm.exe
