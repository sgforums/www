    If you are using Windows XP Professional, try this.

    1) Go to Start, Run and type in gpedit.msc

    2) Navigate to the following path:

    User Configuration\Administrative Templates\Start Menu and Taskbar

    3) Double-click "Remove and Prevent Access to the Shut Down command"

    Select "Not configured" or "Disabled"

    4) Close Group Policy window.
