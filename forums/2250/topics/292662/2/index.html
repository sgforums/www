  • ndmmxiaomayi
    1 Dec 07, 03:00
    WLM 8.0 is the worst beta program I've tested. Buggy, used up too many resources and was pushed out to users before it was properly tested.

    WLM 8.5 codes improved by a lot, it no longer uses much resources. However, the installation method changed, and caused way too much trouble. The installer will download the files from MS servers and install it for you. If you have some blocker or other crap, it won't install properly and troubleshooting is a headache.

    WLM 9.0 does the same as well... and I think it just introduced one of the worst security concepts. The installation went silently. If not for some registry and file monitors, I couldn't track down what the heck is the installer doing.

    This version also phones home regularly... in order to update the contact list in realtime. The change might have been caused by users unable to find their contact list and a few others who wants realtime updates of their contacts.

    As a result of this change, signing in takes a bloody long time. I waited for about 10 minutes to sign in. For the other versions, it takes less than 5 minutes to be up even if on dial-up.

    There are also some other new features like contact's signature sounds... which can possibly lag the whole thing down and eats up some bandwidth, animated GIFs as display picture, multiple devices sign-ins.

    By the way, encryption of contact list is no longer available since the files are moved to the servers. If the server is hacked, good luck. I would suggest everyone on the beta team to back up their contact list in case the server goes down.
