    As a side note, you do NOT need to install the update titled: Windows Genuine Advantage Notification.

    Once installed on your computer, it starts phoning home to Microsoft at a set time-interval to check whether your copy of Windows is a legit or cracked one.

    There is another variant which uses ActiveX controls(works only in internet explorer) and is called: Windows Genuine Advantage Validation. This uses a ActiveX control to check whether your copy of Windows is a legit or cracked one whenever you download items from http://support.microsoft.com or the Microsoft Download Center.
