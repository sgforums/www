    Dear Aishwarya.r, you can Compact and Repair your corrupt .mdb file. But, are you sure that the file database file is with .mdb extension or is it .accdb file? As you are using MS Office 2007, the database file should be .accdb intead of .mdb. Maybe you have done some changes while saving the file. You should use Compact and Repair to repair the file.

    To Compact and Repair your database file in MS Access 2007:

    1. Click on Office button (on top side of the left corner).

    2. From the Menu, click on Manage > Compact and Repair Database.

    3. Now select the database file that you want to repair.

    Within a few seconds the file will be repaired, if possible. If it fails to repair the file, then you can use a third-party access recovery software.
