    I think there is something wrong with sgforums server, because it happened to me. Now I think everything is back to normal.

    My worst computer crashes:

    Well, my computer didn't crash. I boot up the computer as usual. The computer didn't even boot up. It just keep beeping. I then opened the CPU and inspect it. Still find nothing wrong with it. Then I decide to let it rest for a while. After lunch I came back and tried to start it, it is still in the same old condition.

    I then opened up my other CPU and did a comparison. Then I hit my head. The floppy disk drive wire had dropped off, and I set my BIOS to detect the floppy disk drive. No wonder it failed to load. After re-connecting the wires back, the computer work again.

    An advantage of having 2 CPUs --> use the other to compare and see what went wrong with the other one.
