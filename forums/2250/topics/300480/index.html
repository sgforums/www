    just a tip for u all into this kinda tech

    always connect ure wireless projector to a nearby AP...so that when u connect to it...u can surf as well as project at the same time compared to if u use it via peer 2 peer connection

    dun use 3rd party wireless connection software(like thinkvantage access connections) as there will be more often than not....a conflict and no projection of ure PC/Laptop would occur...use the built in windows wireless connection apps...

    and please switch off the firewall...

    or atleast unblock it when the wireless manager is running
