    Okay, I wouldn't suggest doing this method, since EISA partitions are there for a reason, but here goes anyway (I used it to install the 7 RC1 on my lappy)

     

    You will be unable to delete EISA partitions using Disk Manager. EISA partitions are usually recovery partitions used by an OEM. They are, however, recognised as a FAT32 filesystem. This means I can use GParted (not sure if others work) to resize/delete it.

    I ran the GParted LiveCD, selected some boot options, and waited for the host OS to start up. When it does, GParted should launch automatically. You will see 4 coloured bars with different lengths inside a single bar. 2 should show up as NTFS, and have the biggest bars. The other 2 should show up as FAT32.

    Delete 1, or both, up to you. If the free space isn''t enough, shrink one of your normal partitions (Again up to you). Now you should have at least 30GB of free, empty space.

    Format it to NTFS, and once it has been formatted, exit GParted and shutdown the system using the shutdown button (Big, red square. You can't miss it). Take out the CD when prompted, slot in the Win 7 DVD and press enter. The computer should turn off. Switch it back on, boot from the DVD drive, and install Win 7 on the new partition. Done!

     

    If you don't feel like erasing what is probably a recovery partition, then just don't. Win 7 may be faster than Vista, but that depends on your machine as well. Good luck!
