    To save me from typing so many times in shoutbox, I just post the help here.

    For IE users

    Open up IE. Go to Tools, Internet Options. Select the General tab.

    Under Temporary Internet Files, click Delete Cookies. After that click Delete Files. Another window will pop up. Check the 'Delete all offline content' box. Click OK.

    Under History, click Clear History.

    After all this, select OK. Close your browser and restart it.

    Type in www.sgforums.com and see if you can get in.

    For Firefox users

    Press Ctrl + Shift + Del. Refresh your browser. Type in www.sgforums.com and see if you can enter.

    More help if you can't enter through the main site:

    Go to Google and type in sgforums in the search textbox. Click on Cached. Select any forum and log in from there.

    After this, delete your old bookmarks. Re-bookmark this page.

    Last resort (only applies to those who are still using Microsoft Antispyware Beta 1):

    Open up Microsoft Antispyware Beta 1. Go to Tools, Advanced Tools, Tracks Eraser.

    Check these boxes:

    Internet Explorer History
    Internet Explorer Cookies
    Internet Explorer - URL History
    Windows Temporary Files

    Then click Erase Tracks.

    Hope it helps.
