    Open Command Prompt (Start > Run and type in cmd)

    In Command Prompt, type in the following in bold line by line, pressing Enter after each line:

    cd \
    dir

    You will see something like:

    Volume in drive C has no label.
    Volume Serial Number is 3015-B4F7
    The first line is your label.

    A warning though, using the MS method will format away everything.

    Backup everything before using this method.
