    Step 1

    Re-download and save HijackThis to your desktop. Don't run it directly via a browser as it makes backups. All these backups will be lost when your temporary files are cleared.

    Step 2

    Please print out or save this set of instructions as you will be rebooting the PC.

    Please download FixWareout from Subratam or Bleeping Computer.

    Save it to your desktop and double click to run it.

    Click Next, then Install, then make sure Run fixit is checked (ticked) and click Finish. The fix will begin; follow the prompts. You will be asked to reboot your computer; please do so. Your system may take longer than usual to load; this is normal.

    At the end of the fix, you may need to restart your computer again.

    Go to Start > Control Panel.

    Double click on Network Connections.

    Right click on your default connection and select Properties.

    Select the General tab. Double click on the Internet Protocol (TCP/IP) item and select the radio button that says Obtain DNS servers automatically.

    Press OK twice to save the settings and reboot if it asks.

    Next go to Start > Run type in cmd. Press Enter.

    Type in ipconfig /flushdns then hit Enter, type exit hit enter
    (that space between g and / is needed)

    Step 3

    Please post the contents of:

    1. C:\Fixwareout\report.txt
    2. A new HijackThis log
