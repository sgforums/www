    If Mayi's solution doesn't work, then try this. Then, try deleting the upper and lower filters from your registry.

    To load Registry Editor,
    Start Menu -> Run -> Type "regedit"

    When Registry Editor loads,
    My Computer -> HKEY_LOCAL_MACHINE -> System -> CurrentControlSet -> Control -> Class -> {4D36E965-E325-11CE-BFC1-08002BE10318} -> UpperFilters -> Edit -> Delete

    My Computer -> HKEY_LOCAL_MACHINE -> System -> CurrentControlSet -> Control -> Class -> {4D36E965-E325-11CE-BFC1-08002BE10318} -> LowerFilters -> Edit -> Delete

    Do not delete UpperFilters.bak and LowerFilters.bak if you find them on your registry.

    Reboot your computer.
