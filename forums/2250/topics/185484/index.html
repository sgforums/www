Today's Featured Article - XDR DRAM

  • FireIce
    8 Apr 06, 18:23
    The Rambus XDR DRAM device is a high-performance memory device capable of 3.2 and 4.0 GHz data rates with a roadmap supporting 4.8, 6.4, and 8.0 GHz in the future. Geared toward high bandwidth and cost-sensitive systems like graphics processing, consumer electronics, networking, and servers, the XDR DRAM device is a key component of the Rambus XDR memory solution. It uses an 8-bank DRAM core implemented in standard CMOS DRAM process technology and is currently in production with bit densities of 256Mb and 512Mb. Unlike other DRAM devices, the x16 XDR DRAM device implements programmable width and supports data busses that are 8, 4, and in some cases 2 bits wide. With this and many other architectural enhancements, the XDR DRAM device boasts much higher efficiency and throughput when compared with other memoriy technologies. The XDR memory solution gives bandwidth-hungry products the performance they need to succeed in today's demanding markets.
