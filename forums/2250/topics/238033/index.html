    using task manager
    firstly u need to have computers under the same network and group name.
    2. make sure messenger service is enabled on both coms( go to cmd and type net start messenger. if cannot, u need to go to (right click m computer>management>service and enable messenger)

    3.press ctrl-alt delete, under the last tab, right click the computer u want to send messege to. and click send messege.. there will be a field prompt to u for u to type your msg
    ----------------------------------------------------
    OR
    using net send command

    firstly u need to have computers under the same network and group name.
    2. make sure messenger service is enabled on both coms( go to cmd and type net start messenger. if cannot, u need to go to (right click m computer>management>service and enable messenger)
    3. u need a net view. ( go to cmd and input: net view). in netview u will see things like this eg: //mydadcomputer

    forexample if u want to send a messege to your dad com's this is how u go about

    input: cmd

    input: net send //mydadcomputer hi dad!.

    an output from your cmd will say that it send successfully...
    an output from your dad's computer will prompt something like this

    johnscomputer to mydadcomputer 22/6/2007 12:48pm

    hi dad!.

    [OK]
