    Orbit Downloader, an Internet Download Manager. It can grab embedded music or video files, but that isn't so well done. There's a plug-in, Grab++ that is able to do so nicely.

    Apparently supports P2P as well, haven't test it out yet.



    Edit: Works with P2P networks. Can download streaming videos as well. Only tested on Youtube and Crunchy Roll.

    Credits to my sister for playing this software.

    Those using Sunbelt Personal Firewall... you need to tweak the firewall so that it doesn't block Grab++ when it attempts to download the streaming videos or FLV files.

    1. Open Sunbelt Kerio Personal Firewall
    2. Select Intrusions on the left hand side
    3. If the Enable Host Intrusion and Prevention System box is ticked, click on the Advanced button.
    4. A new window will pop up.
    5. Under Buffer Overflow, click on Exceptions.
    6. Add this 2 files in:
      [*]Grab.exe
      [*]orbitdm.exe
    7. Click OK.
    8. Repeat Steps 5 to 7 for the section Code Injection.
    9. Click OK all the way to apply the settings.

    Instructions for using

    1. Open Orbit Downloader.
    2. Click on Tools > Grab++.
    3. Refresh your Youtube or Crunch Roll video.
    4. You will see a file called get_video.flv.
    5. Tick this box and click on Download.
    6. Then install FLV Player to play it. If you have VLC Player, FLV Player is not needed. Just play it directly through VLC Player.

    http://windows-get.sourceforge.net/

    For those using Debian-based Linux, you may be familiar with apt-get if you use the Terminal aka Command Prompt regularly to install programs. It's the Linux way of installing programs through a repository, something like a database.

    This has now ported over to Windows.

    The programs are installed silently, unless otherwise stated in the repository link: http://windows-get.sourceforge.net/listapps.php

    Good for lazy people. You can just write a batch and run it.
