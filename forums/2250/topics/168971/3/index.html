    Time for my contribution for freeware. Some have been posted so I post those not yet posted.

    RegScrubXP A windows registry cleaner that is small and compact. Only for Win2K and WinXP.

    TagScanner A mp3 id tag editor. It provides batch mode editing and renaming.

    NetTerm A telnet like terminal for doing login connection to Unix or Linux server.

    EventCorder A software that can record keystrokes and then re-played them out while you look on.

    WinGrep An equivalent to Unix grep command but with GUI and in Windows environment. Great for searching for keyword in files and folders.

    ExamDiff A very small and simple software for doing file comparison for text files.

    SourceEdit A text editor with FTP save like feature and bundled with a FTP program called File Courier and a hex file editor called Hex Source
