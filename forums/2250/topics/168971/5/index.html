    Crimson Editor is a Notepad replacement for Windows.

    It is used mainly for editing source codes, but you can use it to edit text files as well.

    One great feature that Crimson Editor has but Notepad doesn't is TABS!!!! One window, many tabs, great for editing.

    Since it's a source code editor, it will have features that make editing easier. Take for example the curly brackets ( { } ) in Java. It will mark for you the open and closing tags for you, so you will know where you go wrong. The markings aren't that obvious though, it's just an underline.

    It also has a highlights the current line you are editing.

    If a file that you are editing has changed (by using external programs other than Crimson Editor itself), it will prompt you to reload it

    aTunes

    Open-sourced media player. Nice interface, but can be very laggy. Java needs to be installed before this player can be installed.

    Download Java from Sun: http://java.sun.com/

    One bad point I noticed - while Windows Media Player automatically detects all your songs, this player doesn't. You have to load what is called a repository. For convenience sake, it's best to store your songs in one location and load one repository, otherwise, you will find it a headache, to keep on loading repositories.

    Those who like Chinese or any other Asian songs and the song title is in an Asian language, be aware that all these Asian words will turn into boxes. Mine did, nothing I do could revert them back.

    Edit: Didn't know uninstall is also a pain!
