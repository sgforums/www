    Site Inspector

    You have worked hard on your website and in Internet Explorer it looks fine, but in Firefox it doesn't. With Site Inspector you can easily switch between IE and Mozilla browsers. But especially the online tools are very handy for webmasters. You can check links, CSS stylesheets and HTML and a lot more.

    Civil Netizen for Windows

    Civil Netizen for Mac OS X

    You want to send big files to your little brother, but that doesn't allways succeed with email, ftp and different messengers will also cause problems. Civil Netizen is a software programm that solves this. When both parties (sender-receiver) have installed Civil Netizen you can place folders or files by "drag and drop" into the "send parcel" window. Files are encrypted and zipped. The receiver gets a hyperlink by email to download the files. Civil Netizen also works on the local network and handles routers/firewalls correctly. Only proxyservers at the office can cause trouble.

    MP3Rocket

    MP3 Rocket is p2p software for music, mp3, movies, software and game downloads and uses the Gnutella network. The latest version has the possibility to preview files during download and filters unwanted content and virusses.

    SketchUp

    Google now offers free drawing software SketchUp after they've taken over @Last Software. The SketchUp program is only free for those who use it at home. Business users will have to buy the paid version of SketchUp, but in return they also will receive more functionalities.

    Audio

    QuickTime Alternative

    From its name, you can tell that it's an alternative to QuickTime. This codec is useful if you don't want to download QuickTime Player, which eats up your resources.

    It works with Media Player Classic and Windows Media Player.

    Real Alternative

    Real Alternative is an alternative codec for Real Player. It's free of advertising, no expiry unlike the original Real Player which often shows pop ups while playing a song or video. It doesn't start at startup as well, using up lesser system resources.

    Works with Media Player Classic and Windows Media Player.

    HTML Editors

    ColorCoder

    This is a tool used to specify colours. Good for lazy users who just don't want to find the colour codes online or calculate them and useful for those who wants to double check that their colour coding is correct.

    You either choose the colours from the palette given or type in the RGB values yourself. The program will automatically convert the RGB values into hexadecimal values, so it saves you the time of converting them.

    WebDwarf

    Another HTML editor. This time, no more codings. Completely drag-and-drop. Good for newbies to HTML.

    The only caveat - you need to provide your email in order to download it. The Privacy Policy looks good enough.
