    http://www.winbuilder.net/news.php

    WinBuilder, a program that helps you to build and customize Live CDs. It's similar to tools like BartPE.

    A PE environment is actually a lightweight version of an OS. Specifically, from XP onwards.

    It allows you to recover your system if it crashes (LOL... DOS-based tools are a bit limited and that's where GUI comes into good use).

    Those who suck at DOS will find this useful. It's like booting up to a Linux CD, copy all the things you need and reformat. For the more technically advanced, you can do repairs to a PC without having to resort to reformatting and reinstalling a system.

    Those who have used ERD Commander before (used to be from Winternals before MS bought the company) , then you've used a PE environment.

    A little more info - http://en.wikipedia.org/wiki/Windows_Preinstallation_Environment
