    If it doesn't work, go to Start > Run and type in cmd

    Hmm... try this:

    Go to Start > Control Panel > Administrative Tools

    Double click on Computer Management

    Select Local Users and Groups from the left

    Double click on Groups on the right

    Right click on the right hand side and select New Group...

    Type in a group name and their descriptions

    Click Add to add in members. If you know the computer's name, it will be easy. If not, add a computer name for them

    Click Create, then click Close

    Right click on your folder, and select Properties

    Select the Security tab

    Click on Add to add in the group you've created earlier

    Then change the permissions below...
