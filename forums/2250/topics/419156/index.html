  • kenn3th

    Do this

    • Go to "My Computer"
    • Right click over the hard disk with your operating system (C drive mostly)
    • Click on "Properties"
    • Followed by the "Tools" tab
    • You should come to a screen where they would ask you for Error Checking. 
    • Click that option (UAC needed) 
    • Tick - Auto fix file system errors, and Scan for and attempt recovery of bad sectors
    • It will come out with an option saying Window is not able to check disc as it's running -Schedule it
    • Restart your PC.
    Afterwards try deleting the file again.
