    For those using Outlook Express, this tweak makes OE loads even faster. On this AMD 1GHz and using 256MB of RAM system, it loads faster than it loads IE. Cool tweak. Credits to TweakXP team.

    First, uninstall Windows Messenger.

    Go to Start > Run and type in this:

    RunDll32 advpack.dll,LaunchINFSection %windir%\inf\msmsgs.inf,BLC.Remove

    Make sure you type in this carefully, or you will have to re-install Windows.

    After uninstalling, start Registry Editor (Start > Run, type in regedit)

    Navigate to this key: HKEY_LOCAL_MACHINE\Software\Microsoft\Outlook Express.

    Move your mouse to the pane on the right. Right click in the right pane and select New > Dword value.

    Give it the name Hide Messenger. Right click on Hide Messenger and select Modify and set the value to 2.

    Next, go to Start > Run and type in services.msc.

    Find Messenger. Double click it. Click on the Stop button. Let Windows stop the service. Once it's completed, change the Startup type to Disabled. Click OK and exit services.

    Now press Ctrl + F. Do a registry search for this key: FB7199AB-79BF-11D2-8D94-0000F875C541. After Windows has finished searching, click on the + sign next to it.

    Click on InProcServer32. Move your mouse to the right pane. Right click on Default and select Modify.

    Delete the value in it. Do not delete the Default key. Only delete the values in it so it becomes empty.

    Now move to left pane again. Select LocalServer32. Move your mouse to the right pane. Right click on Default and select Modify.

    Delete the value in it. Do not delete the Default key. Only delete the values in it so it becomes empty.

    Restart your computer for it to take effect.



    The blue box shows that the Default key is empty. It is not DELETED. Only the value is.
