    If you have more than 256MB of RAM this tweak will considerably improve your performance. It basically makes sure that your PC uses every last drop of memory (faster than swap file) before it starts using the swap file.

    1. Go to Start then Run
    2. Type "msconfig.exe" then ok
    3. Click on the System.ini tab
    4. Expand the 386enh tab by clicking on the plus sign
    5. Click on new then in the blank box type"ConservativeSwapfileUsage=1"
    6. Click OK

    Improve XP Shutdown Speed

    This tweak reduces the time XP waits before automatically closing any running programs when you give it the command to shutdown.

    1. Go to Start then select Run
    2. Type 'Regedit' and click ok
    3. Find 'HKEY_CURRENT_USER\Control Panel\Desktop\'
    4. Select 'WaitToKillAppTimeout'
    5. Right click and select 'Modify'
    6. Change the value to '1000'
    7. Click 'OK'
    8. Now select 'HungAppTimeout'
    9. Right click and select 'Modify'
    10. Change the value to '1000'
    11. Click 'OK'
    12. Now find
    13. 'HKEY_USERS\.DEFAULT\Control Panel\Desktop'Select 'WaitToKillAppTimeout'
    14. Right click and select 'Modify'
    15. Change the value to '1000'
    16. Click 'OK'
    17. Now find
    18. 'HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\'Select 'WaitToKillServiceTimeout'
    19. Right click and select 'Modify'
    20. Change the value to '1000'
    21. Click 'OK'[/b][/quote]

    Lots of useless services in Windows NT onwards...

    Disable them...

    Go to Start > Run and type in services.msc

    Right click on a service and select Properties. Click on Stop. Then select Disable from the drop-down list for Startup Type.

    Here are some useless services:

    AdobeLM Service
    Alerter
    ClipBook - only disable this if you are not on a network
    Computer Browser - only disable this if you are not on a network
    Error Reporting Service
    Fast User Switching Compatibility - don't disable this if you want to switch user without logging off
    Indexing Service - don't turn this off if you use Windows search function regularly
    Windows Messenger
    Net Logon - only disable this if you are not on a network
    NetMeeting Remote Desktop Sharing - disable this if you don't use remote desktop
    Network Provisioning Service - only disable this if you are not on a network
    Print Spooler - disable this if you have no printer
    Remote Desktop Help Session Manager - disable this if you don't use remote desktop
    Remote Registry
    Server - disable this if you are not a network
    Smart Card - disable this if you don't use smart cards
    Terminal Services - disable this if you don't use remote desktop
    Windows Firewall/Internet Connection Sharing (ICS) - disable it if you have a third-party firewall like ZoneAlarm
    Windows Time
    Wireless Zero Configuration - if you are not on a wireless network
