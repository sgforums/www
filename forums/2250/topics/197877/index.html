    Before you do this..
    go to my computer
    go to 'tools' , then 'folder options' (all on the toolbar on top)
    go to the 'view' tab then make sure the 'view hidden files and folders' option is checked
    Going back to firefox...
    On the bookmarks tab click 'manage bookmarks'
    then file, 'import'
    select import from file
    in the next menu that pop-up..
    Go to my computer
    go to your main drive.should be C drive
    then 'documents and settings'
    go to the folder with your login name on it
    go to 'Application Data'
    then 'mozilla', 'firefox' and 'profiles'
    go through the last folder which is'xxxxsomethingxxxx.default'
    should see a folder that is named'bookmarkbackups'
    if there are more than 1 bookmark backup file there you should select the one that is the latest created one
