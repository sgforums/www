    1st thing first: This is a old model of printer. Sometimes, old version just dont support X64 of the Windows, even if you use X64 of Windows Vista.

    2nd thing: While theoritically possible, usually it seldom happens, where a driver is certified to be compatible with Vista turns ugly in Windows 7.

    3rd thing: Sometimes some drivers, regardless a printer or not, just a painful installation. Well, I have seen a lot of weird weird file without a exe program to install on. For this, go to Computer -> Properties (Right click) -> Device Manager and look for any exclamation marks or not. If yes, then try to pin point to that folder contains weird files dl from your manufacturer site. If no, then scan for hardware changes and try again. If still cannot, then maybe try adding legacy drivers (Which may not workdue to X64 drivers arent legacy anyway)
