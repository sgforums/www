    If you are using AdBlock or AdBlock Plus, disable it temporarily and refresh the page.

    If it still doesn't work, try this:

    Type in about:config in Firefox's address bar.

    Scroll down to browser.frames.enabled

    Make sure that this value is True. Otherwise, right click and select Toggle.
