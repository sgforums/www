    Under "file", there is a tab called data file management, under data files, all your email regardless archived or not are stored in that personal folder under a file called outlook.pst.

    To access that file, you have to make sure outlook is not running, go to that folder, copy it and put it into your desired folder of your desired computer. To access the emails, your other computer needs outlook, and under the same options above, select to add that database. Its tedious, and sometimes you might screw up.

    Open OE

    goto Tools

    select options

    select Maintenance Tab

    click Store Folder button

    Note : The path to your email data

    Goto to folder and copy the whole OE directory to an ext drive or Thumbdrive

    P.S. you can personalize this folder by clicking on the Change button and

    select a new folder store for your OE data.
