    I am not familiar with the specifics of cpanel but a quick search for 'cpanel open mail relay' threw this up:

    http://help.godaddy.com/article.php?article_id=1456&topic_id=211#exim

    To set the default mail behavior to :fail:, you will want to execute the following command through SSH:
    echo "defaultmailaction=fail" >> /var/cpanel/cpanel.config

    Good luck!

    Hi patgoh,

    You can email the ISP.

    Do a WHOIS of of their IP address and email the relevant ISP.

    There will be an abuse email address, usually something like this:

    abuse@example.com

    I can help you contact another admin who will be able to help you if you need more help.

    If you are using Linux servers, a WHOIS can be done easily.

    whois IP address from the Command Prompt aka Terminal aka Bash.

    If you need more details, do a dig.

    dig IP address

    Or to get some details, try:

    dig -x IP address / server name
