    Turn off modem and router, wait 10 secs, turn on again, see how.

    If successful, good.

    If not successful, go to router configuration page.

    Usually this is 192.168.2.1. Or you can check it from ipconfig.

    Then type the address on your browser.

    Then from there you can adjust the setting of the router.

    Of course you need to be sure it is not a hardware problem, make sure all the cables are connected properly etc.
