    Visual Basic is programming, Visual Studio is the IDE for Visual Basic and other Microsoft-related stuffs.

    There's a script version of VB, known as VBScript, but I don't think it can handle database.

    If you want to use scripting and database, try PHP.

    Download XAMPP and set up your PC as a server to try out the scripts and install MySQL. It's very easy, but there's no such thing as click click click. You write your own scripts.

    Otherwise, use Visual Studio and ASP.Net... not my favourite anyway, I didn't flunk this for fun.

    Or if you want to click click click and done, get Visual Studio Express. It's really click click click and done.

    Or if you want, try Oracle. But it's not free.

    You can learn PHP from these sites:

    http://www.w3schools.com/php/default.asp
    http://www.learnphpfree.com/forum_topics.php
    http://www.phpbuddy.com/
    http://www.tizag.com/phpT/

    Or if you would prefer a book, get Beginning PHP, published by Wrox. It's red in colour, with human faces as the front cover.

    Another good one would be The Complete Reference of PHP... white colour book. It's a pretty thick book, but simple with lots of references.
