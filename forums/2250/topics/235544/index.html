    Don't quite get you.

    If you are using Firefox, and the download got cut off, you can resume it if you are using Firefox's built-in download manager.

    Just go to Tools > Downloads, find the file and click on Resume.

    One thing though, if you set Firefox not to remember anything, you are out of luck.
