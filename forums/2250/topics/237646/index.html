    Pros of using Apple - solve your viruses nightmare, at least for now.

    Video/photo editing are better in Apple. Also don't need to worry about registry and crashing, two big buggers in Windows.

    Cons of using Apple - would be software compatibility. Some softwares are just for Windows only, so if you want to run them, you will need to install Boot Camp, then install Windows via Boot Camp to use it.
