    issue resolved?

    if not, try checking on the items below first

    1 - how much free space is in your c: drive
    if your c: drive is fully used up, on shutdown, your pc would not be able to save your profile so it will become corrupted.

    2 - check if your profile is still there
    go to start> control panel> system>
    click the advance tab (WinXP) or advance system setting (WinVista)>
    Look for user profile> click settings under user profile>
    is your profile still there?

    3 - did you try to do a system recovery?
