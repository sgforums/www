    The new Hotmail is using Ajax, and Ajax relies on JavaScript. If your browser doesn't allow JavaScript, or if a firewall is blocking JavaScript from loading, it will not work properly.

    If you are using Internet Explorer or any browsers that's based on IE, make sure that there are no errors in loading. This can result in a disability to load Hotmail.

    For other browsers, please make sure that JavaScript is allowed to load.

    If nothing works, clear your browser cache.
