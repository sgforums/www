    Released today, Opera 9.0 Preview 2 sports many new features and improvements over its predessor, Preview 1. Among many of the new features are

      [*]Opera Widgets
      [*]New Site Preferences
      [*]Support for BitTorrent and Torrent files
      [*]Improved handling of CSS 3
      [*]Added support for YCCK and CMYK formats in JPEG images.
      [*]Multiple improvements to SVG 1.1 basic and SVG DOM support.

    And much more. Click here to download it now!

    Opera is a faster and more secure browser than Firefox and IE. It uses less memory than Firefox (Firefox has the notorious Memory Leak Bug) and lets users disable content on a page by simply hitting F12 for the contextual menu.

    Note: Opera 9 P2 disables the start bar at default. Right click the URL Bar and hit Customize and check "Start Bar" to get it back.
