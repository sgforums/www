    Never hear of such things before.

    I'm going to ask a few lame questions, hope you don't mind.

    1. What OS are you using? XP or Vista or others?

    2. If using XP, are you using the classic Start Menu or XP Start Menu?

    3. If using Vista, which Start Menu are you using? The new version or old version?

    4. Can you boot up your laptop and run this command (only works for XP and above) . Let me know if it's working OK.

    Click on Start > Run.

    Copy and paste in shutdown.exe -s -t 00

    Press Enter.

    Does it restart your computer or shuts it down?

    Last question isn't lame, but maybe could help find some oddities.

    What's your laptop hardware specifications?

    1. Processor (Example: Intel Core 2 Duo)

    2. Motherboard (Laptop should be using mobo from Intel, but there are some which doesn't)
