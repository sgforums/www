    I tried several things to get rid of it, including:

  • ndmmxiaomayi
    2 Apr 07, 16:21
    Start > Run and type in msconfig

    Select the Startup tab

    You should see quite a few programs

    There will be 2 Adobe programs listed, one is the reader, one is the installer.

    Untick the box for the installer. Click OK and restart PC.

    If unsure, just post back these 2 entries or take a screenshot.
