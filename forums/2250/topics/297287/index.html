    It's either a CD or drive issue.

    If you are sure that your CDs are working, your CD drive has died.

    Drivers can be working, but Windows don't report faulty devices. It only applies to faulty drivers, because they need the drivers to work.

    Another way to test, get your Windows CD. Set the BIOS to boot up from CD. If it could boot up to the CD... something else is wrong then. A dead CD drive most likely won't boot up from the CD drive and will skip directly to the next device - a hard disk or whatever that is set.
