    Most likely.

    You can try one thing though, and I find that it works.

    If you are a Singnet customer and using Firefox, do this:

    Go to Tools > Options

    Click on Advanced at the top.

    Select Network tab

    Under Connections, click on Settings.

    Select Manual proxy configuration.

    In the HTTP Proxy box, type in this: proxy.singnet.com.sg
    Port box: 8080

    Apply the settings.

    Remember to clear your cache before trying to load any websites, or else you will get a time out error.
