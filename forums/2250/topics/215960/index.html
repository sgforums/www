    The chances of a RAM fault and corrupted downloads are low. Most corrupted downloads happen because it is incomplete, due to network problems or files corrupting it. Some downloaders have the ability to split the file into smaller sizes for faster downloading, but don't have the ability to merge them back, causing corruption. That's why I suggested downloading normally without using DAP to be sure that it isn't a software problem.

    If there's a need to test the hardware, try either Microsoft's website or Memtest86.

    There might also be some problems with the hard disk as well. It's very hard to tell. If memory test shows that RAM is OK, you might need to reformat and reinstall Windows. If after reinstalling Windows the same problem occurs, then they have spoilt the hard disk as well.
