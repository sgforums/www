    Find the following files in your PC:

    %System%\XpFirewall.exe
    %System%\bingoo.exe
    C:\funny_pic.scr
    C:\see_this!!.pif
    C:\my_photo2005.scr

    Note: %System% is a variable that refers to the System folder. By default this is C:\Windows\System (Windows 95/98/Me), C:\Winnt\System32 (Windows NT/2000), or C:\Windows\System32 (Windows XP).

    Next, download this file.

    Unzip it and run it.

    You will find find1.txt to find7.txt on C drive.

    Post back find1.txt to find7.txt

    Note: Not all find* files will be on C drive. You may only have find1.txt, find4.txt and find7.txt. That is common.
