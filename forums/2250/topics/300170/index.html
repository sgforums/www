    1. Open MSN Messenger

    2. Tools > Options

    3. On the left, select Security

    4. Check if this box is ticked - Allow links conversation windows and buddy list (sorry, me on WLM 9.0 Beta, but there's a similar option in earlier options of MSN)

    5. If it's ticked, untick this box

    6. Click OK to apply the settings

    If still nothing opens, make sure that Internet Explorer is your default browser.
