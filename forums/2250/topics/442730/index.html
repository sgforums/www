    Originally posted by Siliconchip:

    hmm. okay perhaps I should shift to wordpress. Do you have any advice/guidance to offer about using wordpress?


    Wordpress themes are easier to work with.  I am working with the commercial version of wordpress, so all the limitations are lifted for me. But the free version of wordpress still contains as many tool as the paid version ones.

    If you want to customise the wordpress with your own custom themes, you will have to pay a small fee to unlock the restrictions. Other than that, the free themes available are all good and neat.  I'm using the commercial version, so I do not have any theme restrictions.

    Wordpress allows you to upgrade your account based on your needs basis. If you need more space, you pay a small fee for it. If you want to have your own domain name like siliconchip.com, you will have to pay a small fee too. So it's very customer's centric model of doing business.

    Security wise, it's up to industrial standard and it security framework is constantly being upgraded and reviewed.

    Plugin wise, it's plug-and-use model. very simple and easy, you do not need so much technical knowhow to use a plugin. The framework for wordpress makes it easy for develop to create plugins for users.

    Golden tips when using wordpress: If you're using free version of wordpress, try to save your articles frequently  

     

    3 types of wordpress:

    Free, Paid, and commercial.

    Paid version is mean you pay for addition service to enhance your blog.

    Commercial version is means you download the free wordpress, and customise the blog your own way and host it yourself. This version doesn't get so much support/updates from wordpress, but you still get the major updates. Functions in this version are not updated too. There are some functions in the free version which you can't find in commercial version.
