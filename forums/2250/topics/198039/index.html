    In case your keyboard give up on you while you are navigating through Microsoft Windows........

    Very good piece of info..............

    An alternative keyboard in Windows.

    In case your keyboard or some keys stop working, Microsoft provides
    you with an alternative way to type in using the mouse.
    To work this tool, Click on 'Start' (bottom left of your screen)
    Then
    Select 'Run'
    Type in 'OSK'
    Press OK

    A keyboard will appear for you to use as normal keyboard.
    =)
