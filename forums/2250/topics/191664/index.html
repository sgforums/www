    Is yours a laptop?

    If yes, check the base for a validation sticker. If there's no sticker, that means it isn't genuine. All laptops using genuine Microsoft OS will have that sticker given by Microsoft.

    If it is a desktop, did you bought it from retailers like Harvey Norman and Courts? If yes, then it shouldn't be a problem.

    If you bought it from some other shops in Funan or Sim Lim (branded ones like HP, Acer, Compaq, etc), be sure to check the sides for the Microsoft sticker.

    If you custom built it, double check that the Microsoft OS disc you bought is a real one. There are some fakes on the shelves which looks like the real ones, and Microsoft offers exchange for it. Call up Microsoft and they will gladly help you.
