    Uhh, incompatible mobo?

     

    The processor in question would be a 975, and that needs to be paired up with a Socket 1366 X58 mobo. 

     

    Get a 920 instead, and a nice X58 mobo, like the Gigabyte EX58-UD3R or the ASUS P6T. 975s are not worth it, since they cost more than 1k for the chip alone. 920+good mobo is about $800?

     

    OS: Windows 7 Home Premium = $279?

    CPU+mobo: i7 920 +, say, ASUS P6T-SE is about $760

    RAM:  Team Elite DDR3-1333 CL9 6GB (3x2GB kit) = $186 (3GB kit $118)

    HDD: Seagate Barracuda HDD 2TB (Unsure if Barracuda XT) = $319

    GPU: I would suggest the Radeon HD5870 instead. Cheaper, and about the same performance. HD5870 = ~ $600, GTX295 = ~$800?

     

    And there is no tangible benefit from the Barracuda XT's SATA 6GBit interface right now. Better performance is achieved via a SATA 3GBit interface. It doesn't even saturate the 3GBit interface. You might as well get 2 hard drives and stripe them, a.k.a AID 0 them.
