    Confirmed to be Friendster virus.

    http://friendstertalk.com/viewtopic.php?id=6684&p=1

    Follow the above link to remove the virus.

    Your best bet to remove the virus is to use another browser. Don't use IE, because it works in IE.

    After that, do the following:

    For Windows XP users

    1. Go to Start > Run and type in notepad C:\WINDOWS\system32\drivers\etc\hosts

    2. Add the following lines in:

    127.0.0.1 getitrightnowonline.info
    127.0.0.1 sitejmp.info
    127.0.0.1 adjmp.info

    Save the file.

    For Windows 2000/NT users

    1. Go to Start > Run and type in notepad C:\WINNT\SYSTEM32\DRIVERS\ETC\hosts

    2. Add the following lines in:

    127.0.0.1 getitrightnowonline.info
    127.0.0.1 sitejmp.info
    127.0.0.1 adjmp.info

    Save the file.

    For Windows 98/ME users

    1. Go to Start > Run and type in notepad C:\WINDOWS\hosts

    127.0.0.1 getitrightnowonline.info
    127.0.0.1 sitejmp.info
    127.0.0.1 adjmp.info

    Save the file.
