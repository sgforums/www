    From one computer to another?
    First of all, go to your Network connections, click network setup wizard and choose the option which says "Not connected to the internet".

    If your prompted to restart please do so. Then off your firewall and antivirus.

    Then you need to set a static Ip address, go to your Network connections and double click your LAN connection icon, under Internet Protocol, Select "Manually Set IP address".
    ANy ip address will do, i recommend 111.111.111.111
    As for gateway any will do, as for the other computer do the same steps as above, but set a different IP address but the same gateway.

    Right click the folder you want to send, and go to sharing. You should see that you can choose a option for you to share that folder.

    If you want to receive data from the computer which you set the IP to 111.111.111.111
    You use the other computer and open My computer, ujnder the search bar, type \\111.111.111.111

    And copy paste what you want.
