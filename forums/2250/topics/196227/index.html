    Huh? When windows got MSN?

    It is windows messenger. Yar,
    1. Start>Run>"control">add or remove programs>Add/Remove Windows Components>Uncheck Windows Messenger and click Next
    2. Start>Run>"gpedit.msc">Local Computer Policy>Computer Configuration>Adminstrative Template>Windows Components>Windows Messenger>Enable both:
    i) Do not allow Windows Messenger to be run
    ii) Do not automatically start Windows Messenger initially

    Done
