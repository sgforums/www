    Adminpak - is it the Winternals Adminpak? If yes, then you run out of luck. It works on XP and below.

    You can't upgrade Home Premium to Business edition. See here.

    You can install Vista and XP.

    http://lifehacker.com/software/top/windows-vista-beta-how-to-dual-boot-windows-xp-and-windows-vista-179906.php

    Installing Vista and XP together might cause problems because they use the same file system, which is NTFS. This is an age-old issue, it happened with previous versions of Windows. Windows 95, 98 and ME will never work together, because all uses the FAT32 system. Better still, all uses the same boot file, which literally crashed it. That's why Microsoft will never help you troubleshoot, only asking to you to reformat. One you damage the boot file, that's it, reformat is on the way.

    The same thing for Vista and XP. Since you are talking about domain, I believe you are in a business environment? If yes, get a spare PC to play. I don't think your company is so kiam siap until there's no spare PC.
