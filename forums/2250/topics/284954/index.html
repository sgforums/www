    Could be the fragmentation of Windows files that causes Windows to take a long time to piece back the files.

    Defrag your hard disk.

    There's a thread here about defragmenting tools if Windows Defragmenter hangs.

    http://www.sgforums.com/?action=thread_display&thread_id=284786

    Remember to disable screensaver as well as not use the computer while Windows is defragmenting to prevent corruption of files.

    It can take up to a whole day if you have a lot of files and if the hard disk is heavily fragmented.
