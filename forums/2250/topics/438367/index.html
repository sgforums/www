    Laptops use 2.5" HDDs, so if you have the HDD, all you need is a hard drive enclosure. There are two types, one for IDE drives (older type) and one for SATA (newer type). Make sure you buy the right enclosure. These casings come with instructions on how to install the drive, so no problem there. 

    They will cost you about $15-$20, from typical computer stores.
