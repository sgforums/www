    CD is useless, but manual is useful.



    1. Type in the IP address given to the router into your Internet browser (such as Internet Explorer or Firefox). By default, it is 192.168.1.1. Unsure, check manual.

    2. Log in. Linksys default username is blank (meaning no username), password is admin.

    Configure from there.
