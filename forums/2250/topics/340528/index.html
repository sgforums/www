    1. Open Folder Options in Control Panel.

    2. On the View tab, under Hidden files and folders, click Show hidden files and folders.

    Please backup your windows registry first before trying the following

    Go into registry editor, go to the following key:

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Explorer

    In the right hand pane, see if there is a value called NoFolderOptions

    If yes then change it’s value to 0.

    Reboot your computer and you should be able to see folder options.
