    For me, it depends. I use Chrome most of the time, but Firefox in situations where CHrome won't run.

     

    Chrome is good if you are only going to use 2-3 tabs (Low-end machine) to 5-6 tabs (relatively powerful PC). Once you go over that, Chrome tends to take up more resources given that each tab is a separate process. Useful though, since sometimes Windows will close a process cos it "did something funny", and if you had 8+ tabs in FireFox, *boom*. Gone. Then you have to wait a while to load all the pages again. For Chrome, only the tab that "did something funny" is closed.
