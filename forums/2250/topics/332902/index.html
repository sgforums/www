    Try this:

    Press Windows Key + R, type in "cmd" and press Enter, and a black window will appear. Enter the command "ipconfig /all" and look for the value under IP address for your router (probably listed under "Local Area Connection"). That will be the value for the "Default Gateway" field when setting up the static IP address.

    Regarding the IP address field, just use something like "x.x.x.101" where "x.x.x" is the first three set of numbers from the router's (Default Gateway) IP address.

    Subnet mask: 255.255.255.0

    As for the DNS server fields, use the following IP addresses:

    Singnet
    Preferred: 165.21.83.88
    Alternate: 165.21.100.88

    Starhub
    Preferred: 203.116.1.78
    Alternate: 203.116.254.150

    Last but not least, you will need to disconnect and re-connect the LAN connection for this change to take effect.
