    For web use, I've never heard of DLLs being used. Maybe there are, but users will be wary of it. DLLs are libraries and can be registered with Windows. If done wrongly, it can mess up the registry and Windows and cause Windows to produce errors.

    Rather than use DLLs, use ActiveX for IE and IE-based browsers, and Java applets for non-IE browsers.
