    Originally posted by mayi jie:
    R0 to R2 lines is related to Internet Explorer. The homepage, proxy, etc.

    R3 are some hooks to Internet Explorer to improve IE.

    F0 line are used by viruses to start themselves automatically, rather than putting themselves in MSCONFIG.

    F1 line is using the same method as F0 lines programs, but are usually good. Some printers use this method.

    F2 line is the same thing, except that it may be good or bad.

    F3 line is the same as F0.

    N1 to N4 lines are used by Netscape or Mozilla. Generally harmless.

    O1 are the hosts line. The Hosts file exists in all operating system, and acts like a telephone directory. When you access a website, it will first check this file. If it can find the website in this file, you don't need the Internet. If it can't find, you have to dial out to your ISP, tell your ISP you need to visit this particular website.

    O2 lines are stuffs that programs install on IE to give it some functionality. For example, Spybot S&D uses it. This gives it the ability to protect IE, although there are other ways it's using to protect your system.

    O3 are toolbars. Google, Yahoo! and MSN are common ones.

    O4 are the startup items. All these programs will auto startup when the PC starts.

    O5 are Control Panel items. Usually not affected, but in future might be.

    O6 and O7 are permissions issues. They can prevent you from running certain things.

    O8 is the IE right click menu. Usually used by download managers such as Flashget and Download Express, so that you can download a file easily.

    O9 are internet buttons. Some are visible, like Windows Messenger, others are not. You can choose to hide them, but they will be present.

    O10 is related to TCP/IP. TCP/IP use a chain to make sure that you can surf the Net smoothly. But some viruses hijack this line, leaving you with no internet access. There are special programs to fix this issue and experts need to take a look at it. At no point they should be fixed by yourself as it's already in a very bad position.

    O11, not so sure, but it's used by some programs. More recently is IE7, you can see an extra [International]*International line. On IBM machines that are using IBM version of Java, you will see [IBM]*Java, I think.

    O12 are plugins for Internet Explorer, such as the PDF plugin to read PDF files.

    O13 are hijacks, same for R0 to R2 lines. They can do re-direction. The difference between this and the R lines is the start page. Start page are R lines, whereas for O13 lines, when you type www.google.com, it may re-direct you to some other pages. It often takes a few tries to get to the website correctly.

    O14 are installation files to reset your homepage. It's used by most ISPs and computer manufacturers to set the startup page of IE.

    O15 are the websites in trusted zone of IE. In IE, if you give explicit trust, everything will run. Some malicious sites use this method to make sure they can run properly, even for a secured browser.

    O16 are your ActiveX. Common ActiveX are Windows Updates, online virus scans and games.

    O17 is your domain. They are similar to O1 as in they do re-direction, however, for O17, it's more malicious. You will not notice any changes until you do a search, because the search has been re-directed somewhere else. Legal uses are for companies, such as Singnet (you will see the IP addresses start with 165). When you type in some words in the address bar, it will automatically search Singnet sites.

    O18 are extra protocols for IE or protocol hijacks. Most of the time not needed.

    O19 is the user style page hijack. Some users use a specific style page due to accessibility issue, such as they are near blindness. Some viruses use this method to stop the styling page from being in use and can cause popups.

    O20 are DLLs that load in memory when you login and will be gone once you logout. A common one is WgaLogon.dll, which will check if your machine is a legitimate one. There are other uses for it which viruses use.

    O21 is another autorun method, has been used in most fake anti-spyware programs. They won't load until the O4 load.

    O22 is another autorun method. What it does is out my knowledge.

    O23 are services. Go to Start > Run and type in services.msc. Or using MSCONFIG and select the Services tab.
