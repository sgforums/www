    30 Aug 07, 23:35
    1. Windows is running out of hard disk space. It needs at least 5% free space to run properly. And 20% for optimum performance.

    2. Some programs stopped responding and did not trigger the Crash box (aka XXX Program is not Responding)

    3. Your hard disk is very fragmented. Try defragmenting it.

    Originally posted by fifteen:
    i have like 19.6gb of hard disk space left. and i do not have programmes not responding. now im defragmenting my hard disk as you suggested. whats the meaning of defragmenting by the way?
    19.6GB out of how many GB that your hard disk have?

    Defragmenting is the opposite of fragmenting. Fragmentation of files occur when bits and pieces of it are everywhere, so it takes some time to form back those bits and pieces back to one program and execute it.

    Defragmenting tries to put them right, but moving all the fragmented pieces back into one chunk so they could accessed and executed faster.

    These services can be stopped and disabled:

    1. Wireless Zero Configuration (if you don't use wireless)

    2. WebClient

    3. Windows Time

    4. Terminal Services (as long as you don't use Remote Desktop/Assistance, Fast User Switching, Microsoft mouse (why a mouse needs this I have no idea))

    5. Print Spooler (if you don't have a printer)

    6. SSDP Discovery Service (if you don't use P2P)

    7. Secondary Logon

    8. Windows Firewall/Internet Connection Sharing (ICS) - You already have McAfee installed, so Windows Firewall can be safely turned off

    9. Workstation (not needed if this is your one and only computer or if you are not sharing any files on your home network)

    10. Server (same as Workstation)

    11. Fast User Switching Compatibility (if you don't use Fast User Switching)

    12. DNS Client (only if your computer is owned by you, not part of a company or a school)

    13. Computer Browser (same as number 9)

    14. Windows Audio (if you have no speakers)

    To stop and disable these services:

    1. Go to Start > Run and type in services.msc

    2. Double click on the name listed.

    3. Next to Startup Type, change it to Disabled from the drop-down menu.

    4. Click on Stop.

    5. Click OK.
