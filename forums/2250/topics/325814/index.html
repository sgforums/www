    Go to Services and make sure "Office Source Engine" service is running. It should be OK then. Always make sure this service is running before you update Microsoft Office.

    To get to Services in Vista, press the Window button, in the search box, type "Services". You should see it then and you click on it. For XP, you need to go to Control Panel->Administrative Tools -> Services.
