    Just for fun, for those who always run into trouble unzipping files or who just don't want to download extraction programs. More work for the developers only.

    Download Chilkat ZIP 2 Secure EXE.

    Find the .zip files that you've zipped up.



    Click Open, then Create.

    You will get a successful message.



    The .exe file is in the same location where your .zip file is. Now can upload to your server or other places.

    The only disadvantage - file size is larger...

    My example has a file size of 1.60MB, while after making it to an executable, the file size is 2.0MB.

    For the programming pros/freaks... refer to the website to create scripts for creating a self-executable zip or other stuffs. Very useful.

    Winrar also can create self extracting EXE(SFX)

    1. right click the folder u want,
    2. add to archive
    3. on the left side got alot of checkbox. select "Create SFX archive"
    4. click "ok" if u dont need any advance option

    if u want to have advance option.. like set password etc...
    4. Click onto "advanced" tab
    5. there, on the left side u can click 3 options.
    "compression", "SFX option", "password

    a) compression allows u to select which file type to compress the most. audio, pic color or general
    b) SFX option allows u to ...
    i)extract to which folder
    ii) auto run which file after extraction
    iii)add shortcuts
    iv) delete what files after extraction
    v) prompt question
    vi) unpack to temporary folder
    vii)silent mode
    viii) overwrite mode
    ...and many more for u to explore

    c)password is for you to set password for privacy
