    (for win xp)
    go to:
    -> control panel
    -> date, time, language, and regional options
    -> regional and language options
    -> (under tab) languages
    -> (under text services) click details
    -> (under installed services) click add
    -> (under input language) select chinese (PRC)
    -> click ok

    you should now see the chinese (PRC) under the window for installed services.

    -> click ok

    now, at the bottom right hand side of your screen, there should be a button called "EN". click on it and you should find the chinese (PRC). select it and you can now use it.

    if there isnt any "EN" button at the bottom, go to the text services again and (under preferences) click language bar. A "Language bar settings" should pop-up. Tick "show the language bar". click ok twice. and you should now see the "EN" button.

    On the same tab, click on Details, then on the new window, under the Preferences section of the Settings tab, click on Language Bar, then check Show the Language bar on the desktop. If you have not added the Chinese input before, you can do so at Installed Services section of the same window.
    Click Add and choose the input language you desire. I would let the screenshot do the rest of the talking.
