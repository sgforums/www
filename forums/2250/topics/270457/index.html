    Can you check NoScript and Firefox settings?

    1. Click on the S button, select Options
    2. Select the Advanced tab
    3. Select Untrusted tab
    4. Check that the Forbid Java box is not ticked

    Next, check Firefox settings

    1. Tools > Options
    2. Select Content at the top
    3. Make sure Enable Java box is not ticked

    The 2 are not known to work well... if you block Java in Firefox and block it again in NoScript, it makes no sense...
