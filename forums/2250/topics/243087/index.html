    From Symantec:

    To remove, follow this set of instructions:

    Step 1

    Show hidden files and folders

      [*]Open My Computer.
      [*]Go to Tools > Folder Options.
      [*]Select the View tab.
      [*]Scroll down to Hidden files and folders.
      [*]Select Show hidden files and folders.
      [*]Untick Hide extensions of known file types.
      [*]Untick Hide protected operating system files (Recommended).
      [*]Click Yes when prompted.
      [*]Click OK.
      [*]Close My Computer.


    Step 2

    Download ATF Cleaner or CCleaner to clear all your temporary files.

    Navigate to C:\Windows\System32 to delete this file:

    win[THREE RANDOM CHARACTERS]32.dll

    Navigate to C:\Documents and Settings\username\temp to delete these files:

    cli[TWO RANDOM CHARACTERS].tmp
    win[TWO RANDOM CHARACTERS].tmp
    cli[TWO RANDOM CHARACTERS].bat

    Step 3

    Go to Start > Run and type in regedit

    Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\Notify

    You will see this key: win[THREE RANDOM CHARACTERS]32

    Delete the key shown in bold.

    Next, navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft

    You will see this key: MSSMGR

    Delete the key shown in bold.
    _____________________________________________________________

    Guide to using the registry, using HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MSSMGR as example.

    Step 1



    Click on the + sign (boxed up in red) to expand

    Step 2

    Scroll down all the way to Software (boxed up in blue) and click on the + sign to expand



    Step 3

    Scroll down to all the way to Microsoft (boxed up in brown) and click on the + sign to expand



    Now scroll down carefully and locate MSSMGR (boxed up in pink)



    Right click and select Delete



    Windows will prompt you, click Yes.


    _____________________________________________________________

    This step is required if you are unsure about editing the registry.

    Copy and paste the codes below in bold into Notepad. Please use Notepad and no other editors, or I can't guarantee the safety of your system.

    regedit.exe /e peek1.txt "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft"
    regedit.exe /e peek2.txt "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\Notify    "
    notepad peek1.txt
    notepad peek2.txt

    Click on File > Save As...

    For File Name, type in peek.bat
    For Save As type, select All Files from the drop-down list. Then click on Save

    Copy and paste the contents of peek1.txt and peek2.txt when it opens. You will need to close peek1.txt first, then peek2.txt will open.
