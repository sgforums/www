    Nt sure what your problem really is. But sometimes i run into problems where ppl used chinese input from other developers to write things. Microsoft will not be able to read them if you do not have the software.

    What I do is i save them (messy and funny characters) in a text file (use the one came with Windows Xp, not MS Word). Then, right click on the file and choose open with IE. IE is very universal and should not have problem reading it. However, if it is still in funny characters, just click View ---> Encoding ---> choose Chinese Simplified or Chinese Traditional.

    This never fails me. Hope it will help.
