    1. rthdcpl.exe is a process belonging to the Realtek HD Audio Control Panel and is bundled alongside Realtek sound cards and audio hardware. This program is a non-essential process, but should not be terminated unless suspected to be causing problems.
    if u dont require it u can disable it.. its ok.. if it cause error after disabling it, enable it.

    2.hpbootop.exe is a process belonging to HP Boot Optimizer. This program is a non-essential process, but should not be terminated unless suspected to be causing problems.
    if u dont require it u can disable it.. its ok.. if it cause error after disabling it, enable it.

    3.remind_xp.exe is a process belonging to the SoftThinks CD Creator CD/DVD Writer and serves as a reminder to register for full product support. This is a non-essential process. Disabling or enabling it is down to user preference.
