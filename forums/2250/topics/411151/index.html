  • ndmmxiaomayi
    19 Sep 10, 09:56

    A couple of questions.

    1. Do you have on-board sound card? You can easily verify this in Control Panel (Sound). How many devices did it list for the playing of sounds?

    2. Is the new sound card installed successfully? You can check this in Device Manager. Ensure that it has no yellow exclamation mark.

    Some hardware can't be updated/detected by Windows. You need to go to the manufacturer's website to download the latest driver and install it. A restart is required.

    For the no icon in task bar issue, you need to enable it.

    1. Start > Control Panel

    2. Double click on Sounds and Audio Devices

    3. In the Volume tab, tick this box: Place volume icon the taskbar
