    Do you use a sound card or is it the audio on-board?

    If it's on board, it should be installed by Windows already.

    Try going to Start > Control Panel > Sounds and Audio. In the Volume tab, make sure Place volume icon in task bar box is ticked.

    If you are using a sound card, you need to install drivers for it. It usually comes in a CD. Look at your receipt for more info.
