    Troublesome piece of problem

    1. Uninstall all Apple software if you have it and restart the computer.

    2. Uninstall HP software and restart. Then reinstall HP software and restart again.

    3. Right click on your task bar and select Properties. Select the Start Menu tab. Click on Customize. Select the Advanced tab. Under Recent Documents, untick List my most recently opened documents box, then click on Clear List

    4. If you have an iPod or some storage device, plug it in, restart your computer. After restarting, remove it safely before shutting down your computer and restarting it again.

    5. Check your hard disk for error. Start > Run and type in chkdsk /f. You will see an error, just restart your PC. A blue screen will appear while checking. Don't skip it.

    6. Try booting to Safe Mode to see if you still experience this error.
