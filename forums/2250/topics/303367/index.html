    Uninstall all the Vista updates, restart your computer.

    Then reinstall all of them one by one till you get a problem.

    To uninstall updates

    1. Start > Control Panel or Start > Settings > Control Panel (Classic Start Menu)

    2. Double click on Programs and Features

    3. On the left, click View Installed Updates

    4. Select the update and click on Uninstall. Make sure that you don't uninstall any Office Updates if you have installed Microsoft Office. Only uninstall the Office Updates if you still have the same problem after uninstalling the Windows Updates.
