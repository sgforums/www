    Originally posted by Raraken:

    Apparently, according to my friend (Cuz I don't have iPod Touch/iPhone), 3.0 adds Bluetooth. How the hell did it even get that? Anyone wanna confirm?

    OS 3.0 doesn't ADD bluetooth, but rather it ADDS additional functions that you can use with bluetooth.

    For example, you can now do Internet Tethering via Bluetooth.

    And you can finally use stereo bluetooth headphones because Bluetooth in iPhone 3G/3G S finally supports A2DP.

    And you can also play games with your friend if he/she also has an iPhone and the same game, say maybe Tap Tap Revenge. Last time to play together, both of the users have to be connected to the same WiFi network to play together. Now with bluetooth connectivity, it bypasses that.

    All these I've listed applies to the iPhone 3G and 3G S, not the iPod touch 1st-gen. The iPod touch 2nd-gen will get its bluetooth activated too, so it'll have Nike+, and bluetooth gaming and A2DP support.
