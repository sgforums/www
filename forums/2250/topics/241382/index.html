    Originally posted by RvnRooRon84:

    Yes. Those files have been split, by looking at the part numbers, which is r00, r01, r03, etc

    Also, even if there is no part number as asked by TS, your file name will indicate it.

    Usually, it's named whatever-part1.rar, whatever-part2.rar, whatever-part3.rar, etc.

    All these files need to be in the same location for WinRAR to automate the tasks. If they are not, WinRAR will prompt you to locate the file.
