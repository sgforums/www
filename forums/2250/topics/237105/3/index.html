    Originally posted by FireIce:
    my printer is not printing!
    how??

    help!!!
    Assuming that you are using XP:

    There are lots of causes for a printer that's not printing.

    While users aren't that dumb to say that they didn't on the switch or didn't plug in, it's still an area to check. Make sure the cables are plugged in and you do switch on the printer.

    Next, is to make sure that the printer drivers are installed, particularly for those still using the parallel port cable. Those will need specialized drivers to be installed before they can work.

    For USB printers, there have lots of weird problems. Some reported to have successfully installed the printer driver when it's being turned off, while others have no problems while it's on. Just have to try out whatever suitable for you. USB drivers also need to be installed as well.

    Still, when none works, uninstall everything first. Go to the manufacturer's site to download the latest drivers first. Save them and shut down your PC.

    Turn off the power supply and wait for 30 seconds before plugging in the printer.

    Then switch it on again. When Windows prompt to install the drivers, skip this step. Locate the drivers you have downloaded earlier and install them.

    Make sure the printing spooler in XP is turned on:

    Go to Start > Run and type in cmd

    In Command Prompt, type in sc start spooler

    You should get this message:

    SERVICE_NAME: spooler
    TYPE : 110 WIN32_OWN_PROCESS (interactive)
    STATE : 2 START_PENDING
    (NOT_STOPPABLE,NOT_PAUSABLE,IGNORES_SHUTDOWN)
    WIN32_EXIT_CODE : 0 (0x0)
    SERVICE_EXIT_CODE : 0 (0x0)
    CHECKPOINT : 0x1
    WAIT_HINT : 0x4e20
    PID : 732
    FLAGS :

    If this service has been started, you will get an error message about this service has been started.

    Originally posted by AndrewPKYap:
    How do you tweak windows security so that the power users account can install the USB devices but all other default restrictions remain?
    I assume that you have Windows XP Professional installed.

    Go to Start > Run and type in gpedit.msc

    In the left pane, expand Local Policies.

    After expanding it, select User Rights Assignment.

    Edit everything in the right pane one by one.
