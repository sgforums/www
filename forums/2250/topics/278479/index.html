    1. Start > Run and type in cmd.

    2. Type in ipconfig

    You will see an address called Default Gateway. Type in this address into your Internet Explorer or whatever browser you use. Look around the settings. For changing the network name, it's usually under General, or the first page you see when you logged in.

    For securing it, it's usually under Security. If it supports WPA2 and you are using Windows XP, please download and install this update: http://www.microsoft.com/downloads/details.aspx?familyid=662BB74D-E7C1-48D6-95EE-1459234F4483&displaylang=en before setting up the WPA2 encryption.

    If it doesn't, choose WPA-PSK and type in your passphrase (something like your password) and save your settings.

    After that, try connecting to your wireless network. If you use Windows Wireless, it will prompt you to enter your passphrase. Type in that and tell it to remember.

    If you are using some other software to manage your wireless network, enter those configuration data via the software and connect to your network.
