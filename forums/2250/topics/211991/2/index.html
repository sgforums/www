    Originally posted by duotiga83:
    btw what is the best way to partitition and install win xp into 160gb hdd? thanks
    Boot from Floppy Disk to Dos. Inside fdd disk need to help;
    1)Fdisk.exe
    2)command.exe

    At Dos, type Fdisk, then select 1 (Create), 1 (Primary Part) and when prompt if you want to use whole partition, type n. When finish counting the Hdd size, u can now set how big you want for your primary hdd (Recommended mininium 40GB).

    After installing XP and update windows, right click on my computer and go to manage. Go to Disk management and formate the rest of the unused hdd size.

    Hope it helps.
