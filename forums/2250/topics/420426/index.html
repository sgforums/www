    If the issue persists, let's try to manually reinstall .NET Framework 3.5.1 in Windows 7. 

    Reinstall .NET Framework 3.5.1
    ====================

    1) Type: features in the Start search box.
    2) Select "Turn Windows Features on or off" and click Enter.
    3) Uncheck "Microsoft .NET Framework 3.5.1".
    4) Click OK and restart the computer.
    5) Repeat step 1) and step 2).
    6) Re-check "Microsoft .NET Framework 3.5.1".
    7) Click OK and restrat the computer.

    Then see if the issue persists.
