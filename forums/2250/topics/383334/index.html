    Basically, whenever a computer connects another (be it a web server, a game server, or another peer in P2P networks), the originating IP address can be logged and be used for various purposes. But that's as far as you can go - there's almost no way to find out who's behind an IP address. The fact that residential users are assigned dynamic IP addresses which change from time to time (specifically, it changes when the user re-connects to his/her internet service provider) just makes it even tougher to identify an user.

    If you have a genuine need for the identity of the person using a particular IP address, it's best that you contact the internet service providers (Singnet, Starhub etc.) directly for assistance.
