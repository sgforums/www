    i assume u are not using non static IP

    1. check that the lappy wireless adapter is on and working properly
    2.make sure that both wireless adapter and wireless moden support the same networking type. eg: both wireless G
    3. check that ur wireless moden assign IP address lappy..
    4. check that u have ur internet protocol (TCP/IP) obtain IP and DNS address automatically
    6. check that u have delete all the internet connection settings in internet explorer.(properties>internet option>connections)
    5. check that the data transmission rate from router to wireless adapter is auto.
    6. check the cabls are connected correctly
    7. check that u tap into your network
    8. check that the signal strength is not 1 to 0 bars
    9. check that wireless modem do have wireless mode activated.
    10. check that you do not have mac filter enabled in your wireless modem
