  • takahashi32399691

    no, you just need to look into the language settings in the control panel

     

     from http://sgforums.com/forums/2250/topics/312559 by me:

    First, go to: "Control Panel" > "Clock, Language, and Region"> "Regional and Language Options" > "Keyboards and Languages" > "Change keyboards" > "General"

    From there click on add and a box should appear. Go to "Chinese (Simplified, PRC)" > "Keyboard"  and select "Chinese (Simplified) - Microsoft Pinyin IME". Select "OK". If you want, you can change the default input language

    Now, you should see a language bar at the taskbar near the clock. You can now toggle between typing Chinese and English.

    Slightly different for windows xp. Not sure about Mac OS X or Linux.
