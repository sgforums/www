    RAM is the only cause for your laptop being slow.

    Before consider an upgrade, make sure that you have sufficient hard disk space. Although Windows can run perfectly with 5% free hard disk space, it will be darn slow. 20% free space is recommended.

    Move unimportant stuffs elsewhere, like movies, songs, etc to another HDD.

    Also, a very fragmented hard disk will make a laptop slow down. Defrag it once every few months.

    Next, your startup programs. Some startup programs, like MSN Messenger, are not needed at startup.

    Go snapfiles.com and search for Starter to manage your startup programs or find the thread on What's Running. This will help you better manage your startup programs without ever playing with Windows configuration. If you are comfortable with playing Windows configuration, then you can go ahead with it.

    The programs you run could also be a cause. Some programs don't really shut down properly even after closing it, so that may be a cause of lag. In this case, you have to contact the software developer or company to resolve this issue.

    If none works, upgrade is then recommended, so unnecessary money won't be spent.

    Can range from below $200 depending on the size, brand and type for add on singular. Replacing a complete set, will be more expensive.

    Where can it be done?
    1.) Is ur lappy still under warranty.
    2.) If no, Sim Lim Square have a few shops that can do it for you, if u are not confident in doing urself or DIY.
    3.) If yes, do you want to void ur warranty?
    4.) If yes, most likely u only left a few months, etc, etc then see point no.2
    5.) If no, go back to the lappy manufacturer for upgrade. It will be expensive.

    Do note, some lappy manufacturer, do not cover warranty for third party upgrade, i.e. u DIY or find someone else to upgrade for you other than the lappy manufacturer. So meaning, u add RAM, it screwed up ur lappy, u bring back to lappy manufacturer, they are going to charge u for repairs.
