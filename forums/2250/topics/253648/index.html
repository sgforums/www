    Instructions work for IE6.

    Start Internet Explorer. Go to Tools > Manage Add-ons...

    Select Add-ons that have been used by Internet Explorer from the drop-down list.

    Scroll down to Video__X_MS_ASF Moniker Class. Click once to select it.

    Under Settings, select Disable.

    Repeat for Windows Media Player.



    For Firefox 2:

    Tools > Options

    Select the Content tab.

    Under File Types, click on Manage...



    Look for the Windows Media Player icon. Select each of one by one, and click on Change Action...



    Select Open them with the default application button.



    Click OK all the way to apply the settings. That should stop it from using the plugins and causing it to crash.
