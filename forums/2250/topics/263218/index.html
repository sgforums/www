    Disable thumbnail preview and it will be solved.

    Open My Computer, select Tools > Folder Options

    Select the View tab.

    Tick the Do not cache thumbnails box. Click OK, close My Computer.

    A lot of files in Windows are hidden. They are important Windows files, not meant for deletion.
