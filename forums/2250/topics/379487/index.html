    Non-admin accounts have limited access to networking, core OS files and so on. It's not just the modify bit, because they can't even access some functions of the OS. Same thing happens to me when using school computers, and even things like Removing USB Devices are restricted.

    I'm not sure if there's a workaround for this, because the functions Warcraft 3 requires might allow access to vital data, which is why MS blocked it. IMO, don't use non-admin domain accounts because they are overly restrictive. If you need to keep students in check, and monitor what they are doing, you can try programs such as Junglebyte Teach (Which my PS used) as they allow screen capture and remote locking.
