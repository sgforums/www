    A lesson in how the internet works.

    We found the SGForum using the HTTP referer ID that is sent by all browsers when you visit our web site using a link that appears in the SGForum.

    Not sure what the HTTP referer ID is? See this W3.org page page on the HTTP protocol.

    As is the case for all web sites, this refereral information is logged by our web server. Which is the Apache software running on BSD Unix in our case.

    The log information is written to a file. This is what is looks like,

    code:
    202.156.6.54 - - [23/Apr/2006:01:30:26 -0400]
    "GET /products/pt.htm HTTP/1.1"
    304 - "http://www.sgforums.com/?action=thread_display&thread_id=187313"
    "Mozilla/5.0 (Windows;U;Windows NT 5.1; en-GB; rv:1.8.0.2)
    Gecko/20060308 Firefox/1.5.0.2"

    You can see that from this log that we know your IP address, where you came from, what page on our site you wanted, what browser you are using, which O/S you are running and a few other things.

    Thus it was easy to find this forum talking about our software.

    I would like to note that all browsers and all web servers log this information. It is not something special that we are doing, it is the way the internet works.
