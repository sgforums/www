    You need to submit your website to the search engines so that they can start crawling. That's the first thing. After that will be as Pillow says, with a lot of other things to do.

    Ensure that your website is constantly updated so that they can crawl more regularly, which will help your ranking a bit.

    Another thing to take note of is keywords. If you use a keyword that's very common (e.g. O Levels guides), you will be fighting with a lot of websites that offers that same thing.

    Most, if not all search engines are very particular about links, especially who link to you, how one link in Page X link to Page Y of your website. Learn how to optimize them properly, and make sure that the content is relevant.

    A link such as "Click here" is not a good idea.

    But a link like this: "You can find more Biology books in the Biology section"

    Bold and underline words --> a link

    If your website features rich content such as Flash and JavaScript, be aware that some search engines are very blind to them. They can't crawl such content, so you will have to of alternative ways to workaround it.

    Another thing is re-directs. If your website ever changes and need to re-direct users automatically to the correct links, it will definitely affect your ranking until it's done properly and the search robots index it.

    As for others, Google can teach you. You can refer to their webbie - http://www.google.com/support/webmasters/bin/answer.py?hl=en&answer=35291
