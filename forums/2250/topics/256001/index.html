    How to reduce the photo data size?
    Usually when an application limits your photo quality it should usually affect 2 things.

    (1) your photo file size i.e in bytes (e.g 10kb, 1MB etc)
    (2) your photo size in teerms of number of pixels across and down (e.g 100 wide and 300 tall ~ potrait or 300 wide and 100 tall ~landscape)

    The cheapest way i know with only the MS paint from windows can achieve the file size control with trail and error. But remember to also save the trials as a different name from the results so that if anything coks up you still can start afresh with the original photo. Resizing by interpolation (trial and error) with a percentage of 100 should do the trick. In fact with some creativity MS paint can do quite a number of things.
