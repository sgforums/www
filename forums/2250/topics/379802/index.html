    Click Start, and then type cmd in the Start Search box. In the search results list, right-click Command Prompt or Cmd, and then click Run as Administrator on the pop up contextual menu. If you are prompted by User Account Control for permission, click Continue. At the command prompt, type the following command, and then press ENTER: powercfg.exe -h off Type exit, and then press ENTER. That will turn hibernation off permanently.

    Then it's the mobo/chipset enforcing hibernation/deep sleep mode. IMO, the only way to disable this is to, well, completely delete the hibernation data files. This will remove hibernation, and will stop the problems permanently because now hibernation cannot be enforced because it doesn't exist.

    Of course, if you DO need hibernation, well, you can't have it back.

    Make sure you back-up all data before performing any changes to the core OS files. You might screw something up.
