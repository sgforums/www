    Mine doesn't. It prompts you 2 things:

    Open with a program or save it

    If you choose to open it directly, ou have to navigate to where you installed Foxit Reader.

    By default it's C:\Program Files\Foxit Software\Foxit Reader. Select Foxit Reader

    If it still doesn't work, change the way files are open.

    Open My Computer. Go to Tools > Folder Options

    Click on File Types tab.

    Scroll down all the way to PDF.

    Under Details for 'PDF' extension, click on Change. Click on the Browse button to locate Foxit Reader if it's not there. Then tick the "Always use the selected program to open this kind of file" box.

    Apply the settings.

    This time when Firefox tries to open it, it should prompt you.
