    Turn off UAC feature to enable full control.
    If not, go msn or wadever software properties with the right click, go to security tab -> choose your user account and set your controls mannually.

    Start -> type user -> turn on or off user control.

    There's a well known problem in vista for its sleep mode. It will return back to normal state within sec even though u go to sleep mode. On serious cases, computers will crash and reboot.

    To solve this.
    Go to start -> type device manager -> Network adaptors -> wadever controller -> power management -> uncheck the allow computer to turn off blah blah....
