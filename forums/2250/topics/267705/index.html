    As and when you are able to get MSN Messenger running, do this:

    Go to Tools > Options

    Select General on the left hand side

    Tick these boxes:

    1. Automatically run Windows Live Messenger/MSN Messenger when I log on to Windows

    2. Open Windows Live Messenger/MSN Messenger main window when Windows Live Messenger/MSN Messenger starts

    3. Allow automatic sign in when connected to the Internet

    Click OK to apply the settings.
