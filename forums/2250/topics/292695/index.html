    1. Go to Start > Control Panel and double click on Add/Remove Programs.

    2. Top right hand corner, tick Show Updates box.

    3. Scroll all the way down and find Windows Installer 3.1. Click Change/Remove to uninstall it.

    4. Re-download and re-install a copy of Windows Installer 3.1 here:

    http://www.microsoft.com/downloads/details.aspx?familyid=889482FC-5F56-4A38-B838-DE776FD4138C&displaylang=en

    5. Restart computer.

    See if it happens again.
