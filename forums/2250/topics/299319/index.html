    1. FLAC is probably not supported on most mp3 players, mp4 players and CD players. (in short, not supported on most media players)

    2. You can either convert it back to .wav, at no loss in quality or compress it to a supported format, such as mp3, if you intend to play back the file in a media player.

    3. On the other hand, if you are just making back-ups of the file, without the need to playback the file in a portable media player, just save it verbatim as flac to the CD lor.

    Side note:
    mp3 - 160kbs, 44.1KHz is the human perception limit

    aac/ mp4 (aka iPod format) - 128kbs, 44.1Khz can actually sound good, but varies from user to user's experience

    wma - 128kbs (zune player), 44.1Khz is what Microsoft claims to sound close to CD quality.
