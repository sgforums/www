  • MyPillowTalks
    26 Jan 11, 21:33

    1. Put the motherboard on the table, insert graphics card and only 1 piece of ram.

    2. Wire up power only to the mobo

    3. Plug in vga/dvi/connector into graphics card, short the +/- header for 'power on' and see if it boots

    If still can't, try with other ram, sometimes system won't beep even if it's ram problem.

    Best if you have other system to test on.

    Good Luck!
