    Auto messages are activated once you change your status, which you have set previously. If you didn't set, no auto messages would come out.

    Sign in to Windows Live Messenger. Click on Plus, Preferences. Select the Customize tab. In the left column, select Personalized Status. Add in new personalized status. Under Options, tick the box that says "Display the Personalized Status window when I change my status to Away."

    Note: This only works for Windows Live Messenger. It won't work on other versions of MSN Messenger.

    To have auto messages, do this:

    Open Plus Preferences.

    Select the Conversations tab. In the left column, select Quick text.

    Add in the text you want, set a shortcut for it. Make sure the shortcuts doesn't conflict with your system shortcuts.
