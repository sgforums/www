    DIMM means the number of stick of RAM installed on ur laptop.If lets say the retailer put "2GB RAM (2 DIMM)" It means that the 2GB RAM is from 2 sticks which also means that its 2x1GB RAM stick.Always aim for higher capacity RAM on a single stick since the spare slot can be used for upgrading the RAM.

     

    Also,its more economical since most OEM mobo only have 2 ram slot,which means that if u have 1GB RAM,u need to bin the 2x512mb stick for 2x1GB stick,hence wasting money.

     

    For AMD,its performance is inferior compared to Intel.Dont ask me why,it's just is.

    Like Chery for example(no offense)

     

    RAM will increase performance BUT will not decrease the load of the processor in certain task.Processor-reliant task such as video encoding will only improve slightly with more RAM since it depend on the processor itself.


    Note that Vista 32bit can only recognise up to 4GB RAM(probly only 3.5+GB in ur BIOS) due to technical limitation.If u want >4GB RAM,u need 64bit version of Windows.
