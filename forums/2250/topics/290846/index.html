    If you still have problems, try this:

    1. Start > Run and type in cmd and press Enter

    2. In Command Prompt, type in ipconfig /renew and press Enter

    3. Type in ipconfig /flushdns and press Enter

    4. Clear all your temp files and cache.

    Then try surfing again and see it works again.
