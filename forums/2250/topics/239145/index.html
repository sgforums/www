    Let me see...

    Go to Start > Run and type in control userpasswords2

    Select Users tab

    Under Users for this computer, select Administrator

    Click on Rest Password at the bottom

    That should remove the password.

    One thing to note though, if your PC is from a branded one, this method will reset it back to the manufacturer's password. Refer to manual for the password.

    Try this:

    Start > Control Panel > Administrative Tools > Local Security Policies

    On the left hand side, click on the + sign to expand Account Policies

    Then select Password Policy

    On the right hand side, select Minimum password length

    Right click and select Properties

    Set the characters to 0 (zero)

    Then try resetting the password again
