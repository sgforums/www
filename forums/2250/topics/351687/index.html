    Background story of your problem (?) :
    I suppose the operating system on your old HDD was either XP or Vista or Win 2K? By default these OS will use a file system called NTFS on the HDD, so if you try to access the HDD from another computer, permission settings will lock out other users from accessing the files/folders from another computer.

    Solution:
    Whether the computer you used for accessing the HDD is Vista/XP:
    http://support.microsoft.com/kb/308421/en-us

     

    Follow the instructions there, the instructions there apply to Vista too..
