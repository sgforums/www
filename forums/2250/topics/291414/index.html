    In one week, I've seen enough problems about Singnet's connections.

    Can you guys try this:

    1. Start > Run and type in cmd and press Enter

    2. In Command Prompt, type in the following line by line, pressing Enter after each line:

    ipconfig /renew
    ipconfig /flushdns
    exit

    If you get any error at ipconfig /flushdns, do the following:

    1. Start > Run and type in services.msc and press Enter

    2. Scroll down to DNS Client and double click on it

    3. Click on Start

    4. Click OK to apply the settings and close the Services Control Panel

    5. Then type in ipconfig /flushdns again and press Enter
