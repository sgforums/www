    Essentially, they are Windows files. For those whose copy of Windows isn't too corrupted, you can boot up to DOS, then using DOS commands to copy various files over to the Windows system folder.

    In XP, it will be Recovery Console.

    In a way, they are backup copies of Windows files. They are sometimes updated when Windows update.

    There's also another folder, the Dllcache folder. This folder contains all Windows DLL files, and can be copied over to the Windows system folder the same way via DOS. Same thing, they are updated when Windows is updated.

    You can check whether it's modified by checking the folder properties after you update Windows.

    Also check out Windows KB site for details. They do list the files they update Windows, together with the file sizes and version.

    If afraid... directly copy over from the CD...

    xcopy cd drive\i386 > another windows partition\i386

    I should add this part: from the other windows partition, burn it to CD.

    Another way to check will be to use WinPFind3u... one of the anti-malware tools used.

    It can check for files modification within 90 days (max). You can choose 30 or 60 days if you wish.

    Under Files or Folders Modified Within: Select 90 days, untick Non-Microsoft box. The rest of sections select None, otherwise you get a huge log. Don't select any other options. Then click on Run Scan button.

    After you get the log, you will know if it's been modified.
