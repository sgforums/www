    Plug and Play is enabled by default. If it isn't, go through these steps:

    Go to Control Panel, Administrative Tools. Double click on Component Services.

    Go to Services (Local) and select Plug and Play. Make sure the Startup type is Automatic. If it isn't, right click, and select Properties. In the General tab, find Startup type and select Automatic.
