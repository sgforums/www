    Ever uninstall a program in Windows but it still shows up in Add/Remove Program?

    Here's how to remove it.

    Start Registry Editor (Start > Run and type in regedit)

    Locate this key:

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall

    Find the program, usually they have a similar name. For example, you just uninstalled Winzip, but the entry is stuck at Add/Remove Programs.

    Locate Winzip under HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall.

    Right click and select Delete.

    Once done, delete all Winzip files and folders.

    What if you can't find your program? Easy. Go to C:\Program Files.

    For example, Sunbelt Kerio Firewall. You will notice a Kerio folder within the Sunbelt folder (if you have installed everything to their default settings). That is most probably the name in the registry as well.

    You will find Sunbelt Software under Uninstall instead. Using the same method that you delete Winzip, delete Sunbelt key.

    I bear no responsibility if you crash Windows. Do everything at your own risk. Don't know, ask.

    Alternatively, those who have CCleaner, do this:

    Start CCleaner.

    Click on Tools on the left hand side.

    Find the program that has uninstalled but not deleted from Add/Remove.

    Select it and click on Delete Entry.
