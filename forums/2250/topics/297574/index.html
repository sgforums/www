    Copy and paste the following in blue into Notepad:

    regedit /e "%userprofile%\desktop\look1.txt" "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\ActiveDesktop"
    regedit /e "%userprofile%\desktop\look2.txt" "HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\ActiveDesktop"
    type "%userprofile%\desktop\look*.txt" >> "%userprofile%\desktop\results.txt"
    del /q "%userprofile%\desktop\look*.txt"
    start notepad "%userprofile%\desktop\results.txt"

    Click on File > Save As....

    In the File Name box, copy and paste in look.bat

    In the Save As Type box, select All Files from the drop-down list.

    Click Save.

    Double click on look.bat to run it. Command Prompt will open and close quickly; this is normal. Notepad will open shortly afterwards. Please post the contents of this Notepad file in your next reply.
