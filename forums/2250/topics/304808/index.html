    Boot up to the Windows CD, pirated or not.

    Press R to boot to Recovery Console.

    Log in to the Administrator account.

    Username - Administrator
    Password - Usually blank.

    You will see something like C:\Windows >

    Type in the following line by line, then press Enter after each line.

    cd system32\dllcache
    copy ntfs.sys "%windir%\system32\drivers\"

    Check if there are any errors

    If no errors, type in exit and press Enter to restart the computer

    See if it works.
