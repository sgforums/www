    Erm it's not just now. It has been possible since a few years ago for quite a number of Nvidia cards. Basically SLI is used to link two cards together for a better performance, but most of the time it is not a directly doubled one (just like Quad Core vs Duo Core). There is a similar function with ATI cards as well, named "Crossfire". And usually SLI-compatible motherboards will have "SLI" in their model names.

    http://en.wikipedia.org/wiki/Scalable_Link_Interface
