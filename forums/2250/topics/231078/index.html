  • ndmmxiaomayi
    14 Jan 07, 15:28
    Originally posted by alwaysdisturbed:
    eh, how come my tab is 'sharing' instead of 'security'?
    You need to configure it in a slightly different way.

    Open My Computer.

    Click on Tools > Folder Options

    Select the View tab.

    Scroll all the way down and untick Use simple file sharing (recommended) box.

    Click OK to apply the settings. Then you will get the Security tab shown in GSL picture. Note that it doesn't work on FAT32 formatted drives.
