Open Hotmail with Firefox from Windows Live Messenger

  • kenn3th
    2 Feb 08, 12:21

    One of the biggest problems with Live messenger is it automatically assumes that you like Internet Explorer to view hotmail or any other live mail accounts. This is so wrong because most of us use FireFox as our default browsers. So it’s a bit annoying when clicking on the email button in messenger directly opens IE and starts loading hotmail.

    So here is a nice utility which allows you to customize in which browser you want to open your hotmail page

    even if you set firefox as the default browser, only the links get open. The email button always opens IE to load your hotmail. This is ok if you use IE7 with IE7 pro , and a disaster if you are still using IE6 in older OS, or if you prefer opening hotmail in FireFox.

     

    1 . Install Latest Version of WLM (not a beta, might not work with a beta)

    2. Install Messenger Plus! Live
    Important Note: When install, at the first screen, click Next. Take a careful look at the second screen because that’s where it will ask you whether to install sponsor program or not. Select “I refuse to give my support, don’t install the sponsor“.

    2. Download StuffPlug and install.

    3. Launch Windows Live Messenger and sign in.

    4. Try clicking the email icon and see if your Hotmail inbox will be launched in Firefox. If not, then click StuffPlug 3 from menu and select Options. Go to General tab and make sure the option “Open all URLs in default browser” is checked.

    OK now you are done, you can open your email links in your default browser, may it be opera or firefox….
