    There's another method, you could try if it works.

    I assume that you are using Windows XP.

    Go to Start > Run and type in notepad C:\WINDOWS\SYSTEM32\DRIVERS\ETC

    Whenever there's a popup, note them the website.

    In the Notepad file that opens, add this to the last line:

    127.0.0.1 the popup website

    For example, you get this:

    www.stupidpopup.com/whatever.html

    In that Notepad file, type in: 127.0.0.1 www.stupidpopup.com
