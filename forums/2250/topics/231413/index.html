    Hi,

    First let me introduce myself.

    I am Meat Pao.

    Get VirtualDubMod, which has some extra features than VirtualDub.

    And then download Avisynth, which is a very versatile program to work together with VirtualDubMod.

    You can do almost anything imaginable with Avisynth and VirtualDubMod.

    Of course, in order to read multiple file formats, I recommend you to also download FFDshow, a very powerful engine.

    But, if you are not interested in doing much editing, just want to cut videos and make smaller segments, VirtualDubMod is fine.

    Meat Pao.
