    If yer using WinXP and IT savvy, try this, (I can't remember exactly but I did this something like this before)

    1. Insert yer WinXP OS CD
    2. Boot up from CD
    3. Choose 'Repair' , and 'C drive' when asked
    4. It should bring you to DOS prompt
    5. You will be asked to enter your administrator login
    6. Next, type HELP and enter
    7. Look for something like "Repair restart" (duh, cant recall already)
    8. Boot up
