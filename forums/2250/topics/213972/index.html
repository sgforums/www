    I think your profile may be corrupted. Try creating a new profile and see how.

    To create a new one:

    Click on Start, then Run. Copy/paste firefox.exe -profilemanager into the text box. Click OK.
    Click Create profile
    Click Next
    Choose a name for the profile (OasisBlue perhaps) and click Finish
    Click on Start Firefox

    Firefox should start, using the new profile. Your Bookmarks should be still there, but in the old profile. If you wish to transfer them, navigate to this folder: C:\Documents and Settings\YourUserName\Application Data\Mozilla\Firefox\Profiles.

    Open the folder (double-click) and you should see two folders which start with random characters. One will be called xxxxxxx.default and the other xxxxxxx.rivo (or whatever name you used for the new profile). Open the xxxxxxx.default folder. Right-click on bookmarks.html and click Copy. Go back and then open the xxxxxxx.rivo folder. Right-click on the the right-hand pane and click Paste. You will be asked if you want to replace the existing file (bookmarks.html) - click Yes.
