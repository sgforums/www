  • ndmmxiaomayi
    16 Feb 07, 12:49
    Right click on My Computer and select Properties.

    Select the Advanced tab.

    Under Performance, click on Settings.

    Click on the Advanced tab when a new window opens.

    Under Virtual Memory, click on Change. Select C drive.

    Under Paging file size for selected drive, choose Custom.

    For initial size, set it to the amount of RAM that you have. For example, you have 512MB of RAM, put 512 in.

    For maximum size, multiply this value by 2. You can have more if you want to. Then click on Set.

    Click OK throughout to apply the settings.
