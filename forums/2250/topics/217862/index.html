    Start > Control Panel > Regional and Language Options

    Select Languages tab.

    Tick Install East Asian Languages box.

    Insert Windows CD when prompt.

    After installation, restart PC.
    _________________________

    Go to Start > Control Panel > Regional and Language Options

    Select Languages tab.

    Click on Details button.

    Click on Add button.

    Find Chinese (PRC).

    Tick Keyboard layout/IME box.

    Select Chinese Simplified (US keyboard).

    Click OK.
    _________________________

    Click on Language Bar button under Preferences.

    Tick Show the language bar on the desktop box.

    Click OK.
    _________________________

    You should see the language bar in your task bar. (near your Start menu)
