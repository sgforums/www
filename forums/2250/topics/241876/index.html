    .old.mp3 or .mp3.old

    both the same effect. virus scan still be written on the same directory. maybe i dont quite understand u
    Windows have a funny way of recognizing extensions.

    It only recognizes the last part. You can have a lot of extensions in front, but only the last part is recognized.

    For example, you can name a file this way: whatever.txt.exe.dll.mp3.doc

    Windows will read it as a .doc file

    So if the virus reads a .dll file to infect, it needs to be renamed as whatever.dll.old or whatever.dll.bak

    Because all these extensions are useless in Windows. But never rename it the other way round. Not .bak.dll. The virus will still run, because the part of the extension is .dll
