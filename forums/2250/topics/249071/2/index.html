    Hi.. as a Kaspersky Internet Security user, I have to chip in with my own personal review. I was a McAfee internet security suite user and a Norton user as well during the Pentium 4 days.

    I am currently using KIS7.0 (7.0.0.119 to be exact) which is a beta version. I bought the license from Challenger Funan for $88 cuz I am not a Challenger member. A member can buy the security suite for $68 only.

    When I was using McAfee, I suffered from very slow startups and shutdown. Startup can last up to
    3min--using 2Gb of RAM. Worse still, I noticed my computer is behaving weirdly. I can suddenly get transferred to a porn site. And I get silly messages and my internet connection seemed weird-- I am using Singtel Mio plan 3Mbps.

    After installing KIS6.0 (one of the most expensive internet security suite), without doing any scan, its real time protection immediately caught 3 trojans and a few possible spywares. Also, I get updates every hour, memory usuage when doing full scan is around 26,500k. Much less than using Firefox or Superanti-spyware! I can do a full system scan + surf net + microsoft office applications while enabling real time protection for Superantispyware with no change in speed.

    I upgrade to its beta version 7.0 because it incorporates heuristic analyser as well as using its own signature-based virus database. It also have the new Parental control which I don't need. When I was using McAfee, spybot S & D never fails to pick up numerous threats including cookies and a couple of possible spywares.

    However, one thing I have to warn is the installation part. KIS is very very incompatible with leftover parts that the previous anti-virus has left behind. When I installed and rebooted my comp, I immediately get the BSOD because McAfee uninstaller does'nt fully remove the parts.
