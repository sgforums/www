  • abao
    2 Mar 08, 23:26


    Open Audacity and load 1 of ur music file. Open another one using File>Open put ur second file there first.

    If you want to paste part of the 2nd file onto part of the first file, copy the section you need and return to the first audacity window, then highlight the time where you want it to begin and paste over.

    Done.
    Alternatively if you want them to mix, then note the time where you want the second file to play in conjunction with the first, then place your cursor @ the start of the second file's window and generate silence.

    Then copy the entire second track and Ctrl + V onto the first window, without highlighting the first file.

    You can use the fade in/out to your liking also... ... ... ...
