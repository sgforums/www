    Singnet 10mbps is slow, especially at night after 10pm. It's crawling at dialup speed for me after 10pm. It only resumes normal speeds after midnight.

    As for downloading wise, if you use it during off peak hours, it can be fast. Try using peak hours and it can take a whole day.

    Although ADSL2+ is similar to ADSL, they are two different technologies. ADSL2+ is faster.

    Your wireless router won't be compatible with the current technology, Singnet will help you set it up for the first time if that's your first time using it. If I'm not wrong, it will be free.

    You can't use the same configurations as well, but you will use the same username and password. You can change the password if you want to.
