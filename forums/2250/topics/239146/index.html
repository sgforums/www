    All files have a checksum called MD5 or SHA1

    To check, you use a program called MD5 hash calculator (http://www.diamondcs.com.au/freeutilities/md5.php) or SHA1 hash calculator (http://www.snapfiles.com/opinions/HashCalc/HashCalc.html

    For example, I'm downloading this Knoppix Live CD (700MB)

    After I downloaded the file, I run it through the hash calculator.

    After calculating, I compare it with the MD5 or SHA1 hash given. If it's the same, it means the file had not corrupted. If it's not the same, then the file has corrupted.
