  • ndmmxiaomayi
    21 Mar 07, 16:54
    Your system restore is infected, just need to clear it.

    Right click My Computer and select Properties.

    Select the System Restore tab.

    Tick this box: Turn off System Restore on all drives. Click Apply.

    When everything is being applied, untick this box. This will turn on system restore again, and will delete all previous infected system restore points.

    Click OK and restart your PC.

    Edit: forgotten an extra part.

    Download F-Secure Blacklight

    Select the Graphical user interface version. Do a scan and post the log here. Don't do anything yet.
