    lqcui2.dll is a dynamic link library file for Logitech video file, you could had installed some Logitech devices on your PC and did something to it right?

    This is how you solve it...
    1) Still need to use the driver
    Download this lqcui2.dll from
    http://www.dlldump.com/cgi-bin/testwrap/downloadcounts.cgi?rt=count&path=dllfiles/L/LQCUI2.dll
    and save this file in your WinNT\system32 folder.

    If problem still presist, registry this file yourself...
    Start>Run>"cmd"
    cd c:\winnt\system32
    regsvr32 lqcui2.dll

    2) Doesn't need that driver anymore
    Uninstall from your control panel/add or remove programs,

    unregister it yourself...
    Start>Run>"cmd"
    cd c:\winnt\system32
    regsvr32 /u lqcui2.dll
