    Originally posted by Y_Shun:
    what do i do after installing the XAMP ar?
    like how do i start uploading files and so on..
    XAMPP is a local server. You can install it on your own PC and start using it as if it is a real server. There's no files to upload, put all the necessary files to the HTDOCS folder.

    Once done, type in this to your browser:

    localhost OR 127.0.0.1

    You will see the XAMPP home page.

    To get to your homepage, say blablabla.html in the HTDOCS folder, you will use this URL: localhost/blablabla.html

    The best is to keep all files in one folder, rather than in the HTDOCS folder itself.

    Also, in order not to type out full URLs, make sure that there's a file named index in your folder. It can be index.php, index.html, index.htm.



    So let's say you put your files (including an index.php file) in a folder called Projects.

    Start up XAMPP, point it to the Projects folder. URL is: localhost/Projects

    It will automatically load the index.php page.

    To start XAMPP, double click on the xampp_start.exe file.
