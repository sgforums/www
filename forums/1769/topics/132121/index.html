    Any ideas on how to jazz it up some more?


    Suggestion 1 :
    I was thinking of making it more interesting by making sentences with the words so it's like having some lively conversation.

    For eg. imagine it's the R-H-O-N-D-A Word Association Challenge. It would go something like this :

    1st post :
    I thought that Shangri-La Hotel was RESPLENDANT.

    2nd post :
    HONESTLY, the decor wasn't my cuppa tea!

    3rd post :
    I heard they engaged a very famous DESIGNER to decorate the rooms!


    OR,

    We could make sure that we use the letters in sequence! That would make it more difficult. Eg.

    Post 1:
    There was a ROTTING smell when I walked past my neighbour's flat this morning! (Using R, 1st letter from 'RHONDA')

    Post 2:
    HORROR of horrors! What did you do then? (Using H, 2nd letter from 'RHONDA', so the 3rd post would have to use... that's right! 'O'!)

    Post 3:
    ONE has to be responsible and knock on the door to see if there's any response from the neighbour first! (So the next post will have to use 'N'!)

    Post 4:
    I tried but there was NO response, so I think, there must be a rotting corpse inside!



    Any other suggestions? If not, which one would you prefer?
