    Western-trained docs are good for certain things, sinseh's good for others.

    It does sound more like Carpal Tunnel Syndrome (as diagnosed by Gedanken ), a type of RSI (Repetitive Strain Injury, as diagnosed by Bear).

    This type of injury occurs when there's repetitive hand movements over a long period of time. For eg. if you use the keyboard / mouse for long periods everyday, you actually exert great strain on your wrist and hand.

    Shinta, better go to a western doc to get a proper diagnosis. Arthritis is a serious condition not to be taken lightly because it can be pretty debilitating.

    In severe cases of Carpal Tunnel Syndrome, an minor op can be done which eradicates the problem effectively. I've seen the op done - it's in the Ambulatory Surgical Unit, which means, you check in, the op's done, and you can leave a few hours later on the very day itself.
