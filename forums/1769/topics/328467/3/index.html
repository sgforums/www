    DIR is a diving movement that advocates a specific style of equipment configuration and diving techniques .... it's very very different from the mainstream ... the advocates can be rather ... "passionate" about it ... so much so that people have called the movement the mormons or talebans of diving ..... me, I've been using a DIR configuration every since it first go introduced here in Singapore about 8 year ago, but yeah, I'm a "stroke" ... 

    more information here => DIR diving

    I would love to psycho you to become one of my students, but since you're where you are, I would advise you to learn it properly there ... cold water diving is a whole different ballgame ... you can get certified in cold water and dive in the tropics, but if you get certified in the tropics and want to dive in cold water, they'd want to ask you to go for orientation or skill evaluation dives first ... it's much better to learn it properly there.
