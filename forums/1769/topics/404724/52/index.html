    To LF..

    If you're taking bus to JB for a quick trip next time. You bus fare tabulation.

    To JB (There's fare diff to and fro)

    Using CCL straight to Kranji & x-fer to 170 blue plate bus aka 170X (alighting pt is JB CIQ): $1.88 (MRT) + 9c (170X) = $1.97. Dist: 23.2km (MRT) + 5.8km (170X) = 29.0km.

    Take Yellow bus instead: $2.68 (bus fare 80c).

    Using Sv 135 to Ang Mo Kio, take MRT, x-fer to 170 blue plate aka 170X: 81c (135) + 89c (MRT) + 11c (170X) = $1.81. Dist: 3.7km (135) + 18.2km (MRT) + 5.8km (170X) = 27.7km.

    Take Yellow bus: $2.50.

    Returning from JB

    MRT option: $1.09 (170X) + 89c (MRT) = $1.98. Dist: 6.4km (170X) + 23.2km (MRT) = 29.6km.

    Yellow Bus: $2.23 (MRT fare $1.88, bus fare RM0.80 ~ S$0.35).

    MRT - 135 option: $1.09 (170X) + 68c (MRT) + 4c (135) = $1.81. Dist: 6.4km (170X) + 18.2km (MRT) + 3.6km (135) = 28.2km.

    Yellow Bus: $2.05 (MRT - 135 fare $1.70).
