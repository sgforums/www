    Simply saying, those terminals n interchanges under the areas of purview of SMRT, ie, the northern n northwestern areas, r under the control of smrt.

    Under SMRT:
    Yishun Int
    Sembawang Rd End Ter
    Admiralty Rd West Ter
    Sembawang Int (upcoming)
    Woodlands Reg Int
    Bt Panjang Int
    Choa Chu Kang Int
    Lim Chu Kang Ter
    Bt Batok Int
    plus AMK/WDL/KRJ Depots

    if u wanna know which ints belong to who, go to d ints n see the info boards. If it's SBST's, then d map will have NO SMRT svcs shown. If it's SMRT's, then d map will hav no SBST svcs shown.
