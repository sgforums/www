    System Under Test(static/non-bold)
    Bus-Stop Name + System Under Test(Scrolling and bold)
    continue like this...until reach bus-stop den
    Bus-Stop Name
    den leave bus-stop:
    Bus-Stop Name + System Under Test(Scrolling and bold)
    den a distance away:
    System Under Test(static/non-bold)

    however, for habit buses, things gets more complitcated due to the fact tat it oso acts as a bus-stopping sign

    System Under Test(static/non-bold) >>> If sum1 presses bus-stopping, it will flash this msg den Bus Stopping..den bak to the msg so on...
    Bus-Stop Name + System Under Test(Scrolling and bold)>>> If sum1 presses bus-stopping, it will flash this scolling msg den Bus Stopping..den bak to the scrolling msg so on...
    continue like this...until reach bus-stop den
    Bus-Stop Name>>>normal, gt bus-stopping den flash alternatively...
    den leave bus-stop:
    Bus-Stop Name + System Under Test(Scrolling and bold)>>> If sum1 presses bus-stopping, it will flash this scolling msg den Bus Stopping..den bak to the scrolling msg so on...
