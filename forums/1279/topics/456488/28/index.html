    1. Additional grab pole at where the fares information used to be (same as Batch 2 Citaros)

    3. The panels on the last row of seats are completely flat for both left and right side. (Unlike in Batch 1 Citaros where there is slightly extra headroom for the person on the left). Not sure if it is the same for the standard Batch 2/Early Batch 2 BSEP Citaros.

    4. Speakers using a different type of hinge. Not sure if it is the same for the standard Batch 2/Early Batch 2 BSEP Citaros.

    5. Doors seem to close more gently (same as Batch 2 Citaros)
