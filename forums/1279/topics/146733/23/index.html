    I like to correct some misconceptions here.

    1. "D" does not (always) mean diverted. It usually stands for "delay". However, other airlines have used different characters such as "A" or "B". The usage of this additional alphabet arises from the need to differentiate the previous day's delayed flight from the current day's flight. For example, if SQ1 is delayed on 1/1/05 to 2/1/05, then you would have 2 flights performing the SQ1 routing. Hence, the delayed flight will use SQ1D, while the actual day flight will use SQ1.

    2. NW9xx are cargo flights using B747F aircraft.

    3. SWORDFISHZ5, TRUMPETZ5 are usually used to identify military traffic using Changi's runways.

    4. KA does not operate any scheduled flights (cargo or pax) to Singapore. They are here usually for maintenance since KA purchased (and converted) some of SIA's old B747-300s.
