    I think one point to clarify here is that BSEP services/enhancements as what SMB128B has mentioned are meant to improve bus connectivity and frequencies.

    They are not expected to be profitable (otherwise the bus co. would be operating it and not be labeled as BSEP nor using BSEP equipment). LTA pays SBST and SMRT whatever amount it costs to run the BSEP routes (and enhancements) and all of the fare revenue goes back into the 1.1bil funds, making it profit neutral for both SBST and SMRT.

    Since the goal is to improve connectivity, there is no need for any explainations why some of the loads of BSEP services such as 120 do not have sustainable (or breakeven) loads.

    The exceptions to this are the CityDirect services which are tendered out to pte bus co. which receives a fixed sum of money for running the routes. These are rather expensive to run since they are not running at cost, but at a profit to the pte co. I believe a fellow forumer previously did an analysis showing that each trip costs nearly $500 to run for CityDirect 651.
