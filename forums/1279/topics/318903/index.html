    This is different from "Your own deployment fleet". This is your general idea of deployments, for example, Voith buses on eway services. You should also state the reasons behind the deployments on various types of services. I shall post mine now.

    Express services: It should use Voith-ed buses (Voith version Mark IV DM, DM3500, VSOs, and possibly Mark III) as Voith is more suitable for the Expressway, while ZF is more suitable for starts and stops.

    Cityshopper: Leave it as it is.

    Premium: Leave it as it is.

    Highway hogs (not express, but services like 168 and 36): Also use Voith buses (Voith version Mark IV DM, DM3500, VSOs, and possibly Mark III), as Voith is more suitable for the highway with less gear changes. LO3x, although ZF, can also be used because LO3x is able to sustain well enough on the Highway, as opposed to VO3x.

    Feeders: Use Scania K230. Why? Scania K230 has the capibility to crush load, and people will stand for a very short journey, and might eliminate the need for DDs on feeders. Also, make them full WAB. If feeders are WAB, PIWs can connect to the MRT stations easily and they can use the MRT to travel from Town A to B.

    Non - expressway longhauls (high demand): Can use ZF-ed buses as ZF is more suitable for frequent starts and stops. VO3x should be used here. Striders and ZF ver. Mark IV DM can come here. This is also a good place for Scania K230 to be used.

    Airport services: Most, if not all, run on the highway. 24 which does not really run on the highway can have some K230 and Merz. 27 and 36 is using the Mark IVs now and is very good. 34 has a normal demand so should also use Mark IIIs or Mark IVs.

    Low demand non - eway services: I would recomand Mark III and IVs and the O405s.
