    via PIE, Jln Eunos, Chai Chee

    Service 141: Toa Payoh - Pasir Ris
    via PIE, Tampines Ave 3

    Service 144: Toa Payoh - Kent Ridge
    via Bt Timah, Sixth Ave, Buona Vista

    Service 870: Sembawang - Bedok
    via Yishun, SLE/ CTE, Bedok North Rd.
