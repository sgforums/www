    Originally posted by JaSoNhSu92:


    i used once before when i was in sec sch. if i buy concession for bus only, then if i nid to take MRT, still got stored value and deducts inside my ezlink card?

    Yes. Stored value in your ezlink card is deducted if you don't have concession for that particular mode of transport.

    For example, if you have a bus concession pass, all bus rides on BASIC services are free. Stored value is deducted if you take any premium services (e.g. cityshopper), express, ffw, etc, and the train.
