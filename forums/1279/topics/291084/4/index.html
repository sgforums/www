    Originally posted by Oceane:
    I believe that from the same type of bus model with the same engine, it would sound like a SMRT O405?
    There are some differences between SBST's O.405s and SMRT's O.405s. SBST's O.405s are MkIs which has a default geabox of W3D 080R (for the earliest O.405s)/ W3E110/2.2R/W3E 112/2.2R/W4E112/2.2R (the latter three for later versions). SMRT O.405s are MkIIs which have a default gearbox of ZF 4/5HP500. There are also minor differences in the components they use, for example exhaust pipe (SBS161U seems to have a MkII exhuast pipe, hear it and you'll know why).
