    This thread is intended to post bus bunching observation/spotting. We often see buses of same service number bunch up. I am putting this thread so that we can report them, and this way we can also observe services that often bunch. Please put date, service number and bus type (DD/SD/bendy) when you state your spotting.

    08 Apr

    2 DDs on sv 51 observed bunching at Eunos AM peak towards Jurong Interchange. I was wondering that if the DDs are already sticking to each other, this is probably how they will travel all the way up to Jurong.

    A little more investigation showed there was a non-WAB bus on iris after 04 minutes that would call the stop. My prediction that by the time this bus reaches Havelock, it would touch the other 2 in front, and then all 3 will travel together to Jurong.

    What can BCs do to avoid this? I am sure the wait time for the first of the 3 bunched up buses would be 20+ minutes.
