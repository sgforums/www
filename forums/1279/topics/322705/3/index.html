    Dual Depot Control Duty Codes

    I have explained this somewhere in this forum before but since its been some time I shall explain it again.

    The order for the duty type is still the same i.e. A, P, S, E, T. However, the depot takes precedence over duty type. This means that the first depot's A/P/S/E/T shifts comes first then the 2nd depot's A/P/S/E/T shifts. Which depot duty codes comes first depends on each individual service. There is no standard format. In addition, having majority/minority control means that the depot order is first.

    Let me highlight some examples.

    S21 - amdep majority, bndep minority
    1A-13A, 14P-26P, 27S-31S, 32E amdep* (typo)
    33A-35A, 36P-38P, 39S-40S bndep
    amdep 19 buses
    bndep 5 buses

    S63 - brbp majority, arbp minority
    1A-4A, 5P-8P, 9S arbp
    10A-20A, 21P-31P, 32S brbp
    brbp 12 buses
    arbp 5 buses

    Ever since SBST introduced overnight parking at interchanges, their duty codes, (regardless of duty type) starts from duty no. 81. Any duty code above that number is considered that the duty involves overnight parking at interchanges.

    Examples

    S10 - arbp minority, bndep majority (only Tampines with overnight parking)
    1A, 2P, 3S-4S, 5E-6E arbp
    7A-17A, 18P-28P, 29S bndep BN parking
    81A-84A, 85P-88P, 88S bndep Tampines Int parking
    arbp 5 buses
    bndep 17 buses

    S66 - slbp majority, bndep minority (both sides with overnight parking)
    1A-9A, 10P-18P, 19S-20S slbp SL parking
    21A-23A, 24P-26P, 27S-30S bndep BN parking
    81A-85A, 86P-90P slbp Jurong East parking
    91A-92A, 93P-94P bndep Bedok parking
    slbp 16 buses
    bndep 9 buses
