    Following TIB425R's retirement is TIB426M after 19 years- with near identical specifications save for one major difference.

    Due to the time when non-airconditioned buses were still common, TIBS had likely specified an evaluation unit to be non-airconditioned following the first unit to be delivered together. However when it arrived, it was in original airconditioned specifications with the rounded edge rear window (rounded edge windows were common features for non-aircon buses back then). It has been the only unit to feature this unique rear window since that day...
