    Note you need PowerPoint and GIMP (or any equivalent software)

    The font was made by myself using small circles combined together in PowerPoint.

     As for the animation, I use GIMP (similar to Photoshop) which can be found at http://www.gimp.org/

    Step 1: Combine the different letters in PowerPoint to make the EDS. Use the arrow keys to move esily. Use 1 frame per slide.

    Step 2: Open GIMP

    Step 3: Go to the first frame. Select whole thing and paste it ino GIMP.

    Step 4: Copy the next slide and pasting it into GIMP. This time, click on the New Layer button and click OK.Repeat this step until you have all the frames inside.

    (picture taken from internet)

    Step 5: Save as .gif file, set as animation at 3000 milliseconds (3 seconds)

    The Fonts: http://www.mediafire.com/?wzoyzi2girz
