The ENTRY/EXIT on exit readers function (Automatic mode) has been taken out from the IDFC so there's only 3 options now -- 1) All readers ENTRY mode 2) All readers EXIT mode 3) Normal mode (entry readers show ENTRY/EXIT & exit readers show only EXIT)...
