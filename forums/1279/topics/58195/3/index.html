    If you find it confusing jus read the results from this derivation

    Let the number be pqrs. For any such series of motor vehicles in Singapore,

    12p + 2q + 11r + 1s = N

    To repeat the suffix without repeating the number, we choose to do something to p and q (thousands and hundreds) such that the resulting number has the same remainder when divided by 19, i.e.
    12( p + a ) + 2( q + b ) + 11r + 1s = N + ( 12a + 2b ) = N or N + 38 or N + 76 or N + 114 so

    12a + 2b = 0 or 38 or 76 or 114 ...

    Note that we did not choose 19 and 57 since the multipliers (12 and 2) are both even and it is impossible to obtain an odd number with any combination of a and b.
