    Originally posted by sexy girls:
    SBS9182A (AMDEP 25)(back engine cover using 3rd batch VO3x)

    what does it mean by back engine cover using 3rd batch VO3x? anyone has pics?
    No picture, but can describe.

    The LO3x, VO2x, earlier VO3x batches have different engine covers from the batch 3.

    The LO3x, etc, have the vents on the back cover dented in. There are 3 "dents" horizontally and the vents are placed inside those dents.

    However for Batch 3, the vents are all smooth, with no dents.

    The Batch 3 engine cover "bending point", the place where the cover folds when it is open, is also lower.

    For pictures, you can go see the SBST bus models. This is one of the errors on the bus model. Even though both are Batch 2 Volvos, the SBS livery one has the correct type of vents, but the SBST livery one has Batch 3 type of vents!
