    Before I start these topics, I want to go through the ways these int/routes are rated first, so I can plan these discussion properly.

    Int

    Design - How the int is designed via boarding/alighting booth placement, entry points for buses, bus bay layout, etc

    Location - how is it connected to nearby place of interest? If if is connected to the MRT, how well is it connected?

    Connectivity - How good is the Int serving other places, taking into account the no. of services, and Int size

    (you can suggest some more points here)

    Routes

    Design - How well is the route designed? Is it possible to improve it, or leave it?

    Type - What role does the route play in the area it serve?

    Area served - How well does the route serve the area is ply through

    Overall loading - How well is the route utilised in its overall journey

    Fleet - How reasonable is the fleet used in the route

    Resonable frequency - Is the frequency reasonable enough for this route

    (you can suggest more points here)
