    Originally posted by ButIAmAToilet:

    What is the max loading for SD, Bendy and DD?

    What does SL, TRG means?

    SD = Single decker

    DD = Double decker

    Bendy = nickname for articulated buses from SMRT

    SL = Service leader, aka bus driver

    TRG = Training (Bus)

    For more info, see this thread:

    http://sgforums.com/forums/1279/topics/484393?page=1#posts-11005797

    Hope this helps you!
