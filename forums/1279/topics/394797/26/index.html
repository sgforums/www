  • SBS2601D

    Originally posted by SBS1869ERyan:

    Hello! Have a question. Not sure if this has been asked before, but how come Volvo Olympians sound like they have 5 gears but their gearbox is a ZF 4HP500 4-speed automatic?

    Thanks!

    Rest assure its been asked countless times, but buses using the ZF will have the lock-up clutch engaging in between switches in gears. By default, the fuel is supposed to be cut when that happens to reduce wear on the clutch making it sound like a gear switch, but some buses don't do that resulting in a nauseating grinding sound.

    Voith has the same mechanism, but the engaging is done just before the switch to 2nd in both 3 and 4 speed types, hence it sounds more "normal".
