    18 May 10, 00:34
    Originally posted by Operation//Overkill:

    Guys , a question to u all. Whats all the shift 7S and so on? How to read it?


    SMRT: 966AM01 where 966 = sv no, AM = duty type, 01 = duty sequence number. Duty type consists of AM, PM, S1, S2 and TQ. Since the duty sequence no for each type of duty restarts itself, this means there's PM01, AM01, TQ01 etc.

    SBST: 1A where 1 = duty no, A = duty type. It goes in ascending duty number order, followed by duty type. A, P, S, E, T, N. Each duty number is unique on its own means you wont find a Duty 1A and Duty 1P/1S/1T on the same service. There's no service no tied to the duty code unlike SMRT.

    This is just the basic understanding. There;s still somemore on how to differentiate other stuff.
