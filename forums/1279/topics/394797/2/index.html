    For interchanges, the codes are usually xx009 or xx008, where xx is usually the code for the area e.g. 46 is for Woodlands, so WRI is 46009.

    xx008 is usually for the end-on or another distinct part of the interchange.

    Terminals usually just take on the code of a bus stop ending with 9. Exceptions (not exhasutive) are: Ang Mo Kio Depot (55009), Changi Village (99009), Ghim Moh (11009), Kent Ridge (16009), Lor 1 Geylang (80009), Tuas (25009) and Upp East Coast (94009).
