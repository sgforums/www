    Originally posted by sbs&tibs:
    Did the side EDS also had 222? Did you noticed what happened at that time? How did you felt when you see the desto?
    I would wager that the side desto also displays 222. This is because the route number is controlled only by one out of the 4 fields that the driver has to key in to get the entire desto. Different models have different codes, but in case anyone is interested,

    1st field: Route number ( either the number or route code )
    2nd field: Route details code
    3rd field : Direction code
    4th field: Font code ( eg. Bold like svc 980 UK lance desto )

    For simplicity, the codes for the 4 fields are the same so the driver only remembers 2 codes for trunk svcs, one to and one back. These sets of codes are in sequence. Eg. 434 for forward, 435 for backwards.

    We had even tried and successfully displayed svc 400 on TIB1186Z on our visit to Woodlands Depot last yr.
