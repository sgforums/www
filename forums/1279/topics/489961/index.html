    Hi mr busanalyser, yes true. 

    Both sengkang and Punggol interchange expansion are listed in the govt brochure plans during this general election. The timeframe is stated clearly.

    sengkang interchange extension not even started yet. So not possible by Q4 2015. Since sengkang is already built up, so naturally it starts first. Should complete as per govt plan by 2016.

    punggol interchange extension should start work slightly later around end 2016. It should be in collaboration with the building of Punggol town centre (hawker centre, library and community club). Since most flats around Mrt are just finished or going to complete, it is standard for govt to complete bus interchange extension 2 yrs after residents move in. Currently waterway point mall is nearing completion already and it should satisfy the residents first. So the target is by 2018 as per govt plan. 

    Cheers.
