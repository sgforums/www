    Instead of naming it BSEP Batch x, it should be Batch x BSEP.

    For example, Instead of Volvo B9TL (Wright Eclipse Gemini) (BSEP Batch 1), it should be Volvo B9TL (Wright Eclipse Gemini) (Batch 2B BSEP), since those buses are part of the Batch 2B order, and not a separate order.

    MAN NL323F (BSEP Batch 1) -> MAN NL323F (Batch 1 BSEP)
    MAN NL323F (BSEP Batch 2) -> MAN NL323F (Batch 2 BSEP)
    Mercedes-Benz O530 Citaro (BSEP Batch 1) -> Mercedes-Benz O530 Citaro (Batch 1 BSEP)
