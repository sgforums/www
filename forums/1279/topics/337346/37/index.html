    Originally posted by SMB66X:

    If we keep changing advertisement, we would be pasting a new livery over a old one right? So what would happen to the bus if we keep on changing the advertisement?


    Nothing will happen if you yourself can guanrantee that the livery which is sprayed on, is well on the product.

    If u have used paintmarkers to paint the new livery of the bus, then after tearing down the pasted ads, it will have paint cracks like those on real buses.

    If its the original livery when you bought it, after tearing down the ads, normally there is no problem, but becareful to check how thick is the livery coated first before pasting the first ad.

    Best is to use normal A4 computer paper and transparent watery glue that can be found in NTUCs, cheap ones.
