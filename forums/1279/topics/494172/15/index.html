    I see where you're coming from of wanting to have more 'intra-town' routes from Kampong Bahru but for this to be feasible, if the unused NE2 station along NEL is going to be built near the terminal, perhaps your suggestion of making changes to certain routes can be considered. For example, 121 can be converted to a bidirectional loop service 121A/121B. 121A plies the current 121 route but instead of looping at Telok Blangah Rise, it returns to Kampong Bahru terminal via Kampong Bahru road while 121B plies Kampong Bahru Rd to Telok Blangah then continues back using the original route. So instead of taking 121 from Tiong Bahru, taking 121 from Kampong Bahru can be an alternative and Telok Blangah residents will have options to go to either TB or KB via 121.

    IMO Kampong Bahru terminal is meant to be more of a terminating point for long-haul routes rather than a typical bus interchange where you have feeders, townlinks...
