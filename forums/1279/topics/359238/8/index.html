    Hi everyone,I have some things to share.

    Shader Mod

    This is the long-awaited mod from MM2 Brazil that completely revamps the game in terms of graphics. The shadows and textures for the default cities (you can find that same photo-realistic effect within the custom cities as well) have been completely reworked to make the game look a little more modern.

    Download page:http://www.mm2x.com/files/mods/shadermod2.zip

    Showroom Car Selection Screen

    It replaces the background of car selection screen from ordinary MM2 dark-blue into a stylish car showroom background.

    Download page: http://www.mm2x.com/files/mods/Showroom_Background_(Rest_24).zip

    Fire and Smoke Mod 3.2

    This mod edits the texture of the car damages' smoke making it more realistic with also fire effect.

    Download page: http://www.mm2x.com/files/mods/Fire%20and%20Smoke%20mod-3.2.zip

    Screenshooter .NET 

    A tool to make images of your current screen by pressing the PRINT button. Useful for example at games, so that it's not needed to press PRINT button, minimize the game, paste the image in a program like Paint and finally save it.
    This tool automatically saves the taken screenshots in the selected directory in 4 formats: BMP, JPEG with compression quality adjustment, PNG and GIF.
    Now is also possible to set a custom key combination to take the screenshot, and the settings stay saved after closed the program.

    Note: Requires .Net Framework 2.0 or higher.

    Download page:http://www.mm2x.com/files/files/screenshooter.net.zip

    Midtown Madness 2 Widescreen Tool

         

    Mdtown Madness 2 can work nicely with widescreen without stretching or cropping some top and bottom (also known as VERT-, Midtown Madness 2 is HOR+)

    The problem is that midtown Madness 2 will not list widescreen resolutions above 1440x900 and this is where is the tool is useful, especially cause LCDs run best at their native resolution.

    This tool adds the common widescreen resolutions:
    1280x720 (16:9 - HDTV 720i/p)
    1280x800 (16:10 - typically found in laptops)
    1440x900 (16:10 - typically found in 17"-19" widescreens)
    1680x1050 (16:10 - typically found in 20"-22" widescreens)
    1920x1080 (16:9 - HDTV 1080i/p)
    1920x1200 (16:10 - typically found in 24"+ widescreens)
    2560x1600 (16:10 - typically found in 30"+ widescreens)

    The tool works by editing your gfxconf.dat file, a file that stores the display settings used by Midtown Madness 2 and adds the resolutions listed above to your game.
      Download page:http://www.mm2x.com/files/files/MM2WSTool.zip
