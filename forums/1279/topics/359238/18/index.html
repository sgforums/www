    If you want to know why MM2 has heap overruns... READ THIS:

    • MM2 has a limit of how many polys it could load whenever you start a new game. 
    • The higher the resolution, the less polys it could load.
    • The number of polys it could load includes the vehicle you are loading too. If you are using a highly-detailed 3D bus (w/ interior), and load into a big city (e.g. STC), it will crash. There's simply too much polys for MM2 to load. Besides, it's an old game...
    • If you insists on playing the bus on the map, use a drastically smaller resolution (e.g. 640x480 ~ 800x600; my LCD native resolution is 1680x1050 [wtf, lol])
    • Also, using stopreporter/shader mod reduces the amount of polys it could load, as both uses MM2's memory also. If you wan play buses, do NOT install shader mod. It's the worst freakin' memory hoggers.

    [Source: http://forum.mm2c.com/viewtopic.php?t=14197]
