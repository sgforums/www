    ARM2 for Midtown Madness 2 Help: Step By Step Guide

    Step 1:
    > After downloading ARM2 from StopReporter website and installing it into ur PC, u would come across this interface:

    Step 2 (Reconfiguration of ARM):

    > To startup the console, press the button where the red arrow points, or else it wouldnt start up. Press number '1' button to find the designated route announcement you want, and then follow this button combination:

    If the arrow points '->' at the console screen, follow this combination:
    '6', '3', '3', '3', '3', '7'

    If the arrow points '<-' at the console screen, follow this combination:
    '6', '2', '3', '3', '3', '3', '7'

    After that, please make sure that ur ARM2 is set to Safe Mode. The directory to go there is:
    File > Settings.. > Options Tab > Arguments set to Safe Mode > Click 'OK'

    Then after that, play Midtown Madness 2 Safe Mode in order to start ARM @ MM2.
