    Somewhat a little bit off topic.

    As discussed with ZF employee, the coding is as follows:

    e.g. 6HP604

    1st digit (6) denotes no. of forward gears.

    "HP" means Hydraulic Powershift.

    "60x" means a class of torque rating. It does not need to be 600Nm. e.g. 400, 500, 590.

    "4" means Ecomat 4. It is a generation of the transmission.

     Anthony Lui
