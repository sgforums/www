    Opened this topic specially for SMRT Buses with new Refurbishment.

    (Version 1) Buses with the new refurbishment includes:
    - New set of wheels with BLACK rims
    - OC500LE / O405G Hispano Habit-like A/C ducts
    - New blue/orange seats (Orange for priority seats)
    - 'Priority Seat' stickers
    - Light-blueish walls
    - Milky-white A/C walls
    - New retrofitted lights.

    (Version 2) Buses with the new refurbishment includes:

    - New set of wheels with BLACK rims
    - OC500LE / O405G Hispano Habit-like A/C ducts
    - New blue/red seats (red for priority seats - Same as OC)
    - 'Priority Seat' stickers
    - Light-blueish walls & A/C (Same as OC)
    - New retrofitted lights

    ---------------------------------------------------

    *Please update the buses by putting in '<sup>R1</sup>' for Version 1; '<sup>R1</sup>' for Version 2 at SGWiki to indicate that the bus had been refurbished for 2009.* Thanks!
