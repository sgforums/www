  • sgbuses
    14 Aug 09, 06:31

    Hi Mike,

    Use this to help you: http://sgwiki.com/wiki/Buses

    If the particular model doesn't state anything it is probably pre-Euro, but buses after 1995 generally have a minimum Euro emission standard.

    If you need authoritative sources, either use Internet Archive to retrieve website of the operators (SBST/SBS, TIBS/SMB) which has previously provided these stats, or use LexisNexis/DJ Factiva to retrieve newspaper reports about new purchases (they are usually announced).

    Deregistration dates are obtained from one.motoring database by Land Transport Authority, of which some are listed on the wiki itself.
