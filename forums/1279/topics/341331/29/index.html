    Originally posted by SBS 8529 R:

    Hi

    I heard there is a thread about the calculation of the last alphabet of bus plate numbers. Can anybody fill my curiosity?  Thanks a lot! (:

    For the second and third alphabet, substitute it with a number. A=1, B=2, C=3 etc..

    Using this equation,

    1 - (9*2nd letter) - (4*3rd letter) - (5*thousand place) - (4*hundreds place) - (3*tens place) - (2*ones place)

    Using the result, mod the number with 19, you will get a number from 0 to 18. Refer to the list below for the suffix.

    1 = A, 2 = B, 3 = C, 4 = D, 5 = E, 6 = G, 7 = H, 8 = J, 9 = K, 10 = L, 11 = M, 12 = P, 13 = R, 14 = S, 15 = T, 16 = U, 17 = X, 18 = Y, 0 = Z

    Originally posted by SBS 8529 R:

    Sorry still dun quite understand.. what 2nd and 3rd alphabet? Can give an example? Thanks a millions (:

    For example, you want to find out the suffix for SBS9999.

    2nd alphabet is B, hence equals to 2. 3rd alphabet is S, hence equals to 19.

    Using the equation,

    1 - (9*2) - (4*19) - (5*9) - (4*9) - (3*9) - (2*9) = -219

    -219 mod 19 = 9

    Hence, the suffix is K. SBS9999K.
