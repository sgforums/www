    Cloudy condition: need more light in the photo, so camera set shutter speed to slower to allow more light to be captured.

    Strong sunlight: Photo confirmed overexposed when you set EV +1.0. However, EDS captured because of the same theory. You want more light, shutter speed slower.

    Night condition: Blur because a much slower shutter speed to capture light, even more when you don't use flash, thus a slower shutter speed will result in your hand moving, thus the blurness.

    Evening: Similar to cloudy condition, late evening similar to night condition
