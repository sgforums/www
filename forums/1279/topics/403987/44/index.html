    Anyway, to dan474, cameo in this thread refers to permanent buses which are deployed to serve another service today.

    For example, TIB737R is WLDEP 922, which means it is a cameo only when it serves other services like 178, 961 or any other service (except 922).....

    You can check to see if it is a cameo at www.sgwiki.com
    It is an information guide to help you refer.
