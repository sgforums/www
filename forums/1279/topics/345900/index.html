    After LTA take over the central planning of bus routes, which should be some time this year, we might be able to see some changes...

    Problem of Service 190 or so coined is a long standing matter... Some points: Add buses? Shifts and work hour of many bus captains would be affected. Add more AM/PM shifts? Then the S shifts would have to be taken away (no extra buses), affected the peak hour load. The main point of this is not the peak hour load, but the abnormal off-peak hour load of this service. From 2100-2400 hrs on Fridays and Saturdays and special days such as festive season, there is also evening peak hour-like load on the service. Basically, just visualise a peak hour situation for this service from 1700-2400 hrs That is the unique situtation.

    Adding buses, not the best way, re-arranging shifts, also a headache and affects evening peak hour. Some other solutions would be operate it as a duo-operator, which would have to see LTA's approval or whether it would be considered.
