    Originally posted by SBS1908B:
    Nah.......the pitch of the engine sound never change, only the amplitude of the sound wave (loudness) change.
    actually pitch, for turbo-charged engines and big hub reducers. without turbo, it's only the speed of the turbo.

    regarding low pitch for 4th gear on VO3x's, this is common on certain 94xx-96xx (actually more of 96xx and very few 94xx and 95xx) where the engine picks up faster (due to some mod maybe) and ends up changing to 4th gear at a speed (about 55kph instead of the normal 50kph) close to the speed limiter hence it has a low rpm and sounds very quiet. the downside is that it feels like it's low on power, and this group of VO3x's like to downshift to 3rd but jump to the 4th too quickly, which results in low rpm again and makes the driver will keep push it back to 3rd again to rev it up, which means it keeps on changing between 3rd and 4th when it's supposed to cruise at 60kph.

  • SBS9818A
    26 Sep 05, 16:59
    Each bus captain has their own personal key that is issued to them when they first start work. This key is a universal key, which can be used to start any bus in the company's fleet.

    To turn on the engine, the driver must insert the key and then press a button to start it. If he removes the key with engine running (ie in the case of Sv. 36 where drivers do layover) the bus is immobilised. Gears cannot be changed, so bus cannot move.

    To turn off engine, driver presses a button again. Removing the key prevents the bus from starting up since removing the key activates the immobiliser.
