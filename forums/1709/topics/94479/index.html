    Winamp got 5.3 meh? I using 5.05 registered, so far so good, minimum system resources when running.

    WMP 10, don't really use much. WMP is one problem player when you lack video and audio codecs. Go straight for windows media player classic. Its a standalone executable. No need for installation or folders, it automatically seeks out whatever codecs your system has and tells you where to get if none.

    Realplayer, if possible, don't use at all, its one possible source of adware. Tip of advice, try not to make it your default player. Try Realplayer Alternative. It just gives you the realplayer codecs. Media player classic as mentioned above will do the rest for you.
