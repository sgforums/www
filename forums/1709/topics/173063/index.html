    Saw another thread about this some time ago. ps2.exe is a process for your Hewlett-Packard's multimedia keyboard. If you disable this process, some functionality of the keyboard (multimedia, shortcut buttons) will be lost but the rest will work fine.

    To prevent the error...
    Start --> Run... --> msconfig --> Startup --> uncheck ps2.exe
