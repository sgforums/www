    Originally posted by yakyak:
    would someone elaborate more about the 'save period method'?
    i.e, how to estimate, range from when - when
    basically, for this method you need to chart the girl's period. Usually ovulation for the girl is around the middle of the cycle. 1 cycle is when the girl starts menustrating (sp?) to the start of the next cycle.

    For example, today(9 Dec ) start bleeding until 6 Jan start bleeding again, then the cycle is 28 days. usually the ovulation is abt 14 days after the start, which makes it around 23 Dec. 3 days before this and 3 days after this is when the girl is most fertile, ie UNSAFE, as this is when the egg is released. Any period before and after this should be safe.

    Beware of using this method as the only method as ovulation for certain girls are not very predictable. If is predictable, if the girl is sick or stressed, ovulation can be delayed or early, messing up the safe period.
