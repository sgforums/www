    With the regards of people asking the moderators on how to install this and that in OpenBVE, Samuel Lee have decided to proudly presents --- <OpenBVE for Dummies>. So, if your thing does not work, please try this before asking.

    Download at here:

    OpenBVE for Dummies (Updated on 2010 August 20, contains all latest trains, Harbourvale Line, latest public release EWL and NSL) --- Developers: Please check if I had missed out anything.

    Notes by 26 August 2010: Sorry for the description error --- This 2010 August 20 OpenBE for Dummies release does NOT contain latest Harbourvale Line and did not include Airport Express Line which was released & updated on 2010 June 27.)

    You will have to wait a while before an updated Dummies package is ready.



    Contents included: All except C151 V0.6 and the 093/094 PC advert due to V0.6 class.

    Instructions: Just extract all the files on a suitable folder for you (Meaning, dont ask where is the place to extract because it is ANYWHERE -- Anymore such questions will not be entertained) and you are good to go. Start your routes in \Railway\Route\[May have subfolders for some routes] and using a train in \Train\[Your preferred train] (For mine C651 will have extra subfolder -- All these are the causes of different developers have different ways of organizing their files. But they will work, not for you to worry. )

    You will need to install either 7Zip from here, or Winrar from here where you can install either a 32 or 64 bit version.

    Both works fine, except that the 64bit version can only be installed and run in a 64bit OS such as Windows Vista X64 or Windows 7 X64.
