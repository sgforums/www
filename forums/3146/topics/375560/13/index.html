    Can I ask something... Based on Willis' Developers' Resource: Standards, the C751B should have run sounds when the track is Run Sound #1 and #2. Therefore, it can be concluded that Run Sound #1 is Jointed rails, concrete sleepers and jointed track higher linespeed with wind noise and Run Sound #2 is Continuous welded rail :

    Run sound #      Train WAV File      Description

    0                       Run0.wav               Jointed rails, wooden sleepers

    1                       Run1.wav               Jointed rails, concrete sleepers and jointed track higher linespeed with wind noise

    2                       Run2.wav               Continuous welded rail

    3                       Run3.wav               Continuous welded rail - Tunnel and bridge

    4                       Run4.wav               Jointed rails, wooden sleepers - Tunnel and bridge

    5                       Run5.wav               Passing train sound, continuous welded rail

    6                       Run6.wav               Continuous welded rail on steel bridge

    7                       Run7.wav               Passing train sound, jointed rails

     So, anyway, RouteBuilder gives the description only, so I adjusted the track types for the overground tracks to be Continuous welded rail and the underground tracks to be Continuous welded rail. However, when I play my "track-types-are-already-adjusted" routes and there is no sound at all when the C751B runs on overground tracks. What is the problem, then?
