    Announcements:

    Do put your best to measure the length as accurate as possible (1 to 2 decimal place at most, since this is google earth max). The FHPSD / station will receive a major overhaul soon.

     

    To measure the stop point, you should 141m (From [23m * 4] + [23.5m * 2 end train car] = 141m) + {[Station length - 141m] / 2} = X

    Example:

    If a station X is 150m, then the stopping point should be

    141m + ([150 - 141] / 2) = 141m + ( 9 / 2) = 146.5m after the .sta command, where .sta command is where the start of the station AND just a side note for the logic is because of 2 ends got a staff area, presumely it is the same length, and any extra can extend it backwards in future.
