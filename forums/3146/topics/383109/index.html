    Hey people! You are now able to play multiplayer games with others!

    Requirements:

    • openBVE Multiplayer Plugin
    • Hamachi
    • 1 Hamachi Account
    • [Hosts] openBVE Multiplayer Server

    Once you have all that, open up Hamachi, log in and get your own IP address. Then join one of the below networks. To play with us, initiate a chat and if we're ready we will be all set to go!

    Networks

    1. Network Name: Singapore openBVE Server 1
      Password: sgbve
    2. Network Name: Singapore openBVE Server 2
      Password: sgbve
    3. Network Name: Singapore openBVE Server 3
      Password: sgbve

    In the Multiplayer dialog, the server IP will be the host's Hamachi IP. The host will be decided among us in the chat. Note that the max number of people including the host in one game is 6.

    Have fun!
