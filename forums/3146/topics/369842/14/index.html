    If you are using firefox to download, then, THEN your downloaded file will have a .zip_ as extension. Hence, you will need to remove the _ on your file extension and you can just open it in WinRAR, 7Zip etc file archivers.

     

    BTW, OpenBVE's program are provided in zip form, that does not bar you not to use RAR or 7z etc archive format. The main thing is that get the OpenBVE installer package, download the relevant dependency and put into the same folder as OpenBVE and then, install the routes under Data\Railway folder (Example: If you install OpenBVE on D:\OpenBVE, then your route should be installed at D:\OpenBVE\Data\Railway, which our route package typically consists of Object,Route & Sound sub folders, which specifically means for scenry / any visual objects seen on the game, the route description along with how should the track curves etc details & last but not least, the announcements sound data respectively.)

     

    Ok, you will need a train. Typically any train of Standard Gauge will do BUT for realistic issue, it is recommended to install the C151 train, which is the same as our MRT trains. (C651 & C751B trains are under development, so no such train on BVE as of yet) You will install in Data\Train folder, because the developers of the train will have a folder named after the train's model such as C151 or k_stock (A train used in HK) etc

     

    The moral of the above, just to copy everything from the zip file to the apporpirate directory as described above, if you need help, you may facebook or pm me.
