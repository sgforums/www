    Anyway, I have found that there are 2 solutions to the underground cut through grass issue.

     

    1: Based on HK route, they use a value of .Height 0.

    2: Create a new text file, change the extension to .csv will do. Use that file as your ground.

     

    Eg: Create a new text file, name it as nothing.csv. Then in your declaration, put:

    .Ground(2) nsl\scenery othing.csv

    Then on the start of tunnel, put

    .Ground 2

     

    Of course, if you need to get out of tunnel, then put back the

    .Ground 1

     

    ta da!
