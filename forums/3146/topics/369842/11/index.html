    To fix the very little minor bug, open up the route file with Notepad or Notepad++, see this code?

    ;RANDOMISATION COMMANDS
        ;TRAIN
            With Train,
            ;$Sub(10) = C151
            ;$Sub(11) = C651
            ;$Sub(12) = C751B
            ;$Sub(13) = C151
            ;Train.Folder $Sub($Rnd(10;13)),
            Train.Folder C751B,

    Change it to,

    ;RANDOMISATION COMMANDS
        ;TRAIN
            With Train,
            ;$Sub(10) = C151
            ;$Sub(11) = C651
            ;$Sub(12) = C751B
            ;$Sub(13) = C151
            ;Train.Folder $Sub($Rnd(10;13)),
            Train.Folder C151,

    Fixed! (Please change the one in bold, do it yourselves! Apparently you are lazy by copy-pasting.)
