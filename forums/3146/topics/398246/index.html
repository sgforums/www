    Tell you what, go to the floor.csv, and you can roughly see the -11X+.XX (Negative 110 + meters). That is how long it extend backwards. (The history of FHPSD comes where Willis install the first 4, and only 4 PSD in his CGA, so I do it simple way which just extend it backwards). You will need to divide into 2 (Approximate), which gives you like 50X or so, that is where you need. Look for the direction, and check for the 12nd PSD and you should have a hint or so.

     

    Regarding 3 cars C751B (or any other trains), it is possible.

     

    Go to the specific train folder, find the extension.cfg

    And you will see something like this

     

    [Exterior]
    0 = cars\3237.ANIMATED

    [Car0]
    Length = 23.5

    [Exterior]
    1 = cars\1237.ANIMATED

    [Car1]
    Length = 23

    [Exterior]
    2 = cars\2237.ANIMATED

    [Car2]
    Length = 23

    [Exterior]
    3 = cars\2238.ANIMATED

    [Car3]
    Length = 23

    [Exterior]
    4 = cars\1238.ANIMATED

    [Car4]
    Length = 23

    [Exterior]
    5 = cars\3238.ANIMATED

    [Car5]
    Length = 23.5

    Each [Exterior] means the exterior object (we name it by train car number) and [CarX] refers to the car number, starting from 0 (Any programming language will tell you that index starts from 0).

    See the difference. This is why certain train car between the gaps got difference and I have to research and trial and error until it fits, like now. All number are in meters. (Obviously, you are going to keep Car0, Car1 and Car2.)
