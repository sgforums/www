    So far I know:

    Q -- Increase Break

    A -- Decrease Break / Acceleration

    Z -- Increase Acceleration

    F -- Change train to forward gear (Neutral if it set as back gear)

    V -- Change train to Backwards gear (Neutral if it set as Forward gear)

    Enter - Horn

    1 - - Emergency Break

    Up, Down, Left, right - Change the view in accordance with the direction.

    Controls...

    F5 - Opens/closes the left door.

    F6 - Opens/closes the right door.

    Ctrl + S - Activates the security system. ( ATS, ATP and etc. )

    Numpad Enter - Primary Horn. *

    Numpad Plus ( + ) - Secondary Horn. *

    Numpad Minus ( - ) - Plays/Stops Music Horn. *

     

    * Varied. The horns can be changed to music.
