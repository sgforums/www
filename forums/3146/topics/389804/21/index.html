    This works:

    States = empty.csv, bsharr1.csv, bsharr2.csv, bsharr3.csv, bsharr4.csv, bsharr5.csv, bsharr6.csv, bsharr7.csv
    stateFunction = mod[(trackdistance)/650, 8]

    What it does is it loops between 8 different files. This code assumes 8 min interval, that 650m = 1min, and that preceding trains are set. If they are not, you may get a "train arrived" message when there is no train at the platform.
