    25

    Originally posted by yellowflip999:

    I'm not sure, too many developers and routes of the same object folder confuses me too.

    True, basically this was what happened:

    The EWL used to have 2 teams, EB and WB.

    The EB mainly led by Veyron 16.4 had developed up to CLE - QUE (Before merger) while the WB mainly led by me had developed only BDK - ALJ (Before merger)

    Despite 2 teams are not merged at that time, I have also doing patches for EB so that CLE - QUE is also compatible with WB as well. (Along with few visual errors along the route)

    It was only recently (Within 4 weeks from now as of 2010 June 11), the EWL WB got a sudden increase of co-developers, which each of them doing a few stations. None of these had gotten into release as of 2010 June 11 yet. The PSR - BGS is mainly done by the earlier codevelopers and I which had been planned to release by 2010 May 28 (Due to Vesak Day public holiday + CCL Stage 3's First Anniversery). However due to certain technical errors (Timing and station length), it was re-do and only ready on a few days ago. It was not an official release / found on EWL site is because what Flip said is right, too much developers and different sectors of the route makes it confusing.

    So the plan for EWL is to release an official version 1.0 by September (Subjected to changes) that contains both PSR - JKN and JKN - PSR. Do note that CGL will NOT be included for any short / mid term plans.
