    :: Communication ::

    There are 5 kind of chat communication you can choose in FFXI. Note the color of the text, that is how it looks when you are chatting in the specified channel.

    :: Say

    When you choose this 1 as your chat channel. ALL THE PLAYERS IN THE AREA YOU ARE AT can see it.

    :: Shout

    ALL THE PLAYERS can see it.

    :: Tell

    1 on 1 chat

    :: Party

    Chat with your party members only. No limit to where you are at.

    :: Link Shell

    Chat with all the clan/guild of yours with the clan/guild's Link Shell.

    :: Link Shell System ::

    Link Pearl is something that you can communicate with your Group/Guild/Clan members.

    Link Shell are sold in shops in Town.

    - To use the Link Shell

    Go to your main menu and choose "Link Shell". You are able to input the "Link Name" after doing it. You are able to put maximum of 19 english characters. Link Name will be the name of the Link Group( Guild/ Clan ). After deciding on your Link Name, you will be able to choose the color of your link shell from 16 different color they provide. Link shell will change to "Link Pearl" after you have done so. You can make multiple Link Pearl from 1 Link Shell. If you had the chance to get a "Unopened Link Shell" Try equipping it on you ^^

    - Distribution of Link Pearl

    Players are able to receive Link Pearl via Trade. After equipping it, there will be "Link Group Channel" for the group to chat on. The one that distribute the Link Pearl will be the Leader, the one that receive it will be the member of the group.
