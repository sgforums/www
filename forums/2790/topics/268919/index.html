    Suki = (i like it)

    Daisuki = (i like it a lot)

    Meccha-meccha suki = (I LOVE IT)
    NOTE** Meccha is a Osaka(japan) slang word and very caught in nationwide..

    Anmari-suki-ja nai = (I dont like it very much)

    Suki janai / kirai = (i Hate it)

    Ureshii / Shiawase = ( I'm Happy)

    Sore kiite ureshii = (I'm happy to here that)

    Kanashii = (i'm sad)

    Tsukateta = (im tired)

    Odoroita! / Bikkuri sh!ta! = ( im surprise)
    NOTE** Bikkuri sh!ta is most common to use ..

    Kimochi warui = (I feel sick/ that's sickening)

    Shimpai sh!ta = ( i was worried)

    Gakkari Shichatta = (IM Disappointed)

    Dekiruyo = (i can do it)

    Warui-kedo dame = ( sorry i cant)

    Wakatta = ( i understand)

    Yoku wakatta = (i understand very well)

    wakaranai = ( i dont understand)

    Wakatteru = (i know)

    Shiranakatta = ( i didnt know)

    Shiranai / shiri - masen = ( i dont know)

    nandaka yoku wakaranai = ( im so confused..)

    kawaiso = (what a pity)

    Atteru = ( am i right)

    Machifatteru = ( am i wrong)

    Ochitsuite! = ( calm down!)

    Genki dase yo =( cheer up) for guys to say
    Genki Dash!te! = ( Cheer up) for girls to say
    *note* dont get mixed up..
