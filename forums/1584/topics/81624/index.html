    1) Because of the inherently inaccurate nature of physics spa, and the need for plotting of a straight line to check for anomalous errors, most of the time, when doing Physics Spa Skill B, it is always wiser to fake the results rather than actually doing it..

    i) First plot the tabulate for the 6 values
    ii) Measure experimentally only the first and last values
    iii) Proceed to draw a line (of course you must know it is a line!) using the first and last plots
    iv) Then work backwards to find the measurements for the other 4 values
    v) Because for skill B the teachers will only check for the raw values, so for the calculated values columns, you can anyhow write whatever values (they won't check) to suit the best fit line...

    yes, use this method![/b]
