    no.  1/100 scale means for every 1 inch of the model (or cm if you go by the SI system), it represents 100 inches.  If a particular Gundam is 16m tall, the 1/100 scale model should be 16 cm.  Hence, since not every MS is the same height, not every 1/100 scale model will be the same height!!!
