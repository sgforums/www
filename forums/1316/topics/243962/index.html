    Trivia
    The in-series explanation for the Hyaku-Shiki is also true to life. Early in the design process, designer Nagano Mamoru submitted his idea for the Zeta Gundam, but it was rejected. The Transformation system seems to be defective for this unit, thus, he removed the transformation abilities and reworked the mobile suit, turning it into a high performance mobile suit.
    Hyaku Shiki stands for 'Type 100' in Japanese; this is explained in-series as the designer hoping to create a mobile suit design that would endure for a hundred years.
    In Mobile Suit Gundam SEED Destiny series, a suit belonging to the Orb Union, the Akatsuki, bears a striking resemblance to the Hyaku Shiki. This is probably a homage to the original Hyaku Shiki from Zeta Gundam.
