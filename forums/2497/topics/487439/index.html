    Game Description:

    Top eleven is a football manager game and features  José Mourinho who endorsed this game. Top eleven is mimicking football manager PC game and it’s quite successful at it so far. The game has been out for more than 3 years now and boasts 15 million monthly users. If you are a big fan of football or FM game, this is probably a must have for you. The football manager game type is very addicting and fun, so I hope you don’t judge the game based on how it looks. You will get very hooked. The only problem is that you will be tempted to spend money to compete with top players.

     

    Features: 
    1. Real-time simulation
    2. Big data-base for players

    3. Real-like simulation and managing skills

    4. Requires some sense of professional soccer managing abilities

    5. Requiring depth understanding and knowledge of football
