    Hull City will officially pass Go next month when a Tigers Monopoly board game goes on sale.

    Players and club officials were on hand for the launch of the game at the KC Stadium yesterday, which confirms the Tigers as one of the country's top teams.

    The board features players instead of the traditional street names and gives each of them a value.

    Brazilian goal machine Geovanni is set to be up for sale for £400, with captain Ian Ashbee looking likely to be priced at about £320.
