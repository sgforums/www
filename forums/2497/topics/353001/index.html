    AC Milan's on-loan midfielder David Beckham admits he is flattered by speculation that Real Madrid want him back but says he is not sure if there is any truth to the rumours.

     

    "I don't know if it's true but it's nice to be appreciated," the former England captain told Saturday's La Gazzetta dello Sport. "If Florentino Perez returns as president, he'll make the club better."
