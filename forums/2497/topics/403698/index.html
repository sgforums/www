    Example:

    Germany Vs France

    Germany -1.5

    Result is
    2:1 = 2 (- 1.5): 1 --> 0.5 : 1 (You lose bet)
    3:1 = 3 (- 1.5): 1 --> 1.5 : 1 (You won bet)

    France + 1.5

    2:1 = 2: 1(+1.5) --> 2 : 2.5 (You won bet)
    3:1 = 3: 1(+1.5) --> 3 : 2.5 (You lose bet)
