    Originally posted by Scania N113CRB l0v3r:

    Wanna ask wad is away goal rule?


    Because the FC Porto used away goal rule wan win lolx

    If the aggregate score from both legs results in a draw, they will look at away goals.

    For example, first leg was 0-0 and the second leg is 1-1. On aggregate, its 1-1(draw). But away team qualifies because of that one away goal.

    If first leg 1-1, second leg 1-1. On aggregate 2-2(draw). Both teams scored an away goal each in both legs so the game moves on to extra time.
