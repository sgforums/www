Arteta Targets Top Six Finish

    The Spanish schemer believes that an extension of the Toffees' good run can see them push for a Euro place.

    "Our form is coming together, particularly at home where we have not done so well," Arteta explained to Sky Sports.

    "We want to fight now for a place in the top six. We have been disappointed with some home results this season, but now we seem to be getting that right, too.

    "We are playing so well away from home, so if we can start getting some points at home now we will have a great chance of a top-six finish again.

    "To have had so many players out injured - all four of our strikers - and to have achieved this current run is unbelievable."

    He continued: "We have a platform now to build on. When we have been at our best we have achieved great things over the last few seasons, now we are finding that form again.

    "I have enjoyed it a lot now that I have moved back into a central midfield role.

    "The team have improved and we are more comfortable during games. We aim to keep that going."
