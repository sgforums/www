    Do you find the page loads slower than ever?
    Game is lagging badly?
    Try these:

    My Computer
    Turn off System Restore: System Properties>System Restore>Turn off System Restore for all drives
    Off Error Reporting: System Properties>Advanced>Error Reporting>Disable Error reporting
    Optimise Visual Effect: System Properties>Advanced>Performance Settings>Adjust for best performance
    Enhance Processor Power: System Properties>Advanced>Performance Settings>Advanced>Click all options for Programs.
    Virtual Memory: System Properties>Advanced>Performance Settings>Advanced>Change>Initial = your RAM current, Maximum = 2x your RAM current.
    Off remote assistance: System Properties>Remote>"untick" Allow Remote Assistance invitations to be sent from this computer/Allow users to connect remotely to this computer.
    Disable Indexing: My Computer>$Drive$ Properties(Do it for all drives)>"untick" Allow Indexing Service to index this disk for fast file searching

    Registry (Type "regedit" from Start>Run)
    Boot Defragging: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Dfrg\BootOptimizerFunction\"Enable"\"Y"
    IExplorer Fifth Zone (My Computer): HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Internet Settings\Zones\0\"Flags"\"47"
    Disable Prefetcher: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters\"EnablePrefetcher"\"0"
    StartUp Programs: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
    Disable datetime impression: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Filesystem\"NtfsDisableLastAccessUpdate"\"1"
    Faster Shutdown: HKEY CURRENT USER\Control Panel\Desktop\"AutoEndTasks"\"1" & "WaitToKillAppTimeout"\"1000" & "HungAppTimeout"\"1000"
    Speed up IExplorer page display: DELETE HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\Current Version\Explorer\RemoteComputer\NameSpace\"D6277990-4C6A-11CF-8D87-00AA0060F5BF}"

    Group Policies (Type "gpedit.msc" from Start>Run)
    Unreserve bandwidth: Local Computer Policy>Adminstrative Templates>Network>QoS Packet Scheduler>"Limit reservable bandwidth">Enable>"0"
    Customize desktop/start menu: Local Computer Policy>User Configuration>Administrative Templates>Start Menu and Taskbar/Desktop
    Disable windows messenger: Local Computer Policy>Adminstrative Templates/User Configuration>Windows Component>Windows Messenger>Do not allow Windows Messenger to be run/Do no automatically start Windows Messenger initially

    Run
    Unregistering zip folder dll: regsvr32 /u zipfldr.dll
    Boost Processing Power, ignore background process: rundll32.exe advapi32.dll,ProcessIdleTasks

    Windows Services (Type services.msc from Start>Run)
    Disable the services with respect to:
    http://www.theeldergeek.com/services_guide.htm
    Each PC setting is different from each other, the services you need might be useless for him, so follow this guide.

    Misc
    -Check your harddrives for error at least once per month
    -Defragment your harddrives at least once per month
    -Scan your registry at least once per 3 month
    -Scan for virus at least once per week
    -Uninstall uncessary programs
    -Disable unused devices

    [This is recommended for Microsoft Windows XP User only]
