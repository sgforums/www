    Importing Bookmarks or Favorites to Internet Explorer

    1. Open your browser, select File, and then Import and Export
      (If you are using IE 7 press ALT to see the File menu.)

    2. Click Next then select Import Favorites and click Next

    3. Click Next, you will be prompted for a location within your favorites to store the imported items.

    4. Click Next, and then click Finish.

    5. You should then see a message saying that the favorites were successfully imported. Click OK.
