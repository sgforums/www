    When you load up your MM2, you cannot see a trailer and said something to do with IV50 codec.

     

     

    The solution is to run cmd.exe in administrator, then key in this command in bold:

    regsvr32 ir50_32.dll (For Windows XP or above AND 32bit flavor only)

     

    OR

     

    cd C:\Windows\SysWow64 (Change your System32 directory to point towards the 32bit one, instead of the 64bit one because the codec dll is in 32bit and of course substitiute the C:\ to the actual windows installation directory. Also, the Windows folder may varies as WinNT under some cicrumstances)

    THEN

    regsvr32 ir50_32.dll

     

    Notes to 32bits user. If you see your command prompt not in C:\Windows\System32, you will need to perform the cd command as per 64bit instruction, except substitute the SysWow64 to System32.

     

    This is the way to register the Intel Video 50 Codec without any downloads / installation and being able to watch the trailer. Probably it does not affect the gameplay, but I fix it before I start the game. Also, for my Windows 7 X64 (64bits), it flickers a lot when loading the program, where other games do happen as well.
