    Btw, modern cars speedo normally not accurate one, usually actual speed lower meter show 90km/hr actually only 81-88km/hr something like that..

    Older cars slightly more accurate, cos they base on wheel rotation. If u use stock rims and tyres size same as stock, it will be accurate.

    Modern speedometer all electronic, connected to the transmission, so it takes the average rotational speed of the driveshaft assuming all four wheels have full traction, so car manufacturer normally calibrate slightly higher then actual speed.

    The more expensive euro cars normally has more accurate electronic speedometer as it rely on pulses coming from wheel ABS sensor.
