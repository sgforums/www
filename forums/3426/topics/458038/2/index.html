    But here are some problems i encountered with the cam using for past 3 weeks.

    Problem 01:

    After one week, the plastic piece that is in front of the LCD dropped off. the glue they use never set properly

    Solution 01:

    I placed the plastic back, use a plastic clamp, those hobby ones look like clothes hanger clip, left it overnight now ok liao.

    Problem 02:

    The player program sometimes slow and never show which clip is locked.

    Solution 02:

    I format the SD card again using PC. Remove the "Volume Label" name, default is DR_Disk when u use cam to format, "File System" FAT32, and most important "Allocation Unit Size" must set to "Default allocation size"

    Problem03:

    Sometimes, when u on the camera and GPS connected. The date will change to one day before. ie today 25/9/2012 u on the camera, GPS sync it set to 24/9/2012. Yes u can manually change to the correct date, but everytime it's on will revert to previous day date.

    This could be a problem if let's say u forget to set correct date, for instance morning u record with wrong date, then at night u set to correct date. Then all the morning recording with wrong date will be overwritten as it detects as older file.

    Solution03:

    Simple, Format the card, if let's say u outside can format with the camera. But if there are clips inside u need to save to PC just let the wrong date be. Downside if do like that the time stamp wrong date lah.
