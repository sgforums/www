    Originally posted by Hiei1979:

    Android version only available for passengers leh

    Sorry to hear that. I got the iPhone version and it was so easy to use. From the website http://www.gocatchapp.com.au/, it says Android cabbies need to pre-register by emailing support@gocatchapp.com. iPhone cabbies have it easier because you can register and change the details directly in the iPhone app.

    I used a second iPhone in passenger mode and could see at least 10 cabbies using this app in Singapore. There are cabbies in India and Australia using the same app too.

    For the iPhone app, I suggest the following way to key in the data:

    ID: Use your taxi license plate. No need to key in NRIC. This information will not be displayed on the map.

    Name: no need to key in your real name. Just key in something like 'Mercedes E220' or 'Estima' or 'i30' because it will be shown on the map.

    License plate: Do not key in your license plate here. Just key in something more useful like your handphone number, because it will be shown on the map.

    This is based on my usage of the app as a pax. There is no need to know the TD name or car plate, what I want is to see what type of car, and the hp number. If I decide I want to book the cab, I can call the cabby directly, negotiate the price, and ask for his/her name and car plate.
