    Bro driving CNG taxi take note. If you park overnight and the indicator drop to empty and when you start the engine it goes back to normal means there is a leak in the CNG system. I had recently encounter this problem.

    But I ask around people says it is normal, when I go servicing the ah tiong also tell me it is normal. So I continue because initially the leak is not so bad but after a week it become worst. I notice there is a gas smell when I stop my vehicle and it is coming from the engine bay. Then I notice there is a decrease in range. My gas use very fast.

    After sending in to the workshop last week they suspect the refuelling nozzle because of high usuage so the oring inside might be spoil so they change for me. Still they smell gas and a hissing sound. So they check again and found out the gauge inside leak. After changing the gauge everything ok. The range improve alot!!! So the gauge has been leaking for some time but small leak.

    Then I go find out how the CNG system works. After we switch off out engine there is a solenoid valve that close the supply from the main tank. Only when we switch on electrical supply then it turn on. So what happen is when we stop for the day the gas supply is turn off. So there is a residue gas inside the gasline. So if there is a leak at the gauge it will only drain off the small amount of gas between the tank and the gauge. That's explains why when I start the car in the morning the indicator shows empty. Because there is no more gas inside the gasline. So after starting engine the gas start to flow again so pressure build up at the gauge that's explains why the indicator slowly show gas is full.But in fact when we are driving the gas start to leak again.

    So if you have this symptom of your gas indicator dropping a few bar and returns to normal again please have it check. There is a leak in the system. A small drop means a small leak. If drop until empty like mine then is a big leak. You should be able to hear some hissing sound and smell the gas.
