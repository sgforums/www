    Actually depends on how you manage your time and how much your expected income. For instance if u OMO, some want to make $200 per day and base profit on daily takings after rental and fuel. If want to take sat and sun off then actually should not calculate that way. One example if u want 2 days off per week if OMO: ie; Rental per day - $90 Rental per week - $630 Daily Rental cost if want to drive mon-fri only - $630 / 5days - $126. So daily to break even include fuel must get $156 (base on fuel $30 daily). So set that target break even target, after that all profit. End your shift at same time daily, let's say 11pm then 11pm stop, keep car.. Make less nevermind. Friday night can OT, if friday still cant make your profit target. Then Saturday drive, since you already calculated 5 days week target, saturday you drive minus fuel is all profit. Sunday die die don't drive.. If saturday OT still don't make enough, the following week might get better. If really that 3 weeks of that month still can't make it then last Sunday OT. Want 14 days annual leave, then first 3 months take pay cut, take it as work outside 3 months probation period, set aside 14 days of rental ($90 x 14days = $1260). So during the first 3 months, every month set aside $420, that's about $13 per day for 3 months. If during your "leave" u get super relief cover then better, that $1260 can bring forward to following year
