    [OP] Start and stop meter

    To print a receipt , must [STOP Meter ] , follow by [K1] once . 

    To add surcharge , press [K2]

    *CITY ($3)

    *Chang3 ( Changi Airport $3 )

    *Chang5 ( Changi Airport $5 )

    *Loc100 ( Expo $1 , press twice to make $2 )

    *Loc10 ( in Sgd cents )

    *Advance Register ( Advance booking $5.20 )

    * Current BK ( current booking $3.50 )

    To confirm above selected item , pressed [K3]

    Note : Selected amount will appeared on [EXTRAS] column .
