    17 Nov 05, 18:05
    AETOS is a private security company similar to CISCO.
    Security arms of PSA, STK and CIAS merge into AETOS.

    AETOS, CISCO and SATS security services are all private security companies.

    ATEOS officers can be seen at the airport tearing ur boarding passes. At Ports protecting the Port.

    CISCO officers can be seen more frequently in their vans loading and unloading $$. They can be found in banks also.

    Both AETOS and CISCO are private security companies whose security officers are commonly mistaken as Police Officers because of the similar uniforms.

    CISCO = Commercial and Industrial Security Corporation

    CPIB = Corrupt Practices Investigation Bureau

    CNB = Central Narcotics Bureau

    PCG = Police Coast Guard

    TP = Traffic Police

    NPCC = National Police Cadet Corps

    NPC = Neighbourhood police center

    NPP = Neighbourhood police post

    ISD = Internal Security Department

    SeCom = Security Command

    SOC = Special Operations Command

    STAR = Special Tactics and Rescue
