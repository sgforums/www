    Originally posted by whoop:

    Hi, its my first time to the interview. Is there anything that I need to prepare myself? Is there any girls going next wed assessment session? 

    Hi for the 1st stage, Selection Assessment (SA), there is no interview.

    It consists of:

    1. Psychometric assessment thru MCQ questionnaires (timed and non-timed)
    2. Filling up forms of personal details
    3. Written English essay (random question for each individual, 1.5 page minimum)
    4. Physical Body check up

    Lunch will be after questionnaires, and physical check up will be after lunch.

    Will last approximately from 0900-1700

    Good luck!
