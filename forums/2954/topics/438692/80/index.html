    Lord Buddha ;Dear monks, is it a distress to be poor man..?

    Monks; Yes L.Budda..

    Lord Buddha ; Is it a distress to have a loan..?

    Monks ; Yes, L.Buddha..

    Lord Buddha ; Is it a distress, to know that his interest on loan accumulating..?

    Monks ; Yes, L.Buddha..

    Lord Buddha ; Is it a distress, when loan collector accuses him..?

    Monk ; Yes L.Buddha..

    Lord Buddha ; Is it a distress, if he is imprisoned, unable to pay the loan..?

    Monks ; Yes Lord Buddha..

    Lord Buddha ; Yes you are correct dear monks. Same way..
