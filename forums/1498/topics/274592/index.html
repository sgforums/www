    Each pumper will have 3 numbers: lets use Alexandra as an example

    1 2 1

    the first 1 stands for the division that station is under, which is division 1.

    the 2 stands for the station number. Central is 1, Alexandra is 2, so on and so forth.

    and the 2nd 1 stands for the pumper's number. in which case, this pumper is the Duty Officer's pumper. The first pumper.
