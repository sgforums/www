    The SST4 has only interchangeable soles on its sliding shoe.
    The SST5 has interchangeable soles and heels.
    The SST6 is the same as the SST 5, but has a wider, broader base for more stability.

    For the none sliding shoe, the SST4s and earlier SST5s have only a small rubber patch at the toes for grip, while the later versions up till hte SST6s have almost half the sole covered in rubber.
