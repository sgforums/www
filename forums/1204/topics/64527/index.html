    many should know how to play this game, but just in case, allow me to elaborate a little.

    this game will consists of one empire and at least two players.

    the objective of the game is to be a participant, yet avoiding hitting on the secret number.

    the range of the number always start from 1 to 100 and the range is narrowed as the game goes along.

    e.g.
    there's 1 empire and 3 players, with the secret number: 40

    empire: 1-100
    player 1: 50!

    empire: 1-50
    player 2: 30!

    empire 30-50
    player 3: 40!

    empire: player 3 loses!!

    the game usually start off with an annoucement, from whomever wants to be the empire.

    interested participants basically post join to participate.

    the sequence is naturally selected and the number of participants are determined as soon as the empire prompt the first player for a number.

    while the game is ongoing, new participants can only join when the current game ends and the next game commences.
