    Just to share my knowledge

    coz now i actually seldom train for pull up one, but i used to be zero fighter, but by doing bench press, dumbell row, military press, overhead tricep assisted machine and the occassional dips, i can do ard 5 nw with overhand grip

    Last time in my poly, i got only one month to train for pull-up and i can only do like zero-1 using underhand grip so i rely on machines actually.

    so first u need to improve ur grip strength

    two ones

    Barbell static Hold
    Meaning u just stand there and hold the barbell for as long as possible
    The weight must be heavy (For me, the bar is 20kg and i put two 15kg plates at each side) and i lasted less than 25s usually

    Do ard 2 sets

    or u can try Hanging (Which is just hanging urself on the pull up bar) using ur body weight, no need to lift ur head over the bar (that is another story)
    for as long as possible

    The best way to train pull-up is of course doing pull up, but i feel that its pretty discouraging as for example u can only do one pull up and all u do is repeat the one pull up, whether in gym or at ur fitness station, which is quite boring

    Pull up machines do help, so do lat pull down machines
    but the problem is many people tend to go light on them so effort is not equal to results

    If u are doing pull up machine (do 2 sets)
    the best of course is to set the weight as the lightest weight cause it increase the difficulty level. The lightest weight would be One Chocolate Bar (not sure how heavy it is), if one is not possible, put the pin on the 2nd chocolate bar, or at the very least put at 3rd chocolate bar

    Try to see if u can do 6-8 standard reps and den follow by another set after some resting

    den u can proceed to lat pull-down machine
    In here, u try to go heavy, dun do wide grip pull down, rather do ur shoulder width grip

    maybe u can start with 70-80 kg
    and 8-10 reps per set, standard one, ur body is not suppose to jerk and the bar are suppose to go as low to ur chest level

    i saw b4 some really lousy ones, either they pull the bar down to eye level only, or they use their whole body.

    so pull up machine and lat pull down machine will take 8 sets, and grip 2 sets

    try to do 2 more exericise so total u get to do in gym 16-18 sets
