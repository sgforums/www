    Mathematically the 33.33% is the increase in the percentage point of 1% increase from 3% to 4%.

    cos (4-3)/3 = 1.3333 == which accounts for the 33.33% increase,

    In absolute value, what was $1000 spent in 2002 would yield $30 in GST

    In 2002, $1000 spent == $30 GST
    In 2003, $1000 spent == $40 GST

    So (40-30)/30 = 0.3333 == 33.33% in the increase in absolute value.

    However, if we look at it this way...
    in 2002, we paid $1030 for Item A wif GST
    In 2002, we paid $1040 for item A wif GST increase to 4%
    Hence the increase = (1040-1030)/1030 = 0.0098 == 1%

    we cannot argue that the item is 1000, we have to take the base as 1030. 3% GST has been already implemented some time back liao, and should be taken as the base to start with.
