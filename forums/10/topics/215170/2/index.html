    Actually this might be a blessing in disguise. Currently, Singaporeans use different ISPs, which will have different blocks of IP addresses assigned to them. This means that a user can easily be traced if he/she does something wrong.

    Now, if everybody uses the free Internet, everybody would be using the same block of IP addresses. This would make reading logs near impossible, due to the huge number of people accessing it.

    The spying problem might thus become redundant.

    Edit: However, this doesn't guarantee your privacy. Since it is an opened network, anybody and everybody can just steal your sensitive information.
