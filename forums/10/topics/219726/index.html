Decentralise the tight-fisted cabinet to process management

  • robertteh

    The biggest problem today is concentration of power on a few leaders within the cabinet.

    All major government policies and decisions should be de-centralised as shown by process management one model of which is given below:-

    HOW TO DECENTRALISE POLICIES AND DECISIONS TO BE PERFORMED BY ALL MINISTRIES THROUGH CALL-CENTRE HANDS-ON PROCESS MANAGEMENT?

    First draw up all policies and decisions that need to be implemented.

    Next, subdivide them into 4 macro implementation steps and set detailed functional sub-task processes controlled by a Process file-list, viz:-

    (1) Level 1 - Process File List
    (2) Level 2 - Administrative Forms
    (3) Level 3 - WORK FORMS (Hands-on operations)

    Arrange all these macro- and micro- tasks in forms (administration) and work forms (operations) and process them as in simple task-lines and work towards closing these task-line as shown in the following entries, viz:-

    (1) Serial No.
    (2) Date
    (3) Code [Dept] I,Q,C,and A,E,F,G,H,N,M,S,V,R
    (4) Input task-line - documentation no.
    (5) Output or Closing task-line. - documentation no.

    Adapt existing forms and work forms and set them up in Process Filelist.

    OVERSEEING WORK PROCESSES FROM OSP PROCESS FILE LIST

    All ministries and departments will no longer be giving excuses for any non-performance as all policies and decisions are tracked by form and work form processes ontrolled by the call centre Process File List entries:-

    (1) Idea step: 1X/YY/file no./SV no./document no.
    (2) Enquiry step: 2X/YY/file no./SV no./document no.
    (3) Contract step: 3X/YY/file no./SV no./document no.
    (4) Administration step: 4X/YY/file no./document no.
    (5) Administration step:4X/YY/file no./dept report
    (6) Administration step: 4X/YY/file no./work form no.

    The scholars and highly educated elites should be answerable to results and performance as shown in documentations instead of pushing away non-performance to the people.

    Through the above-stated process management they will then be more conscientious and follow up on all ideas and decisions step by step until they achieve results.

    All detailed or operation activities in any business are basically contracts for which an enterprise needs to perform like project or contract administration.

    Every Friday, each ministry and functional department (see below) will have processed its policies and decisions through tasks and sub-tasks with quantified feedbacks on task variances via cal-centre in standard 4-part TASK-LINE in ten departments:-

    1.(General) 2.(Human Resource) 3.(Revenue) 4.(Accounts) 5.(Marketing) 6.(Services-outsourced activities) 7.(Facility) 8.(Enforcement) 9. (Production) 10.(Compliance).
