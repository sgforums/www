    Well....since a few people complain about the high taxes here, I feel that maybe I should show people these figures. This is not intended to side with anyone. But just to show the figures. You can be the judges.

    Revenue figure for 2003 (In Millions)

    Item Amount Percent
    Income Tax $10,414 ( 42.3% )
    Asset Taxes $1,243 ( 5.0% )
    Motor Vehicles $1,290 ( 5.2% )
    Custom duties $1,802 ( 7.3% )
    Betting taxes $1,566 ( 6.4% )
    Stemp duty $648 ( 2.6% )
    GST $2,724 ( 11.1% )
    Others $1,048 ( 4.3% )
    Fees&Charges $3,587 ( 14.6% )
    Other receipts $320 ( 1.3% )
    Total $24,642

    Note that the income tax refers to both individuals and companies.

    Interesting note: If the total tax amount is divided by the population of 3.3M, then the tax per head is $7,700.
