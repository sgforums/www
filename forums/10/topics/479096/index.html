    The Thai rice buying program

     

    Actually, the intention of this program is good. But unfortunately, the purchase price is too high!

     

    Thus, the Thai government could not sustain this program for long!

     

    There are few options for the Thai government to choose:

     

    Option 1

     

    Print more THB! This can be done if THB has the same stature as USD! Unfortunately, that is not the case!

     

    Option 2

     

    Cancel or modify the program! It is very hard to cancel the program without facing serious consequences! Modifying the program might be easier although it will cause some loss of support!

     

    Option 3

     

    Find more higher value use for the rice! For example, besides selling the pineapple in its original form, you can also sent it to factory for canning or make it into pineapple tarts!
