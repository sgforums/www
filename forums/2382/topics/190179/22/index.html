    You wrote ...(String args[]) - but a better way is ...(String[] args). The main method requires that an array of Strings be passed to it - Thus String[]. The name "args" is convention, it can really be any word; it is the name of the array. For instance, this is valid: ...(String[] theNameOfTheArrayIsFred) - but let's go back to args! Since the main method parameter is a String array named args, then individual elements of the array can be accessed by args[0, 1, 2, 3, ...n ], where args[0] is the first word, args[1] is the second, etc. A space is the delimiter between parameters (words).
