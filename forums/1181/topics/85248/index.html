    D tune - Fujita Enginering silver 8-spoke

    A tune - Initial D Ryosuke/Ry type single pipe muffler

    Rear spoiler/wing
    A tune - none
    B tune - Mazdaspeed

    A tune - Fujita Engineering (standard bumper + spoiler lip) bumper with no. plate on the left
    B tune - Mazdaspeed
    C tune - RE Amemiya
    D tune - Fujita Engineering (same like the one in A tune)

    Sideskirts
    A tune - none
    B tune - none

    Rear bumper
    A tune - none
