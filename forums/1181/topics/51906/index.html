    Hi there, i'm using a controller to play Initial D Special Stage.. I find it ok to control it with a controller, but if you want more realism, you should definitely get a steering wheel. But if you are having trouble controlling with the controller, maybe you should try tapping it lightly while turning. 1 thing about the steering wheel is that you can't turn as much as the arcade can turn and the steering wheel need to be clamp on a table, on your thighs or on the floor. I'm not sure if there's any gear stick for the Steering wheel set or not.
