    Based on the comics and animated films of the same name, Initial D Special Stage's main feature is a story mode in which players will have to compete missions set in several different areas. In addition, the game will have a mode that lets players race for points that they can then use to upgrade their cars. The PS 2 Initial D Special Stage is scheduled for release in Japan on June 26th 2003. Visit the review from gamespot here GameSpot.com.
