    25 Feb 04, 20:21
    For R32, Tune D is the best for beginners.
    For R34, Tune C has been said the best by many.
    For Spirit R, Tune A is better for dry while Tune B is better for wet. Personally i prefer N-Tune for this car.
    For Type R, all the tunes are pretty much the same. But Tune B gets a big single turbo instead of the twin-turbo. This makes it better for high-speed tracks. C-Tune is the wet tune, D-Tune is the drift tune and A tune is both drift and grip.
    For GC8V, you can't choose any tune nor change any parts, because it is Bunta's car.

    Leave ur wheel to the left for 3 seconds and N-tune will appear. This can be done with any car. N-Tune is the stock tune, which means no added parts, it is the stock car from the factory. It is good for certain cars as it is lighter without the the things like bumpers and stuff, and you can also buy parts with it. Errm Type R and Spirit R are basically the same cars, but some say Type R is grippier while Spirit is driftier. The common belief is that Type R handles better while Spirit has better accel. To be honest, there is little difference so just pick the one you prefer.

    For the R32:
    Tune A- powerful start-up but understeer, stable
    Tune B- less understeer, great accel, unstable
    Tune C- for rain
    Tune D- weak accel but best choice for beginners. easy to control
