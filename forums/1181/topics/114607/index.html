    for tsuchisaka, 86 trueno can run this map, just that it's quite drifty..
    try to target a timing below 2'50'xxx for a fully tuned 86T..
    2'47'xxx should be ok for fulu and 2'46'xxx should be fine with wang lu..

    the sharp turn just enter slower, let's say you try 135 first. if too slow den u can enter faster next try.. if too fast den enter even slower next turn..
