    Okay, a question.

    | e^4x/(1 + e^4x) dx

    Let u = e^4x

    --> du/dx = 4e^4x = 4u

    --> dx = du/4u

    Hence,

    | [u/(1 + u)] du/4u

    = 1/4 | 1/(1+u) du

    = 1/4 ln (1 + u) + C where C is an arbitary constant

    = 1/4 ln (1 + e^4x) + C (Note: You don't need to worry about absolute sign in ln (1 + e^4x) since you know that e^4x is always positive)

    Tip: Whenever you see two similar "terms", try using the substitution method.

    Integrate (3-x^2) ^3 2x dx

    T.T
    Do you mean 2x*(3 - x^2)^3? (* is multiply)

    If so, then

    | 2x*(3 - x^2)^3 dx

    Again, you can see that d/dx (x^2) = 2x, so your first thought shall be try to use substitution method:

    Let u = x^2 --> du/dx = 2x --> dx = du/2x

    Therefore

    | 2x*(3 - x^2)^3 dx

    = | 2x*(3 - u)^3*(du/2x)

    = | (3 - u)^3 du (The 2x cancelled each other away)

    = - (1/4) (3 - u)^4 + C where C is an arbitrary constant

    Don't forget to sub back in u = x^2

    = - (1/4) (3 - x^2)^4 + C (ANS)
