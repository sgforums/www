    Theorem: n=n+1

    Proof:
    (n+1)^2 = n^2 + 2*n + 1

    Bring 2n+1 to the left:
    (n+1)^2 - (2n+1) = n^2

    Substract n(2n+1) from both sides and factoring, we have:
    (n+1)^2 - (n+1)(2n+1) = n^2 - n(2n+1)

    Adding 1/4(2n+1)^2 to both sides yields:
    (n+1)^2 - (n+1)(2n+1) + 1/4(2n+1)^2 = n^2 - n(2n+1) + 1/4(2n+1)^2

    This may be written:
    [ (n+1) - 1/2(2n+1) ]^2 = [ n - 1/2(2n+1) ]^2

    Taking the square roots of both sides:
    (n+1) - 1/2(2n+1) = n - 1/2(2n+1)

    Add 1/2(2n+1) to both sides:
    n+1 = n

    Taking the square roots of both sides:

    According to my friend, who is in the International Maths Olympiad squad of Singapore, the problem lies with these 2 lines.

    The first of these 2 is correct. However, the problem is when you try to square root both sides. On the surface, the operation looks valid. However, on the right, the expression can be rewritten as:

    n - 1/2(2n+1) = n - 1/2*2n - 1/2 = n - n -1/2 = -1/2.

    Since the square root of any negative number of undefined, the proof is thus rendered invalid.

    The fallacy is not easy to spot.
