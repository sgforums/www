    I believe that bizad's modules registration works in a different way from arts. Before a semester ends, we are asked to register for the modules that we want the next 2 semesters and we are given the modules we have chosen through balloting. The computer allocates the tutorial classes for each module that we are given before the new semester starts. First week of the new semester, we have this add/drop exercise whereby we can swap the modules and tutorial classes if you dont like what you're given, whether or not you get it is again done by balloting. I think this system is not too bad, compared to the arts system where you have to go queue up physically to sign up for each class. But the downside is we can't access the add/drop from home thru the net. Got to go down to school to do it since everything has to be done in the computer lab, which sometimes can have very long queues. But the good thing is we don't have to worry about not being able to log on to the relevant webpage and spend hours in front of the computer.
