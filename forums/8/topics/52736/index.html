    Linux: Your free and easy option

    YOU'RE worried about getting caught using pirated software.

    Yet you find original software too pricey, especially in times like these.

    What's more, you're upset by the 'if you can't afford it, don't use it' stance of the Business Software Alliance (BSA), formed by IT giants like Microsoft, Adobe and Symantec.

    Do yourself a favour. Get off the whine wagon and on to the Linux operating system (OS).

    Unlike Microsoft Windows, the OS that runs 90 per cent of computers worldwide, Linux is free.

    And there are easy-to-use Linux versions of all your common home and business applications to go with it - all free and available off the Internet at the click of your mouse.

    Save on software licences

    Your dream of running your office without paying a single cent on software licences (even better than paying for cheap bootleg copies) can be a reality. No more BSA bogey men lurking in the shadows.

    Yet last year, half the computers in Singapore were running on pirated business software like word processing and spreadsheet programs, according to BSA. So what's stopping Linux from dethroning Microsoft's Windows operating system?

    Techies have an acronym for it: FUD.

    'Fear, Uncertainty, Doubt,' said Mr Lim Swee Tat, president of the Linux Users Group Singapore (Lugs) which has about 140 members. 'That is the main reason why Linux is not as popular as it should be.'

    FUD has ensured that Linux remains the best-kept secret among a small group of users in Singapore since the man-on-the-street, while he may have heard of Linux, prefers the familiar Windows environment - even at the risk of being prosecuted for using pirated software.

    Most of us can relate to the FUD factor.

    After all, Linux remains on the laundry list of techie terms like Unix, Visual and Perl, jargon that gives you the impression that you need to be a programmer to use it.

    And I was not totally wrong to expect encountering a blank black screen with a bright blinking cursor awaiting my input of some green Greek command line to accomplish a simple computer task.

    Linux techies prefer using this mode. But Linux also offers the familiar-looking graphic interface you see in Windows programs.

    When Mr Lim launched OpenOffice Writer, one of several free Linux-based word processing program available, I recognised the onscreen display immediately as it looks like a more stylish version of MS Word.

    'You can insert a diskette with an MS Excel or Powerpoint file and use OpenOffice Math or Impress to read or edit it without any problems,' said Mr Lim.

    Worth the trouble

    With OpenOffice Writer, however, some text formatting would be inaccurate, but correcting this minor aberration is worth the little trouble when you think about the money you'll save on software licences using Linux.

    Aside from avoiding intellectual property rights pitfalls, your business won't have to buy expensive upgrades to new versions of Windows OS when Microsoft takes away support for your old OS, as it did for Windows NT 4.0 last month. Upgrading can come to a tidy sum if your business uses many computers.

    Just how much can you save on licensing if you're starting a business with Linux software?

    Mr Laurence Liew, manager of SCS Linux Competency Centre (SCS-LCC) calculated that each desktop computer running on Windows would cost you $1,360 in licensing fees. (see Table.)

    If you chose Linux, your total outlay would be zero if you downloaded and installed it yourself, or $40 if you bought a Red Hat 9.0 software package with 5 CDs and a printed installation guide.

    Even if you have 100 desktops in your office, the total cost to run Linux on all your terminals is still $40, since it can be replicated freely. If you had chosen the Windows option, you'd be paying $136,000 - just for licences.

    Another distinct advantage Linux has over Windows is the relative rarity of computer viruses with Linux. Your computer will have immunity against about 60,000 viruses that plague Windows. Linux attracts about 40 viruses.

    Enthusiasts swear that Linux is eminently more stable than Windows. But if you ever needed to fix a problem, help is on hand somewhere in the world through international mailing lists of Linux enthusiasts.

    Help is everywhere

    'Even if you encounter a problem at 3am here, you can get help within minutes from someone in the mailing list somewhere in the world,' Mr Lim said.

    Support is guaranteed, not by an exorbitant contract, but by Linux enthusiasts' egos.

    Said Mr Lim: 'We get a high on being able to fix a fellow Linux user's problem or improving Linux through correcting a bug highlighted by a user.'

    As a result, Linux updates are constantly posted on the Net for free downloading, a feature that has contributed to the stability of the OS. Of late, Microsoft has also taken to this practice.

    Linux lends itself to educational and charitable programmes because it can run on old computers with low specifications.

    'You can run it on just 64Mb of RAM onboard,' Mr Fabrice Marie, vice-president of Lugs pointed out, 'Try running Windows XP on that.'
