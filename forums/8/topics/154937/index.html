    Cover-up rule:

    [(4-3x)/(x+2)(1-2x)]=[A/(x+2)]+[B/(1-2x)]

    A=[(4-3x)/(1-2x)]putx=-2
    B=[(4-3x)/(x+2)]putx=1/2

    If memory serves me right, I think that not all partial fractions can use the cover-up rule.
