    Following is a response to a forummer in some tech support web site:

    Not implying that the following is a universal solution for all laggy issues, but it proves that game servers can mess with your connection.

    =============================================
    click start , run and type cmd then hit enter. Now type tracert (your game server address) and hit enter. This will show you where your connection is going over a maximum of 30 hops. The problem may or may not be the game server or your ISP but could actually be a server in between. Once you find the server that is lagging (causing high pings) you can then contact your ISP and see if they can help you.
    The same problem happened to me awhile back and I found that a server in the chicago area was messing up my connection. I then called my ISP and they contacted them to see what the problem was. the administartor of this server told my ISP that in fact they were having some troubles and it was going to take 2 weeks to correct the problem.
