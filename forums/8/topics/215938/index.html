    For 2nd question

    1) Measure 2 coins per side, lets call them a & b vs c & d. If balanced out, the 5th coin is fake. If not go to step 2.

    2) Measure a&b. If a&b balance out go step 3. If a&b not balanced go step 4.

    3) Replace coin C with 5th coin. Measure 5th coin with D. If they balance, C is fake. If not balance, D is fake. Because at this point either C or D is fake. The 5th coin confirm is not fake.

    4) Replace coin a with 5th coin. Measure 5th coin with b. If they balance, a is fake. If not balance, b is fake. Because at this point either a or b is fake. The 5th coin confirm is not fake.

    All in all 3 measurements needed.
