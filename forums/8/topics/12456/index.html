It's due to the hyper long pipelines that control the flow of information while Imac G4 has very short pipeline. when there's a faulty signal coming thru those pipeline, all the data signal within the pipe must then be clear completely before the new cycle of signal can come in. because of the long pipeline in the P4, significant amount of data will be lost when there are random error signal comming in. while the G4 has very short pipeline which only suffer minor losses in each cycle. that's why a P4 must require super high clock speed cycle to compensate the number of error signal received to catch up with the amount losts in each signal cycle.
