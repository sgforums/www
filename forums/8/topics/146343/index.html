    Originally posted by the.raven:
    OMG n00b, JPG has 256 colors, GIF only has 16 colors, so if you wanna preserve the quality of the pic, use jpg, if the pic only got simple colors, or is animated, then use gif.
    Uh uh - GIF has 256 colours, JPEG supports 16.7 million colours. Use GIFs for logos and images with little need for high colors differentiation, and use JPEGs for photos.

    Use GIF for animated images if you can't use vector-based animations.

    GIFs can be set for transparency, i.e. the GIF background can take the colour of another image that's set behind it.

    GIF supports interlacing, in which the image is first brought up as a low-res picture, and refined over a few passes. that said, Progressive JPEG can do the same thing.
