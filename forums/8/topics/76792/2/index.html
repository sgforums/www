    You downloaded Ad-aware, but did you update it's reference files? I believe the installation comes with a rather old reference file. This is pretty important as the file identifies all the rubbish, works like anti-virus updates.

    Reference file : 01R298 20.04.2004 Loaded

    You should see that line under "Initialization Status". R298 is the latest version.

    Go click : Check for updates -> Connect
    Then restart ad-adware. The reference file should change R298 or newer (bigger number). If the reference file stays at an older version, go it the official webby and d/l the file, then update it manually.
