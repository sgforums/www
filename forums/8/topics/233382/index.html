    THE SCI-FI flick 'Serenity' has the dubious distinction of becoming the first HD DVD movie to be pirated and shoved onto BitTorrent.

    According to Ars Technica, the movie is available as a .EVO file and can be played on most software packages such as PowerDVD. Weighing in at 19.6 GB, the MPEG-4 VC-1 file is huge and pirates are going to need some serious bandwidth if they will have the file within the next decade, or at least a week.

    Pirating the movie, ironically about outlaws, is supposed to be impossible because of the super secure copy protection on HD DVD. Last month a hacker dubbed 'Muslix64' announced that he had managed to bypass the copy protection system and wrote an open-source program called BackupHDDVD to automate the process.
