    Say 2x^2 - 3x - 5 = 0

    2(x^2 - (3/2)x - 5/2) = 0

    2(x^2 - (3/2)x + (3/4)^2 - 9/16 - 5/2) = 0

    2(x^2 - (3/2)x + (3/4)^2) = 49/8

    (x - (3/4))^2 = 49/16

    Hence,

    x - (3/4) = 7/4 or x - (3/4) = -7/4

    x = 5/2 or x = - 1

    The factorization you want is of the form:

    (2x - 5)(x + 1) (since the initial polynomial has coefficient 2 for the highest degree)
