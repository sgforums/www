    Originally posted by launtpc:
    Nooo! Not solve, because behind the algebraic equation there is no "=0" so we cannot assume that.

    We can only factorise.

    x(2x-3)-5

    Or can further factorise?
    By Euclidean Algorithm, for a map f: R -> R[x], x --> 2x^2 - 3x - 5, there exist g such that f = (x - a)g + r where a is a root, deg r < deg g.

    It can be further proved that in this case that r = 0 since we know that f(x) = 2x^2 - 3x - 5 = (x + 1)(2x - 5) where a = 1, g = (2x - 5) and deg r = - infinity which is less than deg g = 1. Therefore the condition is satisfied and thus the factorization holds.

    Though the "let f(x) = 0 and then solve" is not fundamentally correct but what's most important at this particular level is just to get 2x^2 - 3x - 5 = (x + 1)(2x - 5). It is definitely acceptable at this level.
