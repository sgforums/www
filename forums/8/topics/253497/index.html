    Give you the fastest way to learn

    Go to Gym Try those type with the pull up assist.

    Pull from no weight support first... Pull until you can't pull anymore...
    Then put one weight.. And pull until you can't pull anymore..
    Keep doing this until you reached the end where you put all the weights for support.

    This is call overload training... You will strengthen very fast... but for the next few days you will be unable to lift the lightest things not to mention doing any pull up... But go do some push up..

    do this twice a week... do this for few weeks... you'll pull about 12 or more.
