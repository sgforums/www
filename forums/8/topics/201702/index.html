    Go Google and search for a Notepad2.

    This notepad is better than the Windows notepad, it tells you whether you have one less or more extra braces.

    If the number of braces are correct, the code would like this:

    { codes here
    }

    If you put in one extra brace, the code would look like this in Notepad2:

    { codes here
    }
    }

    If there are not enough braces, a combination of the above codes would show.
