    Before installing any OS I think you need to fdisk first to create partitions since nowadays hard disk are so many Giga byte not like my era where hard disk is in Mega byte

    When powering up look for the BIOS setup like Del or F1 key to ensure the boot sequence is set properly and your hardware like hard disk, floppy disks etc are detected by the BIOS.

    Then you use the bootable disk to boot up. Typically depending on what type of OS like Windows or Linux, you need to type in some command. For some OS, it should auto prompt up with some text based screen for you to do installation.

    This may not be the case for Windows NT cuz the bootup screen actually can do the fdisk functionality for you before going ahead with other installation.

    For people who always do installation for a living, they typically use Norton Ghost cloning technique which is definitely much faster and since now CD-R so cheap it becomes more viable.
