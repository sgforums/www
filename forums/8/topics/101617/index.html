    One misconception about cable.

    The district does not share 1.5 megabits per second. The district has a pipeline which can support up to a certain amount. Say 200 megabits per second.

    Imagine 2000 users in that neighbourhood.

    Each is entitled to a 0.1 megabit per second speed if they all log on simultaneously.
