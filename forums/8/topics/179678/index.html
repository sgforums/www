    good practise is to partition your hdd...one for OS + application, the other for your data.

    The problem with some software is they tend to install their dll or some other shared files into the OS partition (usually assume to be C:\) or whatever drive the OS is installed in. This 'contaminate' the OS partition.

    So just do a copy of the software into the application partition will not work after reformatting. You need to reinstall those software to get back those dll and shared files. Data files will have no such problem.

    The safest way is to ghost it using Norton Ghost. A heavily used software for backup purposes. It is like doing a byte by byte copy.
