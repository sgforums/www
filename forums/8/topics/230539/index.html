    Originally posted by Diehard_89:
    how many ways can 9 balls of 4red 4white and 1black be arranged in a line such that no red ball is next to the black

    any one?
    hmm, first, find total number of ways you can arrange all the balls(without the restrictions). you should know how to this if you ask this question. let it be x

    next, find the number of ways a red ball will be next to a black ball. this can be done by making the red and black ball a new 'ball'. this means that you will have 8 balls now, 3 red, 4 white and 1 red/black ball. you can then find the number of ways you can arrange those balls. let it be y. then, within the new 'ball', you can have to ways to arrange it(RB or BR). so, the total number of ways a red ball will be next to a black ball is 2y

    so, total number of ways such that no red ball is next to the black = x - 2y

    case 1:

    (W B W) (6 other balls) => 7 groups to permutate altogether. Treat (W B W) as one group

    combi = 7!/(2! x 4!) = 105 (need to divide becos same coloured balls are identical)

    case 2a:

    (B W) (7 other balls) => 7 groups to permutate altogether. (B W) group is fixed on the LHS

    case 2a:

    (7 other balls) (W B) => 7 groups to permutate altogether. (W B) group is fixed on the RHS

    summing up case 2a and 2b, combi = 7!/(3! x 4!) x 2 = 70

    add these 3 cases you will get 175.
