    If yr in Classic View, double click Regional and Language Options

    Under the Languages tab, make sure Install files for East Asian Languages is checked. Some computers need the Windows XP Installation Disc to be inserted to install, while others dun need...
    Add on:

    Regional and Languages Options --> Languages --> Details --> Add --> For "Input Language", chose "Chinese(PRC)", after you chose this, the "Keyboard Layout/IME", for me, it changes to "Chinese(Simplified) - Microsoft Pinyin IME 3.0" --> Ok --> Apply --> Okay --> Apply --> Okay.

    Another small addition:

    Same as above. After Details --> Left click on the "Chinese(Simplified) - Microsoft Pinyin IME 3.0" --> Properties --> Select "Full Pinyin" instead of double pinyin and "Word" instead of Sentence.

    Now, to type chinese, for me, I can just press "left alt + left shift" to alternate between Chinese and English. For Chinese typing, just type the hanyu pinyin and a row of chinese words will appear. Use the up-down directional arrows to go to the word and press "spacebar". There you go.
