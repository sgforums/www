    Here are some solutions, not guaranteed to work.

    1. Increase your virtual memory again. By default, your virtual memory is twice the size of your physical memory. Let's say you have 256MB of physical memory, the default virtual memory size would be 512MB. Increase this to 1024 or higher. One thing though, you will run out of hard disk space as virtual memory uses your hard disk.

    2. Increase your physical memory size. Before doing so, make some checks. Download CPU-Z and save it somewhere convenient. Unzip it and run CPU-Z. Once it has finished detecting your system specs, click on the Memory tab. It shows the type of RAM you are using. Go Sim Lim or any other tech shops to buy another stick of RAM (from CPU-Z results).

    3. Downgrade your Football Manager to a suitable version or you upgrade your PC.
