    Depends on what you want to do. If you want to reinstall Windows

    1. Put in your Windows CD
    2. Go to BIOS and change bootup sequence to boot from CD
    3. Wait for the Windows setup screen to appear
    4. There will be a screen which asks you if you want to format your drive
