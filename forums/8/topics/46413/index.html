    Here's what a guy who ran a basic computer course said: you may have stuff running in the background that will write some stuff on the HDD so once it is changed, defrag will start again.

    He also said it is best to close all programs except the vital ones, then start defrag. You should be able to do that using the good-ol' Ctrl-Alt-Del to end some of these tasks.

    Hope this helps
