    make sure power off your notebook.

    flip it over and search for the opening to the ram slots.
    consult your user manual if you can or still has it.
    check for maximum allowable upgrade to your notebook's ram capacity.
    last few years notebook manufacturers put 1024 max.
    these recent years seeing 2048 max.
    open up your ram compartment to see how many slots inside.
    usually 2 lah i tell you.

    notebk ram modules are smaller (shorter) in size and height than the desktop type.
    we call them soDIMM. look out for this term when buying notebk ram.
    the same memory capacity may see a higher price in soDIMMS compared to normal desktop DIMMs.

    it is recommended to have a pair, or use RAM modules with the same memory capacity for max peformance.
