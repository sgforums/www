    nid help. anyone noes what this means?

    Make sure you have ethernet interface installed. PPP connection (e.g. xDSL, Dial-up, VPN) is not actually a LAN interface, games like Warcraft3 will not send out any packet (or to be precise, not send packet to the PPP connection) when it can't detect any ethernet interface. You can try to establish a local LAN and forward incoming and outgoing traffic (for WC3, udp and tcp port 6112) to that interface.
