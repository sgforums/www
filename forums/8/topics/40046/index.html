    Usually people display elements in the array by doing a System.out.println(...) to the console. If there is indeed a need to display on a JPanel, you have to use Java 2D Graphics API.

    class CanvasPanel extends JPanel {
    public void paintComponent(Graphics g) {
    super.paintComponent(g);
    Graphics2D g2 = (Graphics2D) g;
    g2.setBackground(Color.white);
    g2.drawString(/*array element*/, x-coordinates, y-coordinates);
    }
    } // end paintComponent
    } // end CanvasPanel

    Now treat CanvasPanel as a normal JPanel class but with the paintComponent method overridden. To refresh the CanvasPanel so that it can draw new strings, do a CanvasPanel.repaint()
