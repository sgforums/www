    trojan horse?

    Trojan horse are a little bit different from virus IMHO. Trojan horse is when it is touted to say open file but instead it delete file. Basically it perform task differently from what the documentation say on that exe when you click it. It does not attempt to disguise itself and hide in the PC like those 'traditional' viruses. The payload is activated immediately and it's presence is detected.

    Virus, on the other hand, are software written meant to spread and hide inside the PC as long as possible to avoid detection. Of course one could argue Trojan horse is virus too. Virus difference is in order for it to spread and to prevent the virus from infecting infected files repeatedly, it embed a signature into infected files so that subsequently those files that are infected will not be infected again.

    Most anti-virus software definition files will contain those identified signature so that when anti-virus came across those infected files they can flag warning some virus has invaded your PC. That is also why one need to constantly update with the latest definition files.[/i]
