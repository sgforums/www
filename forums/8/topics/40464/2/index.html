    how does the ghost software works ?


    For Windows NT, Windows 95/98/ME, get a Norton ghost bootable floppy diskette. Then change your PC BIOS to boot up from floppy drive. Once booted up, a DOS based GUI screen will appear. The Norton Ghost software will auto detect the partitions within a single hard disk or different physical hard disk you have on the PC. Then using some keystroke you will say ghost from where to where. Options include partition cloning like ghost C:\ only and put the .gho image file on D:\ or disk to disk cloning (this requires you to connect up both physical hard disk like their wires and connections to power supply etc etc). Then hit Ok and the ghosting take place with a progress bar displaying the time elapsed etc etc.

    Personally, I have tried partition cloning only. Usually C:\ where I install my OS and drivers etc.

    To de-ghost, use same process but inside choose the Source and Destination the reverse way.
