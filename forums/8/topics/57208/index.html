  • choco B
    16 Sep 03, 13:22
    How many? Infinite!

    Before you just join any dance in public/line dance nights, you gotta take classes to learn. Go sign up with a basic class in the CC, clubs, etc. The teacher will teach you different dances each with its own unique name and accompanying song. Then after you've learnt some dances, you can go for line dance nights, where they have a program of dances set out for the night, ranging from beginners to advanced. If you've learnt alot already, you can dance all of them! If not then have to sit most of them out, or just do the basic dances.

    Usually in a night's of class (about 1hr), can learn about 2 songs, 3 if the group is fast.
