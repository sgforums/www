    Using MPC: Go to File/Save image of the player and current frame of the vid will be saved as jpg

    Using WMP:
    Step1 Go to tools/option/performance/advance and uncheck Â“use overlaysÂ” for video mixing renderer.
    Step2 Print screen using keyboard button
    Step3 Open MSPaint and paste it in.
    Step4 Save it as jpg

    Using Real Player:
    Step1 Go to desktop, right click/properties/settings/advance/troubleshoot
    Step2 Reduce the hardware acceleration to none
    Step3 Print screen using keyboard button
    Step4 Open MSPaint and paste it in
    Step5 Save it as jpg
    Step6 Increase hardware acceleration to Full after job done.

    The reason why some videos cannot be captured via screenshot is the way they were rendered. If its rendered in YV12 format, no problem, but most renders disable this format, so you usually end up with a blurry shot or no shot at all. If your player has the option to play in RGB mode, no problem, use the printscreen will do just fine.
