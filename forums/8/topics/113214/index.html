    Under folder options.
    Click on the "View" tab.
    Uncheck "Hide file extensions for known file types"
    Apply changes and click OK
    Behind the "." of the file you selected
    Change the extension from "dat" to "mpeg"
    Click ok or yes when an error box pops up and tell you it may cause the file to become unstable
    tada! you're done
