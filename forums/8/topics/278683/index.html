    Need to apply drilling license to HDB first.

    You must use a type approved drill with PSB sticker.

    The drill bit must be certified masonry bit by PSB.

    The bolt you want to use must conform to BCA regulations.

    since you are drilling ceiling, you using ladder, ensure this are safety certified.
