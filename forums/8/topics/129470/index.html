  • ditzy
    30 Apr 05, 23:52
    Put all the mp3s into one playlist. In your winamp playlist editor, ctrl+A to select all the mp3s, then right click and select view file info or Alt+3. This will be ley chey but its to remove your mp3 tags and your mp3s will be named as per file default file name. Uncheck both ID3v1 and ID3v2 tags. Click update and do so for all the other mp3s that will follow consecutively since you selected all. Its only tedious unless you got like what a few thousand mp3s in that playlist.

    Tip of advice, make sure you renamed the mp3 file name to something you recognized first. Coz if you did above, both tags are gone permanently, and winamp reads tags as priority.
