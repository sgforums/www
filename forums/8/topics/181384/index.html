    Common ones are 96, 128, 160, 192 bitrates. 128kbps is near CD quality. Lets use 96kbps as example.

    Download iTunes

    Throw your mp3 songs into the library. Then go to Edit --> Preferences --> Advanced --> Importing.

    Select MP3 Encoder. The default bitrates are 128, 160 and 192. If you want 96 or 112kbps, select custom and a window will pop up. When you are done, press ok

    Select or batch select the songs you wanna convert. Right click on the song(s) and 'Convert Selection to Mp3'
