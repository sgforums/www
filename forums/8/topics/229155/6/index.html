    HERE WE GO AGAIN!!

    game senta ni ikou = ( lets go to video arcade)

    itsu hima? = ( what time are you free )

    okasi = (Sweets)
    okasi tabetai = ( want some sweets)
    NOTE** if you want to offer someone something , u can add in
    ...........(food/sweets/things) tabetai = (e.g) want some coffee..

    onaka suita = ( i'm hungry)

    nanka tabetai = (i'd like to eat something)

    mada tabetenai = ( i havent eaten yet)

    tabetai = ( do you want to eat? )
    **note** you can ask a friend **tabetai** whether he wants to eat or if you want to ask whether he/she wants to drink something :
    Nanka nomu = ( do you want to drink something)

    Biru = ( BEER )

    wain = ( WINE )

    Kora = ( cola ) known as "coca cola"

    ocha = ( green tea )

    kohi = ( coffee )

    Chuhai **means** alcoholic drinks based on shochu, a kind of japanese vodka. they come in at about 5%, in sweet and sour flavours and are easy to drink.

    nodo kawaita = (im thirsty )

    *note* when u enter a restaurant, pub or anywhere that sells *drinks* in JAPAN, you can reply the following as Sofuto dorinku ______ <--- (drinks).
    *means* (e.g) Sofuto dorinku kora.. (COLA) ** i'd like a cola

    Nihon-no tabemono-no naka-de, dore-ga ichiban suki?
    = ( whats is your favourite japanese food )

    shokuji dekita / gohan mada = ( is the meal ready ? )

    Itadakimas! = ( Grace )
    *why??*
    Almost every japanese says this before eating, in almost situation. The closest English equivalent is *grace*, but it has lost its religious meaning. Dont hesitate to say this before a good meal. *WHY*? because you MAY EVEN BE COMPLIMENT FOR YOUR GOOD MANNERS. its very regonize around japan where people say itadakimas before any meal =)
