    depends on the shop, some of them sell in pairs (usually under 40bucks for a pair) while the others sell in a box (one year) which costs like 100 plus. Ask them before buying.

    You can get the disposibles one also but chances are you wont be able to see properly at night. I don't recommend you getting the coloured one because if you wear it, you're looking through a hole which is so bloody irritating.

    and one more thing, although a monthly contact lens is to be used for one month only, it can last for about 3 months before it looses its shape. 3 months that is if you take care of it properly.

    hope this helps =)
