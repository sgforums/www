    there r two approaches. both use gotta r-forumale..

    (1) Start with cosine

    R = (4Â²+3Â²)^0.5=5

    3cos2t + 4sin2t
    = R cos ( 2t - b )
    = R cos2tsinb + Rsin2tcosb

    where b: basic angle

    comparing coef,
    Rsinb = 3
    Rcosb = 4

    so b = arc tan (3/4) = 36.9
    => 4sin2t + 3cos2t = 5cos (2t - 36.9)

    ----

    (2) Start with sine

    4sin2t + 3cos2t
    = Rsin (2t + b)
    = Rsin2tcosb + Rcos2tsinb
    tanb = Rsin2t/Rcos2t = 4/3
    b = 53.1
    4sin2t + 3cos2t
    = 5sin (2t + 53.1)
    = 5cos (90+2t-53.1)
    = 5cos (2t - 36.9)
