    I'm no tech guy but this is what I would do.

    First, I'll probably try to back up the data. Perhaps you can do this, by booting up through another hard disk and transferring the data to it because sometimes even though you can't boot up through that hard disk, you can still access it.

    Then, boot up through the Win XP cd and go to recovery console. Run chkdsk. Run the fix boot record and master boot record, just in case the boot record got messed up due to improper partitioning.

    Then, reboot. If cannot, then most probably the hard disk is not functioning anymore. Can try reformatting I guess. I know the computer shops have a diagnostic program to check whether the hard disk is still functioning but I forgot what it is. I think can google for it.

    If confirm dead, check warranty.
