    Help you with the second one:

    Imagine:

    Jone = 1st prize
    Smith = 2nd prize
    No. of different ways = 1 x 1 x 8 x 7 = 56

    Jone = 1st prize
    Smith = 3rd prize
    No. of different ways = 1 x 8 x 1 x 7 = 56

    Jone = 1st prize
    Smith = 4th prize
    No. of different ways = 1 x 8 x 7 x 1 = 56

    Altogether oredi has (56x3) = 168 choices

    If Jone = 2nd prize, then no. of choices still same. 168 choices.
    If Jone = 3rd prize, then no. of choices still same. 168 choices.
    If Jone = 4th prize, then no. of choices still same. 168 choices.

    Altogether u add up (168+168+168+168 ) = 672
