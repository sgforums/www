    Hm. Try uninstalling MSN Messenger and reinstall it, or run the installation file again and select the Repair option. (You may wish to disable any anti-virus program resident in memory during reinstallation. Some anti-virus programs may interfere with the proper installation of certain components.)

    If that still doesn't solve your problems, open Internet Explorer and go to Tools > Internet Options > Temporary Internet files > Settings > View Objects.

    In the Downloaded Programs folder, you should see some ActiveX components for the games you've installed (e.g. Solitare Showdown Class). If you see any component with a text string like {F6BF0D00-0B2A-4A75-BF7B...} it is likely corrupted during installation and has to be removed. To remove it, right click the file and choose delete. Then uninstall and reinstall MSN Messenger.

    HTH

    Normally the latest version of Javascript would be installed together with the latest version of your browser, which I assume is Internet Explorer 6.0 (with SP2 if on Windows XP).

    Try checking your Internet Security Settings:

    Open Internet Explorer, go to Tools > Options > Security > Select Internet > Custom Level.
    Under Settings, scroll to Scripting section and ensure that all options are enabled.

    You may also wish to upgrade the Java Runtime Environment and Java Virtual Machine in your system.
    Go to the link below and download the installation file:

    http://java.com/en/download/download_the_latest.jsp
