    I am using Sony DSC W1.

    The pros:

    Great battery life - and it uses AA batteries!

    Great 2.5" LCD display which is very usable in daylight.

    Excellent shutter lag and power-up time.

    Very good video recording capability - in MPEG format too.

    Has manual control.

    The Cons:
    Not the most slim camera. However it uses AA batteries, so size is what it has to compromise on. If you want a truly slim camera that uses AA batteries, try Ricoh Caplio R1. Was deciding between these two cameras but the Sony won me over due to its reputation.

    Memory Stick is EXPENSIVE.

    The pic with the best resolution/size is 2.2MB in size.

    The video, due to its good quality, uses tons of storage space. A 256MB card can store only 10" of movie - and this is not the highest frame-per-second movie.

    In a nutshell, the only disadvantage of the W1 as far as I'm concerned is the cost of Memory Stick.
