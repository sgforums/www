    simplest method .... not the best, but simple

    1. open up your Bitmap pic using MS Paint (every Windows PC should have this)
    2. goto : File > Save As ...
    3. use the drop down menu to select your file type (the drop down menu is located at Save as type, eg. .jpg, .gif
    4. voila!!
