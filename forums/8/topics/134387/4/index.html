    Hello,

    Before u decide to find out the details on how to trade, u have to have a clear fundamentals first. Things like what brokerage firms to choose, whether to trade online/ offline, etc, are not really important. Once u have ur fundamentals strong, u understand the concepts, u know the different types of traders, the different analysis used, know ur objectives/ time frame, etc, then u can start to think about the details on how to start the actual trading. It shouldn't be a focus.

    If u manage to know all the trading details, all the commision rates, all the nitty-gritty rules, but ur fundamentals is shaky, u will only set urself up for trouble. They're not important at all.

    Unless if u have gone through all the homework, and now starting to trade.
