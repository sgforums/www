    For your O2 device to function properly, both main and backup battery MUST be full, or the main battery promptly recharged. Otherwise, it will trigger itself into a Hard Reset, resulting in total lost of all settings and data stored in ROM.

    Ideally, you'd be better off upgrading to an O2 Atom Exec, or Stealth or even the latest Zinc, which runs Windows Mobile 5 which is much more stable.
