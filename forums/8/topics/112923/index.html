  • CoolMyth
    16 Jan 05, 22:03
    Formal solution:

    Let x be the number of members in 1998.
    Let y be the membership fee in 1998 for each member in cents.

    Hence,

    In 1999,

    no of members = x + 20
    membership fee = y - 10

    Therefore

    xy = 4200 ------------- (1)
    (x + 20)(y - 10) = 4500 ------------- (2)

    From (1)

    y = 4200/x -------------- (3)

    Sub (3) into (2)

    (x + 20)(y - 10) = 4500
    (x + 20)[(4200/x) - 10] = 4500
    4200 - 10x + 84,000/x - 200 = 4500
    10x + 500 - 84,000/x = 0
    x^2 + 50x - 8,400 = 0
    (x - 70)(x + 120) = 0
    x = 70 or x = -120(not possible)

    Lastly,

    y = 4200/70 = 60

    In 1998, no of members = 70, membership fee = 60 cents.
