    ii) (1+ cosec x)(1 - sin x) = cos x cot x

    LHS
    = (1+ cosec x)(1 - sin x)
    = 1 - sin x + cosec x - 1
    = cosec x - sin x
    = (1 - sin^2 x)/sin x
    = cos^2 x/sin x
    = (cos x)(cos x/sin x)
    = cos x cot x
    = RHS
