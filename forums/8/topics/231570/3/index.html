    Originally posted by hegu:
    what does it mean by 'homogenous equation' in case of differential eqns?

    how do we see whether the DE is homogenous or not? and if it's not we must make it homogenous to proceed?
    example:
    (dy/dx)+p(x)y = r(x)
    if r(x) = 0 it is homogenous 1st order linear DE
    if r(x) not = it is non homogenous 1st order linear DE

    Homogenous equations are easier to solve as there are proven methods available. (normal separation of y and x, substitution of y=vx, integrating factor, bernoulli's method for non linear DE.)
