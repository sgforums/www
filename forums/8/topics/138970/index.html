    Look like it is a consensus that HP ink is expensive.

    Do you know that actually other printer's ink is expensive too?

    You need to know the actual cost per page for the ink. For example, if you buy $30 ink and it prints you 1000 pages -> 3 cents per page.

    Do your math first!
