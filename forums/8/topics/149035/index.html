    open all yr pictures u want to make into a gif in PS first.
    move all yr pics into a single picture by using the move arrow as circled.
    now each pic is called "layer"

    >>now to avoid confusion, i suggest u open the pics in order, eg pic1 then pic2 and so forth. then u move pic 2 into pic 1, so pic 1 has 2 layers

    make sure all yr pics are aligned or they will look weird. to check for alignment, use the Eye icon in the Layers palette beside the layer name. if it isnt there, means it is off.

    >> to align all pics, uncheck all the Eye icons (i know there's a name for it but i simplify it) then click on each layer and use the Move button to move it ard.

    Once u are done


    go to Image ready

    U shld see the following palettes
    Layers, Animation,



    click on the duplicate frame to add more frames. as u can see, frame 46 and 47 are the same because i clicked on 46 then duplicated a frame

    Now, as mentioned before, the Eye icon controls visibility of each frame. so
    click on frame 1, uncheck all other Eye icons(in the Layers palette) except the Background layer's icon.

    this is yr first frame.

    continue until u have finished all the frames

    then now u can change the time delay between each frame



    in this pic, i selected a few frames and clicked on this number below each frame. this number denotes the time delay before moving off to the next frame

    when u select more than 1 frame, u change the timing for all the frames selected, rather than individually.

    u can do a preview prior to saving

    [img[http://img.photobucket.com/albums/v215/tarui/Untitled-7.jpg[/img]

    using the Imageready to preview isnt accurate(unimportant step)



    once u are done, see above, then save as gif.
