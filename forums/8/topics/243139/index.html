    Really, we need English lessons. I can barely understand.

    If you are using Windows, get a card reader. Plug the card reader in, and let Windows install the drivers.

    Next, reboot. Then slot in your memory card. Windows will detect it. Now, open it by double clicking on the drive that appears the list of removable drives.

    Press Ctrl + A to select all the songs. Press Ctrl + C to copy the files to memory. Now navigate to the folder where you want to transfer these songs. Press Ctrl + V to paste them into the folder.

    In Linux, same thing. Plug in the card reader, and Linux will automatically install drivers. If it can't detect, you are out of luck. Go and install a new version of Linux, since all newer versions of Linux contains USB drivers, especially the kernel.

    After you plug in, double click on it to open. Those using Xubuntu, open Thunar. Click on Filesystem. Locate your card reader. Double click all the way till you find your files. Again, press Ctrl + A to select all, press Ctrl + C to copy them. Next, navigate to the folder. Press Ctrl + V. Other versions, go and explore yourself.

    In Mac, an icon should appear once the card reader has been installed properly. Double click on the icon to open. Then select all, copy and paste. Can't remember the exact buttons on Mac.
