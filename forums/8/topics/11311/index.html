    Palm IIIxe is basically cheap Palm Vx.
    Both same processor at 20MHz.
    Older palms use the Morotola Dragonball processor at 16MHz only.

    New Palm M505, Visor Prism, Visor Platinium uses the NEW Motorola Dragonball at 33MHz.

    By using software like TRG FlashPro.
    You can make use of the UNUSED space in the 2MB FLASH ROM.
    The 2MB ROM is used to store the Palm OS, but the remaining is left unused.
    By using FlashPro, you can use the remaining space to store your data/application.
    But normally only about 700Kb or so...
    Depending on the OS version and the Palm model you use.

    With a Palm III(old one, only 2MB), running OS 3.3, you can get additional 800Kb.

    Even If you use a 32MB Compact Flash card in the Visor Deluxe now, the 32MB only can be used to store data.
