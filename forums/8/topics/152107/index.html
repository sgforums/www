    If you are using WIndows XP Home, in Control Panel you can configure fax, with the Printer and Fax icon.

    You will need to configure the Fax console to pick up a call after a number of rings, from 1 - 99 rings. The Enable Send and Enable Receive should be ticked.

    Also, when entering your fax number, do not include country-code. Same in Fax Console, don't use country code.

    Mine's working fine and dandy. I can receive fax and it will be saved in a separate InBox which I created, and a copy sent to printer to be printed out.

    I can also attach the fax to be emailed.
