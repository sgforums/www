    You could try using PHP and mySQL as your database, they are all free and they are pretty gd to use, lots of resources abt them.Go and google for
    PHP Triad, it comes with a built in Apache Server ( You need this to run your PHP and mySQL ) and also the mySQL database itself.

    Alternatively you could use MS Visual Studio 2003 Edition and write a .NET application either in VB or C#, C# is better as it has more functionality connecting to web services etc connecting to web based systems.

    Alternatively, you could use J2SDK to develop a small CORBA application that is able initiate database calls through the Object Request Broker and use the database. You will need a Apache Server or Tomcat Web Server to set up the HTTP to allow your client to discover the IDL and invoke the methods.

    The 1st 2 options are for small and medium. Last one is for enterprise applications. but running a small application for yourself can be a mind rewarding experience learning how to set up the web server and do remote object discovery.
