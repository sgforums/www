    Okay... Go to the start -->Control Panel-->User (assuming you're using WinXP).

    Set up 2 account, one is already set up, because you're using it now.

    There would be 2 account then, the old and the new.

    Give the new account administrative rights.

    And remove the rights from the old and current account. This is to make sure that you are using your current settings.

    Set the password to the new account (the admin one) and don't forget that password.

    Take note that some programs might not be able to run, like ICQ and Nero if you don't know how to configure their right,
