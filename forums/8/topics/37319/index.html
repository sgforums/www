    Originally posted by larren:

    quite misinformed...
    not really... SCSI harddrives are internal like normal IDE counterparts. the only difference is that it offers amazing multitasking, self correction as well as insane seek/read times.

    fastest way to recognise a SCSI harddrive is to read the information printed on the top of the drive itself. they should specificially state "SCSI Hard Disk" or some tech jargon like "U160, U360 and so on"

    the U prefix indicates the SCSI interface while the number gives the specification for bandwidth overhead in Megabytes per second

    only high end macs use scsi drives. scsi drives are insanely expensive. i know because i considered it but settled for IDE RAID since it was significantly cheaper. a top end SCSI drive like the Seagate Cheetah X15 will run into $500+ for a measly 36 gigs or even less. not to mention one needs to buy a dedicated SCSI drive controller.

    scsi is too expensive for a typical home user, IMHO.
