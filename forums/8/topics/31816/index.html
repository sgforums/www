  • chamenos
    1 Jan 03, 18:41
    make sure you have less than four devices occupying your IDE ports first. in every computer, there are two IDE ports. the primary IDE port, and the secondary port. on each port, you can have two devices. one is configured as the master device, the other as the slave. make sure the hard disk with your OS on it is the primary master, as the computer usually looks for devices in this order (for hard disks): primary master > primary slave > secondary master > secondary slave. to let the computer know which is which, you have to set the jumper on the hard drive accordingly. its usually indicated on the sticker on the hard drive, but if its not there you can try to see if its indicated on the controller board underneath.

    before you install it and boot up, you first have to make sure you set the bios to auto-detect all IDE devices. after you boot up into windows (i'm assuming you're using win9, open a dos box, type fdisk, and set the partitions as you wish...you can just make the whole hard drive one partition, or you can split it up into a few; its up to you. after you're done partitioning it, you have to format it. to do this, just go to the dos box again and type format d: or whatever the drive letter is.

    lastly, try to install the hard drive away from other devices so it doesn't heat up excessively...that's about it. anymore questions just ask here
