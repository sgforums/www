    Originally posted by kendoboi:
    Can anyone tell me How ip address is being assigned to to each pc? I'm learning something about networking now and met these problems..please asisst
    IP address can either dynamically generated or statically definded. The idea of using an IP address is to enable the communication between machine using Internet Protocol (IP) other than using the networking card's physical MAC address.

    Most Network uses a DHCP server to dynamically assign IP address. The dynamic generated IP address is used to ease network administration by defining a pool of IP address that can be obtain "as and when" needed. When a PC connects itself to a network, the network card checks the DHCP server for an unused IP address and request for it. When a PC disconnect itself from the network, the IP is release back into the pool.

    In static/assign mode, the user needs to specify the IP address bound to its network card.
