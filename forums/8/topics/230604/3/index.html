    Of course can survive. Can save some money some more.

    1 meal $2.50 x 3 meals a day = $7.50
    $7.50 x 30 days per month = $150
    1 MRT trip $1.20 x 2 trips a day= $2.40
    $2.40 x 30 days per month = $56
    Room Rent = $200 per month
    $1000 salary - ($150 + $56 + $200) = $694
    Still can save $694 each month.
    12 months x $694 = $9586

    Still can save almost $10,000 each year.
