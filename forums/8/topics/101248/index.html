    5 Nov 04, 11:20
    I'll use | to signify integration sign:

    | x^2 ln x dx

    Let u = ln x, dv/dx = x^2

    --> du/dx = 1/x, v = (1/3)x^3

    Therefore,

    | x^2 ln x dx

    = [(1/3)x^3 ln x] - (1/3)| x^3(1/x) dx

    = (1/3)x^3 ln x - (1/3)| x^2 dx

    = (1/3)x^3 ln x - (1/9)x^3 + C where C is an arbitrary constant.

    Let y = 3 - [11/(x-3)]

    --> 11/(x-3) = 3-y

    --> x-3 = 11/(3-y)

    --> x = 11/(3-y) + 3

    --> x = 3 - [11/(x-3)]

    Therefore,

    f(x) inverse = 3 - [11/(x-3)] = f(x)

    As for symmetry, note the following is a "thought process"

    Since you know the symmetry of the curve y = 1/x is y=x, note as well, that the graph of f(x) is similar to the curve of y=1/x move to the right by 3 units. So, the symmetry of f(x) or f(x) inverse shall be y = x - 3, if I'm not wrong. Feel free to correct me.
