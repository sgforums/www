    I use BSplayer to play all my video files.

    For screen capture:

    Options
    Preferences
    General
    Check YV12 and YUY2 formats
    Set desired directory
    Set image format
    Set capture file name format
    Play video
    Press "p" to capture video frame original size
    Or press "shift-p" to capture what you see
    Go to set directory and find captured frame

    guess i'll share a tip here... for this, u do not need any external programs of any sort.

    1) Go to Tools > Options in the Window Media Player.

    2) Go to the Performance tab

    3) Set the Video Acceleration to NONE.

    4) Play your video then pause at any time u want.

    5) Press Shift + Print Screen on your keyboard.

    6) Open MSpaint via Start taskbar > Accessories > Paint

    7) Press Ctrl + V and viola
