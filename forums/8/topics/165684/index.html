    Wait... i think i know why the diff...

    It depends on what filesystem your hard disk has... NTFS or FAT32. FAT32 is the older filesystem also used for windows 95, 98 and ME, so it has no sense of file / folder security, any user can access any file with the same rights.

    NTFS is the windows NT/ 2000 filesystem so it has knowledge of user priviledge. so admin protected file cannot be viewed by user on NTFS drives.
