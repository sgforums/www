    Someone need to host the chat server.

    Install the server: Currently the port is hard-coded to 1234
    1. Copy Chat Program to install the server.

    Run the server:
    1. Open up a command prompt
    2. Navigate to the folder when you copy the server/client files
    3. Execute it like C:\\>java chatsvr

    All you clients need to install Java JRE on your PC to run the client.

    Install the client:
    1. To install Java JRE click on Java 1.1.3 JRE to install on PC first.

    2. Copy Chat Program to install the client.

    Run the client:
    1. Open up a command prompt
    2. Navigate to the folder when you copy the server/client files
    3. Execute it like C:\\>java chatclnt
    4. Make sure you fill in your nickname, IP of the server and Port No set to 1234. Then click Connect
