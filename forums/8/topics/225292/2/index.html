    not everybody have access to E-mart.

    buying running shoe, shouldn't just about cost and good(what is gd for you might not be gd for others).

    Ask yourself what you wanna use your running shoe for? leisure run ard ur housing block for 10mins? run in stadium for 30mins? east cost run for 1 hr? Macritchie trail running for 3hrs? Are you going to use the shoe for other activities? Eg badminton, basketball etc.

    What are your foot type? pronated? Supinated?

    Injury occurs when one just grab the cheapest pair of "good" looking/feeling shoe off the shelf. There is small invisible print that states injury not included if use inappropriately that most are not aware of.
