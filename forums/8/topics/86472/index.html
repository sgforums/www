    Originally posted by Parka:
    Wah. So ex. Haha. Antoh's PHP server only cost $25 for a .com

    Why is there such a big difference.
    $15-$25 is for the Domain Name only(example: sgforums.com or abc.com, .net, .org, etc).

    What you will be paying for are:

    1. Domain Name and registration ($20+)
    2. Web space/bandwidth/etc (example: maybe $100 per year for 200MB space)

    If you want cheaper domain, do the registration yourself at namecheap.com. And then just get a web host to host your domain and provide you web space.
