    Please see this , and this may also help.

    You can by the way but as hmaster said you should search.

    c:\documents and settings\user\application data\microsoft\msn messenger\account number\custom emoticons.

    Also you need to have hidden files and folders showing.
    If you search the forums, I have answered this before.

    The emoticons are stored on your computer as .DAT files in that folder. But you can change them back to .gifs so that you can use them somewhere else.

    1. First open the .DAT file in internet explorer

    2. Then you should see the emoticon that the .DAT file contains. Right click and save this emoticon. You must save it as different name than the .DAT file.

    3. Now open the file you just saved, again in internet explorer. You should see the emoticon again and now you can right click and save it as a .gif file.

    Its a lengthy process but it works.

    Hope this helps.
