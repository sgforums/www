    Using | for integrate symbol:

    Note: | sin x dx = - cos x + c where c is an arbitrary constant.
    | cos 2x dx = (1/2) sin 2x + d where d is an arbitrary constant.

    Therefore:

    | 3 sinx - 4 cos 2x dx
    = 3 | sinx dx - 4 | cos 2x dx
    = - 3 cos x - 2 sin 2x + e where e is an arbitrary constant.
