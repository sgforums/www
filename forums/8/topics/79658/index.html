    First, before you all read the rest i hope you all just think for yourself, what do you think zero divided by zero gives you. Most of you probably would think it is 0 or undefined. But i choose to stick to 1. My reasoning is as follow.

    First, i will give a simple number pattern to prove it:

    5 / 5 = 1 (reads 5 divided by 5)
    4 / 4 = 1
    3 / 3 = 1
    2 / 2 = 1
    1 / 1 = 1

    thus

    0 / 0 = 1

    also, here's a logarithemic formulae

    0 / 0 = x
    lg (0 / 0) = lg x
    lg 0 - lg 0 = lg x
    lg x = 0 (lg 0 - lg 0 cancels each other out)
    x = 10^0 (reads x = 10 to the power of 0) (changed form using index form formulae)

    thus, since anything to the power of 0 is 1

    x = 1
