  • CoolMyth
    12 Apr 05, 20:25
    I'll use | for integrate symbol:

    | 1/cos x dx
    = | sec x dx
    = | [sec x(sec x + tan x)]/(sec x + tan x) dx (ie, multiply (sec x + tan x) top and bottom)
    = | (sec^2 x + sec x tan x)/(sec x + tan x) dx

    Observe that d/dx (sec x + tan x) = (sec^2 x + sec x tan x), hence you can apply the logarithm derivative rule:

    d/dx (ln [f(x)] = f'(x)/f(x)

    Therefore,

    | (sec^2 x + sec x tan x)/(sec x + tan x) dx
    = ln (sec x + tan x) + c where c is an arbitrary constant.
