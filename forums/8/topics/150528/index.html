    WINDOWS WORM WARNING: 'W32.Reatle.I@mm' exploiting MS05-039 and MS04-011 Vulnerabilities

    http://www.microsoft.com/technet/security/Bulletin/MS05-039.mspx
    http://www.microsoft.com/technet/security/bulletin/ms04-011.mspx

    ACTION: Windows users need to apply MS05-039 and MS04-011 updates immediately. Anti-virus definitions should also be updated.

    SUMMARY

    Previously Microsoft announced vulnerabilities in Windows and Internet Explorer. One bulletin, MS05-039, described a vulnerability in the Windows Plug-and-Play feature. Another bulletin, MS04-011, described a vulnerability in the Windows Local Security Authority Service.

    This week a worm was released that exploits these vulnerabilities. This worm, known as 'W32.Reatle.I@mm,' is quickly spreading across the Internet.

    Microsoft systems lacking the Windows updates released to address the vulnerabilities described in MS05-039 & MS04-011 are at the greatest risk of being victimized.
