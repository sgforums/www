    one: you need a CD/DVD Burner Software that supports CUE sheets.
    a good example: Nero Burning ROM

    two: Open the burner program, just Ctrl + O, select the CUE file.

    then.. customise the burning options to your preferences.

    and... Burn.

    done.

    if it doesnt work, open your CUE sheet with any text editor eg Notepad or Wordpad.

    you should see something like: "C:\FOLDER\FOLDER\FILE.BIN"
    delete the directory part, leaving just the file name i.e.: "FILE.BIN"
    save and close.
    (make sure your BIN and CUE files are in the SAME directory!!)

    and do the steps above (CD Writing part).
