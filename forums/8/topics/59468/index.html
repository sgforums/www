    erm win 9x CAN run VB. VB is meant for all windoze platforms... cos the kernel is basically the same. of course NT and Win9x kernels work differently but the command subsets for both are approx the same

    i suggest u try to up ur budget. but if u dun intend to get a monitor... perhaps can squeeze in

    get a nforce 2 mbo. got more than adequate integrated graphics + sound (though i dun like Nvidia cards/chipsets).. it should be at a sane price since it has be around for sumtime. get a LARGE hdd... i cant stress it enuff... try to get 60 gigs minimum. programs like visual studio.net etc eats up GIGABYTES of HDD let alone MSDN (which weighs in at 2 gigs)... Seagate, Maxtor are the cheaper ones from the last time i checked. better ones like Western Digital/IBM can be considered. here im using WD.

    get a AMD processor for its the only one compatible with e Nforce mobo. suggest u look at the 1.6 GHz range and above.. thats where you are really getting value for $$$ cos the cheaper ones... which runs at much lower clock speeds arent that worth it IMHO

    finally, try to get a minimum of 256 MB of RAM... bare minimum. i would suggest running windows XP since the windows 2000 is a update nightmare. it doesn't really install right outta the box (at least for me) and the configurations will surely confuse someone who is new at this.

    and i think $600 is cutting it close but it can be done if u forgo a monitor...
    but on the other hand i would recommend $1000 as a ballpark for the most value for money system. i stress the 3 words value for money here cos u are paying for what u get.
