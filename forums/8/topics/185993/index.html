    Your download speed is restricted, usually in P2P, because of a router that you are using and Windows firewall.

    First of all, you need to configure Windows firewall to accept your P2P client, which is usually done by default when u install the client, so Windows Firewall doesn't really matter

    Now, you need to do a simple port forwarding for your router. ALL ROUTERS block out P2P by not allowing port forwarding, by configuration of the router, u can then cancel this protection (Completely Legal). Routers only block it as a security precaution.

    And so far, Pac and Sing net DOES NOT block P2P in anyway. Starhub do affect P2P speed by shaping ur speeds in ratio to your peers in the gateway. However, this does not really affect much.
