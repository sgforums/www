    New version of Mydoom virus found

    HELSINKI -- Internet security experts on Wednesday found a new version of the Mydoom computer virus, dubbed the Mydoom.B, that evades detection measures for the original virus, Finland's F-Secure said.

    'The new virus has been modified so that the original Mydoom anti-virus protection does not detect it,' director of computer security firm F-Secure's anti-virus division Mikko Hyppoenen, told AFP. He added that computers are getting infected but in smaller numbers than by the original Mydoom virus so far.

    The Mydoom.B virus is designed to attack Microsoft Corp's main website, as well as the website of US-based software vendor SCO, which is the sole target of the original Mydoom virus.

    In addition, it has a new feature, rendering it impossible for infected computers to access the websites of several anti-virus software firms.

    The new virus has already been cracked however, taking F-Secure's experts just 90 minutes and posing far less trouble than the first one as they had already analysed cracked the first version, Mr Hyppoenen said.

    Computer users should update their virus protection software immediately, he urged, adding that several anti-virus firms have posted the necessary software and procedures free of charge on their websites.

    The new virus must have been created by the same person or group that authored the first Mydoom virus, as the modifications were made to the original source code, Mr Hyppoenen said.

    Buried in its programming code -- and only readable after it has been decrypted -- was also the message 'Andy; I'm just doing my job, nothing personal, sorry' from the creator, Mr Hyppoenen said.

    F-Secure Corporation is a leading provider of security for companies' computer systems, including anti-virus software. -- AFP
