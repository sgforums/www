    boolean checkRow(int[][] aa, int i, int j, int n)

    }
    return true;

    for the best perfect solution to solving sudoku. for all empty spaces, u just write possible numbers down by eliminating the numbers horizontally, vertically and inside the box. u will need quite a few practices before u make no mistakes in filling in the numbers. and note that the flaw of this solution is u must make NO mistakes at all. after u managed to fill in all the empty spaces with possible numbers, u may want to solve either within the box, verically or horizontally. look for the empty spaces with only 1 possible number, or the number that only present at one space, that will be the correct number. following that, eliminate that number in all the spaces within the box, horizontally and vertically. repeat the process and there goes ur ans.
