    You'd have to convert .sub file (binary) to .srt file (ASCII) for the softsub to work.

    The only reliable freeware I know of is Vobsub.

    1. After installation of Vobsub, launch SubResync.
    2. Open the *.sub file.
    3. Select your language option, then save as *.srt file.
    4. You may have to enter matching letter(s) for subtitle text.

    HTH
