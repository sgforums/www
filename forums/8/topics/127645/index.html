    Originally posted by the.raven:
    prove, for that the limit lim x-> infinity 1/x + 3 = 3 exists
    It suffices to prove that lim (1/x) = 0

    Therefore, we have to prove given e > 0, there exist an integer N > 0 such that

    x > N --> |(1/x) - 0| < e

    Hence, from |(1/x) - 0| < e --> |(1/x)| < e --> x > 1/e.

    Choose the smallest integer N such that N _> e, therefore x > N _> 1/e, hence this implies

    |(1/x) - 0| < e Therefore lim (1/x) = 0 and hence lim (1/x) + 3 = 3.
