    If everyone is a bear,

    Let number of bear = X

    Each bear has 2 eyes,

    Hence, total number of eyes = X x 2 = 2X

    There's a saying, an eye for an eye.

    A bull also has 2 eyes.

    Number of bull's eyes is equivalent to the number of bears' eyes.

    However, i play darts just now and got 6 bulls' eyes.

    Which makes our total number of bulls' eye = 2X + 6

    Since you want to know how many bulls are there,
    all we need to do, is to divide our sum " 2X + 6" into 2, and we will get the number of bull.

    The answer is X + 3,

    I think you should change the question to :

    How many bears are there?

    in order to find X, and ultimately, you will get the answer on how many bulls are there.

    Cheers
