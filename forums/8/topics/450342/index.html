    script writing is an art...but it can be learnt as well.... first, u start by selecting and reading a good script that you are familiar and interested with (preferably one that has won awards).....highlight any memorable lines u come across..how each scene/plot, each character develops, how each flow converge with one another..write down afterthoughts if u have any.... next, u find an award winning film that was based on the script...its ok if they are adapted...to get the feel of how the script was played....never mind about the styles of director, acting dynamics, costumes, props...focus on the differences between the film and the script that you have read....this is very useful as you write your own script once you have some ideas and how you would like it to flow in real life.... keep developing, find all ways to develop till each scene achieved your heart's desire...till the end of the script.... This will not make you an overnight genius, but it is one of the best way to begin script writing...best of luck!
