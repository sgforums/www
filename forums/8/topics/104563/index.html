    The phishing site looks exactly like DBS Bank's website, with the following exceptions:

    1. The DBS logo is wrong.



    2. The URL of the site is wrong. The URL for the same page on DBS Bank's website is http://www.dbs.com/sg/personal/.



    3. The login page for Internet Banking is different from that of DBS Bank's.



    We strongly urge all customers to carefully verify that they are at https://internet-banking.dbs.com.sg before carrying out any Internet Banking transactions.
