    there's a software call recovery studio that can preform recovery of file even after it has been deleted cos to the harddisk, files are not actually deleted immediately when the operating system delete the files.

    things is going to get a bit technical here...
    what is deleted is actually the pointer that points to where the actual file is and when this pointer is deleted, the OS will recognise as the file is deleted..

    recovery studio or similar software can recover files deleted from the harddisk or even files in the harddisk when the partition has already been deleted.

    recovery studio can managed to recover up to several instance of the harddisk that has been formatted several times.. (as long as the data blocks have not been overwritten)

    so recovery studio can be your last resort to restore your files!
