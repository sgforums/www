    My answers to Question 5.

    1. Label the bags of marbles from 1 - 10.

    2. Extract 1 marble from bag number 1, 2 marbles from bag number 2, 3 marbles from bag number 3, 4 marbles from bag number 4 and so on till you extract 10 marbles from bag number 10.

    3. Weigh the 55 marbles at one go.

    4. If the last digit of the total weight will tell you which bag contains the heavier marbles. For example, if the total weighs 551gs, number 1 bag contains the heavier marbles. If the last digit of the total weight is a 0, the number 10 bag contains the heavier marbles.
