    CD-RW (Compact Disc - Rewritable)
    A storage device that uses laser technology and is re-recordable.
    Commonly mistakened for: CD-RW Writer/Reader

    CD-R (Compact Disc - Recordable)
    A storage device that uses laser technology and can be recorded once.
    Commonly mistakened for: CD-R Writer/Reader

    CD-ROM (Compact Disc - Read Only Memory)
    A storage device that uses laser technology and is non-recordable.
    Commonly mistakened for: CD-ROM Reader
    Another term: CD

    CD/CD-R/CD-RW Writer/Reader
    A peripheral used by a PC to read CD/CD-R/CD-RW and write CD-R/CD-RW.
    Wrong Terms used for this device: CD-Burner, Burner, CD-ROM

    CPU (Central Processing Unit)
    A peripheral required in a basic PC used to perform calculations, making comparisons and check logical functions.
    Commonly mistakened for: Tower Casing, Monitor
    Another term: Processor, Processing Chip

    Monitor
    A peripheral required in a basic PC used to display visual data.
    Wrong Terms used for this device: TV, projector

    Modem (Modulator/Demodulator)
    A peripheral used by a PC to connect to the Internet.
