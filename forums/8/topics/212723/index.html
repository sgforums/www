    row 2: 4 cans, total of 6 cans
    row 3: 6 cans, total of 12 cans
    etc etc

    now, lets look at these numbers, and make some ratios here, skipping right to row 2.

    so in row 2, we had a total of 6 cans, so if we take the number of rows over the number of cans, we can simplify the fraction: 2(row #)/6(total cans) = 1/3

    now, in row 3, we have the total of 12 cans. so lets do that, 3/12 = 1/4

    row 4, total of 20 cans, 4/20 = 1/5

    and so on.

    so lets pick one of these ratio, we can use row 2. 2/6 = 1/3. what can we figure out from here?

    row/total cans = 1/row+1

    so lets do some math to figure out which row we have 1190 cans in. let X be the row.

    X/1190 = 1/X+1

    lets do some simple math..

    1190(1) = X(X+1)

    a little further..

    1190 = X^2 + X

    0 = X^2 + X - 1190
