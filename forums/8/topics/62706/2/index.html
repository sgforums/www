    Hi Riff,
    The most important thing to note is that it's almost impossible for you to try to change someone. Even though you think and you know that it's for his/her own good. There are a few things you can do:

    1. Compromise: Strike up a deal with her. Let her know how you feel and meet somewhere in the middle. Meaning, you change a little... she changes a little.

    2. You make all the adjustments yourself. This requires the biggest sacrifice. For this, you have to try and see if you can accept this particular character trait you're having a problem with. Give it some time.

    If none of the above works, I'd suggest that you break it up. Harsh as it may sound, but seems like this issue is the kind that will stay on for a long time.

    Hope that helps.
