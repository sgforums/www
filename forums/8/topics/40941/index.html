    1. if your notebook is wireless, then just go straight into the set-up. If not, plug in the lan cable.

    2. Go to network connections and check whether ur lan/modem card is functioning properly. It should state "the device is functioning normally" and "enabled".

    3. Go to settings if modem. If using ADSL then follow the settings given step by step.

    4. If u're on the Lan, go to Lan connections and make sure your protocol Tcp/IP is present. If not have to install from win-xp disc. Then do the appropriate proxy settings for ur LAN system. Usually proxy.XXX.XXX.XXX.sg or something like that.

    5. Reboot b4 trying (if it still doesn't work)

    otherwise shd be up and running ...
