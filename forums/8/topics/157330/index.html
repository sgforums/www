    if i'm not mistaken, 3G has more to do with the complete network. 3G is supposedly the replacement for GPRS, and is based on either the CDMA2000, WCDMA, or the TD-SCDMA. GPRS has always been regarded as the interim technology till a better and faster technology can be developed and implemented. as such, the infrastructure for a 3G network is different from the infrastructure of supporting GPRS.

    3G networks allow continous streaming while GPRS, as it's name sake suggests, transfers data in packets.
