    Originally posted by Big Bot:
    If say want to connect several LAN in 1 building together...

    Can use Switches instead of Hub?

    Switches offer dedicated bandwidth, hubs do not.

    Do you need to block broadcast traffic between the LANs? If yes, hubs will not work, as you can't define separate broadcast domains with them.

    Do you need separate VLANs for security? If yes, then switches that support VLANs are needed.
