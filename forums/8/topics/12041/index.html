    for those of you who wanna display but don't know how to, here are a few crude steps..

    set up your screen exactly as you want it to look like in the shot.
    hit the print screen button.
    go to windows paint.
    select edit>> paste.
    say yes if it asks you if you wanna enlarge the bitmap.
    save the file.
    upload it to a server somewhere.
    link it in your msg by using the [ i m g ] tag (without the spaces).

    if you can convert the files to .jpg beforehand, even better.
