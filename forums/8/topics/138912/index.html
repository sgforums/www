    haha ok i know this reply is pretty late but these are some points to take note of if u ever encounter this problem again.

    if the error msg u see is something like " memory full, try closing some applications", this means that u've too many applications running in the back ground. just press and hold the menu button and press "C" button to close them 1 by 1.

    if you saw the msg while transferring files, this means that your PHONE memory is full. you may have tons of space left on your memory card but the files you are recieving is going to your phone memory, therefore, you can either clear up your phone memory or adjust the settings so that recieved messages go to your memory card instead of the phone's memory.

    hope this helps
