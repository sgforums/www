    generally all brands have similar quality. if you heard that brand x got more problem, usually it is because more pple are using brand x.

    the numbers behind the number indicates some sort of performance. used to be ATA33, then 66 and now 133. std for desktops hard disk to spin at 7200 rpm now.

    memory, pc-100 is quite old already. thought pc-133 was already the norm like a year ago. if your pc is running at 133 fsb, then putting in pc-100 memory won't work. now the std ram for pentium 4 is DDR.

    any way, hard disk is going through a major change in the middle of this year. the drive interface will change from parallel ATA to serial ATA, which means that whatever har disk you have now will not be able to use in newer systems launching later this year.

    so if you are going to upgrade for long term, think twice.
