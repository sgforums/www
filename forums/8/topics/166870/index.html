    You cannot make them show up.

    When a program is installed, the Installer writes a Registry value in the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall with the values of the Name, Uninstall Path and Icon.

    When you upgrade, Windows 2000 apparently overwrites the whole Registry's Uninstall section.

    Therefore -- you have to hunt around your Program Files, uninstallin the unneeded programs...
