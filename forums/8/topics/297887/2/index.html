    I don't think there is actually such a thing as American English, or Canadian English at all ... (British English, Aussie English too ? )

    if you stay there long enough, you'd be soon be able to realize that there are so many different regional accents and nuances of speech, that almost no two ang mohs speak the same way ... it's especially true at cities with high migrant turnovers, high inter provincial and city-rural movements ....

    prairie farmboys would have a different accent from newfies, people from ontario would sound almost american, some vancouver ang mohs now speak like CBCs, with flatter, with more precise tones compared to others ... etc etc ....

    there is an acrolectal form of English ... but I don't think there is an "acrolectal" accent that one should strive towards ... just speak clearly and well, you'll be understood everywhere ...
