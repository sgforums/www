    Q1 re: excel

    For data in your main sheet to be have the same data on your other sub sheets (all within same xls file), you should use a forumla like
    ='sheet1'!cellnumber

    E.g. Let main sheet = sheet1 and supporting sheet = sheet2
    and you want sheet 1 cell A1 to take the value of sheet 2 cell A1
    in sheet 1 cell A1 type: = , then select sheet 2 cell A1
    the resulting formula in sheet 1 cell A1 will look like this :
    ='sheet2'!A1

    So if the data in sheet 2 cell A1 changes, it will be reflected in sheet 1 cell A1
