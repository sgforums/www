    Here's a table of can and cannot....

    Toshiba fridge (made after 2005) : ok for deciduous plants
    Mitsubishi (most models) : ok for green plants, but set thermostat first
    General Electric : not recommended
    Panasonic (or any matsushita brands) : ok if there is no silver ion techology
    Sharp : best result if plasmacluster disabled because pollen will be filtered

    it's best to contact the manufacturer direct for advice.
