    boot up using ur XP CD... after all the loading, press "R" to go into the Recovery console. Select "1" for Windows XP.. then after that you should see a "C:\windows" after that...

    type "cd system32\drivers"... then type "ren update.sys update.sp2"...

    type "exit".... your system should reboot.. and now you should be able to go into windows.... (remove your XP CD first)

    Go to microsoft website and read all articles there on how to solve all the problem... remember to rename update.sp2 back to update.sys after that...

    It works for me.. so It should work for u too !!
