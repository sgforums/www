    also try to learn when to use:
    "liao"
    (meaning is like "already")
    eg. "he go off liao" "I eat finish liao"

    "lah"
    (Cant explian exactly when to use, but it's very commonly used.)
    eg. "Cannot Lah" "Dont like that Lah" "please Lah"

    "meh/ah"
    (end with it when ask a question)
    eg. "You dont know meh?" "You dont know ah?"

    "Wah"
    (exclamation, like "Wow!")
    eg."Wah, so many people!" "Wah, today you wear so smart ah?"
