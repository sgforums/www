    Yes, Command Prompt is a lookalike of MS-DOS.

    But there are some differences between them.

    The first is the usage of long file names. It doesn't exist for Win 9x. Not too sure for Win ME...

    Windows 2000 and above support the use of long file names.

    Win 9x uses a convention called 8.3

    By 8.3 it means the file name has a maximum of 8 letters, and the extension has a maximum of 3 letters.

    For example this file:

    internet explorer.exe

    In Win 9x it would be - interne~1.exe

    On a Windows 2000 machine, it will be internet explorer.exe

    Also in Win 9x PCs, you don't enter in cmd. You enter in command.com

    The first window is command.com, the command line in Win 9x and ME.

    The second window is cmd.exe, the command line in Windows 2000 and above.

    Look at the folder naming convention. That's one difference.

    Take a look at the years. 2001 was the last year when MS support MS-DOS, when the first NT family was released (if I'm not wrong).

    Open Task Manager now. You will find a program called ntvdm.exe. That's MS Virtual Machine, and is used to run 16-bit programs, without interfering with the other 32-bit applications. Otherwise, you will get an error trying to run a 16-bit program.
