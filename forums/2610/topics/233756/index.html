    Some basic command in Command Prompt..

    cd.. ---> Go up one level
    cd\ ---> Go straight back to the main (eg. C:\)
    cd document and settings ---> eg. c:\ ---> C:\Document And Settings
    dir ---> Directory (show all files)

    Since we are on directory, let's continue on about directories.

    You can make a new directory aka new folder by doing this:

    mkdir path | directory

    For example, you want to create a directory called whatever in C:\Program Files

    mkdir C:\Program Files\whatever

    A folder called whatever will be found in C:\Program Files
