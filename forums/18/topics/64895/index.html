    with assumption that you have same number of columns in both tables, below are the steps i could advise:

    - copy and paste the second table right below the first one.
    - delete the space in between until they're joined.
    - adjust each column width (by adjust the border line, not by the top ruler) so that it matches exactly across both tables.
    - once this is done, your table is completely joined as one.
