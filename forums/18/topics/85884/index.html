    Chlamydia

    Chlamydia is an *STI which is passed on during unprotected sex. The symptoms include stomach pains, irregular periods, a fever and an abnormal vaginal discharge. Often, there are no symptoms at all, but if it's left untreated, it may spread to the *fallopian tubes. If that happens, it often results in *PID, which can make you infertile.

    If chlamydia is diagnosed, it's easily treated. So if you've had unprotected sex, make an appointment with your GP (aka doctor, not General Paper) immediately. Chlamydia is on the increase amongst teenagers, so if you're in a sexual relationship, use a condom everytime you have sex.

    STI = Sexually transmitted infection (Not Singapore Trade Index)
    Fallopian tubes = A body part which carries an egg from the ovary to the womb
    PID = Pelvic inflammatory disease
