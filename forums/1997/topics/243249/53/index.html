    Originally posted by hisoka:
    port scanning means what?
    Port scanning lor...

    Port scanning can tell what ports are opened or closed, therefore, revealing the type of operating system or programs in use.

    For example, when you connect to the Net, it automatically uses port 80. It's the default port. If you have set your firewall to prompt you each time, take a closer look at the prompt. It will say XXX program attempting connection to www.whatever.com port 80.

    Port 80 is too common, so it makes no sense scanning it.

    But what if you have something like port 6346 opened? It gives clue that a P2P program is in use. Specifically Shareaza. Shareaza uses this port by default. So if you are using Shareaza, this port will be open and when Singnet does a scan, Singnet will know you are using P2P.
