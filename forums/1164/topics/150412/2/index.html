    Together with the pictures shown the website seem to imply that we have M60A3 MBTs.

    But...

    The only M60A3 vehicles we operate are AVLB - Armoured Vehicle Launched Bridge, and the CEV - Combat Engineer Vehicle.

    The AVLB is armed only with machineguns.

    The CEV has front-mounted dozer balde and a very short stubby canon - probably 105mm - for demolition of walls and other obstacles etc. Not for anti-tank role.

    These 2 versions probably make up the number 30 - probably 15 each, which is a fair figure. Is it possible that among this there are MBT versions? The answer is no.

    The existence of these 2 types of M60A3 variants are well-known facts, not rumours, speculations, etc.
