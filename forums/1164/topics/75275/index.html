    The Aegis phased array radar system is to deal with saturated missile attacks which was Soviet naval doctrine. Given their relative inaccuracy, it was hoped that a ship's defences would be swamped by missiles coming from all sides. The Aegis system works in conjunction with the vertical launcher system to provide for a more rapid response to incoming missiles.
