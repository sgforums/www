    Building planes is done in batches, or in this case block. Block can be view as are version number.

    Each block defines what configuration the F16 of that batch have, like engine, cockpit layout, incorporate later technology and weapons systems.

    The later block will alway consider more advance as it will incorporate the new technology, engines, configuration, softwares and weapons.

    The F-16 C/D comes in 4 versions. Block 25, 30/32, 40/42 and 50/52. C are single seater while D are double. RSAF is using the Block 52 F-16 C/D. The latest is the F-16 E/F Block 60.

    Go to http://www.f-16.net/f-16_versions_article10.html to see what improvements Block 60 has over its predecessors.
