    Considering that the AGM-65 is heavier than the Aim-7, weight is not an issue for F-5 integration. Also, Elbit has been integrating missiles like the Derby onto Indian air force planes eg mirage 2000H so it is not inconceivable that BVR missile capability can be incorporated. However, in order to happen, an upgrade to the launchers must happen or the aircraft must be able to be configured with the latest LAU-128/9 launcher series or the older LAU-106 series (for the Aim-7) or upgraded to -D config (for the Aim-9X). I don't think the technology is beyond ST/Elbit. But whether such integration actually happened is a state secret. The point for many observers is that if it was merely an avionics upgrade, it wouldn't have needed a HMS sight (as highlighted by others many times before). HMS sights complements off bore missiles and the current AIM-9s (excluding the -X) are far from off-bore sighted. Chile's upgrade involved the Python and possibly the Derby as well. Factor in ST eng's boast about its F-5 upgrade being the most advanced and bringing it to F-16 standard (see annual reports). Considering the closeness of the Israel-Singapore relationship and the value vs cost of such integration, I would have to seriously factor in such capability in mission planning, if I was a potential anti-SG aggressor. But if it did happen, there is also an issue with training, live firing testing etc. Having a missile without tests and training reduces the utility of the missiles. Such training can be compensated in ground simulators, ACMI (the RSAF have 4) and consistent training with the HMS which is something that the RSAF does. Further with integration of paveway II and III capability to the F-5s, one wonder if something like this could already be in inventory.... http://www.sargentfletcher.com/bus_dev/flyer_lite_pod.pdf More added capability and shooters.....
