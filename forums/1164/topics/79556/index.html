    Remote weapon system to be more cost effective will be cannon-based type rather than MG-based - because might as well use human gunner to operate the .50 MG immediately rather than via remote TV - eye-hand co-ordination is worse, quicker to bring weapon to bear if aiming directly through iron sights.

    But cannon-based is different because the typical target type (soft-skinned vehicles/shelters), engagement envelope (detect,recognise,identify,engage in same general sector over 3-4 secs) and weapon effects (AP/HE) justify the use of electronic targeting methods to improve accuracy round for round rather than pepper the enemy senselessly

    In the first place, should not use the LSV as a base reference vehicle when you want to apply the operation context like that in Iraq/Somalia/Bosnia etc where the main operating terrain is in BUA.

    LSV as it's name suggests is already meant for strike missions - surprise/stealth, speed/agility, overwhelming firepower. These attributes are in direct conflict with the level of protection that is required for vehicles used for peace containment missions in BUA with heavy concentration of hostiles just like in Iraq

    So in essence the LSV serves its job adequately if not surpassing its equivalents (compare US Humvee fitted out for the same role). The Humvee should fare better if adequately protected but then it will never become an acceptable strike vehicle in any sense. With the armour that is available today, there can never be a heavily armoured, well protected LSV - you just have to accept it.
