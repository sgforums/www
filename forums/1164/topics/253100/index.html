    My reservist 2PDF infantry unit were issued with 20rd mags ONLY for M16 users and 30rd for SAW gunners.

    We were all very curious as none of us had ever used a 20rd mag before. Even the brand new webbing issued to us were made just for the 20rd mags. This webbing is the IDF-type and is still in current use (pending replacement by the vest). Although yours would hold 30rd mags.

    I found this to be really puzzling cos if someone handed you a bunch of 30rd mags you would have no way to store them as the pouches are shallow.

    I don't know if this is because SAF felt we do defence only so 20rd is better in a sandbagged trench or something. But we only did one defence ex and all the other time we did FIBUA and FOFO eventually trial for V200 mounted.
