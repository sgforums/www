    First time synchronization

    First time synchronization

    When synchronizing a Nokia phone and a PC for the first time, you may want to use the following instructions. To be able to use Nokia PC Sync, you must have a Nokia phone connected to the PC. The phone must be in standby mode.

    1. Connect the phone to the PC.

    2. Open Nokia PC Sync.

    3. In the Nokia PC Sync dialog, click Synchronize Now. The Create Settings wizard opens.

    1. On the first page of the wizard, select the PIM &lt;!-- FilePopupInit('a1'); //--&gt; application you want to synchronize the phone with from the drop-down list. Follow the instructions on screen to finish the wizard process.

      If you choose to skip the wizard process, a set of default settings will be used in the synchronization. It is recommended that you follow the wizard to get your personal settings right for the synchronization.

    2. On the What To Synchronize page, you can specify what information to include in the synchronization.

    3. On the Synchronization Range page, you can specify the settings for synchronizing Calendar/To-do items. For more information, see Calendar/To-do. If you did not select Calendar / To-do Items on the What To Synchronize page, this page will not be shown.

    4. On the E-mail page, you can specify settings for synchronizing e-mails. For more information, see E-mail. If you did not select E-mail Items on the What To Synchronize page, this page will not be shown.

    5. On the Bookmarks page, you can specify the settings for synchronizing Bookmark Items or Bookmarks and RSS Feeds depending on the phone model. For more information, see Bookmarks. If you did not select Bookmark Items or Bookmarks and RSS Feeds on the What To Synchronize page, this page will not be shown.

    6. On the File Synchronization Settings page, you can specify the settings for synchronizing File and Folder Items. For more information, see Files and Folders. If you did not select File and Folder Items on the What To Synchronize page, this page will not be shown.

    7. To view and change the settings for automatic synchronization and conflict resolution, see the advanced settings used in the synchronization by clicking Advanced...
       

    1. Click Finish on the last page to run the synchronization. If you do not want to start the synchronization, clear the selection in the check box at the bottom of the page.

     

    Following synchronizations

    • If you want to run the following synchronizations with the same settings, follow the instructions in the topic Running a synchronization.

    • If you want to change general synchronization settings, see Defining general settings.

    • If you want to change the settings for the PIM you have been using, see Defining PC application specific settings.

    • If you want to change the PIM, see Creating a new synchronization task.
