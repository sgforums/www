    Yes, I assume you're a beginner, so you can start with this workout.

    This is known as Rippetoe's workout. Do it 3 times a week, remember to feed your body so that it grows. You'll notice yourself packing on mass if you're doing it religiously and eating and sleeping well. If you don't know how to do the exercises you can always google them. If still not sure, you can ask us here. :)

    Week 1:

     

    • Monday - Workout A
    • Wednesday -Workout B
    • Friday - Workout A
      •

    Week 2:

     

    • Monday - Workout B
    • Wednesday - Workout A
    • Friday - Workout B
      •

    Workout A:

       

    • 3x5 Squat
    • 3x5 Bench Press
    • 1x5 Deadlifts
    • **2x8 Dips (if you cant do these or no assist machine then do Decline Dumbbell Bench Press with your hands Facing each other)
      •

    Workout B:

       

    • 3x5 Squat
    • 3x5 Standing military press
    • 3x5 Bent Rows (or power cleans-> ignore this for now)
    • **2x8 Chin-ups (recommended mainly if doing the cleans)
      •

      Note: This doesn't include warm-up sets (**)Means this is OPTIONAL
