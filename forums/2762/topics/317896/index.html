    I normally replace my running shoes every 1 to 1.5 years, because I feel that the rubber compound in the shoe changes it's properties after that time. Sure enough this year I used my running shoes for 2 years, injured my ankles, I think it's because of the changes in the rubber compound in the shoes.

    I normally run with Asics Kayano series, but this year I was adventurous and slightly richer and decided to try Asics Kinsei 2. After using my new shoes for 2 runs, I find that I can run better. Perhaps it's psychological, I don't know, but it sure feels like the shoes are absorbing the shocks from the roads much better.
