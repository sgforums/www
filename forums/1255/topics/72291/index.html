    yeah bismarck is right... if u want to print 4R the ratio should be in 3:2 since the actual dimensions is 6"x4"

    if u want different R sizes then they'll be differet ratios.

    ur digicam probably takes in 4:3 format (most digicams do), so when they put it on 4R, they'll chop the top and bottom. if the person at the photo lap has a little brains, they'll choose how much of the top and how much of the bottom to crop, not necessarily in the centre. the best is to save the format urself, so no cropping need be done.

    i always give my digital photos on CD in 1800x1200 format. (when i print 4R or S8R - both has 3:2 ratio) for 4R this works out to be 300dpi... more than enuff resolution. if i need to print S8R i would send in at 3000x2000 pixels. my cam is about that resolution (6mp)

    yo thanx for ur comments, yes i'm using a digital camera

    don't worree about the original resolution when u first open it up in photoshop. wat u can do to crop it to 3:2 ratio is use the crop tool (hotkey c) and insert the values on the bar on top. for me i put in 1800 width and 1200 height. this way, no matter how big or small u crop, the crop will be contraint to that number of pixels, then u can save the file and send it for printing.

    dun worree about wat dpi setting it is. if u have 1800 pixels and tell the shop u wanna print 4r, the full 1800 pixels will be used to fill 6", which means u ahve 1800pixels in 6" which equates to 300dpi.
