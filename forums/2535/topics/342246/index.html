    The first comes from XiaoFeiChong. According to the forums, this is a recompiled M33 firmware. The most interesting thing about this release, however, is that it does not support backups. No ISOs, no CSOs, no nothing. If you want to run CFW but don't want the lure of piracy,you may consider giving this a try.

    The next one is from a group calling themselves Team Omega. Again, it's 5.02, but it still has support for backups. There's also some "new support for PSX games" mentioned on the forum post along with bug warnings for the pops, but the explanation is a bit hard to wade through. If you update to this, be aware that a lot of your homebrew and plugins will end up broken, as is usually the case with firmware updates.
