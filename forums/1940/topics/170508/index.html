    MSI K8N Diamond Plus Technical Specifications :

    CPU Support Supports AMD socket 939 Athlon 64 CPU
    Supports AMD socket 939 Athlon 64 FX CPU
    Supports AMD socket 939 Athlon X2 CPU
    Supports AMD Cool 'n' Quiet Technology

    Chipset NVIDIA nForce4 SLI X16 SPP
    Bi-directional 1GHz (2000MT/s) HyperTransport interface to processor and MCP
    18 lanes flexible PCI Express configuration
    NVIDIA nForce4 SLI X16 MCP
    Bi-directional 1GHz (2000MT/s) HyperTransport interface to SPP
    20 lanes flexible PCI Express configuration
    NVIDIA SLI Dual GPU architecture
    Supports up to 6 PCI v2.3 slots
    Supports up to ten USB 2.0 ports
    Dual controller SATA II architecture
    Dual-channel Ultra ATA-133 IDE controller
    IEEE 802.3 NVIDIA MAC for 1000BASE-T

    Memory Dual channel memory architecture integrated into processor
    Four 184-pin DDR DIMM sockets
    Supports 2.5v DDR SDRAM DIMM modules
    Supports up to 4GB using PC1600 (DDR200), PC2100 (DDR266), PC2700 (DDR333) and PC3200 (DDR400) unbuffered DDR SDRAM DIMM

    Storage NVIDIA nForce4 SLI X16 MCP
    4 x Ultra ATA 33/66/100/133 (data transfer rate up to 133MB/sec.)
    4 x SATA II (Serial ATA II 3.0Gbps interface)
    NVIDIA MediaShield Technology
    Supports RAID 0, RAID 1, RAID 0+1
    Supports RAID Morphing
    Supports cross-controller RAID

    Audio Onboard Creative Sound Blaster Audigy SE
    24-bit / 192KHz sampling rate
    106dB SNR clarity
    Up to 7.1 channel surround sound
    Coaxial, Optical S/PDIF out on back I/O port

    Networking NVIDIA Gigabit LAN co-supported by Marvell Alaska 88E1115 PHY chip (Transceiver)
    Marvell Yukon 88E8053 PCI Express Gigabit LAN

    IEEE 1394 (FireWire) VIA Fire II VT6306 Host Controller
    3 x 1394a ports @ 400 Mbps speed

    I/O Interface 1 x Floppy connector
    2 x Ultra ATA IDE connectors
    6 x SATA connectors
    10 x USB 2.0 ports (4 x rear, 2 x bracket, 4 x header)
    3 x IEEE 1394a ports (1 x rear, 2 x header)
    2 x RJ45 LAN ports
    1 x Coaxial S/PDIF Output
    1 x Optical S/PDIF Output
    8-Channel Audio I/O ports
    1 x CD-IN
    1 x 9-pin serial port
    1 x parallel port (SPP/EPP/ECP)
    1 x mini-DIN-6 PS/2 mouse port
    1 x mini-DIN-6 PS/2 keyboard port

    Expansion Slots 2 x PCIe x16 slots, PCI Express Bus specification v1.0a compliant
    1 x PCIe x2 slot, PCI Express Bus specification v1.0a compliant
    2 x PCIe x1 slots, PCI Express Bus specification v1.0a compliant
    2 x 32-bit PCI slots, PCI 2.3 compliant

    Special Features D.O.T. 3 (Dynamic Overclocking Technology)
    CoreCell
    Live Update 3

    PCB ATX Form Factor, 12"x 9.6"(30.5cm x 24.4cm)
