    Firefox addons I like

    Flashblock - Blocks Flash. Very neat addon for those websites who uses lots of Flash as advertisements that can't be blocked. It's also good for telling a phishing site, since they have started using Flash to trick people into believing that it's a real site.

    Adblock Plus - Blocks advertisements and irritating pictures. Some Flash can be blocked as well.

    Blocksite - Works like IE restricted zones. This allows me to do lots of testing on a bad website without getting harmed myself (when I accidentally click on links)

    NoScript - Blocks all sorts of scripts from running, except those in whitelist. This is also useful while I play around with bad websites.

    ShowIP - Tells me if a site is bad.

    CookieSafe - Handles cookies. Found out by chance that Mozilla developers have tweaked the way Firefox handles cookies, which I don't like. CookieSafe helps me to handle cookies in a better way.

    IETab - Switch to IE while using Firefox. For testing of websites and running Windows Update.

    JSView - View JavaScript and CSS codes. Lazy to write codes people are like that. View their source codes using this addon, copy and paste the codes and modify it to suit myself. Only works with client-side scripts. Server-side scripts like PHP will not work.

    ViewSourceWith - View the HTML source with my favourite editor or choose other editors when one editor doesn't work.
