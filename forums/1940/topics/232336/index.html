WinRAR 3.71 Beta 2

  • GSL
    18 Jan 07, 15:36
    WinRAR is a powerful archive manager. It can backup your data and reduce size of email attachments, decompress RAR, ZIP and other files downloaded from Internet and create new archives in RAR and ZIP file format.
    Archive file formats supported are ZIP, RAR, ACE, CAB, ARJ, LZH, TAR, GZ, BZ2, UUE, JAR, ISO, 7Z and Z.



    What's new in this version:
    - Positions of "Mode" and "Cancel" buttons in archiving dialog
    have been exchanged.

    Bugs corrected:
    - beta 1 could truncate names of files decompressed from ISO 9660 archives by one character;
    - progress bars in archive conversion window did not work correctly
    with "Windows progress bars" option turned on;
    - "Load WinRAR settings from" dialog did not displayed if
    settings.reg was not present in %appdata%\WinRAR folder.
