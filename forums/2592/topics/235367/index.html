    shizuka = (quiet)

    chiisai = (small, little)

    atarashii = (new)

    furui = (old)

    ii = good

    warui = (bad)

    atsui = (hot)
    note** it can be use for weather or you drink hot stuff.

    samui = (cold) **weather**

    tsumetai = (Cold) **things**

    muzukashii = (difficult)

    yasashii = (easy)

    takai = (expensive , high )

    hikui = (low)
