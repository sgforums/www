    For PS2 owners, please be careful when buying original games from shops. Most shops like comics connection carry NTSC/J and NTSC U/C games. If you buy the wrong version, your PS2 will not run the game.

    How to check your set for its version/region. On your PS2 box flap there should be either "NTSC U/C", "NTSC/J" or "PAL" wording on it. That is the region that your set came from.

    NTSC U/C set is commonly known as US Set
    NTSC/J set is commonly known as Asian Set
    Pal set is commonly known as European set

    This wordings will also appear on the top right hand corner of the box of the game.
