    open a new game room and make a password for it so that only you can enter

    leave you character in the room for at least 5 mins.

    After that is done,the game room will be saved and still available for a char to enter it for the next 5 mins.

    Drop all the stuff you want to transfer onto the floor.Log off this char,loggin new char and into the previously made game room,pick up items.
