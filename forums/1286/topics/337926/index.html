It is very family theme based and has a number of games able to play 4 players at once . One such game was Raving Rabbit series . I own the Rayman Rabbit part 2 and basically it is a mini game based game where you have to compete in some mini games to unlock more mini games if you are playing it for the first time . I would recomend this console if you are looking for some family based games as the games are more general as compared to PS3 which have quite a number of serious type gaming for adult .The WII doesn't have any built in harddisk but is able to save games by using some flash memory thingy i supposed but one good thing abt it is you can insert or use SD card to backup your saved games . . ( But of cos theres a limit much smaller then PS3 )And last but not least , the highly anticipated Monster Hunter 3 is coming to the WII soon .
