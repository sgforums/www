    Game commands:

    Here is a table of the rate settings recommended for use with their
    corresponding connections:

    14.4 Modem Get a new modem
    28.8 Modem 2500 to 3000
    33.6 Modem 2900 to 3900
    56k Modem/1 Channel ISDN 3600 to 5300
    2 Channel ISDN 5000 to 7000
    Cable Modem 5600 to 10000
    xDSL, T1+ 7500 to 20000
    LAN (10MBps or 100MBps) 20000

    Source: Tweak 3d

    I recommend that when playing ANY game online that ALL of the games
    detail settings are on low, and this will help ping as well.

    The following commands only work in Half-Life, and not other games such
    as Quake. I recommend that you all download the FULL Half-Life patch and
    the FULL Counter-Strike. Uninstall Half-Life and then re-install again, and
    use the FULL patches.

    I personally use the following settings. Just open up your config.cfg file in
    notepad.

    This command sets up the rate at which your modem sends and receives
    data:

    rate "5300"

    Enable MMX with the following command if your CPU supports it:

    r_mmx "1"

    These commands affect Half-Life's visual performance, and in some cases
    improve ping:

    r_decals "300"
    r_dynamic "1"
    r_drawviewmodel "1"
    r_shadows "0"
    gl_texsort "0"
    gl_texturemode "GL_LINEAR_MIPMAP_LINEAR"
    precache "1"
    fps_max "30"
    cl_himodels "0"
    gl_polyoffset "20"

    These commands turn joystick support off and mouse filter on, for
    smoother mouse movement:

    joystick "0"
    m_filter "1"

    This sets how fast the console comes down:

    scr_conspeed "3000"

    These commands turn of view bobbing when running, and improves most
    peoples aim (NOTE: This is NOT cheating):

    cl_bob "0"
    cl_bobup "0"
    cl_bobcycle "0"

    In Counter-Strike (CS Only) set the following value to 1 if your machine
    slows down when rendering a smoke bomb. You could also try setting it
    to 2 for an even further performance boost:

    fastsprites "0"

    The value of the pushlatency command should be:

    -3/4 of your ping

    So if you have an average ping of 180, you'd set it to:

    -90

    These commands set up your player prediction:

    cl_nopred "0"
    cl_predict_players "1"
    cl_resend "2"

    pushlatency "-35"

    These commands turn downloading and uploading of player logos off. This
    improves ping considerably:

    cl_allowdownload "0"
    cl_allowupload "0"
    cl_download_ingame "0"

    These commands set up your sound ingame. If your running a sound
    blaster Live card or something that supports EAX, set the EAX value to 1.
    I'd recommend leaving reverb on for all sound cards (So you can hear
    echoes):

    s_a3d "0"
    s_eax "1"
    s_reverb "1"

    This command turns the netgraph on:

    net_graph "1"

    There will now be a graph at the bottom of your screen - this is a net
    graph. It shows your connection and how good/bad it is, similar to the
    graph in Quake 3. When you're not monitoring your internet connection,
    to improve your FPS use this command instead:

    net_graph "3"

    If you're on a Dial Up Modem (e.g a 56k) replace X in the following line
    with 20. If you're on ISDN or faster replace X with 30. Once you've
    replaced X with 20 or 30 add the line to your config file.

    cl_updaterate "20"

    For modem users, set the following to 30. If you're using ISDN or higher,
    set the value to 40:

    cl_cmdrate "30"

    These commands set how long text messages are displayed:

    hud_saytext_time "20"
    hud_deathnotice_time "10"

    This command enables fast switching. For example, in Counter-Strike, with
    this command enabled all you need to do to switch to the knife is press 3:

    hud_fastswitch "1"

    These commands are, in my opinion, the most important to add. These
    change some of the settings so that the client side controls them. These
    settings default to on. To change them to server side (like how the old
    netcode worked) change them to 0:

    cl_lc "1"
    cl_lw "1"
    cl_lb "1"

    To get some cool affects that may mess up your framerate/ping, type the
    following into the console (Counter-Strike only):

    gunsmoke

    So, once I've set all of my commands up I add the following to my config
    file:

    rate "5300"
    r_mmx "1"
    r_decals "300"
    r_dynamic "1"
    r_drawviewmodel "1"
    r_shadows "0"
    gl_texsort "0"
    gl_texturemode "GL_LINEAR_MIPMAP_LINEAR"
    precache "1"
    fps_max "30"
    cl_himodels "0"
    gl_polyoffset "20"
    joystick "0"
    m_filter "1"
    scr_conspeed "3000"
    cl_bob "0"
    cl_bobup "0"
    cl_bobcycle "0"
    fastsprites "0"
    cl_nopred "0"
    cl_predict_players "1"
    cl_resend "2"
    cl_allowdownload "0"
    cl_allowupload "0"
    cl_download_ingame "0"
    pushlatency "-135"
    s_a3d "0"
    s_eax "1"
    s_reverb "1"
    net_graph "3"
    cl_updaterate "20"
    cl_cmdrate "30"
    hud_saytext_time "20"
    hud_deathnotice_time "10"
    hud_fastswitch "1"
    cl_lc "1"
    cl_lw "1"
    cl_lb "1"

    These commands only work in Unreal Tournament. For better internet
    play, select 16 bit colours and low on both skin and world detail in
    preferences. Also, turn view bobbing off. The netspeed command alters
    the rate at which your modem sends/receives data. Press the tab key
    and then enter:

    netspeed 5300
    netstat

    These commands only work in Quake 3 Arena. For the best internet play,
    in the video options set everything onto low. The rate command alters
    the rate at which your modem sends/receives data. Bring down the
    console and enter these commands, or add them onto the bottom of your
    config file:

    The following command alters the rate at which your modem
    sends/receives data:

    rate "5300"

    This command alters your maximum FPS:

    com_maxfps "30"

    The value of the cl_timenudge command should be:

    -3/4 of your ping

    So if you have an average ping of 180, you'd set it to:

    -135

    These commands set up your connection:

    seta snaps "30"
    seta cl_maxpackets "30"
    seta cl_packetdup "1"
    cg_nopredict "0"
    seta cl_timenudge "-135"
    cg_deferPlayers "0"

    This command enables the netgraph:

    cg_lagometer "1"

    So once I've set all of my settings up, I add the following to my config:

    rate "5300"
    com_maxfps "30"
    seta snaps "30"
    seta cl_maxpackets "30"
    seta cl_packetdup "1"
    cg_nopredict "0"
    seta cl_timenudge "-135"
    cg_deferPlayers "0"
    cg_lagometer "1"

    Install the Quake 3 Arena 117 dll's for a performance boost. When looking
    for servers, use Arena Spy.

    These commands only work in Quake 2 and Soldier Of fortune:

    The following command alters the rate at which your modem
    sends/receives data:

    set rate "5300"

    This command sets your maximum FPS:

    set cl_maxfps "30"

    This command turns the netgraph on:

    set netgraph "1"

    So once I've set all of my settings up I add the following to my config file:

    set rate "5300"
    set cl_maxfps "30"
    set netgraph "1"
