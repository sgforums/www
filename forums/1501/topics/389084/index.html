    ShowNearby, a IT company is looking for Singaporeans to fill the following positions with the salary range on offer behind:

    1. Web Evangelist (Developers) :: $1.8k-4.5k

    2. Web Designers :: $1.8k-3.5k

    3. Business Development Manager :: $2.5k-4k

    4. Sales Manager :: $2.5k-4k

    5.Marketing Manager :: $2.5k-4k

    Working hours are from 930am to 630pm

    Qualifications expected:

    1.Web Evangelist (Developers) :: Any. Must be good in PHP, Ajax, MySQL, JavaScript and has applied OO Programming before.

    2. Web Designers :: Any. Portfolio more important. Must reflect ability to create good user flow, experience and interfaces.

    3.Business Development Manager :: Any. Wide customer contact base in the retail industry is a must. Min 2 yrs experience.

    4. Sales Manager :: Any. Hunger is the best qualification.

    5. Marketing Manager :: Experience in internet marketing and social media marketing for at least 2 yrs.
