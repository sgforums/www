    VB.Net/SharePoint Developer

     

     

    Job Descriptions

    -          Creating high quality Visual Basic.Net programs from written specifications

    -          Design sessions or verbal instructions within specified timeframes and budgets;

    -          Participate in design sessions of new software programs and applications; 

    -          Documentation of software programs and applications;

    -          Ensuring the highest quality programs and reports by unit testing and other methods; using best practice methods;

    -          Providing input on how to improve business processes and systems.

     

     

    Requirements:

    -          Bachelor’s Degree in Computer Science or equivalent technical degree/experience required

    -          At least 4 years experience developing web applications

    -          Capture requirements, design and develop new web applications and enhance/revise existing web applications.

    -          Strong knowledge/experience with the following development tools and technologies:                                                               ASP.NET, ADO.NET, HTML, XML, AJAX, VB.NET, MOSS, Oracle, SQL Server, web services

    -          Knowledge and experience object oriented programming techniques

    -          Strong analytical and communication skills.

    -          Demonstrable understanding of application administration

    -          Experience with formal software development lifecycle methodology

    -          Good project and time management skills

    -          Prepare software testing & engineering documentation

    -          Enjoy working with others in a fast paced team atmosphere

    -          Have an innovative programming approach and a passion for exploring new ideas

    -          Strive to enhance code in order to meet changing customer and industry needs

    -          Ability to handle several projects at any given stage
