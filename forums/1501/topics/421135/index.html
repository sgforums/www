    Technical Support Engineer

    Job Description:
    • Troubleshoot and follow up with user problems
    • Conduct computer health check (apply all latest security patches, upgrade to latest revision of software, ensure anti-virus signature is updated and ensure Mobile Backup is performed etc) for users
    • Conduct Blackberry/Apple device health check (upgrade of software/firmware)
    • Rollout new IT solutions/services to user community (for example Windows 7 etc )
    • Support conferencing solutions used by users (Audio Conference and Video Conference)
     
    Work Scope:
    • To provide 3rd level support for Singapore IT Operations
    • To conduct IT training sessions to Singapore office users
    • To provide technical advice to end users on technical solutions and hardware recommendations based on corporate guidelines and standards
    • To troubleshoot Blue Screen of Death for Microsoft operating system
    • To test new software package to ensure that the software package work in Singapore for users
     
    Job Requirements:
    • Minimum Diploma in IT or related
    • At least 2-3 years of relevant experience
    • Good technical knowledge in Laptop and PC hardware
    • Good technical knowledge in Microsoft Windows XP, Windows Vista and Windows 7
    • Good technical knowledge in Microsoft Word, Excel (Pivot table, Macro and advanced formula) and PowerPoint
    • Good technical knowledge in Apple MAC OS
    • Good technical knowledge in SmartPhones (Blackberry devices, Apple iPAD, Apple iPhones and Windows Mobile devices etc)
    • Good technical knowledge in Printers and Multi Function Printer (for scanning, printing and copying etc)
    • Good technical knowledge on the wired LAN, wireless, VPN and 3G data services

    Remuneration: Gross monthly: $2500-3000
