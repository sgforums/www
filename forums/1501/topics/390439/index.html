    IT Network / Server Engineer

    Main duties / Responsibilities:

    ·         Assist in the installation and troubleshooting of IT hardware & software

    ·         Troubleshoot and configure network related issues

    ·         Creation & Maintenance of IT hardware & software licensing

    ·         Perform company’s data backup in daily, weekly & yearly basis.

    ·         Manage, plan, design, deploy and support of network technology

    ·         Administration and maintenance of servers and LAN & WAN operations

    ·         Active monitoring of system and network performance

    ·         Manage daily off-site backup, restoration and recovery operations

    ·         Provide troubleshooting and system configuration

    ·         Provide user support 

    ·         Provide support on software/hardware installation

    ·         Other ad-hoc duties when assigned

    Requirements:

    ·         Skills Required: Win XP, Win 2K, Win Vista. Basic networking, dreamweaver

    ·         Preferably Junior executive specializing in IT/Computer

    ·         Diploma in Computer Science or IT related field

    ·         At least 1-2 years IT technical support experience in supporting user and server maintenance

    ·         Must be able to work independently

    ·         Position is opened for Singaporean/ PRs

    ·         Prefer someone who can start work immediately /within short notice

    ·         Good interpersonal and communication skills
