    If you are granted a student pass and you wish to find/apply for a job in Singapore, you will need to apply for a work pass. This pass can be apply from the Ministry of Manpower website. If you are caught working in Singapore without a work pass, you will be deported back to your country.

    Empolyers in Singapore tends not to hire student pass holder because not all student pass holder holds work pass permits. If the student is caught, both the student and the company will be in trouble. However, student pass holder is very attractive to empolyers because of the low starting pay.

    => Apply for a work pass permit @ Ministry of Manpower website. You will be able to find a job in Singapore.
