    Title: Software / SQA Engineer

    Job Summary:
    Individual contributor who works with Developers and SQA team to ensure successful, timely and good quality product release.

    Skills Required:
        Strong knowledge in software development life cycle and SQA methodology
        Extensive experience with white box testing in Java for both unit and API testing
        Working knowledge of JUnit and/or TestNG
        Good knowledge of Java, XML, HTTP web services, SQL, C
        Ability to install and configure MS SQL Server and Oracle
        Working knowledge of Subversion and Eclipse
        Experience in functional test automation especially with Selenium
        Familiarity with the Hudson continuous integration server
        Knowledge of Apache Ant build making tool
        Good communication and presentation skills

    Skills Desired:
        Experience with the code coverage tools
        Familiarity with Open Source test automation tools
        Familiarity with the TestLink test management system
        Experience with JMeter performance testing tool
        Familiarity with Mantis defect tracking system

    Responsibilities:
        Implement functional test automation framework in DocuShare Business Unit
        Perform white box testing in Java
        Automate functional testing in Selenium
        Design and document test cases in the TestLink
        Execute manual and automated tests and report test results into the defect tracking system
        Participate in implementation of continuous integration with Hudson, Subversion, Ant, and Selenium
        Conduct root cause analysis of the product defects

    Education/Experience:
        BS or MS in Computer Science or related technical discipline, and more than 4 years of working experience in software engineering / test automation.
