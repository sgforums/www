    3rd base is not forced to run in this case, so no questions there. The main consideration is 1st base.

    If infielder purposely drop a potential fly-ball, then 1st base will be tricked not to run. Two scenarios:
    a) he throw 2, 1st base out.
    b) he throw 1, batter out.

    Hence, potentially double-play. Hence in-field fly will apply here

    Infield fly rule only applies when
    - at least 1st and 2nd base are loaded, with less than 2 outs.

    Thus if only 1st base is loaded, or 1st and 3rd loaded, infield fly rule does not apply. This is because when only 1st base is loaded, if the infielder purposelly drop the high pop fly, the batter would be assumed to have reached 1st base already and thus the double-play is avoided as the fielder can only play the runner out at second.
