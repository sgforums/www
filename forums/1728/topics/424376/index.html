    Here's wat i could think of for now.

    1. At google homepage, click the advance search option.

    2. Enter the chinese sutra name (stating the obvious, u could copy and paste there) at the 'find web pages that have' form.

    3. Scroll down, at the language option, instead of using default 'any languages' choose 'English'.

    4. Click the advance search button.

    5. Browse thru the return result page. The english name of the sutra might be somewhere in the result page.
