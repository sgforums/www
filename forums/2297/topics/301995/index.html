    First, if you could not solve such questions or cannot see where it is going, you could actually try to write it out like the following

    f(2) = f(2/2002) + 3 (2)
    f(2/2002) = f(2/2002^2) + 3 (2/2002)
    f(2/2004) = f(2/2002^3) + 3 (2/2002^2)
    and so on

    You will find that it will end up something similar to

    3(2) + 3 (2/2002) + 3 (2/2002^2) + ....

    Similar to a GP
    first term 6
    rth term 1/2002
    and Sum to Infinity

    Thus,
    f(2) = a / (1 - r) = 6.003
