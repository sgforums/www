    experiment...
    say need voltmeter, ammeter, and either a variable voltage supplier or a fixed voltage supply with a potentiometer to vary the input voltage

    Use a few voltages (>6), and record the corresponding current values
    Plot graph

    sketch graph
    Think a bit. power increase -> resistance increase for lamp
    R = V/I
    so u know V increase more proportionately than I

    Hence, if you plot V-I graph, graph will curve upwards
    if you plotted I-V graph, graph will curve downwards
