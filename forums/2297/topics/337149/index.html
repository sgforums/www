    Question 7:

                   2NO2 (g)  <=> 2NO (g) + O2 (g)

    Initial:            2                     0             0

    Final           2 - x                   x            x/2

    (2 - x + x + x/2) / 2 = 1.2

    2 + x/2 = 2.4

    x/2 = 0.4

    Therefore ans = D.
