    A : B = 1 : 3 = 2 : 6

    B : C = 2 : 3 = 6 : 9

     

    D = C / B x A = 9 / 6 x 2 = 3

    A + B + C + D = 2 + 6 + 9 + 3 = 20

    3 u -> 30

    20 u -> 20 / 3 x 30 = 200 cm2

     

    The area of rectangle WXYZ is 200 cm2
