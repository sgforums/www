    To evaluate, use your calculator
    ½log2(5) + log4(0.4) + log½(25) = -4.144

     

    If it is to simplify:

    ½log2(5) + log4(0.4) + log½(25)

    = ½log2(5) + log4(2/5) + log½(5²)

    = ½log2(5) + [log2(2/5)] / log2(4)+ [log2(5²)] / log2(½)

    = ½log2(5) + [log2(2) - log2(5)] / 2 + 2log2(5) / (-1)

    = ½log2(5) + ½log2(2) - ½log2(5) - 2log2(5)

    = ½log2(2) - 2log2(5)

    = ½  - 2log2(5)
