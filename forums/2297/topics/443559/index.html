    Hi,

      Starting with 8^(2x -1) = (x + 1)^(2x - 1)

      => (2x - 1) ln 8 = (2x - 1) ln (x + 1)

      => (2x - 1) ln (8/(x + 1)) = 0

      => 2x - 1 = 0 or 8/(x + 1) = 1

      => x = 1/2 or x = 7.
