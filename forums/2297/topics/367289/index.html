    Hi Jt061952,

      Let y = ln [x + sqrt(x^2 - 1)].

      Then dy/dx = [ 1 + (2x)/sqrt(x^2 - 1) ] / [ x + sqrt(x^2 - 1) ], which simplifies to your said answer.

      In general, dy/dx = f'(x) / f(x) if y = ln f(x). Thanks!

    Cheers,
    Wen Shih
