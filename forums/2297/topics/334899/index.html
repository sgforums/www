    H = 4/5(220-n)
    H = 176 - 4/5n
    4/5n = 176 - H
    n = (176 - H) / (4/5)
    n = (176 - H)(5/4)
    n = 220 - (5/4)H

    I think you got the answer but remember your brackets are important. :)

    Maith, got technique to check one. And very quick.

    Sub in values. For example,

    H=4/5(220-n)
    Sub n = 10,

    H = 0.8 * 210 = 168

    n = 220 - (5/4)H

    Sub H = 168,
    n = 220 - 210
    n = 10

    Since you get back n = 10, it means it is correct.
