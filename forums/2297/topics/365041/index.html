    Belt B l____l____l

    Belt A l____l____l____l____l__144__l   ( first scenario )

    Belt A l____l________432_________l    ( second scenario )

    The above model is made possible only after you have realised one thing, Paul's egg picking speed was twice of Peter in the first scenario, and half of Peter's egg picking speed in the second scenario.

    3 units + 144 = 432

    = > 1 unit =96

    Belt A = 1 unit + 432 = 96 + 432 = 528 eggs

    Belt B = 2 units = 96 X 2 = 192 eggs
