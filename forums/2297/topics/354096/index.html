    Hi, welcome to Homework Forum for your first post.

     

    a) cost of 50 units = $24 + $0.25 * 50 = $36.50

    b) fixed charge = $24

    $44-$24 = $20

    Number of units used = $20 / $0.25 = 80 units

    c) $A = $(24 + 0.25n)
