    Hi,

      From x2 - 4x + 2 = 0, you can find α + β and αβ yes?

      To form the new equation, you need to find

      1. sum of roots = (α2 + β2) + 2αβ = (α + β)2 yes?

      2. product of roots = (α2 + β2)(2αβ) = { (α + β)2 - 2αβ } (2αβ) yes?
