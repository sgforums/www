    When a teacher sets this type of question without giving a clue for the students, it will be difficult to get the relationship between the x and y for the students.

    TS has not given an obvious relationship between x and y for the question 

    However, with the help of the calculator Casio Fx 95SG Plus, we can find the relationship between x and y.

    Use the equation mode to calculate the x answers and the y answers and form the relationship between x and y.

    Next, use this relationship to solve for the y answers.
