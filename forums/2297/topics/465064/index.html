    Hi,

      1. Press "APPS".

      2. Select "PlySmlt2".

      3. Select "SIMULT EQN SOLVER".

      4. Select 2 for "EQUATIONS", select 3 for "UNKNOWNS", press "GRAPH".

      5. Enter the values in the matrix.

      6. Using your values, we obtain, from GC, x = -6 + z, y = -5 + 6z, where z is a real number.

    Hi,

      For the function y =(-x^2 -x +1)/( x^2), we can just simplify directly:

      y = (-x^2)/(x^2) - x/(x^2) + 1/(x^2)

         = -1 - (1/x) + 1/(x^2).

      Note that the curve has asymptotes y = -1 and x = 0.

    The new Casio graphic calculator fx cg 20 has solved the problem.

    It can graph the function (-x^2 - x + 1) / x^2 without expressing it into partial fractions. In addition, the function can be keyed as a normal fraction form into fx cg 20 just like the other Casio calculator eg fx 96 sg plus

    The calculator can calculate in surd form too ie 2 sqrt 3 x 3 sqrt 5 = 6 sqrt 15
