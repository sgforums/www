    sin 2x = sin (x - 4)
    2x = x-4, π - (x-4), 2π + (x-4), 3π - (x-4), 4π + (x-4), 5π - (x-4) => Generate round the quadrant
    x = -4, (π+4)/3, 2π-4, (3π+4)/3, 4π-4, (5π+4)/3
    x = -4, 2.38, 2.28, 4.47, 8.57, 6.57

    since 0 ≤ x ≤ 5,
    x = 2.28, 2.38, 4.47 (do check for careless mistakes)
