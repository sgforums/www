    Originally posted by SBS n SMRT:

    Hi all, can methyl - enols shows positive iodoform test?

    If the methyl group is adjacent to the sp2 hybridized C atom of the alkene, then no, because the molecule tautomerizes to an ethyl ketone, not methyl ketone.

    If the methyl group is adjacent to the sp3 hybridized C atom with the hydroxyl group, then yes, because the molecule tautomerizes to a methyl ketone.
