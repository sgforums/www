    You went wrong on your first part

    "2x ln xe = -x ln 30e"

    That's not how logarthims work.

     

    Ans:

    (xe^x)^2 = 30e^-x
    2 ln(xe^x) = ln (30e^-x)
    2 ln x +2 ln (e^x) = ln 30 + ln (e^-x)
    2 ln x + 2x = ln 30 - x

    so
    2 ln x = -3x + ln 30
    ln x = -1.5x + 0.5 ln 30

    so a = -1.5, b = 0.5 ln30 = 1.7
