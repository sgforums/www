    Q1  Solve the inequality |x + 3a| > 2|x - 2a|, where a is a positive constant. Hence deduce the solution to the inequality |sin x + 3/2| > 2|sin x - 1|.

    Q2  A set of 20 pairs of bivariate data (x, y) is summarised by

    sum x = 200, sum x^2 = 2125, sum y = 240, sum y^2 = 8245.

    The product moment correlation coefficient is -0.992.

    Find the equation of the regression line of y on x.

    Hi,

      Here are some interesting trigo-related problems. Solve them without the use of a calculator.

      1. Find the value of (sin pi/8)^4 + (sin 3pi/8)^4 + (sin 5pi/8)^4 + (sin 7pi/8)^4.

      2. Find the value of [ (tan 40 deg) x (tan 60 deg) x (tan 80 deg) ] / [ (tan 40 deg) + (tan 60 deg) + (tan 80 deg) ].

      3. Let m = sqrt(2). Find the value of [ log_m (cos 20 deg) + log_m (cos 40 deg) + log_m (cos 80 deg) ]^2. Note: log_m k refers to the logarithm of k with base m.

      4. If cos 2A = -sqrt(5) / 3, find the value of 6 (sin A)^6 + 6 (cos A)^6.

      Enjoy!
