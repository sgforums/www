    If you're talking about topics, there isn't much change actually except that in emaths now there's vectors,sets and matrices which were previously in amaths (correct me if I'm wrong).

    As for amaths, there are new topics like partial fractions, angle properties of circles (intercept theorem etc.), higher level of trigonometry & differentiation. (Sorry but me myself ain't sure which topics are taken from which, I just follow what the teacher teaches :P)

    But the main difference in emaths I can see is only that now both Paper 1 and 2 we can use calculators, the format of questions seem to be the same.

    As for amaths, there is no more either or question, meaning all questions are compulsory. Unfortunately, it means that if the last 10 mark question is a topic you're weak in, all the best. Nevertheless, format of questions seem to remain the same.
