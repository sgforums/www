    de original question was correct.

    d [ln (ln x^e)] / dx

    = d [ ln (e ln x) ] / dx

    = d [ ln e + ln (ln x) ] / dx

    = d [1 + ln (ln x) ] / dx

    = (1/ln x ) . 1/x

    = 1 / (x ln x)
