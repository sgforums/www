    3/x - 4/y = 1... (1)

    7/x + 2/y = 11/12... (2)

    (2) X 2 is 14/x + 4/y = 22/12... (3)

    (1) + (3)

    3/x - 4/y + 14/x + 4/y = 1 + 22/12
    3/x + 14/x = 34/12
    36 + 168 = 34x
    x = 6

    Sub x =6 into (1)
    3/6 - 4/y = 1
    4/y = -1/2
    y = - 8
