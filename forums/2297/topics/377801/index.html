    Originally posted by Snoopyies:

    Solve 2a^(lg 3) x 3^(lg a) - 5a^(lg3) = 3 where a is a constant.

     

    Hi,

    one thing which you must understand is that a^(lg3) is the same as 3^(lga).

    Thus, you can rewrite the whole equation as 2X3^(lga) X 3^(lga) - 5X3^(lga) = 3.

    If we let 3^(lga) be u, we will have 2u^2 - 5u - 3 = 0, which is the same as (2u+1)(u-3)=0

    u = -1/2 is rejected in this case, and you should be able to see why.

    Thus, u=3, 3^(lga)=3,  lga=1, and a=10.

    Originally posted by Snoopyies:

    Hi,

    How come a^(lg3) is the same as 3^(lga) ?

     

    Hi, 

    a^(lgb) = b^(lga) for all a and b > 0.

    Proof:

    Assuming that the hypothesis a^(lgb) = b^(lga) is true,

    a^(lgb)=b^(lga)

    lga^(lgb)=lgb^(lga)

    (lgb)(lga)=(lga)(lgb)

    There you go.
