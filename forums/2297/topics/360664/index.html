  • SBS261P
    19 May 09, 21:53

    Use the theory of a^2 - b^2 = (a+b)(a-b)

     

    99^2 = 99 x 100 - 99 = 9801

    98^2 = 98 x 100 - 98 x 2 = 9604

    Consider 98.5^2

    99^2 - 98.5^2 = 0.5 x (99 + 98.5) = 98.75

    => 98.5^2 = 9801 - 98.75 = 9801 - 100 + 1.25 = 9702.25

    98.5^2 > 9700.

    Consider 98.49^2.

    98.5^2 - 98.49^2 = (98.5 + 98.49) (0.01) = 1.9699

    => 98.49^2 = 9702.25 - 1.9699 = 9702.25 - 2 + 0.0301= 9700.2801

    Given that difference between 98.48^2 and 98.49^2 will be >> 0.2801 (by deduction by observing 98.5^2 - 98.49^2),

     

    HENCE distance AC is 98.49km (to 2 d.p)
