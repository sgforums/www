    The current ratio is 1 : 3

    The old ratio is 1 : 4 ( let this be 1 unit : 4 units )

    Hence, (1 unit + 46) X 3 = ( 4 units + 63 )

                 3 units + 138 = 4 units + 63

                 1 unit = 75

    Original cakes = 1 unit = 75 cakes

    Original buns = 4 units = 75 X 4 = 300 buns
