    dy/dx = x - (24/x^2)

    Integrate [x - (24/x^2)] dx = (0.5 x^2) + 24/x + C

     

    Since (3, 1) is a point on the curve,

     

    (0.5*(3)^2) + 24/3 + C = 1

    0.5(9) + 24/3 + C = 1

    4.5 + 8 + C = 1

    C = -11.5

     

    Equation of the curve would be:

    y = 0.5x^2 + 24/x -11.5
