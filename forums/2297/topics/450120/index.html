    If you explain in simple terms, you can read it like this.

    Original Number of Pencils - 84 (to 5A) = 5 units remaining.

    2 out of the 5 units left give to 5B. 3 units left.

    3 units = 1/4 number of original number of pencils

    Original Number of pencils = 3 x4 = 12 units.

    12 - 5 = 7units (84 pencils - 5A)

    Original Number = 12/7 x 84 pencils = 144 pencils.
