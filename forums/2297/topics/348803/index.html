    (a) P(all of them have opinion) = 0.85 X 0.85 X 0.85 = 4913/8000

    (b) Let X = not disapprove; P(X)=0.6
           P(at least 2 disapprove) = P(DDX) + P(DXD) + P(XDD) + P(DDD)
                                                  = 3 (0.4X0.4X0.6) + 0.4X0.4X0.4
                                                  =44/125

    (c) Let NA = not approve; P(NA) = 1-0.45 = 0.55

        P(at most 1 approve) = P(all NA) + 3 P(A.NA.NA)

                                           = 0.55X0.55X0.55 + 3(0.45X0.55X0.55)

                                           = 2299/4000
