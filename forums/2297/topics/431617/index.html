    Hi eagle, got it. Thanks.

    (logx 2011 + logy 2011) / logxy 2011

    = [(ln 2011 / ln x) + (ln 2011 / ln y)] / (ln 2011 / ln xy)

    = (ln xy)(ln x + ln y) / (ln x)(ln y)

    = (ln x + ln y)² / (ln x)(ln y)

    = 4{[(ln x + ln y) / 2] / √[(ln x)(ln y)]} ²  > 4

    Hence, the smallest possible value is 4.
