    The smallest number which can be multiplied by 560 to give a cubic number is 2^2 x a^2 x b^c. find the values of a,b and c.

     

    Thanks in advance. (:

  • eagle
    22 Sep 08, 20:40

    560 = 2^4 * 5 * 7

    for a cubic number, the smallest number that can be multiplied is
    2^2 * 5^2 * 7^2

    so that we will get 2^6 * 5^3 * 7^3 in the end, which is a cubic number

    Hence,

    a = 5, b =7 and c = 2
