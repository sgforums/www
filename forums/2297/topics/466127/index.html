  • wee_ws
    22 Feb 13, 08:01

    Hi,

      For (a):
      1. B ~ Po(14) for a 200-page document.
      2. Then B ~ N(14, 14) approximately since mean > 10.
      3. To find P(B > 21) = P(B >= 22), apply continuity correction first.

      For (b):
      1. A ~ Po(1) for a 10-page document and B ~ Po(1.05) for a 15-page document.
      2. Consider A + B ~ Po(2.05), via addition of the means.
      3. Then find P(A + B < 4) = P(A + B <= 3).

      For (c):
      1. A ~ Po(1) for a 10-page document.
      2. Find P(A = 0).
      3. Let X be number of documents, out of 8, that are error-free. Then X follows a binomial distribution.
      4. Find P(X <= 2).
