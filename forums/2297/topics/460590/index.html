    Your textbook actually has a lot of questions that you can try. I know that some of them were probably set by your teacher as homework questions before, but you can try those that you've not done before. No need to spend money on assessment books, the questions there are very similar to the textbook questions.

    If you are able to do most of the textbook questions, you would be at a very good level already. If you want to challenge yourself further, you can try ten-year-series questions or questions from other schools' prelim papers if you can obtain them.
