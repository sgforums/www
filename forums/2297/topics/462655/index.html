    To illustrate the different approaches of factorisation

    Question

    Factorise completely x^3 + 3x^2 - 4

    Approach 1

    x^3 + 3x^2 - 4

    = x^3 + 3x^2 - 1 - 3

    = x^3 - 1 + 3x^2 -3

    = (x -1)(x^2 +x+1) +3(x^2 - 1)

    = (x -1)(x^2 +x+1 +3(x - 1)(x + 1)

    = (x - 1)[x^2 + x + 1 + 3(x +1)]

    = (x - 1)(x^2 + 4x + 4)

    =(x - 1)(x + 2)^2

    Approach 2

    x^3 + 3x^2 - 4

    = x^3 - x^2 + 4x^2 - 4

    = x^2(x -1) + 4(x^2 -1)

    = x^2(x -1) + 4(x -1)(x + 1)

    = (x - 1)[x^2 + 4(x+1)]

    = (x - 1)(x^2 + 4x + 4)

    =(x - 1)(x + 2)^2

    Appraoch 3

     x^3 + 3x^2 - 4

    = 4x^3 - 3x^3 + 3x^2 - 4

    = (4x^3 - 4) - 3x^3 + 3x^2

    = 4(x^3 - 1) - 3x^2 (x - 1)

    = 4(x^3 - 1^3) - 3x^2 (x - 1)

    = 4(x - 1)(x^2 + x + 1) - 3x^2 (x - 1)

    = (x - 1) [4(x^2 + x + 1) - 3x^2]

    = (x - 1)(x^2 + 4x + 4)

    =(x - 1)(x + 2)^2

    Approach 4

     x^3 + 3x^2 - 4

    = x^4 - x^4 + x^3 + 3x^2 - 4

    = x^4 + 3x^2 - 4 - x^4 + x^3

    = (x^2 + 4)(x^2 - 1) - x^3(x - 1)

    =  (x^2 + 4)(x + 1)( x - 1) - x^3(x - 1)

    = (x - 1)[ (x^2 + 4)(x + 1) - x^3]

    = (x - 1)(x^2 + 4x + 4)

    =(x - 1)(x + 2)^2
