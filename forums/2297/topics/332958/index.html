    x= 2y + 11

    4x = 8y + 44 -------(3)

    y= - 4.

    Sub y=-4 back into (2)

    4x - 12 = 0

    4x = 12

    x = 3

  • tinuviel07

    T^4 / T^1/2 = T^n

    For e.g. if the base is the same like

    4^5 / 4^2,

    you can change it to 4^ (5-2) = 4^3,

    so for division, you subtract the power,

    if it's 4^5 x 4^2, you change it to 4^(5+2) = 4^7,

    so for multiplication, you add the powers together.

    T ^ (4-1/2) = T^n

    T^ (3.5) = T^n

    n = 3.5
