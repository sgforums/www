    To know what's metal R, you must know the answer for green ppt U.

    Metal R + Sulphuric acid = R sulphate + Hydrogen

    R sulphate + Sodium Hydroxide = R hydroxide + Sodium Sulphate

    If green ppt is obtain, it's Iron(II) hydroxide.
    If red/brown ppt is obtain, it's Iron(III) hydroxide.

    From this Sodium hydroxide test, we can tell the the Cation ion is Fe2+
    Therefore the Metal R is Iron.

    First you must know that the use of acidified Barium chloride is to test for presence of the of sulphate ions. If sulphate ions is present, then white ppt will be obtain.

    Background information that you might need to know.

    Chlorine is an oxidizing agent. Which mean, it'll cause something to oxidized. So in this case, Fe2+ is oxidized to Fe3+

    The brown solution after bubbling is Iron(III) sulphate.
