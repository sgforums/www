    Casio FX 95 SG PLUS is a lot better than Sharp EL 509 WS or VM.

    Both calculators have the equation function.

    FX can input data in a table form so that any mistakes can be corrected easily

    FX 95 can input data in a table form especially useful for doing standard deviation question

    FX 95 can input data in real fraction form and hence it can give the correct answer without the addition of brackets

    Press minus sign (not small minus) 2 into 509Ws, we will see 0 minus sign 2

    while FX 95 will show correctly minus sign 2.

    Press minus sign (not small minus) 2 minus sign (not small minus) 3, FX 95 will give correctly minus sign 5 while 509WS will give us error.
