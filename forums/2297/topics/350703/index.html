    Hi,

      Sum of roots A + B = 4p -- (1)

      Product of roots AB = 15p -- (2)

      In (a), we are given that A : B = 3 : 5, so A = 3/5 B.

      From (1), 8/5 B = 4p, so B = 5/2 p and A = 4p - 5/2 p = 3/2 p.

      From (2), (3/2 p)(5/2 p) = 15p, from which p can be solved.

      Try part (b), thanks!
