    i) { V() e^10k } / { V() e^0} = 2
    Hence, e^10k = 2
    10k = ln2
    k = 0.1 ln 2

    ii) { V() e^kt } / { V() e^0 } = 3
    e^kt = 3
    kt = ln 3
    t = 1/k ln3
