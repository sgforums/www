    Hi,

      For Q1:

      1. When x = 2, y = ?

      2. dy/dx = ?

      3. When x = 2, value of dy/dx = ?

      4. Gradient of line joining the point where x = 2 and (1, -10) = ?

      5. Now value of dy/dx = gradient of line, from which p = ?

      For Q2:

      1. Rewrite T as T = {2 pi (g)^-0.5} (l)^0.5, noting that the expression within braces is a constant.

      2. Use the fact that the derivative of x^n is nx^(n-1), to find dT/dl.
