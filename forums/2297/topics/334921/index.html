    4 50-cent coin equal to 10 20-cent coin

    treat X, stuff inside bracket() as boxes

     

    before change

    50-cent  X X
    20-cent X X X

    after change

    50-cent  X X                ===> take away 4
    20-cent X X X             ===> add 10

    50-cent  (X-2 )(X-2)
    20-cent  X X X (10)            

    since new ratio is 2:7
    1 unit is equal to (X-2)
    7 unit is (X-2)(X-2)(X-2)(X-2)(X-2)(X-2)(X-2)

    50-cent  (X-2)(X-2)(X-2)(X-2)(X-2)(X-2)(X-2)
    20-cent  X X X (10)

    ===>

    50-cent  X X X X X X X (-14)
    20-cent  X X X (10)

    ===>

    50-cent  X X X X (-14)
    20-cent  (10)

    ===>

    50-cent  X X X X 
    20-cent  (10)(14)

    ===>

    50-cent  X X X X 
    20-cent  (24)
    ===>

    50-cent   X  X  X  X 
    20-cent  (6)(6)(6)(6)

    X = 6
