  • secretliker
    7 Jun 08, 22:10

    (1-x)(1+ax)^6
    (1-x)(1+6ax+15a^2x^2)
    1 + 6ax + 15a^2x^2 - x - 6ax^2 - 15a^2x^3
    1 + (6a-1)x - 6ax^2 + 15a^2x^2 = RHS

    1 + (6a-1)x - 6ax^2 + 15a^2x^2 = 1 + bx^2

    Now it can be shown that 6a-1 = 0

    Thus a = 1/6

    15a^2 - 6a = b

    b = -7/12
