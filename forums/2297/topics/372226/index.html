    Hi Gunner,

    No, the median is NOT green.

    To find median, we need to arrange all the items in ascending order

    (or descending order) first.

    Next, the median will be the item in the middle.

    In this question, the item is colour and we cannot say which colour is smaller

    or bigger ie we cannot arrange the different colour in ascending order (or

    descending order).

    Since there is no ascending order, there is no middle item ie no median

    for this question.

    This is the fun part of the question.
