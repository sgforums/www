    z varies q^2 + p is the same as z = k (q^2 + p)

    Subbing in z= 3 and q =4, we will get

    3 = 16k + kp -------(1)

    Subbing in z = -1 and q = 2, we will get

    -1 = 4k + kp -------(2)

    (1) - (2) will result in

    12k = 4

    Hence, k = 1/3.

    Subbing k = 1/3 back into (1),

    we will get :

    3 = 16/3 + p/3

    9 - 16 = p

    = > p = -7
