    nvm i had figured out the solution...

    1.
    (2a + 1)³ + 2(2a + 1)² + 2a + 1
    (2a + 1)³ + 2(2a + 1)² + (2a + 1)
    let b = 2a + 1
    so it becomes:
    b³ + 2b² + b
    = b(b² + 2b + 1)
    = b(b + 1)²
    = (2a + 1)(2a + 1 + 1)²
    = (2a + 1)(2a + 2)²
    = (2a + 1)(2)²(a + 1)²
    = 4(2a + 1)(a + 1)²

    2.
    (p + q)² + 2(p + q)(p - 2q) + (p - 2q)²
    let p + q = a, p - 2q = b
    so it becomes:
    a² + 2ab + b²
    = (a + b)²
    = (p + q + p - 2q)²
    = (2p - q)²
