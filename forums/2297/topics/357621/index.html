    "Find the points of intersection of a straight line 2x +3y = 10 and the curve 2/x + 3/y = 5.
    Answer: (4, 2/3)(1/2, 3)"

    2x + 3y = 10 ----- ( 1 )

    2/x + 3/y = 5 ----- ( 2 )

    From ( 1 )

    2x = 10 - 3y

    x = (10-3y)/2 ----- ( 3 )          <<<< can either use x or y as subject. 

    Sub ( 3 ) into ( 2 )

    2/((10-3y)/2) + 3/y = 5

    4/(10-3y) + 3/y = 5

    (4y + 30 - 9y)/(10y - 3y² ) = 5

    4y + 30 - 9y = 50y - 15y²

    15y² - 55y + 30 = 0

    3y² - 11y + 6 = 0

    (3y - 2)(y - 3) = 0

    (3y - 2) = 0 or (y - 3) = 0

    y = 2/3, 3 

    Sub in Y values and get x

    When y = 2/3, x = 4

    When y = 3, x = 1/2
