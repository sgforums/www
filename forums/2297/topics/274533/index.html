    Since curve is a parablioc curve, it is highly likely the curve is trig function.
    Assume that I take the left edge of the curve to be the origin, the curve will be a sine curve.

    Amplitude of curve = 3. This implies that there is coefficient of 3 in the sine function.

    Let y = f(x)
    f(0) = 0
    f(2) = 3
    f(4) = 0

    Recall that sin 0 = 0, sin (pi) = 0, pi is the next value of 0.
    For this situation, 4 is the next value of 0.

    Therefore, imagine sin (pi) = 0
    Sin(pix 4/4) = 0
    Sin(pi/4 x 4) = 0

    When x = 4, y = 0
    Hypothesis: y = 3 sin [(pi/4) x]

    To confirm, check with f(2) = 3 and f(0) = 0

    After it has been confirmed, the parabolic curve has the eqn y = 3 sin [(pi/4) x]

    Integrate it from 0 to 4 since I have taken the left edge of the curve to be the origin.

    u will get
    [-12/pi cos(pi/4) x ] from 0 to 4.

    Area of the curve = 24/pi
    = 7.639 = 7.64 m^2
