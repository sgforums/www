    if im not wrong, m = 3

    960 / 5m = 192 / m

    192 = 2 x 2 x 2 x 2 x 2 x 2 x 3

            = 4 x 4 x 4 x 3

    a cube has equal sides throughout, or it's three factors must be equal. in order for 192 / m to be a cube, at least the factor 3 must be removed.

    so m = 3
