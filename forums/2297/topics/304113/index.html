    f(x) = (1 + x^0.5) ^ 0.5

    f'(x) = 0.5 * (1 + x^0.5)^(-0.5) * 0.5 * (x^-0.5)

    f'(x) = 1/{4 sqrt[x + x sqrt(x)]} (shown)
