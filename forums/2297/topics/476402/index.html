    Q28. correct answer is B

    mariogirl19 wrote, "it should be 3d5 4s1 instead of 3d4 4s2"

    Actually, it is neither of the above. The correct configuration for Fe2+ is 4s0 3d6 (or 3d6 4s0 if you prefer).

    During ionization, electrons are always first removed from the outer 4s subshell, before removing the inner 3d electrons.

    Fe : [Ar] 4s2 3d6

    Fe+ : [Ar] 4s1 3d6

    Fe2+ : [Ar] 4s0 3d6

    Fe3+ : [Ar] 4s0 3d5
