  • ^tamago^
    12 Apr 08, 10:22

    x²-y² = 37
    (x-y)(x+y) = 37

    Since 37 is a prime number, the 2 factors can only be 1 and 37.

    x-y = -37 ----- (1)
    x+y = -1 -----(2)

    (1)+(2): 2x = -38, x = -19, y = 18

    x-y = -1 ----- (3)
    x+y = -37 -----(4)

    (3)+(4): 2x = -38, x = -19, y = -18

    x-y = 1 ----- (5)
    x+y = 37 ----- (6)

    (5)+(6): 2x = 38, x = 19, y = 18

    x-y = 37 ----- (7)
    x+y = 1 ----- (8)

    (7)+(8): 2x = 38, x = 19, y = -18

    So the solution pairs are (18,19), (18,-19), (-18,19) and (-18,-19).
