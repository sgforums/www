    clockwise moments = 300 (value of P) x 1.2m = 360Nm

    anticlockwise moments = ( 0.8Fb + 1.6Fc ) Nm, where Fc is T, Tension, Fb is upward force at point B

    but Fb = T sin 40

    Thus, (0.8sin 40 + 1.6) T  Nm = 360 Nm
    T = 170.3 N
