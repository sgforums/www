    Rulers to be kept horizontal using spirit level

    Timing is started only when oscillations are steady.

    Ensure metre rule used to measure height is vertical by using a set square.

    Height is measure using increasing and decreasing values of L.

    When measuring height, ensure that the vertical ruler is placed close to the end of clamped ruler, reading at eye level.

    Ensure height is measured at the same point.

    Repeat timing for each value of R to reduce random error.

    Check the wire connections to ensure that there are no loose connections in the circuit.

    The reading on the multimeter is changing rapidly, hence do a few trials before taking the actual measurements.

    Check that the reading on voltmeter is zero before closing the switch.

    Constructing a table.

    Always construct with a heading magnitude+ a state symbol.

    If your heading doesnt have symbols, always make sure your readings have a symbol.

     

    Example, you write a value of 5.

    5 what?
    5 monkeys? 5cm? or 5 metres. 

    Marks will be deducted.

     

     

  • kenn3th
    15 Oct 08, 17:18

    Accuracy.

    burette: 0.1cm3

    pipette: 0.1cm3

    difference between burette and pipette.

    burette: measures readings of up to 0.1cm3, like 23.5cm3 ect.

    pipette: measures SET volumes - 25cm3, 10cm3, 15cm3.

    Measuring cylinder: 1cm3 accuracy - dont be a blur cock and write 1.0cm3 - marks will be deducted for in accurate accuracy.

  • kenn3th
    15 Oct 08, 17:20

    Measuring lengths:

    check for zero errors. 

    you can do so by closing the jaws of the apparatus, and look at the if the "zeros" of the 2 scales are at the same point.

    if they are not, then - or + the readings.
