    Hi,

      Since α is a root to the equation, then α^2 = 2α + 1, which is useful for our later proof.

      One (correct) result we can show is α^3 = 5α + 2.

      Proof:

      LHS = α^3

      = α. α^2

      = α (2α + 1)

      = 2α^2 + α

      = 2(2α + 1) + α

      = 5α + 2

      = RHS.

      Thanks.
