    recursive formula ba... cos integrate or differentiate e^x always e^x.

    so just take
    INT e^x(cos x) =e^x(cos x) - INT e^x(-sin x)
    =e^x(cos x) + INT e^x(sin x)
    =e^x(cos x) + e^x(sin x) - INT e^x(cos x)

    so 2 INT e^x (cos x) = e^x(cos x + sin x). then divide by 2.
