    Find the coefficient of x^6 in the expansion of (4+x^2)^1/2

    since the question asked for coefficient of x^6, u should expand in ascending powers

    (4+x²)^½
    = 2(1+x²/4)^½
    = 2[1 + (1/2)(x²/4) + (1/2)(-1/2)/2! * (x²/4)² + (1/2)(-1/2)(-3/2)/3! * (x²/4)³ + ... ]
    = 2[... ... + (1/16) * (x^6 / 64) + ... ... ]
    = x^6 / 512 + ...

    coefficient of x^6 = 1/512
