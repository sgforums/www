    Originally posted by PUrePLs:

    Find the value of a and b for which x^2 + 4x -1 is a factor of x^3 + ax^2 - 5x + b.

    Anyone can help ? Thanks.

    Let x^3 + ax^2 - 5x + b = (x^2+4x-1)(x-b)

    Let x = 1

    1+a-5+b = 4(1-b)

    a+5b = 8 ------(1)

    Let x = -1

    -1 + a + 5 + b = -4(-1-b)

    a-3b = 0----- (2)

    Eqn (1) - (2)

    8b = 8

    b = 1

    From (1)

    a + 5 = 8

    a = 3
