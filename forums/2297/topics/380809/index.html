    I'll be sharing here a little of my analysis on STI index using the things learned in H2 Maths: Linear Regression

    We have here the following data from the monthly charts (interim peaks and bottoms):

    Mar 1994 (zeroth month): 2482.9
    Feb 1996 (23rd month): 2504
    Mar 2000 (72nd month): 2582.94
    May 2006 (145th month): 2666.33
    Mar 2008 (168th month): 2745.96

    In my linear regression analysis, the variable x shall represent the number of months after Mar 1994. The variable y is the value of STI, excluding inflation.

    A simple excel calculation when we consider the linear chart:


     

    As we can see, the calculations point to 2754.31 as the resistance for this month, Nov 2009, the 188th month since Mar 1994.

    The linear product correlation for both the linear stands at a high 0.988, signifying a very strong linear correlation between the Straits Times Index and time line.
