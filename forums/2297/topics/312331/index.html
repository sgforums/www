    |a| means a is positive.

    ie
    |2| = 2
    |-2| = 2
    |3| = 3
    |-3| = 3

    thus, |x| >= 0 always.

    so |x| < 2 is the same as 0<= |x| < 2
    if x = -1, |x| = 1 < 2
    it is still valid... and so on...

    so, -2 < x < 2 will all fit the equation |x| < 2

    Your images are not loading, otherwise eagle has provided a good overview of the absolute function.

    |x| < 2 = -2 < x < 2 but why not 2 < x < -2? simply because there is no value of x that is larger than 2 AND smaller than -2.

    -2 < x < 2 simply means "from -2 to 2, exclusive". the absolute of the numbers within this range is indeed wat we want (absolute or only numerical portion less than 2), e.g. -1, -1.5, 0.5, whose absolutes are 1, 1.5 and 0.5 respectively.
