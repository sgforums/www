    Sounds like an okay idea.

    You will have to argue about how this idea will actually benefit the user instead of making it more troublesome for him/her. Here are some food for thought:

    1) More partitions = more space needed to fix in the partitions

    2) How can you be sure that each partition space is enough for putting a certain subject? (E.g. Maths may take up a lot more space than english, and if you make all partitions the same size, the bag will end up over-sized, with a lot of space unused.)

    3) Can this bag be used by everyone? A tailored-made bag is not a good idea, since it has to be made according to each individual's needs.
