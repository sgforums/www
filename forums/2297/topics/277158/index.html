    Trigonometry

    Sine, Cosine and Tangent

    using toa, cah, soh

    toa --> tan = opp/adj
    cah --> cos = adj/hyp
    soh --> sin = opp/hyp

    opp, adj and hyp are the 3 line that form a triangle


    For this triangle , ACB is a right angle.

    Different angles have different Sine, Cosine and Tangent...
    As you can see, using toa cah soh to solve can only be used on a right angled triangle...

    ABC is a right angle triangle...

    If the question ask, "Find Angle A"
    You will need to find what is Tan A first then find what is A
    Then you will have to used the tangent formula toa, tan = opp/adj.

    the opp line for A is a, the hyp line is c, the adj is b...
    How to tell which line is which?

    Opp, Opposite... as the name say the one that is opposite the angle...
    Hyp, Hypotenuse... The longest one... This is opposite the right angle
    Adj, Adjacent... Also as the name says, "lying near, close" which mean this line will touch the angle given...

    Hyp and Adj is different... Hyp is the longest one, while Adj is the other one which touch the angle... Hyp will also touch the angle...

    The fomular toa, tan = opp/adj.
    Tan A = opp/adj
    Tan A = a/b
    A = tan^-1 a/b (you need a calculator for this)

    Tan^-1 mean Tan to the power of -1, which mean tan inverse...

    Well thats all...
    It is the same for other Formula or Angles...
