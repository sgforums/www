    If the casino decides to set the machine to have 3 apples appear in a row once randomly in 121 plays on the machine, how much the casino is expected to make ? 

    The casino will win $0.21 for every 121 plays on the machine ie

    Expected value per 121 plays = $0.01 x 120 - $0.99 x 1= $0.21 or

    Expected value per play = $0.01 x 120/121 - $0.99 x 1/121 = $0.00174 (3sf)

    What if the 3 apples appear on the 88th play and the player wins the one dollar and stop playing, will the casino lose money ? 

    The casino will not lose money as long as there are many players who wish to play on the machine ie the next time the 3 apples will appear once randomly in the next 154 plays.
