    For this type of integral:

     ∫ 1/y(1−y) dy

    use partial fractions, break up like this:

     ∫ 1/y + 1/(1−y) dy

    Then easy to get:

     ln|y| − ln|1−y|

    After that still got some algebra, but should be ok already ..
