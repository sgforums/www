    Double-Angle Formula:

    sin 2x = 2 sin x cos x

     

    f'(x) = 2 cos x - 2 sin 2x = 0

    ==> 2 cos x - 2 (2 sin x cos x) = 0

    ==> 2 cos x (1 - 2 sin x) = 0

     

    cos x = 0 (90, 270) and sin x = 1/2 (30, 150)

    The other 2 points are reference angles.
