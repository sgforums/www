    Area of SQ = 70cm2 as given

    i.e. (AB x AB) or (BC x BC) = 70cm2

     

    Pythegoras Theorem,

    AC x AC = (AB x AB) + (BC x BC)

                = 70 + 70

                = 140

    therefore AC = square root of 140 which is also the radius of circle

     

    Area of circle = (22/7) x radius x radius

                       = 22/7 x AC x AC

                       = 22/7 x 140/4

                       = 110cm2
