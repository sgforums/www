    12x^2 - 5y^2 = 7

    2x - 5y = 7

     

    2x = 7 + 5y
    (square both sides and expand)
    4x^2 = 49 + 70y + 25y^2
    (whole eqn multiply by 3)
    12x^2 = 147 + 210y + 75y^2

    (sub into 1st eqn and simplify)

    147 + 210y + 75y^2 - 5y^2 = 7
    70y^2 + 210y + 140 = 0

    (take out common factor 70)

    y^2 + 3y + 2 = 0

    (y+1)(y+2) = 0

    y = -1 or y = -2

    when y = -1,
    2x = 7 -5
    2x = 2
    x = 1

    when y = -2
    2x = 7 - 10
    2x = -3
    x = -3/2

    solutions -- (1,-1) and (-3/2,-2)
