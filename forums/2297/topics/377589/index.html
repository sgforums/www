    really you should ask your teacher about the link skills. For example, for globalization, if they ask you this

    How far do you think education can solve the environmental impact? DYA. EYA. [12]

    ( btw, this is an adapted qns from N level 2009 question in globalization)

    This is a weak link

    Globalization causes environmental impact. By education and promoting the 3 R (Recycling, Reuse, Reduce), less plastics will be use

    WHY is this a weak link?

    • Never explain what precisely is the impact of using plastic on environment.
    • How does globalization link to plastics (The increase packaging, the increase in goods produced)

    For SBQ, i not that sure, but just note these

    NEVER false-match, this is just matching by either words in source or the face value

    X-ref is required by these questions

    • How far does.......prove that ..........
    • How useful/reliable.........

    In addition, when doing X-ref, do this

    A and B differs in (issue). A says "Quote", B says "quote", therefore A seem to suggest (what) and source B seem to suggest (what)
