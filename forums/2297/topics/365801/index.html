    For Q2,

    (m+n/m-n)^1/2 = p+q/p-q

    (m+n/m-n) = (p+q/p-q)^2

    (m-n)(p+q)^2 = (m+n)(p-q)^2

    m(p+q)^2 - n(p+q)^2 = m(p-q)^2 + n(p-q)^2

    n(p+q)^2 + n(p-q)^2 = m(p+q)^2 - m(p-q)^2

    n = [m [(p+q)^2 - (p-q)^2]] / [(p+q)^2 + (p-q)^2]
