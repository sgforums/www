    a)

    pKa = 10.61
    pKb = 14 - 10.61 = 3.39
    Kb = 10^(-3.39)

    [OH-] = sqrt(Kb*conc. of N-butylamine)
    = sqrt(10^(-3.39) * 2*10^-3 / 0.025)
    pOH = 2.243455007
    pH = 14 - 2.243455007 = 11.76 (to 2 dp)

    b)

    pOH = pKb + lg ( [salt]/[base] )
    14 - 10 = 3.39 + lg( [salt]/(2*10^-3 / 0.050) )
    [salt] = 0.163 mol dm^-3 (to 3 sf)
