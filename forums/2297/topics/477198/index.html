    d2n/dt2 = 10 − 6t
    ⇒ dn/dt = 10t - 3t2 + c
    ⇒ n = 5t2 − t3 + ct + d

    Given n = 100 when t = 0. Sub above, get d = 100.

    ⇒ n = 5t2 − t3 + ct + 100.

    The given info enables us to eliminate one of the arbitrary constants only (d). The other one (c) cannot eliminate.
