      We could use the units method to solve Q2.

      Ali: 1u

      Bibi: 6u

      Cathy: 1u - 60

      Now

      1u - 60 = (1/15)(1u + 6u + 1u - 60)

      ...

      1u = 120

      6u = 720

      Bibi scored 720 marks.
