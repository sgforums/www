    Q1

    dy/dx=[(2ax)(x-1)-(ax^2+2)1]/(x-1)^2=(ax^2-2ax+2)/(x-1)^2

    When dy/dx=0,

    ax^2-2ax+2=0

    The curve has no stationary point if this equation has no real roots.

    b^2-4ac<0

    4a^2-8a<0

    4a(a-2)<0

    0<a<2
