    For Class 2B there is only one Theory Test. (RTT)
    You must attend 2 lectures, (useful tips are given so please attend them).
    Then book the test date.
    You are allowed to do practice tests before the test date.

    You need the RTT cert to get the PDL.
    You need valid PDL for Prac. Lesson 5 & 6 & 8 ( Riding outside circuit)
    For TP Class 2B test you need valid RTT Cert and valid PDL.

    RTT cert is valid for 1 year, so plan to pass the RTT.

    If you pass too early, by the time of your TP test theres only a few months left before expiry. If you do not pass on your first attempt, the next test might be a month later and it gets uncomfortably close to the expiry date of the RTT cert. If your RTT cert expires before you pass your TP test, you need to take the Theory Test again.
