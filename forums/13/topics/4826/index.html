  • MS
    4 Nov 01, 19:58
    PGM4 maximum bHP is only 45. Its set that way already. Even with the aid of the HRC card Dog fight exhaust, it can only push up to 65bhp. However, the RS250 is originally set at 60bhp. Assisted by 2 arrows or gianelli RS250 can push up to 65bhp also. But with other improved versions like a dry clutch, It can bring up more pick up. PGM4 requires a steering damper to have more stable steering while RS250 does not need to. RS250 potentially can push up to 75bhp whil PGM4 can only push to the limits cornered dead at 65bhp. Pick up is faster for PGM4 is due to the dry clutch and lighter body. But with more power and a more stable crafting of frame design, RS250 is more stable. If aided by advanced modification parallel to PGM4 It can beat the PGM4 straight in the face flat on a simple straight. As for track, it depends on the rider.
