After a minute or two, choke can be disabled, then you can continue warming the bike for another minute. Then can start moving slowly, increase the RPM bit by bit until the engine reaches the normal running temperature.

    If the bike can't start in the morning without choke, I think it's quite normal. Some big bikes have automatic choke, which will be automatically enabled when the needed. Of course the bike designers wouldn't make it that way if there'll be problem in the future, would they?
