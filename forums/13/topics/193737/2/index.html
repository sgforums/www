    Originally posted by X9_panel:


    [b]How to check the EFI signal - X9-500cc

    S08: EFI- Fuel injection warning lamp ( 250 & 500 cc )



    This digidash is only to display the EFI signal - simple indication lamp.

    So you have to check the sensor side. This senor is providing the ON/OFF switch i.e. ON/OFF status

    It is using negative logic.

    Normal state : 12V sending to digidash = EFI Lamp OFF
    Warning state : 0 V = EFI Lamp ON

    for example, if the sensor is disconnected then EFI Lamp is ON

    Hopefully, this may help you to understand how to troubleshoot.

    x9.dashboard@yahoo.com[/b]
