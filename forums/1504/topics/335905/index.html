    SIA charges for exit row seats on economy class

    SINGAPORE: Some customers always opt for the exit row seats on flights because of the extra legroom.

    From now on, these customers will be able to get a guaranteed exit row seat on Singapore Airlines flights, but they will have to pay US$50 per sector for the privilege.

    This is applicable for economy class bookings.

    For example, customers going on a long-haul flight to San Francisco can end up paying US$200 more for a return flight.

    Previously, such seats were only available on request, and subject to availability.

    SIA says it may extend this Preferred Seat Selection programme to other seats that offer more leg room in future.

    - CNA/yt
