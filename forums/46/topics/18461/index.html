    Lance18,
    u can use FDISK to partition ur harddisk.
    follow these steps. but before u do this, make sure u have created a boot disk so that u can boot up ur system. and oso make sure that u can install ur cdrom drivers after u have deleted ur partitions so that u can run ur win98se cdrom

    1 - type FDISK in dos prompt in c:\
    2 - select YES when ask if wan LARGE Disk Support
    3 - select option 3 to clear ur exisiting partition
    4 - in menu 3, select option 3 to clear logical dos drives (if u got any)
    5 - now still in menu 3, select option 2 to clear ur extended dos partition (if u got any)
    6 - now u select option 1 to clear ur primary partition (bear in mind if u proceed with steps 4 - 6, u will destroy ur exisiting data in the harddisk, so make backups of things u need!)
    7 - return to main menu and select option 1
    8 - in menu 1, select 1 to create ur primary partition (if i did not remember wrongly, after this the computer will ask u to specify how much u wan to use for primary partition, u can put any number, eg 50% of ur hardisk of 2GB)
    9 - now u select 2 to create ur extended partition. after this the com will ask u how much u wan to use for extended partition, u juz put the remaining of ur hardisk space, eg 50%
    10 - now select 3, to create ur logical drives in ur extended partition. u have to assign the size of each logical drives u wan to create ur self.(eg juz now out of 2GB, u assigned 50% which is 1 GB to ur primary partition and assigned another 50% to extended partition. now ur logical drives are using the 50% or 1 GB in ur extended partition. maybe u wan to assign 50% of the 1 GB u can use into the 1st logical drive anf another 50% into the second logical drive)
    11 - after creating all these, return to main menu and select option 2 to set ur partition active
    12 - after this u can take a look at the partitions u made by selecting option 4 in the main menu. u should see something like this... (base on the example i given)

    Partition ....... Mbytes ..... Usage
    C:1................1000.........50%
    D:.................500..........25%
    E:.................500..........25%

    13 - once u see this, exit fdisk completely
    14 - now use the boot disk to reboot ur com.
    15 - in a: u type FORMAT C: after formating c:, type FORMAT D: to format d: drive and so on...
    when u finish formating all the drives, reboot ur sys again
    16 - now install ur cdrom drivers
    17 - after installing ur cdrom u can put ur win98se cd into ur cdrom and type SETUP.EXE from the folder win98. and now u follow the rest of the instructiuon given.

    hope that this will help u..
    but i have to say this, u have to make sure u know what u are doing before formating ur exisiting drive.

    happy formating...
