    Read the help section on targeting the frames..

    if you are using dreamweaver, look out in the properties box for a little dropdown menu labelled "Target"

    to click a button and have the page change in the frame on the right (ie. the main content of the site).. select _Main from the Target dropdown menu, or whatever name you have named your frame.

    To prevent the left menu from popping up.. check that you link to just the .html file with the correct content and not the a page which already contains the left menu.. this way you will always end up with two menu bars... if you really have to select the page with the left menu bar.... on the "Target" dropdown menu... click on _parent.. this will refresh the entire page and display only the page AND the menu bar you want.

    hmm.. think it might be a bit confusing here... if my explanation is a bit confusing then maybe you can just try reading up the help section on frames... it contains the necessary solution you need....
