    maybe you have not installed the correct protocols needed. some games like Quake for example , need the IPX protocol to be included in your networking option in your control panel. [netcpl.cpl] Go to control panel >> Network >> "ADD" >> select "Protocol" >> choose Microsoft as Manufacturer >> choose "IPX/SPX-compatible protocol" >> choose OK. Have your windows 98/ME CD ready. insert it & restart your computer after the installation completed.

    When you want to play network games, just go the option : IPX gaming mode or something similar. Happy playing on Your Network !
    Hope this helps.
