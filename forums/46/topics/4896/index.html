    Existing Messages
    Open a message that you have received.
    Resize this form so that it fills up the screen. Do not use the Maximize button to maximize the window.
    To resize the form, move the mouse pointer over the corner of the window until a double-pointed diagonal arrow appears.
    Click and hold down the mouse button, and then drag the pointer to enlarge the window.
    Close this form window.

    After this, each time you open up a message, the read form is displayed in the same location and size, and will appear maximized.

    New Messages
    Open a new message form by clicking the New Message button on the toolbar.
    Resize this form so that it fills up the screen. Do not use the Maximize button to maximize the window.
    To resize the form, move the mouse pointer over the corner of the window until a double-pointed diagonal arrow appears.
    Click and hold down the mouse button, and then drag the pointer to enlarge the window.
    Close this form window.

    Each time you bring up the new message form, it is displayed in the same location and size, and will appear maximized.

    Charles
